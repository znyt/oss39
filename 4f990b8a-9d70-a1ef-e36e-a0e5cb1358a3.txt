　　他一个人，实在是精力有限，无法顾及全部。
<br/>　　但他可以炼制独立分身，而且是强大的分身，比他本体都不差多少的分身。
<br/>　　分身可以为他守护苍穹神宫，帮他发展苍穹神宫，也可以出去赚取大量的资源。
<br/>　　而他自己，就能抽出时间，去做其它的事情。
<br/>　　“不知道我能炼制多少分身？”
<br/>　　苏莫低声自喃，炼制独立分身难度虽然不大，但这里面牵扯的东西确实不少。
<br/>　　因为分身需要分化他的神魂，太多的话，会对他遭到伤害，而且分身有自己的生命，但所有记忆都和他相通，一旦太多，会对精神也造成不小的压力。
<br/>　　当然，这是只分身太多的情况下，一两个分身对他并没有丝毫的压力。
<br/>　　“看来要尽快炼制分身！”苏莫心中暗道，不过，目前并没有合适的肉身。
<br/>　　普通的武者，根本不适合做他的分身，只有最顶尖的武帝境武者，甚至是武尊境的武者，才是最理想的分身。
<br/>　　少倾之后，苏莫便不再多想，抛却心中的杂念，开始了修炼。
<br/>　　接下来的日子中，他终于有了空闲的时间，每日除了静心修炼，便对苍穹神宫中天赋较高，并且近两年修为进展够快的天才弟子，给予指导。
<br/>　　反正等派出去的弟子，查探到聂乾和韩盈的消息之前，他都不会离开，会一直的待在苍穹神宫之中。
<br/>　　时间悠悠，大半个月的时间过去了，陆续有弟子返回。
<br/>　　海明江派出去了三十名武皇境的弟子，在一个月之内，尽数返回了苍穹神宫。
<br/>　　他们带回来的消息，全部都是一样，聂乾和韩盈并没有被古天意救出来，依旧在镇魂之地。
<br/>　　这让苏莫的心，沉重了下来。
<br/>　　果然！
<br/>　　古天意以为他死了，不会再管聂乾和韩盈。
<br/>　　“怎么办呢？”
<br/>　　苏莫坐在阁楼中，低声自语，难道还要他找到古天意，向对方表明身份？
<br/>　　但是，他却不想暴露身份，不想人知道他还活着。
<br/>　　苏莫陷入了沉思之中，目前，聂乾和韩盈已经在镇魂之地快两年了，不能再等下去了！
<br/>　　思索良久，苏莫也找不到好的方法，似乎只有亲自去找古天意了。
<br/>　　只是，他一旦亲自去找古天意，以后怕是不好脱身了。
<br/>　　古天意拉拢他，只是想让他加入皇祖圣朝，估计他一旦现身，必须要加入皇祖圣朝，不然很难全身而退了。
<br/>　　苏莫思虑了良久，加入皇祖圣朝就加入吧，日后再想办法脱身便是。
<br/>　　不过，这其中还要和古天意好好商谈一番，达成一个协议。
<br/>　　思虑清楚之后，苏莫和父亲苏洪、以及帝释等人告别，离开了途山镇。
<br/>　　几日之后，他来到了一座叫东黄城的城池之中。
<br/>　　此城位于神武学府的东部，算是距离神武学府最近的城池之一，平日有很多神武学府的弟子来到此城。
<br/>　　苏莫在城中一家叫‘天黄’的豪华客栈住了下来，一边修炼，一边等待。
<br/>　　他没有等待多久，仅仅是半天的时间，就等到了要等的人。
<br/>　　“来了！”
<br/>　　房间中，苏莫睁开了眼眸，他一直在注意着客栈的大堂，发现了一名皇祖圣朝派系的弟子。
<br/>　　这名弟子他曾经见过，只是并不清楚对方叫什么名字，但也无所谓了。
<br/>　　随即，苏莫起身离开了房间，向客栈的一楼大堂走去。
<br/>　　很快，他便来到了大堂，见到了一名身形微胖的紫衣青年，正端坐在大堂的窗边，独自的饮酒。
<br/>　　此人修为不高，只有武帝境五重的境界。
<br/>　　“这位兄台，在下可以坐吗？”苏莫收敛了全身的气息，走到紫衣青年的身前，指着对方对面的位置问道。
<br/>　　紫衣青年闻言看了看苏莫，又看了大堂中，见大堂中空位多的是，不禁眉头微皱。
<br/>　　不过，他见苏莫气息深邃不可测，修为极高，却也没有拒绝，默默的点了点头。
<br/>　　苏莫在紫衣青年的身边坐了下来，目光望着对方，开门见山的说道：“这位兄台，想必你是神武学府的弟子吧，在下有一事相求。”
<br/>　　“何事？”紫衣青年皱眉问道，他和眼前的人毫不相熟，对方居然开口就说有事相求。
<br/>　　“这里有两万下品灵晶，算是给兄台的报酬！”苏莫没有立刻回答对方，而是手掌一翻，取出了一枚储物戒指，放在了紫衣青年的面前。
<br/>　　“两万下品灵晶？”
<br/>　　紫衣青年闻言双眉一挑，但是他并没有拿取面前的储物戒，这种报酬不算小，但也要看是什么事情。
<br/>　　“兄台，只是让你帮个小忙而已，希望你到神武学府，通知皇祖圣朝派系的圣子古天意，就说有朋友在天黄客栈等他！”苏莫沉声说道。
<br/>　　“你要见圣子？”紫衣青年闻言恍然，不过，他不明白，这等小事，对方为何不直接去神武学府，让人通传圣子古天意。
<br/>　　居然还搞的如此藏头露尾，难道有什么诡计不成？
<br/>　　他就是皇祖圣朝派系的人，当然不能让别人加害圣子。
<br/>　　“不错，希望兄台帮个忙！”苏莫郑重的点了点头。
<br/>　　“我怎么知道你有什么目的？”紫衣青年不为所动，淡淡的说道。
<br/>　　“呵呵，兄台无需担心，你只要告诉古天意一句话，他自然便能明白了！”苏莫微笑着说道。
<br/>　　“什么话？”紫衣青年疑惑问道。
<br/>　　“你就告诉他——约定和救人，他就会明白了！”苏莫沉声道，这正是他当初和古天意的约定，约定他在真圣碑留名，对方请动皇祖圣朝的圣皇，帮忙救出聂乾和韩盈。
<br/>　　“是吗？”
<br/>　　紫衣青年闻言，面露沉吟之色，随即便点了点头，道：“那好吧，我帮你这个忙！”
<br/>　　他同意了，只是传一句话而已，便能赚取两万下品灵晶，这么好的事情，他没有理由拒绝。
<br/>　　至于苏莫的目的，他虽然不知道，但想必圣子自有定夺，不需要他操心。
<br/>　　随即，紫衣青年拿起面前的储物戒，稍一检查，发现其内果然是两万下品灵晶，便立刻收了起来。
<br/>　　“我现在就回神武学府，告辞了！”紫衣青年起身，向苏莫微微抱拳。
<br/>　　“不送！”苏莫同样抱拳回礼。
<br/>　　随后，紫衣青年便大步的离开了天黄客栈，向神武学府而去。
<br/>　　苏莫望着对方离去的背影，叹了口气，便返回了房间之中，静等古天意的到来。
