　　就在混沌道君在遮天界中如火如荼地进行着主神计划之时，元始的本尊在洪荒之中也是修为精进不少。
<br/>　　昆仑山顶，玉虚宫中。
<br/>　　元始高坐云台之上，身边云雾缭绕，道音袅袅，神光闪烁，各种异象惊人，显然他的修为已经到了一个不可思议的地步。
<br/>　　“锵——”
<br/>　　忽然，两道金光闪烁，破开重重云雾，元始睁开双眼，眼中异象纷呈，生死幻灭，最终归一为元，恢复到最初的常态。
<br/>　　“呼——”
<br/>　　轻轻呼出一口气，气云如一条白色匹练，飞射出五丈开外，才开始缓缓消散于空间之中。
<br/>　　“终于突破混元大罗金仙九重天了，距离混元圆满已经不远了。”元始嘴角微微翘起，露出一丝不易察觉地笑意，显然，修为的突破，让他心情愉悦。
<br/>　　“《神元九转》是时候突破到下一层了。”良久，元始喃喃低语道。
<br/>　　《神元九转》是元始刚夺舍玉清是所创的元神修炼功法，当初只是草创，并不完整，这些年修为精进，不少地方都被他完善了，而且最重要的是最后两转也被他彻底创造出来了。
<br/>　　《神元九转》第八转——轮回，顾名思义，此转若是功成，他不久可以掌控轮回之道，更能让他元神更进一步。
<br/>　　第九转归一，才是真正的强大，九转归一，回归最初的一，最初的元，那是元始为了突破天道境所准备的。
<br/>　　“以神御法，以魂轮转，以心悟道，三千红尘，万千轮回，苦海无尽，沉沦其间，法海无边，万变归一……”
<br/>　　“魂兮——离兮！”
<br/>　　随着一道神秘的道音响起，在元始的神魂之中，孕育出千万个新生灵魂，他们纷纷飞出魂界，向着地府飞去。
<br/>　　元始眼中精光一闪，默默道：“不经历万千轮回，如何悟得轮回，不入三千红尘，如何洗涤天心，不游无尽苦海，如何超脱苦海？”
<br/>　　看见众多新生魂魄进入幽冥地府，元始便没有再多管其他。地府每日都会有新生魂魄产生，魂魄并不一定只有生灵死亡后才会产生。如今多了他这千万新魂，说多不多，说少也不少，地府也不会吃多了去管这些事。
<br/>　　这些新生魂魄将会在三生大帝的有意或无意的安排下，或进入六道轮回进行转生，或进入幽冥界，开始鬼的生涯。
<br/>　　他们将会无止境的轮回下去，或为人，为草木虫鱼鸟兽，为神，为魔，为佛，为仙，为鬼，为修罗，更甚至会成为一块石头，一滴水，一块土，一缕火……
<br/>　　他们中，有的甚至会魂飞魄散，消散于天地间，而有的或许会长生，会普普通通，但不管怎样，他们都不会知道他们的上一世。
<br/>　　直至未来的某一天，等待一个无上存在的召唤，他们将一一归来，无尽经历，无尽轮回，无尽领悟，终将归一……
<br/>　　做完这些，元始神色不变，这些魂魄，对于他的修为，没有丝毫损失，或许他早已开始准备，对于领悟灵魂大道的他来说，这是微不足道的事情。
<br/>　　元始此刻也并没有继续修炼，他将目光投入洪荒大地之中，一个人族中型部落内，一间简陋的房屋内。
<br/>　　那里，有他感兴趣的人，或者是事情。
<br/>　　洪荒继神农尝百草，开创《神农百草经》，修为大进之后又过了数百年，神农证得地皇之位，禅位于有熊氏公孙轩辕，自己飞入火云洞中，镇压人族气运。
<br/>　　神农终究是没有天皇伏羲那般气运，最终只是成就半步帝境的地皇，而没能完全破入极道帝境，证得混元果位。
<br/>　　公孙轩辕继位人族共主之位后，开始大力发展军事力量，念人族部落不一，文明不同，过于分散，欲一统人族，结束人族纷乱之态。
<br/>　　但人族部落众多，人心不一，终究是难以驯服，经过连年征战，最终只剩下了两方强大的势力，一方是人族公孙轩辕氏领导的黄帝部落，一方是魔神蚩尤领导的九黎各部落。
<br/>　　经过一连番的战争，因公孙轩辕氏占据大义，有着各路仙神的帮助，一路上由弱逐渐开始壮大，吞并诸侯，斩妖除魔，最终占据上风。九黎蚩尤终究是靠强大的战力使得对方屈服，战力也只是暂时的，强大的战力也使得他内心野心的膨胀，最后也只能一步步丧失人心，落入下风。
<br/>　　最终的决战，已经不远了。
<br/>　　公孙轩辕一方势力中，一个中心部落的一个简陋屋中，一个满脸沧桑的老者形象的人从屋中走了出来，在他的嘴里，有些痴巅的说着：“我成功了，我成功，我终于成功了！哈哈！”
<br/>　　他的声音越来越大，也越来越激动，显示着他内心的喜悦。
<br/>　　老者只有四五十岁，但他的模样却已经是百八十岁的形象了，在这武道盛行的人族之中，这种情况显然是极为稀少的。显然，他将自己的全部精力投入到了其他事情上，没有时间来修炼武道。
<br/>　　老者名为仓颉，出生在人族部落之中，从小就具有一双敢于发现的眼睛，他感念人族各部落文明的混乱，导致人族的纷乱，于是就想要发明一种各部落之间能够共同交流的东西。
<br/>　　这时，他想到了远古时期妖师鲲鹏创妖文，传妖道一事，他深受其影响，决定创造一种万族使用的文字，特别是人族。
<br/>　　于是，他感悟天地，游走大地，观察万物，记录众生，回到部落中，整理分析，模仿天地万物之形态，于今日，终于创造出了一种特别的文字——甲骨文！
<br/>　　来到人族部落中央，仓颉取出一块巨大的骨匾，手持一柄锐利小刀，坐于中央道台之上。
<br/>　　“轰！”
<br/>　　仓颉柔然身躯一震，一股庞大的气势从体内冲天而起，这股气息不似武道那般霸道，也不像仙道那般缥缈，更不像妖道那般诡异，而是蕴含着一股天地正气，宁静，祥和，如春风拂面，如细雨无声，似有天地至理，又有无穷智慧蕴含在其中。
<br/>　　这一刻的仓颉，哪还有老者的颓废，这一刻，他就是一个充满智慧的智者！
<br/>　　“吾，人族仓颉，有感万族文明交流不易，感悟天地万物，为造福万族，今日造字功成，传于众生！”
<br/>　　仓颉仰头望天，大声喊道，声音如同天地之音，又似大道之声，传遍洪荒大地，上彻九霄，下贯九幽，浩大无边。
<br/>　　……
