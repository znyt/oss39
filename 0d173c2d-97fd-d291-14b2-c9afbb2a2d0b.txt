
.LA，最快更新天命赊刀人最新章节！<br/>　　<br/>　　丁武跑了，用回老家去看望他妈当借口跑了，一听王惊蛰提亲的对象可能是李世民不知道多少代孙女他就感觉自己腿有点软，然后义不容辞的把友谊万岁抛在了脑后。<br/>　　<br/>　　王惊蛰之后跟林羡鱼说，你什么时候把丁武泡到手了，我什么时候就答应为你家镇宅，然后王惊蛰也跑了，他挺怕这姑娘一直跟在自己屁股后面不走，毕竟他要去陇西提亲了，林羡鱼跟着算怎么回事啊。<br/>　　<br/>　　他得为聘礼的事着急了，提亲的日子迫在眉睫，耽误不得。<br/>　　<br/>　　提亲聘礼这种事，几千年来在中华上下的历史中都是一直存在的，古时可能送的是猪马牛羊，后来就是绫罗绸缎金银翡翠，到了现在不是房子就是车，总之你提亲的时候得带点什么。<br/>　　<br/>　　但巧了，王惊蛰就属于那种两手空空一穷二白什么也没有的，除了背后那包菜刀和剪刀，但拿这玩意儿当聘礼也不合适吧？<br/>　　<br/>　　“这可怎么办呢，得想个法子混点聘礼啊，不然空着手去总归不是太好的·····”王惊蛰惆怅的感叹了一句。<br/>　　<br/>　　京城东三环南路潘家园桥西南，王惊蛰溜达在大街上，这里是老京城最鱼龙混杂的地方，特别是到了后半夜的时候，这里俗称叫鬼市，专门卖一些不知道真假的老货，要是真够眼力的话在这里淘到的东西用一句话来形容就最贴切了，叫一切都有可能。<br/>　　<br/>　　此时已近黄昏，王惊蛰在潘家园大街上来回的溜达了两圈之后挑了一处挺偏僻的角落然后席地而坐，打开包从里面拿出一块红布铺在了地上，然后摆放出了几把菜刀和剪刀，又用捡来的粉笔头在前面写了一行字。<br/>　　<br/>　　“只赊，不卖”<br/>　　<br/>　　字写的很不错，龙飞凤舞铿锵有力。<br/>　　<br/>　　过了会，王惊蛰可能觉得光凭这行字可能不太吸引眼球，也表达不出自己的意思，于是在后面又另外加了一句。<br/>　　<br/>　　“只赊不卖，或者以物换物，懂的来”<br/>　　<br/>　　这是正统赊刀人走民间的标准把式，随便在哪出个摊子摆放几把菜刀，然后也不吆喝叫卖，不像算命先生似的，碰见个路过的就上前一把拉住人家，告诉他说你最近要有血光之灾了，你过来我给你算一卦我有破解之道，也不会说小伙子我看你骨骼清奇马上就要发达了，来我送一句话祝你鹏程万里什么的。<br/>　　<br/>　　赊刀人是从来不会叫卖的，他们只算有缘人。<br/>　　<br/>　　而且这一次摆摊，王惊蛰想的是用一把菜刀，来换取一份能够拿得出手的聘礼。<br/>　　<br/>　　摊子摆好之后，半天了都没有人上前过问，主要就是地上那几把菜刀和剪刀卖相实在太一般了，和现在的王麻子，张小泉出的菜刀简直都不是一个层次的，想买菜刀的都不一定有心思问一句。<br/>　　<br/>　　王惊蛰也不着急，背靠着墙闭着眼睛打盹，以往赊刀人开张十天半月才接一单生意的时候也有，毕竟他们不是真卖刀的。<br/>　　<br/>　　“啥意思，赊刀，不卖？”王惊蛰快睡着的时候，感觉有人走到自己面前蹲了下来。<br/>　　<br/>　　他半睁开眼睛扫了一眼又闭上了，摊子对面蹲着个四十多岁的中年，穿着倒是挺得体的，但说话的时候就喷出一股酒味来，一看就是没少喝，明显是过来闲逗闷子的。<br/>　　<br/>　　看王惊蛰不吭声，中年从身上掏出钱包抽出一张一百的扔在了地上：“买一把菜刀，我还就不信了，钱都扔这里了，你几块钱一把的菜刀还能真不卖么？”<br/>　　<br/>　　王惊蛰连眼皮都没抬，抱着胳膊继续打盹。<br/>　　<br/>　　“哎？”中年看着地上的一百块钱愣了愣，又抽出一张扔在了地上。<br/>　　<br/>　　王惊蛰依旧不搭腔。<br/>　　<br/>　　“啪”又是一百。<br/>　　<br/>　　“啪······”<br/>　　<br/>　　这中年一共往地上扔了五百鲜红的大票，王惊蛰始终都没睁开过眼睛，嘴里就嘟囔了两个字“不卖！”<br/>　　<br/>　　“你当这是屠龙宝刀呢？我看他么杀个鸡都费劲·····”中年明显被整出脾气来了，将钱包里的钱全都扔在了地上，零散的加起来能有一千多了：“我就非得要买你这把菜刀了，你到底卖是不卖？”<br/>　　<br/>　　被中年这么一搅合，意想不到的效果初显了，摊子前面围了不少看热闹的人，这里有一部分是闲着没事过来随便逛逛的，也有不少是附近卖老货的摊主，也有一部分人是带着自己家的老物件过来兜售的，看见有人拿出一千多块钱要买一把破菜刀但人家还不卖的时候，路过的人都好奇了。<br/>　　<br/>　　“这是炒作呢？”人群里有人呲牙笑了，觉得王惊蛰这个路子挺吸引眼球的，把这中年当成了托。<br/>　　<br/>　　“只赊不卖？或者以物换物？这破刀谁愿意要啊，傻子才会换呢·······”<br/>　　<br/>　　王惊蛰也不解释，被人吵的睡不下去了后，就靠在墙上点了跟烟随意的抽着，眼神扫着人群里的人，然后落在了几个带着老物件的人的手里，这帮人手中拎着的东西千奇百怪，可能有乾隆的鼻烟壶，雍正帝的夜壶，就是有人跟你说他手里拿的是唐寅的真迹都有可能。<br/>　　<br/>　　有人不解的蹲在地上，拿起一把菜刀摆弄了几下后，诧异的说道：“也不是古物啊，挺寻常的东西，历史最多超不过百年，哪有值钱的地方啊”<br/>　　<br/>　　王惊蛰看人多了，第一次主动开口说道：“刀不值钱，也不卖，我只赊给有缘人”<br/>　　<br/>　　“啥叫有缘的人？”<br/>　　<br/>　　这时候人群外面溜达过来个六七十岁的老头，穿着内联升的纯手工布鞋和隆庆祥的褂子，看见这边围了不少人后就哼着小曲挤了进来。<br/>　　<br/>　　这老头一进来王惊蛰的眼神就瞟了过去，首先就看见了对方身上的隆庆祥褂子还有脚下的内联升布鞋，因为他身上穿的藏青色长袍也是隆庆祥出的。<br/>　　<br/>　　早几百年，隆庆祥是专门为皇帝王爷和达官贵人做衣服的，到了近代的话，隆庆祥的衣服也都是量身定制的，市面常见的虽然价格不菲但也能买得到，不过隆庆祥有一种衣服是不售卖的，就是专供，这种衣服一年也出不了多少件，是由隆庆祥大掌柜级别的人亲手定制的。<br/>　　<br/>　　所以这衣服很少见，只有能穿上的人才认出来，因为在王惊蛰长袍的下面还有老头褂子的袖口上都有个隆庆祥的特殊标志。<br/>　　<br/>　　妥了，两个老少爷们瞬间对眼了，他俩都认出了对方身上的隆庆祥特制款。
