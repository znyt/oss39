　　“好了，就这么定了！”苏莫站了起来，直接拍板，不容两人反对。
<br/>　　“这件事就交给你们二位了，记住，一定要以最快的速度，将消息传到黄泉魔宗。”
<br/>　　苏莫知道两人肯定有办法，能最快速的传递消息。
<br/>　　想到黄泉魔宗随时可能杀来，他又出声叮嘱，道：“消息传的越快越好，越快苍穹神宫才能越安全。”
<br/>　　“这……！”帝释和海明江面露迟疑之色，苏莫这是铁了心要邀战巫森了啊！
<br/>　　沉默了一番，海明江开口道：“这样吧！苏宫主，我们为你传递这个消息，不过，你不用去应战，趁着这个时机，将苍穹神宫所以人转移，然后隐藏起来。”
<br/>　　苏莫闻言面露沉吟之色，海明江的话，不失为一个方法。
<br/>　　这对他来说很简单，只要将所有人全部收进虚界神图便可。
<br/>　　不过，苍穹神宫的山门，他却是带不走，而且他也不可能隐藏起来。
<br/>　　因为他还要保护东海蛟龙一族，他若躲藏起来，自然能安心修炼，继续提升，但东海蛟龙一族就会遭受灭顶之灾。
<br/>　　东海蛟龙一族对他有大恩，他不可能舍弃东海蛟龙一族。
<br/>　　最主要的是，就算是与黄泉魔宗宗主一战，他也不惧怕，就算真的不敌，他也不至于被对方击杀，逃命的本事还是有的。
<br/>　　“海阁主，我意已决，不用再说了！”
<br/>　　苏莫摇了摇头，道：“事不宜迟，你们速速去办吧！”
<br/>　　言罢，苏莫不再多言，径直的离开了苍穹神殿。
<br/>　　海明江和帝释面面相觑，最后只能无奈的离开，按照苏莫的要求，前去传递消息。
<br/>　　苏莫离开苍穹神殿之后，便去找到了金一等傀儡人，询问了一番炼器材料的事情。
<br/>　　上次，他也让金一等傀儡人，为他准备炼制本命灵剑需要的材料。
<br/>　　这些材料早已准备好了，苏莫拿到了炼器材料之后，便径直离开了苍穹神宫。
<br/>　　离开了苍穹神宫之后，他便向东方飞去，准备前去东海。
<br/>　　苏莫御剑而行，速度奇快无比，如同一道飞驰的电光，瞬息千里。
<br/>　　他要尽快赶到东海龙宫，以免对方遭到黄泉魔宗的攻击。
<br/>　　至于苍穹神宫，他不是太多担心，毕竟浑天洛水大阵非同小可，就算黄泉魔宗真能破开，也不是短时间能做到的事情。
<br/>　　在苏莫前往东海之时，他要约战黄泉魔宗宗主巫森的消息，也在苍穹大陆上传开了。
<br/>　　这是一个无比劲.爆的消息，在中洲大地上直接炸开，并以难以想象的速度，向四方传播。
<br/>　　这个消息就是，苏莫已经拥有了武帝级的实力，要挑战黄泉魔宗宗主，巫森。
<br/>　　这个消息是在是太劲.爆了，黄泉魔宗宗主巫森，那可是刚刚突破到武帝之境，已经成为了苍穹世界无敌的存在。
<br/>　　而苏莫居然在这个时候，主动向其挑战！
<br/>　　苏莫有了武帝境的实力？真的这么夸张？这也太快了吧！
<br/>　　这件事情，无疑于九天惊雷，轰动了天下，震动了亿万武者，一时间整个苍穹世界彻底沸腾了。
<br/>　　各大城池、每一条大街小巷、每一家客栈酒楼，只要是有人的地方，便有人在讨论此事。
<br/>　　这个消息的传播速度非常之快，几乎在苏莫刚刚到达东海之滨之时，便已经传遍了整个苍穹大陆，并向四方海域传播。
<br/>　　本来，有很多大势力，都在考虑向黄泉魔宗示好了。
<br/>　　毕竟，巫森成为了苍穹世界唯一的武帝，已经无敌于天下，任何势力都要为自身考虑了。
<br/>　　不过，听闻了这个消息之后，所以的势力都暂停了动作，准备先静观其变。
<br/>　　虽然没有人认为苏莫是巫森的对手，但苏莫敢和巫森一战，想必也有些底气。
<br/>　　若是苏莫能在巫森手中不死，他们与黄泉魔宗交好，肯定会遭到苏莫的报复。
<br/>　　殷家就是最好的例子，几百万族人尽皆惨死。
<br/>　　所以，听闻了这个消息之后，所以准备有动作的势力，都不得不停止了动作，
<br/>　　这个消息传遍了天下，自然也传到了黄泉魔宗。
<br/>　　紧接着黄泉魔宗便有消息传出，巫森应战。
<br/>　　于此同时，黄泉魔宗更是昭告天下，待巫森与苏莫一战之后，所有势力必须臣服，否则杀无赦。
<br/>　　黄泉魔宗此举，嚣张不可一世，简直就是直接向整个苍穹世界的大势力下命令了。
<br/>　　而且，很显然，黄泉魔宗根本没把和苏莫一战放在心上，已经认为必胜无疑了。
<br/>　　此消息一出，天下再次哗然，沸腾一片，很多大势力的人都是愤怒不已。
<br/>　　但他们可不敢和黄泉魔宗叫板，只能寄希望于，苏莫能打败巫森，虽然这个可能性非常的低。
<br/>　　一时间，整个苍穹世界，亿万武者都充满了期待，期待苏莫和巫山一战。
<br/>　　这是苍穹世界最巅峰的战斗，绝对的巅峰之战。
<br/>　　胜者，便是苍穹世界无敌的存在，便是苍穹世界之尊。
<br/>　　无数的武者，离开了家族、离开了宗门、离开了城池，向约战之地帝玄山脉而去。
<br/>　　……
<br/>　　浩瀚无垠的东海之上，苏莫急速飞行，他已经快要到达东海龙宫所在的海域。
<br/>　　一路风平浪静，又过半日之后，他如愿的到达了东海龙宫。
<br/>　　幽幽海底，苏莫身形落在了东海龙宫之前，只见眼前的龙宫，被一座大阵笼罩。
<br/>　　苏莫见此，顿时松了口气，看来蛟龙一族无恙。
<br/>　　苏莫的到来，直接就惊动了东海龙王，对方亲自将他迎进了龙宫之中。
<br/>　　恢弘的龙殿之中，龙海龙王和苏莫分宾主落坐。
<br/>　　“苏莫，你居然要与巫森一战，是不是有些冲动了！”东海龙王开门见山，直接询问苏莫约战之事。
<br/>　　“龙王前辈已经知道了？”苏莫闻言愕然，这海明江和帝释两人，将消息传递的这么快？
<br/>　　他才刚到东海，消息居然就已经传到了龙宫！
<br/>　　苏莫不知道的是，海明江和帝释传递消息，全部用的是传讯符。
<br/>　　两人作为苍穹世界的巅峰强者，熟人非常的多，在传讯符的传讯范围之内，他们能将消息传送给很多人，然后再通过别人，将消息再次传递出去。
<br/>　　如此周而复始，想将此事传遍天下，根本要不了多长的时间。
