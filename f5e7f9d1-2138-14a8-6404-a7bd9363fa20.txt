　　“呵呵！小小进步不足挂齿！”苏莫面带笑容，谦虚的说道。
<br/>　　“苏宫主，久仰大名！”这时，天涯海阁阁主海明江站了起来，向苏莫微微抱拳。
<br/>　　现在的苏莫，整个苍穹世界已经没有任何人敢小觑了，就算是天涯海阁的阁主，面对苏莫亦是客客气气，一副平辈论交的态度。
<br/>　　“前辈是？”苏莫望向老态龙钟的海明江，面带疑惑之色，他并不认识对方。
<br/>　　不过，苏莫心中已经猜测出了对方的身份，但还是要确认一番。
<br/>　　“老朽天涯海阁阁主海明江！”海明江自我介绍道。
<br/>　　“原来是海阁主，幸会！”苏莫向对方抱拳，客气的说道。
<br/>　　“一直听闻苏宫主少年英才，天赋盖压万古，今日一见果然名不虚传！”海明江口中吹捧，将苏莫抬的非常高。
<br/>　　“呵呵！海阁主折煞晚辈了！”苏莫轻笑着摇了摇头。
<br/>　　“苏宫主，你将要大婚，我天涯海阁作为苍穹神宫的盟友，自然也带来了贺礼，请笑纳！”
<br/>　　海明江手掌一翻，掌心中出现了一枚储物戒指，随即储物戒自动飞到了苏莫的身前。
<br/>　　苏莫拿起储物戒，意念一探，随即便将储物戒收了起来。
<br/>　　对方的贺礼，着实不少，单单是极品灵石便有三十多亿，其他的灵草灵药、兵器的等宝物也是奇多。
<br/>　　苏莫没有拒绝，他现在急缺修炼资源，这些对于他还说虽然是杯水车薪，但聊胜于无嘛！
<br/>　　“多谢海阁主了！”苏莫向对方道谢。
<br/>　　两人寒嘘了几句之后，苏莫略作沉吟，脸色凝重了下来。
<br/>　　他目光扫视海明江和帝释，沉声道：“两位，即然我们三方结盟，便意味着要共同进退，有些事情我要告知二位！”
<br/>　　“苏宫主请讲！”海明江道。
<br/>　　帝释没有开口，但目光却是凝视着苏莫，等待他继续说。
<br/>　　两人看到苏莫略显凝重的脸色，都知道估计有什么重要的事情。
<br/>　　苏莫的眸中陡然迸射出了凌厉的光芒，他声音洪亮，道：“我苍穹神宫，欲一统苍穹世界，所以，待这次大婚之后，便要准备动手了，还需要两位的辅助！”
<br/>　　众人闻听苏莫之言，顿时心中狂震，纷纷口中倒吸凉气。
<br/>　　一统苍穹世界？
<br/>　　苏莫的胃口，未免太大了吧！
<br/>　　若说几年之后，苏莫有了天下无敌的实力，想一统苍穹世界，还在情理之中。
<br/>　　但目前，苏莫虽然强大，但想一统苍穹世界，可能性并不大。
<br/>　　帝释和海明江沉默了起来，眸中均是闪过思索之色，他们也已经猜测到了，苏莫定然会想办法争霸天下。
<br/>　　从苍穹神宫选在这个时间点出世，就能看得出来，但两人也没想到，苏莫如此的心急。
<br/>　　“各位，想必你们也知晓，苍穹世界大劫将至，试炼者将要到来，只有集合整个苍穹世界的力量，才能更好的应付此次劫难，所以，一统苍穹世界刻不容缓！”
<br/>　　苏莫眸光如电，朗声说道，此地均是帝玄宫和天涯海阁的高层，他也没有什么避讳，将自己的目的说了出来。
<br/>　　“苏莫，你准备如何动手？”帝释沉声问道。
<br/>　　对于大劫，众人当然知道，试炼者来到苍穹世界，目的是为了掠夺资源，以及收集武者的武魂。
<br/>　　没有人想成为试炼者的猎物，但对抗试炼者，众人心中根本没有底气，因为这简直是以卵击石。
<br/>　　以往试炼者降临，苍穹世界强者的应对之法，就是尽量躲藏，甚至将宗门都暂时解散。
<br/>　　这个方法虽然很窝囊，但不少强者都能顺利的躲过一劫，所以，虽然大劫将至，有些人并没有多大的压力。
<br/>　　到时候躲藏起来便是，运气好的话，便能安然保命！
<br/>　　听闻帝释的询问，苏莫沉声道：“首先诛灭黄泉魔宗，只要灭了黄泉魔宗，以我们的实力，便能轻易扫荡天下。”
<br/>　　“黄泉魔宗！”众人闻言心中一凛，同时也理解苏莫的想法。
<br/>　　黄泉魔宗实力最强大，而且又与苏莫有大仇，若是灭了黄泉魔宗，其它势力除非联合起来，不然只有归降的份。
<br/>　　“苏宫主，据我天涯海阁得到的消息，黄泉魔宗最近与北堂世家、殷家以及独孤世家走的较近，估计也是在密谋合作，想灭黄泉魔宗，太难太难！”海明江道。
<br/>　　“北堂世家、殷家、独孤世家！”
<br/>　　苏莫闻言眼眸微眯，他上次从阴霆口中得知，已经有两大超级势力与黄泉魔宗合作，难道不是两大势力，而是这三大势力。
<br/>　　“海阁主不用担心！”
<br/>　　苏莫微微摇头，道：“就算有这三大势力的帮助，我们也丝毫无惧，现在我已经有了堪比准帝的实力，要不了多久，就能有真正武帝强者的实力！”
<br/>　　其实苏莫本想等一短时间再动手，毕竟，等他有了武帝境的实力，诛灭黄泉魔宗还不是轻而易举。
<br/>　　但是，他不想再等了，因为黄泉魔宗也在策划，说不定就率先动手了！
<br/>　　“武帝境的实力！”众人闻言心中一震，个个眸中露出惊骇之色。
<br/>　　若是苏莫有此实力，整个苍穹世界，将无人再能阻挡。
<br/>　　不过，想到苏莫的成长速度，众人倒是没有太多置疑。
<br/>　　帝释和海明江对视一眼，无声交流了一番，随即对苏莫道：“苏莫，此事太过重大，还是容我等考虑几日吧！”
<br/>　　对黄泉魔宗出手，此事太过重大，弄不好会导致帝玄宫和天涯海阁灭亡，所以，帝释和海明江不得不郑重考虑。
<br/>　　“无妨，给你们几日时间！”
<br/>　　苏莫点了点头，随即心中沉吟，若是不给两人一点甜头，怕是两人心有顾虑。
<br/>　　“这样吧！你们只要全力辅助苍穹神宫，一人一门帝级中品功法！”苏莫许诺道。
<br/>　　“帝级中品功法！”众人闻言心中再次一震，而帝释和海明江两人，则是眼眸中闪过璀璨的精光。
<br/>　　这话若是别人说出来，众人根本不会相信，但苏莫说出此言，众人没有丝毫怀疑。
<br/>　　帝释和海明江两人，心中均是暗暗激动了起来。
<br/>　　帝释之所以和苏莫结盟，最大的原因就是想从苍穹神宫得到一门帝级中品的功法，如此他将来冲击武帝之境有望。
<br/>　　而海明江，对于帝级中品功法的渴望，比帝释还要迫切，因为他的寿元已经不多了。
<br/>　　海明江已经九百岁了，而武皇境强者，即便是准帝，也活不到一千岁。
<br/>　　他的寿元不多了，所以他对于帝级中品以上的功法，更为渴望，因为只有修炼更高等级的功法，他才有希望在有限的生命里，突破到武帝之境。
<br/>　　一旦突破到武帝境，他的寿元便能大增，轻松活个两三千年。
