感觉到怀里女孩的乖顺，陆寒霆身上的怒气消了一大半，他伸出健臂抱住了她纤柔的身体，将她护在自己的怀里，“夏夕绾，你怎么这么喜欢惹事，你能不能消停一会儿？”
<br/>
<br/>        夏夕绾突然想到了什么，她抬起澄亮的水眸看着陆寒霆，“刚才你过来的时候看到了什么？”
<br/>
<br/>        如果这里可以让人看到心底最渴望的幻象，她看到的是妈咪的宝箱，那他看到的是什么？
<br/>
<br/>        陆寒霆垂下英俊的眼睑看着夏夕绾，“什么我看到了什么，我看到的是你啊，我看到你奔向悬崖了！”
<br/>
<br/>        他看到的是她？
<br/>
<br/>        夏夕绾一震，他看到的是她，难道是因为他心底最渴望的是她，还是说他独独不受这个针法干扰，这个困不住他？
<br/>
<br/>        “想什么呢，在我面前还这样分心，恩？”陆寒霆不悦的伸手捏了捏她水嫩的小脸蛋。
<br/>
<br/>        夏夕绾迅速意识到自己还像只八爪鱼般粘挂在他的身上，她迅速松了手，往后退了好几步，拉开了彼此的距离。
<br/>
<br/>        陆寒霆的大手僵在了半空，他眸色深了深，“夏夕绾，你真够现实的，有危险就黏在我怀里，没危险了就一脚将我踹开。”
<br/>
<br/>        “…我不是这个意思，你回去吧，我还要找宝箱。”夏夕绾转身就走。
<br/>
<br/>        但是陆寒霆快速的伸出健臂一把箍住了她盈盈一握的纤软腰肢，将她用力的扯进了自己的怀里。
<br/>
<br/>        夏夕绾猝不及防的向后跌去，软若无骨的身体撞进了他结实的胸膛里，像撞在一堵墙上，她当即疼红了双眼，“陆寒霆，你…”
<br/>
<br/>        “闭嘴！”陆寒霆凶了她一句。
<br/>
<br/>        夏夕绾迅速察觉到了他的异常，她顺着他的目光向前看去，这一看倒吸了一口冷气，只见前方的黑暗里多出了两双绿油油的眼睛，然后两头狼闯入了视线里。
<br/>
<br/>        嗷呜。
<br/>
<br/>        两头狼盯着他们，发出了森然的嚎叫。
<br/>
<br/>        这是夏夕绾第一次看到狼，她吓得双腿一软，两只小手紧紧的抱在了陆寒霆精硕的腰身上，死也不撒手了。
<br/>
<br/>        陆寒霆单手护住了她莹润的肩头，将她搂抱在怀里，这个女人的现实程度已经一再刷新了他的认知。
<br/>
<br/>        “陆寒霆，我们会死在这里吗？”夏夕绾问。
<br/>
<br/>        陆寒霆戒备的看着那两头狼，“不知道，我生，你就生，如果我死了，你就跟我一起死。”
<br/>
<br/>        夏夕绾抬起小脑袋看着他，“为什么没有第三个可能性，你死，我生，英雄救美不都是这样的吗？”
<br/>
<br/>        “做梦去吧！我陆寒霆从不是什么英雄，你是个不安分的，我死了，让你跟别的男人好？如果我死了，你必须给我陪葬！”陆寒霆霸道强势的不容置喙道。
<br/>
<br/>        夏夕绾心思一动，心里已经软的一塌糊涂了，两只小手紧紧的抱着他，她突然就心安下来了。
<br/>
<br/>        他在时，全世界的风雨绕过了她，往他身上倾倒。
<br/>
<br/>        这时黑暗里的那两头狼嗖一声如闪电般扑了过来，想将他们撕成碎片。
