　　第508章
<br/>　　史文恭和单雄信碰上，当真是上山虎碰上下山虎，过江龙碰上出海龙。都想着人前显贵，在蛤蟆寨众英雄面前卖弄一下本事。
<br/>　　军营中诸人都围在校场，口中连声怪叫，为二人的战斗叫好。史文恭初来乍到，单雄信也是刚来不久，对于二人的本事，谁也不知深浅。
<br/>　　而蛤蟆寨的将官，有一个算一个，都和关鱼一个脾气，将眼睛长在头顶，从来都是目中无人。
<br/>　　擂台上的二人，先以拳脚过招，打得越来越急，一个力沉势猛，一个步法出奇，交手数个回合，口中呼喝出声，却是分不出高下。
<br/>　　打得越来越急，情知一时间分不出胜败，同时跃到擂台两旁。擂台两边摆着兵器架，二人各自抽了一条木棍，又互相打在一起。
<br/>　　虽然是木棍，但二人的力气，一旦打中打实了，轻则断一两根骨头，重则有性命之忧。
<br/>　　擂台下诸人眼睛都看直了，有人伸出舌头，好久收不回来。没想到啊没想到，本以为二人武艺不如何，谁曾想二人各个都有如此惊人的武艺。
<br/>　　擂台上的二人也打得急了，用棍棒分不出胜负，抽身退后，抓住了各自的兵器。一个举起方天画戟，另一个握住金顶枣阳槊，又杂在一起缠斗。
<br/>　　刘悲挨着程大雷而立，虽然他的武艺不如何，但眼光却是有的。口中道：“大当家，这要是打得急了，谁有个闪失，都损我山寨一员大将啊。”
<br/>　　程大雷轻轻点头，眼睛也直直盯着二人的战斗，不敢有任何疏忽。
<br/>　　擂台之上，史文恭越打越心惊，他本想得明白，轻而易举击败这蓝脸大汉。为的是在蛤蟆城立威，也不让诸人小瞧自己。却是没想到，竟碰到一个如此难缠的对手。交手几十个回合，竟然奈何不了对方。
<br/>　　如此下去，岂不是令众英雄觉得我史文恭没有本事，罢罢罢，莫要怪某家心狠手辣，我先结果此人，哪怕山寨降下惩罚，估计看在自己武艺面上，也不会过多惩罚自己。
<br/>　　一刹那，史文恭杀机已动，要斩杀单雄信立威。
<br/>　　偏偏，单雄信也是如此想的。他来到山寨也有段时间，一直没有卖弄本事的机会，今天好不容易碰着一个，他怎能放过。
<br/>　　罢罢罢，发狠杀了此人，反正他初来乍到，想程大雷也不会过多惩罚自己。
<br/>　　二人几乎同时动了杀机，眼睛一对，便各自明白对方心意，互相都拿出了看家绝技。
<br/>　　一个使出了霸王盖顶，将金顶枣阳槊照着对方脑门砸去。另一个来一招毒蛇出动，方天画戟扎向对方胸膛。
<br/>　　擂台之下，雅雀无声，都感觉到扑面而来的杀气。
<br/>　　“呔，住了吧！”程大雷忽然喝了一声，身子平地跃起，不偏不倚，正落在二人中央。
<br/>　　“大当家，小心！”擂台之下诸人惊叫。
<br/>　　间不容发之机，程大雷左手握住单雄信的大槊，右手拦住史文恭的方天画戟。二人都已打得急了，一刹那感觉兵器落在大山中一般，下意识都往外挣脱，却感觉如撼一座大山相仿，不动分毫。
<br/>　　如此，二人才回过神来，看清楚立在二人中央的程大雷，二人都倒抽一口凉气，刚才若是有一个不对，程大雷便要丧命在擂台上。
<br/>　　程大雷面寒如铁，忽地抽了抽眼角，长笑出声。
<br/>　　“哈哈哈。”程大雷大笑道：“二位英雄莫再动手，你们的武艺大家都已看到，留在山寨日后必有重用。”
<br/>　　如此，史文恭才对程大雷心服口服，撒开兵器，与单雄信同时跪道，口中道：“愿听大当家差遣，此心永生不二。”
<br/>　　“二位英雄快快请起。”程大雷将二人扶起，道：“你我皆是兄弟，不需行此大礼。来来来，今日山寨摆下大宴，专门请二位兄弟吃酒。”
<br/>　　暂且将史文恭安排在关鱼手下，单雄信由张肥统领，目前山寨暂时无战事。不过，千日养兵，只为一日用兵，此刻也不能松懈了训练。
<br/>　　到得晚间，城主府内大摆宴席，各军头领都到了。今天这场大宴，为的是史文恭和单雄信，而两人的武艺皆能服人，大家举起酒杯，杯杯敬给二人。
<br/>　　程大雷吃了几杯酒，听得高飞豹埋怨着，我手下也却大将可用，大当家为何偏心，不分给我一人。
<br/>　　如今山寨五路军，秦蛮的蛮字军，关鱼的鱼字军，张肥的张字军，赵子龙的龙字军，高飞豹和高飞虎兄弟的队伍合一，称为虎豹军。听得高飞豹的话，程大雷心道这又何难，为再召唤出几个不就罢了，也壮大一下军中的中层将官。
<br/>　　酒过三巡，菜过五味，和珅悄悄给程大雷打了个眼色，程大雷离开宴席，来到城主府的后院书房。
<br/>　　“大当家，你可还记得将军府的来人？”
<br/>　　程大雷点点头：“我让你查他们的来意，不知你查清楚没有？”
<br/>　　“查清楚了，我正是因为这件事，要向大当家禀报。”
<br/>　　“喔，他们是为什么来的？”
<br/>　　“大当家容禀。”和珅轻咳一声：“如今李乐天坐了龙庭，封下十八路诸侯王，凉州杨龙停也被封为凉州王。但这宋伯康是相府的人，李乐天是因将军府的支持坐上龙庭的，自然不可能看着相府的势力坐大。”
<br/>　　和珅左右看看，压低声音道：“长安传来的消息，将军府愿意支持我们，联手除掉宋伯康。”
<br/>　　“啊呀呀。”程大雷长叹一声：“和大人当真是一语惊醒梦中人，如此说，上次相府的人来见我，也是为了这件事。”
<br/>　　“大当家英明。”和珅道：“如今双方的代表都在蛤蟆城，当然，不会让他们见着面，大当家你看，咱们先和那家谈？”
<br/>　　“啧啧啧。”程大雷抿了口醒酒茶：“我这一个山贼，倒成了香饽饽，不过，依我看，他们的目的都是一样的，就是要把我当枪使，好让我们狗咬狗，一嘴毛。”
<br/>　　“那么大当家，咱们要不要见他们？”
<br/>　　“见为何不见？”程大雷道：“反正也闲着没事，就陪他们耍一耍。”
<br/>(本章完)
