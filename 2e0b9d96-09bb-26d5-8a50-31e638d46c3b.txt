　　在众弟子言语争锋相对之时，不远处，六大王宫的六位长老，已经在着手开启轮回古城了。
<br/>　　整个轮回古城，都笼罩在各种阵法之中，他们也只能开启一小部分。
<br/>　　只见六位长老，齐齐出手，各自结出复杂的符印。
<br/>　　而后，六只符印组合在了一起，形成了一个磨盘般大小的古怪符印。
<br/>　　开启轮回古城，唯有六宫之人联手，才能打开，缺一人不可。
<br/>　　符印之中，传出深邃莫测的气息，如同浩荡的大海。
<br/>　　“去！”
<br/>　　六位长老轻喝一声，便见磨盘般大小的符印，如同流光一般，向轮回古城的青铜巨门飞去。
<br/>　　嗡嗡~~
<br/>　　符印融入了青铜巨门之中，少倾，青铜巨门缓缓的打开了。
<br/>　　一片迷蒙的光芒，从青铜巨门之内照耀而出。
<br/>　　迷蒙的光芒，照耀万丈，在青铜巨门之前万丈之处，急速凝聚，形成了一个高达千丈，宽约五百丈的虚幻之门。
<br/>　　虚幻之门中，有奇异的力量在波动，玄妙而不可测。
<br/>　　“轮回古城已经开启，小辈们，天门关等着你们！”
<br/>　　天王宫长老天有为，环视众人，朗声高喝。
<br/>　　“哈哈!”
<br/>　　天尘山大小一声，一马当先，向巨大的虚幻之门飞去。
<br/>　　这巨大的虚幻之门，便是轮回三关的第一关，天门关。
<br/>　　他作为轮回神朝，百岁之内第一人，自然要当仁不让，第一个穿过天门关。
<br/>　　嗖！
<br/>　　天尘山进入了虚幻之门中，身形瞬间顿住。
<br/>　　但是，仅仅是一个呼吸的时间，其便畅通无阻的穿过了虚幻之门，身形缓缓的落在了虚幻之门后方。
<br/>　　六大王宫的六位长老，见到天尘山如此快的穿过天门关，均是暗暗点头。
<br/>　　一个呼吸的时间，已经是快到极致了，这说明天尘山的元神天赋，差不多是顶级的存在了。
<br/>　　天门关，就是检验弟子元神天赋，元神天赋越强，则通过的越快。
<br/>　　嗖嗖嗖！！
<br/>　　而后，一众六宫的弟子，纷纷身形闪掠，向虚幻之门飞去。
<br/>　　不过，并不是所有人，都能畅通无阻的穿过虚幻之门。
<br/>　　只见，有些弟子，进入虚幻之门后，很快便被弹了回来。
<br/>　　而且，人数还不少，足有五分之一的比例。
<br/>　　当然，更多的人，是轻松的穿过了虚幻之门，跨越了天门关。
<br/>　　最快的，就是一个呼吸的时间，慢的则需要三个呼吸的时间左右。
<br/>　　骆妤和姬月松两人，均是轻松的便穿过了虚幻之门。
<br/>　　姬月松所用的时间，也是一个呼吸，并不比天尘山差，而骆妤则是一个半呼吸的时间。
<br/>　　“顶级的元神天赋！”
<br/>　　苏莫没有急着动身，他的目光，看到了那名天王宫高高瘦瘦、面色古板的青年。
<br/>　　此人，所用的时间，隐隐不到一个呼吸，便穿过了天门关。
<br/>　　此人，居然是顶级的元神天赋，貌似比天尘山和姬月松等人，还要厉害，只是之前并没有人关注。
<br/>　　他适才，和沁儿聊了一番轮回三关，对于轮回三关的情况，已经了然于胸了。
<br/>　　苏莫不清楚这名青年是谁，但是，有一丝熟悉之感，他便稍微留意了一下。
<br/>　　“又为长老，那位小辈是谁？”
<br/>　　不远处，六大长老伫立在一起，也在观看着众人通过天门关。
<br/>　　地王宫的姚长老，望着天王宫那位高高瘦瘦、面色古板的青年，向天王宫天有为问道。
<br/>　　他们几位长老，一直在观察着每一位穿过天门关之人。
<br/>　　目前来说，能用一个呼吸的时间，穿过天门关之人，也就七八人。
<br/>　　比如人王宫的姬月松，天王宫的天尘山，兽王宫的花小雀等等。
<br/>　　而这名天王宫的高瘦青年，却是从未见过，声名不显。
<br/>　　“呵呵，姚长老很快就会知晓了！”天有为微微一笑，卖了个关子。
<br/>　　其他几位长老见此，均是好奇了起来，此子貌似极不简单。
<br/>　　“沁儿，我们快去闯天门关吧！”
<br/>　　褚彦来到了沁儿和苏莫的身前，先是瞥了一眼苏莫，然后对沁儿说道。
<br/>　　“嗯！”
<br/>　　沁儿点头，长身而起，随即对苏莫道：“段惊天，我们也去吧！”
<br/>　　苏莫闻言，便也站了起来。
<br/>　　“段惊天，这天门关你可别闯不过去啊！”褚彦面带讥讽之色。
<br/>　　看到沁儿一直和苏莫坐在一起，他心中非常的不爽。
<br/>　　然而，苏莫根本没理会褚彦，身形一展，便向虚幻之门飞去。
<br/>　　“可恶！”
<br/>　　见自己被无视，褚彦心中不禁暗怒，这个段惊天嘴脸太可恶了，不教训一番，实在咽不下这口气。
<br/>　　随即，他身形一闪，同样向虚幻之门飞去。
<br/>　　他要和苏莫比比，谁能更快的通过天门关。
<br/>　　一般，元神天赋越好的人，通过的越快。
<br/>　　褚彦的速度极快，几乎和苏莫同一时间，进入了虚幻之门中。
<br/>　　进入虚幻之门，一股奇异的力量，包裹住了褚彦。
<br/>　　他意念一动，用体内的元神，沟通了这股奇异的力量。
<br/>　　约莫一个半呼吸的时间，奇异的力量消失了。
<br/>　　褚彦穿过了天门关，他的嘴角，勾起了一丝笑意，因为，段惊天还没有通过。
<br/>　　虚幻之门中，苏莫感知到了那股奇异的力量。
<br/>　　然后，稍稍等待了一番，他才用万化神剑元神和不败战气元神之力，沟通了那股奇异的力量。
<br/>　　而后，感知到他两种元神的强大，那股奇异的力量，立刻消散了。
<br/>　　随即，苏莫闪身穿过了天门关，身形落在了地面。
<br/>　　整个过程，苏莫用了约莫三个呼吸的时间。
<br/>　　这倒不是他故意拖延时间，他只是有些担心，担心天门关检测出他的吞噬元神，从而闹出动静。
<br/>　　毕竟，这是轮回天尊的轮回古城，他的吞噬元神就和轮回天尊有关，不得不防。
<br/>　　好在，他的担心有些多余了，天门关不会自动检查他的元神，更不会有异像显现。
<br/>　　随即，苏莫身形一闪，便穿过了天门关，身形缓缓落在了地面。
<br/>　　这个时候，沁儿也穿过了天门关，用时一个呼吸出头，时间比姬月松稍长，但是，比褚彦和骆妤要短。
<br/>　　不远处，玄天长老眉头微皱，他特意留意了苏莫。
<br/>　　本来，他猜想，苏莫所用的时间，可能会在一个呼吸之内，没想到却多达三个呼吸的时间。
<br/>　　“哈哈哈！！”
<br/>　　就在此刻，一阵爽朗的大笑声，突然响了起来。
<br/>　　只见褚彦指着苏莫，满脸嘲弄之色，大声道：“段惊天，你也太废了吧！居然用了三个呼吸的时间！”
<br/>　　褚彦心中大爽，难得找到打击段惊天的机会，他怎么可能错过？
<br/>　　姬月松和骆妤，适才也在关注苏莫，此刻，两人也不禁微微摇头。
<br/>　　虽然说，只要能穿过天门关，元神天赋都很高。
<br/>　　但是，三个呼吸的时间，与其他妖孽一比，那就差的太远了。
<br/>　　本来，他们两人，还算重视苏莫，但是，现在不仅莞尔。
<br/>　　这个段惊天，天赋与顶级妖孽想必，还是有很大的差距。
<br/>　　沁儿微微皱眉，她可是清楚，苏莫有万化神剑元神，就算只此一种元神，也不可能用三个呼吸的时间啊！
<br/>　　褚彦的嘲笑声，不禁引得很多人注视苏莫。
<br/>　　不过，并没有人在意，毕竟，苏莫只是一品高级修为而已，不值得关注。
<br/>　　天王宫那位高高瘦瘦，面色古板的青年，轻轻瞥了苏莫一眼，眸中闪过隐晦的光芒。
<br/>　　不过，他并未有什么表示。
