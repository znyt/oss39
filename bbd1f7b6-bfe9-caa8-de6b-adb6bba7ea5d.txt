　　神树在炼化吞噬这那五件破碎的半仙器，楚风眠也是快要将这五人空戒之中的宝物全部探察一遍之后，突然他从这道寒的空戒之中，发现了一枚石头。
<br/>　　这一枚石头，呈现出一种青蓝之色，隐隐约约之间，散发着一股独特的灵力。
<br/>　　这一股灵力并不算强，可是却精粹无比，比起楚风眠凝炼出的神品灵石来，蕴含的灵力还要精粹。
<br/>　　不单单是精粹，更像是一种质变。
<br/>　　这一枚青蓝色石头之中蕴含的力量，仿佛是跟灵力截然不同的一种力量，楚风眠将这一枚石头握在手心之中，其中的一部分力量融入到了楚风眠的身躯之中。
<br/>　　楚风眠都清晰的感觉到，随着这一枚青蓝色石头之中蕴含的力量融入到身躯之中，他的境界居然是距离半步天人境，又近了一步。
<br/>　　这一枚青蓝色石头中蕴含的力量，足以来帮助楚风眠来冲击半步天人境。
<br/>　　“仙力！这是仙力！这石头，就是仙石？”
<br/>　　楚风眠仔细的感受着这一枚青蓝色石头其中蕴含的力量，立刻反应过来了，这青蓝色石头，到底是什么来历。
<br/>　　仙石！
<br/>　　神品灵石，已经是灵力凝聚的极点了，在往上的，便是仙石，以仙力凝聚而成的结晶。
<br/>　　仙力，据说都不是寻常的天人境的仙人，可以凝聚的，只有踏入到天人境第四重，仙士境界才可以真正掌控仙力。
<br/>　　而想要凝聚仙力，化为现实，至少是天人境第七重，仙将级别的强者，才可以做到。
<br/>　　正因如此，这种仙石哪怕是在天人境的仙人之中，都有着极高的价值。
<br/>　　楚风眠曾经也只是听闻过仙石的存在，这还是他第一次亲眼见到。
<br/>　　这五个人的来历，的确是惊天，楚风眠都没想到，道寒的空戒之中，居然有着仙石。
<br/>　　“我想要冲击境界，必须要吞噬大量的仙物才行，这种仙石，对我冲击境界，有着大用！”
<br/>　　楚风眠心神一动，急忙的打开灵识，再度寻找着这种仙石的踪迹。
<br/>　　楚风眠感觉的，他想要突破境界，仙石有着大用，一旦是让楚风眠得到大量的仙石，楚风眠就可以借此，来突破到半步天人境。
<br/>　　楚风眠打开灵识，全力寻找着，就在这五人的空戒之中，接连又寻找到了数枚仙石。
<br/>　　这些仙石，都是被隐藏在了空戒最深处的角落之中，更是在上面被布下了密密麻麻的无数禁制。
<br/>　　楚风眠第一次探察，许多都没有发现，还是这一次仔细探察，才终于发现了。
<br/>　　楚风眠张开手，就在楚风眠的手心之中，一共是有着九枚仙石的存在，这一枚枚仙石，都只有大拇指大小，比起灵石要小得多。
<br/>　　不过这一枚仙石蕴含的力量，乃是千万神品灵石，都无法比拟的。
<br/>　　因为其中蕴含的仙力，乃是再多的灵力，都无法比拟的，是截然不同的力量。
<br/>　　“这九枚仙石，如果吞噬下，最多也就提升百分之一的突破几率，难道说我要突破半步天人境，需要一千枚仙石？”
<br/>　　楚风眠看到这九枚仙石，感受着其中蕴含的力量，还是失望的叹了一口气。
<br/>　　这一枚枚仙石之中的仙力，的确是可以帮助楚风眠突破境界之用。
<br/>　　可是九枚仙石，还是太少了，对楚风眠来说，起码是要将这个数字，在多上百倍，才有让楚风眠成功突破的可能。
<br/>　　可一千枚仙石，楚风眠想要寻找，谈何容易？
<br/>　　整个九域，大罗天星域，可都没有仙石的踪影，楚风眠这一次还是杀了这五名九华天的天才，才从他们的空戒之中，得到了九枚仙石。
<br/>　　以这五人的身份，半步天人境的境界，居然一共才只有五枚仙石，都已经是足以证明这仙石的珍贵了。
<br/>　　只怕是对着五人来说，这仙石也是让他们未来突破境界所用的，平日里根本不舍得动用。
<br/>　　“除非是前去九华天，不然只怕是没有寻找仙石的机会。”
<br/>　　这五个人的身上，既然是有仙石，那九华天之中，肯定是有着仙石的存在。
<br/>　　可是现在楚风眠杀了这五个人，现在去九华天，无异于是自投罗网，自己找死。
<br/>　　不过这几枚仙石，也算是给楚风眠提了个醒，大罗天星域没有，也许是在其他星域之中，有着仙石的存在。
<br/>　　只要是楚风眠得到大量的仙石，就可以借助其中的仙力，突破半步天人境。
<br/>　　这也总算是给楚风眠多了一重突破的希望了。
<br/>　　“看来是要必须离开大罗天星域一趟了。”
<br/>　　楚风眠脑海之中不由的想到。
<br/>　　大罗天星域之中的宝藏，多半都被四大势力所瓜分了，现在楚风眠再去寻找，也找不到多少了，而且在四大势力的宝库之中，也没有仙石这样的仙物。
<br/>　　楚风眠只能够是离开大罗天星域，前往其他星域去寻找了。
<br/>　　当然在九域之中，还是有些可能会藏有仙石的，不然古籍之中也不会记载。
<br/>　　但是九域卧虎藏龙，以现在楚风眠的实力，都无法探察的清楚，在楚风眠看来，都是一片迷雾。
<br/>　　更何况之前大师兄剑凌的提醒，楚风眠也不得不担心天九域的问题。
<br/>　　与其是在九域畏首畏尾，还不如前去其他星域痛快一些。

<br/>　　楚风眠一股脑的将刚刚炼制的血丹，还有从这五人空戒之中所得到的那些圣药，统统交给了寒月漓。
<br/>　　楚风眠不在，剑道门真正的门主，就是她。
<br/>　　这些圣药，统统交给寒月漓保管，楚风眠也放心。
<br/>　　“你要离开？”
<br/>　　寒月漓看向楚风眠举动，轻声问道。
<br/>　　她的目光，已经是看透了楚风眠的想法。
<br/>　　“不错，我要前去沧海星域一趟，至少要两个月的时间，才可能归来，甚至时间更久。”
<br/>　　楚风眠开口说道。
<br/>　　“现在大罗天星域的四大势力，我都已经降伏，纵然是我离开，有着心魔之王坐镇，也无人可以反抗，以后大罗天星域的四大势力，都并入到剑道门之中。”
