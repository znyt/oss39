<br/>				谢老爷子抬起的那只脚一直无法跨上那阶梯，他盯着前一阶的阶梯良久，终是叹了一口气，将脚缩了回来，看向凤九，道：“看来我也只能到这里了，量力而行，我就不上去了。”<br/>　　<br/>　　他的声音一顿，看着凤九，道：“前面的路还很长，你自己小心。”<br/>　　<br/>　　“好。”凤九应了一声，这才转身继续往前走去。<br/>　　<br/>　　在后面的老者看着他步伐轻松，一迈便是一步没有停留，就仿佛没有什么可以阻挡到他一般，看着他从那些修士的身边经过，看着他的身影渐渐远去，他终是轻呼出一口气，这才往下走去。<br/>　　<br/>　　走到这里，心境发生的变化仿佛又上一层，体内的灵力气息在涌动着，不得不说，就算是没有上了登仙梯上面，但对于他来说，此行也是有收获的。<br/>　　<br/>　　凤九往上走去，越往上面，越看不见前面的路，也不知还有多少阶梯，反正未到尽头，她也就往上走着，也不知是什么原因，她看不见前面和后面的人，也听不见其他的声音，在这条路上，仿佛就只有她一人一样。<br/>　　<br/>　　步伐缓缓而行，忽的狂风吹来，吹刮得她睁不开眼睛，身体也仿佛要被吹倒一般，她稳住脚步停顿了一下，朝前面看去，只有呼呼狂风四处吹刮着。<br/>　　<br/>　　紧接着，狂风中夹带着暴雨打来，那暴雨伴随着风力打落在身上如同就如同被一把石子砸到一般，有些疼痛，身上的灵力气息仿佛在这一刻起不到作用，竟无法隔绝开那些风雨的袭击。<br/>　　<br/>　　她只知道，暴雨落在身上，墨发沾湿的贴在脸颊，身上的红色衣袍也湿渌渌的贴着身体，因身上衣裳湿透，举步更是难行。<br/>　　<br/>　　然，她的步伐没再停留，因为不知从何时开始，她身后阶梯便在消失着，仿佛只要她停下来就会摔向下方一样，因此，哪怕往上走的速度并不快，她也依旧一步一脚印的走着。<br/>　　<br/>　　也不知过了多久，狂风停了，暴雨也停了，取而出现的却是灼热的艳阳，就仿佛那个太阳就在她的头顶暴晒着她一样，灼热的气温将她身上衣服和墨发上的水份烤得一干二净，水气蒸发在天空之中。<br/>　　<br/>　　原本湿渌渌的贴在身上的衣服干了，可是，汗水渗了出来，一滴滴的往下滴着，背上的汗水也沾湿了衣服，她的口极渴，伸手想要去空间中取出水来喝时，却发现什么也拿不到。<br/>　　<br/>　　她只是怔了一下，便继续往前走着。在这条路上，仿佛没有黑夜一般，她仿佛走了很久，可就是没有看到头。<br/>　　<br/>　　她的体力消耗极大，步伐也渐渐的放慢了下来，而在这时，这条登仙梯上的天气又是一变，前一刻还艳阳高照，晒得她睁不开眼睛，下一刻却仿佛坠入了冰天雪地一般，雪花飘飘寒风呼啸。<br/>　　<br/>　　雪越下越大，冰寒刺骨，就连脚下的路也被雪覆盖了厚厚一层，口中极渴的她忍不住的弯下腰抓起一把雪在手中，然，未等她将雪塞入口中，就见雪在她手中消失了。<br/>　　<br/>　　：。：<br/>　　<br/>
