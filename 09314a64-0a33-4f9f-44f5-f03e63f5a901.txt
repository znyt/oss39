　　“嘁，你那是看到那个小妹妹快哭了才把棒棒糖还给人家的。”小云悠依旧对自己弟弟的说辞表现的很不屑，说道：“如果人家不是因为你那句话要急哭了，那根棒棒糖早就进你肚子里去了。”
<br/>　　云熙用手指点了点罗昊的腰，轻声问道：“罗小昊，你小时候是不是也这么调皮。”
<br/>　　“差不多吧。”罗昊若有所思，说道：“我在孤儿院的时候就是孩子王，连院长阿姨都管不住我！不过像小战这么大的时候，师父找到了我，并且把我接走了，送进了红旗学校。”
<br/>　　“听说师父接你的时候，还给你出了一个考题，拿一个假的包裹炸弹骗你，你想都没想就抱着那个包裹跳进了孤儿院外的河里，你当时是怎么想的？”
<br/>　　云熙好奇地看着罗昊，毕竟那个时候罗昊也只是一个七八岁的孩子，可能对“包裹炸弹”这个概念都没有清晰的认识，可就是在这样的情况，他能果断的抱着可能带有炸弹的背包跳进河里。
<br/>　　“当时也没怎么想。”罗昊说道：“我身后是我长大的地方，还有跟我一起玩耍的小伙伴，我又是他们的‘老大’，有事情当然是要我站出来承担！”
<br/>　　云熙脸上挂着笑容，握紧了身边男人的手。能遇见他，是自己最大的幸福。
<br/>　　推着一辆手推车，带着两只小的买了鱼、鸡、蔬菜后，经过冷冻柜的时候，姐弟俩瞪大着眼睛看着里面的冰淇淋。云熙走上前，打开冷冻柜拿出两个冰淇淋递给姐弟俩，说道：“一人只能吃一个！”
<br/>　　“好！”
<br/>　　姐弟俩兴高采烈的点头。
<br/>　　云熙平时很少对他们发火，可是云熙说得话，姐弟俩是肯定会听的，这也可能是姐弟俩从出生就一直是云熙带着的关系，知道自己妈妈不容易，所以从来不会跟云熙执拗。
<br/>　　回到家，让姐弟俩自己在客厅里玩，罗昊和云熙进了厨房做饭。姐姐云悠从带回来的背包中拿出黏土坐起了沙盘；而弟弟罗战则是拿着一支笔在本子上画啊画啊的，并且不时在本子上涂涂改改。
<br/>　　从厨房出来的罗昊看到姐弟俩做的事情，不禁有些好奇。自己女儿强大的空间思考能力他是知道的，所以看到云悠在用黏土做沙盘，他也不奇怪，赵天宇早就有计划要对云悠进行定向的培养；罗昊好奇的是罗战在干什么。
<br/>　　走进一看，才发现罗战竟然在本子进行着一支自动步枪的设计！从外观上看，罗战画在纸上的自动步枪有点像AK，但是外形却要比AK更显得现代化，除了画出了枪的外形，本子上还标出了各种详细的数据，虽然不能跟正规的武器设计图相比，但是能在罗战这个年纪画出他手中这样的自动步枪图样可以说非常不容易了。
<br/>　　“这是你设计的？”罗昊坐在罗战身边，问道。
<br/>　　“是啊。”罗战点了点头说道：“这是我理解中的AK自动步枪。AK适应性强，但是它的设计是几十年前的，已经不适合现代特种作战了，所以我根据自己的想法做出了修改。”
<br/>　　“就连卡拉尼什科夫集团那些专家都不能解决的问题，你这小子能解决？”罗昊点了点自己儿子的额头，笑着说道。
<br/>　　罗战不服气的皱了皱鼻子，说道：“那些老毛子不行，不代表我不行！红旗学校有轻武器设计和改装这门课程，我现在先学习起来，等到了红旗学校，惊呆他们！”
<br/>　　“你别太得意了。”云悠这个时候又对罗战打击道：“红旗学校里藏龙卧虎，那是从全国挑选出来的精英，而且轻武器设计和改装是高年级的课程，能在红旗学校坚持到高年级全都是精英中的精英，所以你的自我感觉也不要太良好。”
<br/>　　云悠这点倒是没有说错，要想在藏龙卧虎的红旗学校里做到出类拔萃太难了，要知道从红旗学校出来的普通学员放到部队里，那也是骨干！
<br/>　　罗战闻言，眨巴了一下眼睛，随后握紧小拳头说道：“我一定会是最厉害的那个。”
<br/>　　看到自己弟弟充满斗志的样子，云悠微微一笑，继续摆弄着自己手上的黏土，不再说话。
<br/>　　坐在旁边沙发上的罗昊看着姐弟俩，感觉这辆小家伙就像是自己跟云熙的翻版。姐姐云悠就像她妈妈，话不多，但是却充满了智慧；弟弟罗战就跟自己一样，被人一激立刻就跟打了鸡血一样，充满了斗志，别人说自己不行，偏偏要证明给别人看自己可以做到。
<br/>　　而罗战现在所做的略显幼稚的设计图，后来还真被他捣鼓出来了一支能适应现代化特种作战的AK自动步枪，只不过这种AK全世界只有他这一把，从枪管到扳机都是订制的，当然这些都是后话。
<br/>　　吃过晚饭，带着两个小家伙在小区内散了会儿步，就让两个小的分别去洗澡，准备睡觉。
<br/>　　其实这也不需要罗昊和云熙刻意去安排，在龙隐部队生活的日子，已经让他们养成了军人式的作息时间，作息非常规律，而这也对以后两个小家伙进入红旗学校有很大的帮助，可以更快的适应红旗学校里的生活。
<br/>　　房间里空调吹着冷风，罗昊就穿着个大裤衩躺在冰丝席上，一身睡裙的云熙擦着头发从浴室中走出来，看到罗昊，嗔道：“回家了就没点‘龙刺’的样子了，也不知道把衣服穿一下。”
<br/>　　“回家了还要那么一本正经干嘛，‘龙刺’也是人，又不是真的战斗机器。”罗昊看着身上只穿了件睡裙的云熙，咽了口口水，说道：“再说了，反正一会儿还要再脱，何必这么麻烦？”
<br/>　　听到罗昊的话，云熙立刻就意识到不妙，往后退去，说道：“今天晚上我跟小悠儿一起睡。”
<br/>　　“云小熙，你现在才想到跑，是不是太晚了？”
<br/>　　罗昊从床上蹦了起来，跳下床，把正打算想要逃跑的云熙给抓了回来，仍在床上，如饿狼看着小绵羊一样看着云熙，说道：“云小熙，明天你要是还能走路，就算我输！”
