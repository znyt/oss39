　　林峰等人一离开特等区，第一件事情就是建造据点，采用野火燎原的模式，不然的话，分开行动，漫无目的的行动，就他们这点人，很可能会被吞没。
<br/>　　而且林峰也相信，肯定还有些散兵游勇在抵抗，这些散兵游勇也是一股非常厉害的力量，必须要把他们全部凝聚在一起，因此，林峰他们就第一时间内拿下了人人都知道的综合办事处，不过这一次行动也付出了几十头生命。
<br/>　　不过好在这群人没有白白牺牲。办事处已经建立了初步防线，林峰就让周华呆在办事处保护据点保护陆勇，林峰就出来寻找幸存者，让他们前往办事处聚集。
<br/>　　这批大学生是他遇到的第三支队伍，也是目前人数最多的队伍。
<br/>　　林峰斩杀掉那个七阶变异丧尸后，继续在平民区内四处奔波，不管是什么变异丧尸也好，超级怪物也罢，全都被林峰一刀砍死，一些皮厚的，几刀下去也都可以轻松斩杀。
<br/>　　最后。
<br/>　　陆勇他们就看到了陆陆续续，不下于百来人出现在综合办事处。若不是有林峰出手，其中一大半人估计都已经身死当场了。
<br/>　　第一时间就是检查有没有被感染，然后再整编成为一股生力军。这群人都是实力不错的，最弱的都是三阶，最强的乃是六阶，足足好几个。
<br/>　　“一切听陆将军安排！！”
<br/>　　每个人心中都抱着一股视死如归，不破楼兰终不还的决心。
<br/>　　陆勇非常感动，“你们都是好样的，若是这一次，基地能够夺回来，而我陆某人又可以侥幸不死，到时候请你们吃大餐！”
<br/>　　“好！那我们可要等着您的大餐！”
<br/>　　“上将请我吃大餐，哈哈，这恐怕是我一辈子都不会去想的事情吧！”
<br/>　　许多人都哈哈大笑。在这种时候还能谈笑风生，无愧于男儿本色四个字。当然了，这其中也有些女子，也是巾帼不让须眉。
<br/>　　而此时林峰还在外面找幸存者。
<br/>　　几乎是绕着平民区一大圈。林峰把一些能找的全部都找回去了。
<br/>　　可能还有不少幸存者躲起来了，但是这个时候，根本不可能再去地毯式的仔细搜索了，尤其是平民区这种本来就大街小巷，密密麻麻跟个老鼠窝的地方。
<br/>　　林峰一路杀了不下于百头丧尸回到了军部综合办事处。
<br/>　　“林峰战神回来了。”
<br/>　　陆勇看到林峰是越看越喜欢，多亏了林峰，他许多事情都根本不需要去操心了，本来这些事情绝对是足够让他操心许久的。
<br/>　　“林峰战神？”
<br/>　　许多人都是第一次听到林峰的名字。
<br/>　　“这不是排行榜上那个大神吗？”
<br/>　　“好像是战士会的。”
<br/>　　“就是那个林峰，还真是厉害啊。多亏了他，不然我现在已经死了。”
<br/>　　“没错。他也救了我们一家三口。是我们家的大英雄，儿子，快谢谢大哥哥的救命之恩。”
<br/>　　许多人都无比感激林峰。
<br/>　　“林峰战神你来的正好，我们正在讨论如何把驱逐之墙的破洞给补上。”
<br/>　　陆勇道。
<br/>　　“不过我们遇到了一些麻烦。”
<br/>　　“什么麻烦？”
<br/>　　林峰问道。
<br/>　　“要想补上这个破洞，我们肯定现在没有办法炼制出之前那样的大钢门了，只能用一些其余的材料代替，比如说石头。”
<br/>　　“必须是那种大石头，若是用人力来搬运的话，非常的消耗时间，而且搬运石头，那就是等于给怪物们当靶子了。”
<br/>　　陆勇道。
<br/>　　林峰点了点头，现阶段，倒是不需要去考虑什么搬不搬的动问题，几千斤的石头算大吧，他们现在只要是五阶上下的，都可以搬得动，尤其林峰，几万斤都不是问题，完全就是一个人形挖掘机。
<br/>　　“所以我觉得最好的办法还是用运输机来运输石头，不过运输机现在都在军用机场，那个地方也不知道被丧尸攻入了没有，若是被攻破了，那我们只能用第一种最蠢也是最为原始甚至是最危险的方法了。”
<br/>　　陆勇道。
<br/>　　“值得试一试，这样吧，我去军用机场看看。”
<br/>　　林峰道。
<br/>　　“嗯，对了，这是小李和小王，他们两个都是飞行员，这是小张小宋，他们两个都是开挖机的，在军用机场那里有许多废弃的材料，一些石料钢筋都有，本来是寻思着给军用机场周围做一个简易防御带，但是这个计划还没有实施，就遇到了眼下这个问题。”
<br/>　　陆勇道。
<br/>　　“你们几个跟我走吧。”
<br/>　　林峰扫了他们一眼，都是些三阶的。
<br/>　　“林峰，这是一百个精锐的突击战士，希望能够帮助你。”
<br/>　　陆勇认真道。
<br/>　　“林峰，虽然我也很想去，但是你也知道的，这个地方的重要性。”
<br/>　　周华道。
<br/>　　“行了，我去就我去，反正到时候，你给我的物资可不能少。、”
<br/>　　林峰道。
<br/>　　“哈哈，放心吧，到时候肯定少不了你，物资和弹药库全部都建造在地下，目前肯定保存完好。”
<br/>　　陆勇笑道。
<br/>　　林峰没有任何废话，直接带着一群人朝军用机场奔去。
<br/>　　在建造曙光基地的时候，就特别留了一大块空地，专门来修建机场，上面听的都是些运输机直升机之类的，至于歼击机这种战斗机早就已经全部被毁掉了，往往几百头变异鸟就可以干掉一个造价几十亿的先进战斗机。
<br/>　　林峰带着一群人从综合办事处杀了出来，这一百个突击队员每个都是骁勇善战之辈，其中不少人之前参加过XJ反恐，XJ这个地方，也算上国内这个大和平环境之下，唯一一个暗流涌动的地方了。
<br/>　　一群人在专业素养下，很快来到了军用机场。
<br/>　　军用机场这里，还有大概五十个看守军官，此时此刻。
<br/>　　隔着一个铁丝网竖立起来的警戒带，林峰可以清楚看到他们已经全部牺牲了，有的人成为了尸体有的人成为了丧尸。
<br/>　　在铁丝网一边，还有个巨大的口子，就好像被什么玩意给撞开的。
<br/>　　“看来有什么强大的变异怪物啊。不过好在运输机和直升机都还在。”
<br/>　　林峰松了口气，其余人也都松了口气。
<br/>　　“都小心点。”
<br/>　　林峰对身后那群跟着他的人喊了句就直接第一个进入了铁丝网。
