　　这一次七大宗门失利之后，短时间内，应该不会来找剑道门的麻烦了。
<br/>　　这也算是给楚风眠以及剑道门一些休养生息的时间。
<br/>　　剑道门如今建立起来，不过一个月的时间，现在楚风眠塑造出这一方圣地出来，剑道门弟子在其中修行的速度，将是一日千里。
<br/>　　只要是给剑道门一段时间，剑道门弟子，也不会像是现在这般的弱小了。
<br/>　　纵然是不如七大宗门，可也依然是足以成长成为九域之中的强横势力之一了。
<br/>　　楚风眠已经是将这剑源界之中的种种重新塑造，建立起新的护山大阵，剑道门的事，便是暂时告一段落了。
<br/>　　不过楚风眠心中清楚，纵然是七大宗门短时间内，不会再来攻打剑道门，但是一旦是让七大宗门喘息过来，弄清楚楚风眠的底牌之后，这七大宗门，定然还会卷土重来的。
<br/>　　楚风眠的实力越强，这七大宗门的强者，也就会越是想要杀了楚风眠，得到曾经剑道门的底蕴。
<br/>　　提升实力，才是对楚风眠来说最为重要的事情。
<br/>　　楚风眠已经是踏入了九劫古帝的境界，一身实力比起之前要强横了数十倍。
<br/>　　现在的楚风眠已经是真正足以跟半步天人境的强者，平起平坐了。
<br/>　　不过这还远远不够。
<br/>　　楚风眠击败那荒鎏金，都是耗费了不少的力气，最终还是因为剑鼎的缘故，楚风眠才可以找到机会，轰杀了荒鎏金。
<br/>　　可如果说是换做其他半步天人境的武者，哪怕是实力比起荒鎏金稍弱的五行刀圣，楚风眠想要杀了他，都没有那么容易。
<br/>　　更何况，半步天人境之中，还有着少帝那般的存在，少帝的实力，纵然是突破之后的楚风眠，都没有把握可以战胜少帝。
<br/>　　现在的楚风眠，最多只能够是在少帝的手中，全身而退，跟他抗衡。
<br/>　　不过这还是楚风眠没有真正见识过少帝的全部实力的推测。
<br/>　　身为半步天人境的强者，谁的身上，都有着不为人知的底牌。
<br/>　　像是这荒鎏金身上的剑鼎，就是他最为强横的底牌。
<br/>　　剑鼎一出，哪怕是同为半步天人境的强者，都会被镇压。
<br/>　　楚风眠还是沾了他出身自剑道门的便宜，这剑鼎毕竟是剑道门的灵器，所以楚风眠有着办法，来对付这剑鼎。
<br/>　　可如果是少帝，五行刀圣等人身上的底牌，只怕是楚风眠就没有这么好的办法可以对付了。
<br/>　　他现在的实力，还远远不够。
<br/>　　七大宗门之中，这荒鎏金的实力，也绝对不算最强者。
<br/>　　起码楚风眠可是知晓，在七大宗门之中，只怕是还会有着一些恐怖的老古董的存在。
<br/>　　荒鎏金不过是万年之前荒古门之中的天才弟子，现在便是就已经是成为了半步天人境，巨头一般的存在。
<br/>　　那若是在七大宗门之中，曾经万年之前的巨头，经历这万年之后，又该如何强大？
<br/>　　所以纵然如今楚风眠是突破了，但是对力量的提升急迫感，楚风眠丝毫没有懈怠。
<br/>　　他的实力，还不足以让楚风眠懈怠，只有不断的提升实力，才可以化解一切的危机，甚至是未来为剑道之主报仇。
<br/>　　剑鼎！
<br/>　　楚风眠看着落入到楚风眠掌心之中的剑鼎，一股灵力其中，将剑鼎收入到楚风眠的身躯之中去。
<br/>　　现在楚风眠想要提升实力，最好的办法，便是炼化剑鼎。
<br/>　　这剑鼎可是剑道门的至宝之一，被荒鎏金所得到，就足以在助他从九劫古帝的境界，一步踏入到半步天人境的境界。
<br/>　　这还是荒鎏金无法炼化剑鼎，无法发挥出剑鼎威能。
<br/>　　楚风眠可是剑道门弟子，更是剑道之主的亲传弟子，对着剑鼎，无比熟悉，荒鎏金无法炼化，楚风眠却是可以做到。
<br/>　　剑鼎之中，蕴含着无比玄妙的力量，其中的玄妙种种，楚风眠都没有办法全部了解，但是一旦是炼化了剑鼎，楚风眠的实力，绝对会提升许多。
<br/>　　楚风眠身形连连闪烁，飞入到了云羽天宫的深处去，进入到一座密室之中，楚风眠这才将剑鼎放了出来。
<br/>　　这剑鼎虽然是剑道门的灵器，但是想要炼化起来，可没有那么容易。
<br/>　　越是强横的灵器，想要认主炼化，便越是困难，这一点楚风眠也是清楚的。
<br/>　　所以楚风眠早已经做好了完全的准备。
<br/>　　一道道的剑意，在楚风眠的背后凝聚，突然化为了一枚枚金色的小剑，对着剑鼎猛然飞了过去。
<br/>　　极道剑印！
<br/>　　想要炼化这剑鼎，最基本的一点，便是需要剑道之主的力量，也就是极道剑印。
<br/>　　曾经剑道之主，炼制这剑鼎的时候，也就是为了镇压剑道门的气运。
<br/>　　所以只有剑道门的弟子，只有足以凝聚出极道剑印的人，才有资格降伏这剑鼎，让剑鼎认主。
<br/>　　这极道剑印，无比神奇，是剑道之主自己创造而出的一种玄妙禁制。
<br/>　　除了剑道之主，还有楚风眠等剑道之主的弟子之外，任何人都无法模仿的出，极道剑印的气息。
<br/>　　哪怕是在强横的剑修，甚至是上剑宗之中的那些老古董，都无法模拟的出，极道剑印的气息来。
<br/>　　也就是因为这极道剑印的原因，像是巨阙神剑，这剑鼎，虽然是落入到了其他人的手中，可其他人却是根本无法将他们认主。
<br/>　　一道道极道剑印，在楚风眠的手心之中凝聚，向着剑鼎之中打去。
<br/>　　这极道剑印，接近剑鼎，不会去轰击剑鼎之上的禁制，反而是开始融入其中。
<br/>　　这是在唤醒剑鼎。
<br/>　　那荒鎏金，为了控制剑鼎的力量，其实是动用了一些手段，让剑鼎沉睡了起来。
<br/>　　如此一来，剑鼎的力量，便是十不足二三，不过相对而言，也没有那般反抗的力量了。
<br/>　　对荒鎏金来说，他哪怕是能够得到剑鼎的两成力量，也比起一个无法控制的剑鼎要好得多。
<br/>　　如此一来，这剑鼎已经是沉睡了万年。
<br/>　　想要让剑鼎认主，楚风眠第一件事，就是要先唤醒剑鼎。
<br/>　　只有唤醒了剑鼎，才可以让剑鼎真正的认主。
