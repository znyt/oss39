　　“白。”黑无常来到白无常的尸体旁，脸色惊骇。
<br/>　　他万万没想到，自己这个老伙伴，竟然几个照面就被杀了。
<br/>　　“不要再大意了，这易霄很强，不愧是天才猎妖师。”黑无常冷冷地盯着萧逸。
<br/>　　天残地缺也一样冷眼看着萧逸，“小子，我承认你很强。但，别以为你杀了白无常就能在我们手上逃掉，我们只不过是稍微大意罢了。”
<br/>　　“易霄，你今日必死。”
<br/>　　三人冷喝一声，再度出手。
<br/>　　这一次，他们没有再大意。
<br/>　　三人没有分开，而是联手起来。
<br/>　　黑无常和天残再次放出了鬼藤和寒冰铁链。
<br/>　　地缺则在伺机而动。
<br/>　　嗖，萧逸的身影再次消失。
<br/>　　他既然之前明知那两个茶客就是杀手，却还敢出城，自然是有把握的。
<br/>　　他知道自己的实力在什么程度。
<br/>　　这时，天残地缺和黑无常三人一边操控武魂，一边大手一挥，喝道，“灵气禁锢。”
<br/>　　霎那间，方圆百米内的灵气暴走起来，并禁锢住这一小片空间。
<br/>　　本在快速移动的萧逸，顿时被停下了脚步，显露出了身影。
<br/>　　“在那里，鬼藤，寒冰铁链，捆住他。”地缺喝了一声。
<br/>　　然而，萧逸不屑地身躯一震，轻松便震碎了灵气禁锢，而后身体快速移动，再次消失在原地。
<br/>　　鬼藤和寒冰铁链再次扑了个空。
<br/>　　“该死，那小子之前连寒冰铁链都能强行挣脱，灵气禁锢困不住他的。”地缺谩骂了一声。
<br/>　　半步洞玄，毕竟不是真正的洞玄境，他们只是初步地能掌控天地灵气。
<br/>　　他们发出的禁锢，欺负一下先天八重以下的武者还行，随便一个先天九重都能轻松破开，更别说萧逸了。
<br/>　　半步洞玄，真正强大之处，在于他们体内的真气远超先天九重，自然实力也更强。
<br/>　　但，如今的萧逸，单靠体内真气，加上升龙，就有先天九重战力。
<br/>　　若加上灵器，实力早就达到半步洞玄，甚至更强一些。
<br/>　　论单打独斗，他比天残地缺、黑白无常任何一人都要厉害。
<br/>　　只不过，面对四人的联手，他才落于下风。
<br/>　　但别忘了，他还有形意五绝。
<br/>　　同为半步洞玄，萧逸使用豹形后的速度，远非这四人能比的。
<br/>　　对付这四人，他很多底牌都不必用上，就能轻松战胜。
<br/>　　嗖嗖嗖，萧逸快速移动着，脚步发出阵阵破空之声。
<br/>　　“哼。”天残冷哼了一声，“易霄，你不过是借着灵器，才有如此实力罢了。”
<br/>　　地缺冷笑道，“论真实修为，你比不过我们，只要将你束缚住，你必死无疑。”
<br/>　　“易霄，我看你能如何躲我这一招。”
<br/>　　黑无常大手一挥，冷喝道，“满城荆棘。”
<br/>　　霎那间，遍地鬼藤，完全覆盖了方圆数十米的范围。
<br/>　　嗖嗖，萧逸只能立刻后退到数十米之外，也停止了快速移动，显露了身影。
<br/>　　“武技？”萧逸眉头一皱，那让鬼藤瞬间覆盖这么一大片范围的，显然是武技，而且似乎等阶不低，起码是玄阶武技。
<br/>　　“看来要麻烦一些了。”萧逸暗道。
<br/>　　这时，天残地缺和黑无常再次攻来。
<br/>　　“满城荆棘。”
<br/>　　“你觉得同样的招式，对我有用吗？”萧逸冷笑一声，打算后退躲过。
<br/>　　然而，天残却也同时大手一挥，喝道，“天罗地网。”
<br/>　　顿时，数十条寒冰铁链凭空而现，彻底封锁了萧逸后退的道路。
<br/>　　“又是武技？”萧逸眉头一皱。
<br/>　　暗影楼的杀手，果然非泛泛之辈。
<br/>　　“小子，这下你躲不了了。”地缺瞬间出手，独脚横扫而至，“大黑风斩。”
<br/>　　“好强。”萧逸一惊，显然从地缺的这一脚上感受到压力。
<br/>　　这就是大势力武者和普通武者的差别。
<br/>　　普通武者，缺乏武技，战力弱得多。
<br/>　　大势力的武者，个个都有武技，而且武技等级不低，战力极强。
<br/>　　像如今天残地缺和黑无常三人的武技，随便一招都是玄阶，那大黑风斩，甚至是玄阶中级。
<br/>　　一道巨大的黑风斩凭空而出，并以极快的速度劈向萧逸。
<br/>　　“噬火百刃。”萧逸大喝一声，同时释放出八十道火刃。
<br/>　　轰...轰...轰....
<br/>　　八十道火刃不断的碰撞，最后以双双消散而结束。
<br/>　　大黑风斩虽然消散，但无数鬼藤和寒冰铁链可没有闲着，迅速缠向萧逸。
<br/>　　“噬火百刃。”萧逸反应极快地再次打出火刃。
<br/>　　鬼藤瞬间被火焰和爆炸轰碎，寒冰铁链也被震退。
<br/>　　这时，大黑风斩再次劈来。
<br/>　　“该死。”萧逸暗骂一声，“三人都使用武技，一个接一个地来，难缠至极，一旦我其中一道攻击没接住，要么被捆绑住，要么被攻击中，都必然陷入极麻烦的境地。”
<br/>　　“看来要速战速决了。”萧逸暗暗下了决定。
<br/>　　“噬火百刃。”
<br/>　　又是八十道火刃打出。
<br/>　　“易霄，又是同样的招数吗？”黑无常冷笑一声。
<br/>　　“再这样下去，他必然会手忙脚乱，只要一招没挡住，他必死无疑。”天残充满了自信。
<br/>　　地缺则是一道大黑风斩劈出，准备挡下火刃。
<br/>　　果然，火刃和大黑风斩再次同时消散。
<br/>　　正当天残地缺和黑无常满脸自信之色时，忽然，十数道火刃在空中划过一个弧度，拐着弯朝他们劈去。
<br/>　　“哼，这些小把戏也敢在我们面前卖弄。”
<br/>　　“八十道火刃尚且奈何不了我们，更何况区区十数道。”
<br/>　　“哼，甚至不用释放武技，我一巴掌就能拍碎。”
<br/>　　三人不屑地说着，然而，一秒后，他们的不屑，忽然化作了惊骇。
<br/>　　那十数道火刃，在即将打到他们面前时，忽然瞬间融合成一道，变成一道更为巨大的火焰。
<br/>　　“不好，是武技融合，快躲开。”三人顿时大惊。
<br/>　　然而，他们此时才反应过来，已经迟了。
<br/>　　轰的一声爆炸，火焰彻底淹没了他们三人。
<br/>　　萧逸这数月修行，可不单单是赶路，在无数场战斗中，他也在磨练着自己的实力。
<br/>　　融合武技，便是他花费时间最多的一部分。
<br/>　　融合武技，很难，没有人成功过，也没有人敢去尝试，因为失败后，力量暴走会后果很严重。
<br/>　　当初在狂血玄君的洞府内，足足融合了五十道火刃，但非常不稳定，最后在自己面前爆炸，席卷整个洞府，要不是有幽魂面具救了自己一命，自己死定了。
<br/>　　但，他既然尝试过一次，便有了经验。
<br/>　　之后，在历练时，他尝试着两道融合、三道融合、四道、武道....
<br/>　　一直到现在，他能融合十道，而且是如臂指使，不单止不会不稳定，还可以远程操控。
<br/>　　火刃的融合，威力可不是一加一这么简单。
<br/>　　就像同时释放出两道火刃，和两道火刃融合成的一道火刃，两者显然不是同一个层次的攻击。
<br/>　　威力，必然是一加一等于十。
<br/>　　两道火焰融合出来的巨大火刃，其威力不亚于分散地释放十道火刃。
<br/>　　如今十道火刃融合出来的巨大火刃，其威力便是百道火刃齐发，等同于噬火百刃大成。
<br/>　　噬火百刃虽然是玄阶低级，但火焰可是绿色武魂层次的，加上灵器的增幅，威力直逼玄阶高级。
<br/>　　天残地缺三人不过是黄色武魂，就算有玄阶中级武技，也远远比不上萧逸的攻击。
<br/>　　修为、武技、武魂等级等等，都是决定武者实力的因素。
<br/>　　言归正传，爆炸的火焰淹没了天残地缺三人。
<br/>　　数秒后，三人从爆炸中跳跃而出，满脸狼狈，口吐鲜血，显然受了不轻的伤势。
<br/>　　“该死，这个易霄是个变态吗，竟然能融合武技。”天残地缺恨恨地骂了一声。
<br/>　　“天才猎妖师，果然名不虚传。”黑无常暗暗咬牙，头一次对击杀易霄这个任务产生了没有把握的想法。
<br/>　　另一边，萧逸冷冷地盯着他们，自语道，“看来，十道火刃的融合还杀不了你们，那便试试二十道。”
<br/>　　话音刚落，他的声音再次消失。
<br/>　　“那小子想做什么？”天残地缺三人谨慎起来，不敢大意。
<br/>　　嗖，萧逸瞬间出现在他们面前。
<br/>　　“嗯？”三人先是一惊，而后大喜。
<br/>　　“小子，你敢在我们三人面前现身，找死。”
<br/>　　他们千方百计要困住萧逸，不就是想和萧逸正面战斗嘛，这样他们的胜算才更大。
<br/>　　毕竟他们三人现在可是联合起来，而并没有分散。若三人联手，同时近战对付萧逸，都打不过的话，那其它手段也就没用了。
<br/>　　三人同时发出手，鬼藤和寒冰铁链如同两条长鞭，被舞得威势凛凛。
<br/>　　地缺的黑风血镰也是刀影重重。
<br/>　　嗖嗖嗖，萧逸脚步不断闪躲，如同一条灵蛇。
<br/>　　密布的攻击，每每被他堪堪躲过。
<br/>　　“怎么回事，好诡异的身法。”三人顿时一惊。
<br/>　　在他们眼中，萧逸的身影不断变化，时而像一头火焰猎豹，时而像一条灵动的火蛇。
<br/>　　嗖，当萧逸靠近他们三人时，二十道火刃凭空而现。
<br/>　　“合。”萧逸大喝一声。
<br/>　　“又是武技融合，该死，快躲。”三人脸色大变。
<br/>　　二十道火刃，萧逸可就没办法远程操控了，只能近距离释放。
<br/>　　一道巨大火刃被凝聚而出，但非常不稳定，瞬间爆炸。
<br/>　　天残地缺三人来不及躲避，再次被爆炸淹没。
<br/>　　萧逸在脱手放出火刃时，已经马上后退，但始终还是被爆炸的余威给波及了。
<br/>　　“噗。”萧逸脸色一白，猛地吐出一口腥血。
<br/>　　而后，他擦了擦嘴角的鲜血，自信地看着后方被彻底淹没的三人。
<br/>　　当火焰和爆炸消散，露出了天残地缺和黑无常三人的身影。
<br/>　　此时，三人已经衣衫破烂，浑身是血，倒在地上大口大口地吐着鲜血。
<br/>　　虽然未死，但显然已经重伤，再无战力。
