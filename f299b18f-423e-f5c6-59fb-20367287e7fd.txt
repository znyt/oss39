　　陆玟丞连忙把胡梦浩拉进机舱，直升机立即全部起飞离开，从高空往下看还能看到无数浓烟从地缝中钻出来，不断翻滚着涌起层层黄土。
<br/>　　让狼牙情报部及暗部颇为头疼的这个曹氏研究院，竟然就这么毁于一旦，让人有种世态无常之感。
<br/>　　“你不是说半小时后才会毁么？”陆玟丞心有余悸的开口。
<br/>　　旁边的胡梦浩喘着气道：“我说的毁是指消除痕迹。”
<br/>　　陆玟丞一时间没听懂，问道：“消除痕迹是什么意思？”
<br/>　　胡梦浩随口而答：“让这整个地下研究院不复存在，恢复成寻常土地，哪怕挖地三尺都找不到任何研究痕迹，包括器皿都全部分解消失。”
<br/>　　机舱内顿时一片安静，不少人都向胡梦浩投去惊悚的目光，顷刻间将如此巨大的一整个地下研究院销毁成这种程度，这还是人干事吗？
<br/>　　沉默让机舱内陷入良久的诡异氛围，陆玟丞咽了好几次口水后，这次平复了情绪自我介绍道：“您好胡博士，我是狼牙情报部部·长陆玟丞，军衔中将。”
<br/>　　胡梦浩双眼一亮：“原来你就是陆玟丞，多谢你们情报部的帮助。”
<br/>　　陆玟丞也不废话，直接说出最关心的问题：“现在人都救了出来，但我们长官的父亲却还是在对方手中，所以任务还是没有完成。”
<br/>　　胡梦浩叹口气：“我跟楚涵达成的协议也本就是潜入这里营救楚云天，没想到最后还是失败了，而且我打听不出来他们去了哪里。”
<br/>　　“剩下的事我们情报部会继续努力。”陆玟丞拍了拍胡梦浩的肩膀：“但还是感谢您的付出，狼牙基地也有研究部门，诚心希望您加入。”
<br/>　　胡梦浩双目发亮：“我对那位提出循环论的女士很感兴趣，哦不，我是说我对循环论很感兴趣，想跟袁熙叶女士见一面。”
<br/>　　陆玟丞顿时有些神色古怪，道：“您不会不知道袁熙叶院长是我们长官的母亲大人吧？”
<br/>　　咔！
<br/>　　胡梦浩一下子楞在那里，紧接着猛然想到了什么，结巴道：“那，那楚云天岂不是？”
<br/>　　“对，他们是一家三口。”陆玟丞说着，对周围惊讶的神色已经习以为常：“不用意外，这一家子都是变·态。”
<br/>　　一架架直升机快速将这批研究人员接回狼牙，原本已经空无的基地再次热闹起来，狼牙研究部门更是暴涨好几十倍的成员。
<br/>　　袁熙叶并不知道自己丈夫的行踪和情况，楚涵早就下令将这件事隐瞒下来，但随着循环论的出现，很多事情也难以再继续当做没发生。
<br/>　　虽然并没有对世界公开，但在曹氏研究院是神秘家族罗家的秘密研究场所，一旦有新的研究命题出现，都会被第一时间送到那里，所以胡梦浩等人虽然没有去过外面，研究的进度却走在前列。
<br/>　　袁熙叶被接回狼牙基地研究院后，出乎所有人的意料，并没有对忽然出现的大批研究人员赶到意外，而是颇为大气的先做自我介绍：“你们好，我是袁熙叶，狼牙研究部的部·长。”
<br/>　　胡梦浩见到袁熙叶本人的当场手都在颤抖，其余人就更别说了，激动到下意识都没注意袁熙叶气质与美貌。
<br/>　　只是很快有一个年轻女孩被送了过来，并且还是与身份特殊的袁熙叶同乘一辆车。
<br/>　　当这个女孩出现的一瞬间，胡梦浩的神色便立即变化，激动之情不在，取而代之的是想要逃离的慌乱感。
<br/>　　她便是卫安。
<br/>　　那个楚涵给胡梦浩的承诺。
<br/>　　卫安在狼牙呆的很好，更是狼牙资料部的重要成员，与丁雪一起在安罗城工作，已经是很默契的搭档，同时也能够独当一面。
<br/>　　这次被调来狼牙基地，卫安原本还以为是工作调动，帮助袁熙叶院长整理大量新进资料。
<br/>　　但看到夜色里一个快速消失的轮椅身影，卫安便一下子心头咯噔一下，一种震惊的情绪开始暴涨。
<br/>　　“这怎么可能？”她呐呐开口。
<br/>　　那个背影实在太像胡梦浩，她已经从楚涵的口中知道当时那个让她失望透顶的人，是胡梦浩的孪生兄弟胡梦瀚。
<br/>　　只是……
<br/>　　楚涵没有告诉过她胡梦浩还活着，其他人更只字未提。
<br/>　　“胡博士。”忽的角落里，一个人轻拍了躲在暗处的胡梦洁肩膀。
<br/>　　胡梦浩很激动，小声的呵斥道：“你们怎么把她接来了！我现在怎么见她！”
<br/>　　那人笑道：“胡博士请随我来，长官给您留了东西。”
<br/>　　说罢，那人便直接推着轮椅，将胡梦浩推进了一处房屋。
<br/>　　外边卫安已经追了过来，她并不知道自己是不是眼花，但哪怕一丝可能她都要确认。
<br/>　　她看到有人推着坐在轮椅上的人进入了原科技部，没有多想，卫安便上前敲门。
<br/>　　叩叩！
<br/>　　叩叩叩！
<br/>　　先是缓慢，再是急促，显然卫安已经心神不宁。
<br/>　　“开门！”到最后卫安更是大声喊出口。
<br/>　　这时‘吱呀’一声，门从里面打开，昏暗的光线让人看不清里面有什么，只能模糊的看到一个人影站在门口。
<br/>　　“你推了什么进去，这位少尉？”卫安有些严厉的出声，她在此之前看到了那位狼牙防守队成员的胸前徽章。
<br/>　　但出乎意料的，一个令她熟悉至极的声音响起，带着一丝颤抖：“我不是少尉。”
<br/>　　卫安愕然抬眼，恰好对上了眼前这人的双目。
<br/>　　时间就此停止，四周一片安静。
<br/>　　啪嗒！啪嗒！
<br/>　　屋内的人有些动作迟缓的走出，走到了月光之下，让眼前这个年轻的女孩看到了他的全貌。
<br/>　　卫安倒吸一口凉气，捂住嘴满目惊讶，眼眶更是一下子布满泪水。
<br/>　　胡梦浩！
<br/>　　“你……”卫安声音哽咽着，惊讶的看着眼前这个人，双目更是下意识的看向胡梦浩的双·腿。
<br/>　　原本空空荡荡的裤腿处，此时竟然有着双·腿，穿着鞋袜，与正常人一样站在她的面前！
<br/>　　胡梦浩更感觉自己好似活在梦境里，他有了双·腿，甚至双·腿开始有了知觉。
<br/>　　这时候忽然一声轻咳声，屋内那名少尉走出来，将一个东西递给胡梦浩道：“抱歉打断一下，但是这个您拿着，高科技玩意儿，先看看使用说明书。”
