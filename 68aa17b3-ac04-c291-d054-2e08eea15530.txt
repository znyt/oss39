　　任山按照他的吩咐，去外面灌上了一壶白水，不过也算他聪明，并没有在这个时候耍什么心思。
<br/>　　等谁烧开之后，刘哔拿出一个茶杯盛了一点，在储物戒指中撒上了一撮细盐。
<br/>　　他对盐没有什么抵触感，也比较喜欢这种咸咸的味道。
<br/>　　说起来，这些盐还是他打算用在女鬼身上的，可惜当时没能用到，他也只能随手放在了储物戒指里面。
<br/>　　一口气喝光了茶杯中的盐水，刘哔抹了一下嘴巴，身体上的毛孔舒张开来，只觉得有些畅快。
<br/>　　他对于食物没什么需求，但是水也是不能缺少的，如果长时间不喝水，身体也会出现很多症状。
<br/>　　放下茶杯，他发现任山站在一边，坐立难安的样子，心中微微一动。
<br/>　　“你不打算解释一下吗？”直视着任山的双眼，刘哔平淡的开口说道。
<br/>　　微微叹了口气，任山脸上泛起一抹苦涩：“刘先生，我之前向您隐瞒这里的情况，实际上，这个地方我来过。”
<br/>　　停顿了一下，任山小心翼翼的看了一眼刘哔，看到他并没有计较的意思，方才放下心来，继续往下诉说。
<br/>　　“这个地方确实是鬼鹰先生的老巢，但是整个村子，只有他和他的徒弟两个活人，其余的，全部被制作成僵尸傀儡了，所以，这也是我不想来这个地方的原因。”
<br/>　　颔了颔首，刘哔示意自己在听，但是等待了一会之后，任山却没再继续说了。
<br/>　　“你想活吗？”看着任山的眼睛，刘哔语气有些严肃：“我和你没什么仇怨，我只是来找尸丹的，帮我找到尸丹，我会带你离开。”
<br/>　　看到任山眼中的不信任，刘哔耸了耸肩，无奈解释道：“我没理由杀你，任婷婷我都放过了，帮我找尸丹，找到之后，我们各走各路，恩怨一笔勾销。”
<br/>　　看到任山有些松动的表情，刘哔嘴角勾起了一抹笑意，他知道，任山最终会答应他的。
<br/>　　只要有生的希望，又有谁会选择去死呢。
<br/>　　之所以找尸丹，也是为了完成系统给出的任务，对于鬼修的修炼资源，他还是有些了解的，无非是借助尸体，鬼气以及尸丹之类的东西。
<br/>　　鬼气和尸气根本不会长时间存留下来，那么，能积攒下来的，也就只有尸丹了。
<br/>　　刘哔计划的没错，任山虽然表面平静，但内心却是无比挣扎了。
<br/>　　再三犹豫之下，他重重的点了点头，答应了刘哔提出的交易。
<br/>　　他觉的刘哔所说有理，细算起来，他和刘哔也没有什么仇怨，如果按照他的吩咐去做，那他或许还真有可能活下来。
<br/>　　常年负责运输货物，他大致知道哪些尸丹藏在什么地方，这个任务，对他来说也算不上难。
<br/>　　只是有些风险罢了。
<br/>　　看到任山答应下来，刘哔笑了一下，满意的点了一下头，重新倒了一杯盐水，端着回到了炕上。
<br/>　　现在天色还早，不太方便行动，趁着这段时间，他想要和任山制定一些计划。
<br/>　　而任山也难得的上道了一次，温顺的和刘哔商量起来，一番推敲之下，两人总共找出了五个可能存放尸丹的位置。
<br/>　　比较难办的是，这个村子里僵尸实在太多，尤其是到了晚上，失去了阳光的庇护之后，就更加肆无忌惮了。
<br/>　　对此，任山提议白天的时候行动，却被刘哔果断给否决了。
<br/>　　那个叫豺的少年，也不是什么好相与之辈，白天行动的话，也太容易被发现了。
<br/>　　虽然晚上僵尸要多一些，但他们小心一些，还是有机会窃得尸丹的。
<br/>　　敌人实力不明，如果可以的话，还是尽量不要产生争斗好些。
<br/>　　天色逐渐漆黑了下来，刘哔端着茶杯，静静的盘坐在床上，身处这样一个环境，他没办法修炼。
<br/>　　任山也是，紧张兮兮的站在刘哔身边，眼睛不时向外张望着，似乎是在担忧着什么。
<br/>　　当最后一点太阳消失在地平线，光亮逐渐消失，村子里忽得刮起一阵阴风，将院子里的小物件吹的‘哐啷’作响。
<br/>　　刘哔表情没有任何变化，半睁着双眼，手指在茶杯口上缓缓滑动。
<br/>　　反观任山，此时却像是热锅上的蚂蚁一般，在房间里翻来覆去走个不停，脑壳上不断有冷汗流落。
<br/>　　看着心急火燎的任山，刘哔嘴角轻轻一扯，也就是他，能在这个情况下笑出来了。
<br/>　　“走，我们过去看看。”唤了一声任山，刘哔从床上下来，披上外套，悄摸的打开房门，纵身跳到矮墙上观看者。
<br/>　　外面此时，和白天所看见的场景已经不一样了，那些原本散落在地上的破旧白幡，此时被狂风刮到天上，伴随着树叶和一些细小的树枝，肆意狂舞。
<br/>　　破旧白幡之下，一片漆黑如墨的黑暗之中，影影绰绰的可以看见一些黑影，动作僵硬，像是人形。
<br/>　　诡异的气氛让刘哔背后升起一层细密的疙瘩，收起脸上漫不经心的表情，他的眼神无比凝重。
<br/>　　一旁的任山刚从底下爬上来，抬头就看见了前方不断游荡的僵尸身影，瞳孔猛然一缩，连绵的冷汗从其额头流淌了下来。
<br/>　　细密的汗珠顺着皱纹，一路落在任山的衣领中，但他像是一点没有觉察到一般，直愣愣的看着前方僵尸群，不敢发出半点声音。
<br/>　　他不是第一次见到这种场景了，但心中依旧是震撼的说不出话来。
<br/>　　刘哔做了一个噤声音的手势，专心致志的查看着前方的情景。
<br/>　　片刻之后，双眼终于适应了黑暗，借着刚刚升起的微弱月光，他终于看清楚了那些僵尸的真身。
<br/>　　和他之前见过的僵尸不同，这些僵尸的服饰，虽然也是清朝官服，但颜色是暗黑色的。
<br/>　　就像是被腐血浸泡了很久一般，粗略看去，看不出一点原本的颜色。
<br/>　　也只有在那些身穿宫女服的女性僵尸头上的扁方上，方能看出一缕暗红。
<br/>　　空气中，充斥着浓郁的血腥味道，明明并不好闻，却该死的有些甜美。
<br/>　　舔了舔已经初成的獠牙，受到血腥气味的影响，刘哔此刻的双眼，都变成了一片猩红。
