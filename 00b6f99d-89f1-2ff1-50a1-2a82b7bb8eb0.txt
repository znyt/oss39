　　苏莫陷入了沉默之中，现如今，不仅是帝一魂在找他，终极神王殿肯定也在找他，四处皆敌。
<br/>　　而且，紫霄肯定也不会放过他，比如会想办法胁迫他交出终极剑道和终极神剑。
<br/>　　他躲在天龙神朝，目前还算安全，短时间之内，无论是帝一魂还是终极神王殿，都不太可能找过来。
<br/>　　他现在最为担心的是紫霄，或者说是芊寻月和绫裳。
<br/>　　不用想，紫霄肯定会用芊寻月要挟他现身，只是不清楚紫霄会不会将芊寻月一事公布于众。
<br/>　　“苏莫，你倒是说话啊，接下来我们怎么办？”司空炎见苏莫沉默良久，忍不住问道。
<br/>　　“苏莫，你如今遇到了什么困难？”李峰面色古板，他能从苏莫的脸上，看出苏莫深深的担忧。
<br/>　　苏莫没有回答两人的询问，又思忖了片刻之后，环视众人，沉声道：“龙腾，你就留在龙族吧，至于你们几个，我另有任务。”
<br/>　　“怎么安排？”闻人天都问道。
<br/>　　“宫主，我们听你的吩咐，全力配合你！”清雅沉声道，他们都知道苏莫在神域，有帝一魂这个大敌。
<br/>　　“目前的情况，是这样的……！”苏莫徐徐开口，向众人讲解了帝一魂和拓跋青云的情况，并且将芊寻月和紫霄一事，也告知了众人。
<br/>　　“除了你和帝一魂之外，居然还有一个分魂？”龙腾面色凝重不已。
<br/>　　“这两人修为太高了！”彩芸话语不多，但是同样面沉如水。
<br/>　　“苏莫，这紫霄实在可恶，也是目前最大的威胁！”姜枫然说道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即环视众人，道：“我心中有一个想法，你们看是否可行？”
<br/>　　“你说。”众人异口同声的说道。
<br/>　　苏莫心中组织了一番语言，随即目光看向龙腾，道：“龙腾，你想办法在龙族弄些特制的传讯符，传讯距离越远越好，我们一人一枚。”
<br/>　　“这个没有问题！”龙腾点了点头，龙族作为神域的巅峰大势力，各种传讯符很多。
<br/>　　“清雅，你去人王宫求见人德宫主，加入人王宫，若是见不到人德宫主，那就见玄天神王；若是见不到玄天神王，那就见其女姬月沁，只要你表露出和我有关系，他们会让你加入人王宫。”
<br/>　　苏莫眸光闪亮，继续道：“人德宫主，对我有恩，目前也算是支持我。”
<br/>　　“好！”清雅点了点头。
<br/>　　随后，苏莫又看向闻人天都和战乱两人，道：“你们两人加入地王宫，以你们的天赋，这个应该不难！”
<br/>　　闻人天都和战乱两人，均是点了点头，他们没有询问，到了地王宫具体怎么做，因为待会苏莫肯定有交代。
<br/>　　“姜枫然，你加入天王宫。”苏莫又看向姜枫然，他相信以三人的天赋，加入地王宫和天王宫不是问题。
<br/>　　毕竟，三人的天赋，毫不夸张的说，比之神域那些万古天骄，都丝毫不差，甚至更甚一分。
<br/>　　“苏莫，你安排我们四人，分别加入人王宫、地王宫和天王宫，具体是要做什么？”姜枫然疑惑的问道，他心有了猜测，但并不确定苏莫的想法。
<br/>　　清雅和闻人天都、战乱，同样好奇的望着苏莫。
<br/>　　“清雅，你的任务比较简单，只要在人王宫安稳成长就好，必要的时候，为我获取人王宫的资源；至于姜枫然你们三人的任务比较艰难，但只有一个目标，那就是挑起地王宫和天王宫的仇恨，让两宫不断的对抗。”
<br/>　　“挑起仇恨？”众人闻言，均是眼眸微亮。
<br/>　　“不错，无论是帝一魂还是拓跋青玉，都不能让他们安生，让两人开启战端，两宫大规模的对抗！”苏莫眸中精光一闪，两人不是都在争分夺秒的冲击高级神王吗？他不能让两人如愿。
<br/>　　只要两宫大规模的开战，就不会有这么多心思对付他了。
<br/>　　“此事不算困难，但是要有详细的计划！”闻人天都沉吟着说道。
<br/>　　“嗯，此事我们稍后再详细商议，你们的安全必须保证！”苏莫点了点头，随后又看向彩芸，道：“彩芸，你是角魔族，去亘古魔国吧！目前没有任务，但我希望将来，你能有所作为。”
<br/>　　彩芸身为角魔族，算是魔族的分支，去往亘古魔国，比留下来更好。
<br/>　　若是对方在亘古魔国，能够得到重用，将来或许也能成为他的一大助力。
<br/>　　“可以！”彩芸点了点头，她现在实力低微，帮不上什么忙，既然苏莫如此安排，必有用意，她也不会拒绝。
<br/>　　况且，去亘古魔国，的确是对她成长有益。
<br/>　　“李峰，你去往霸天神朝吧，霸天族才有修炼肉身的上乘法门，不过，单纯的肉身之路不是长久之计，待会我赐你元神！”苏莫又转头对李峰说道，至于对方的神魂强度，能不能承受的了元神，还需要一试。
<br/>　　“好！”李峰同样点头，并未拒绝，他虽然想以肉身证道，但是有了元神，也不耽误他修炼肉身。
<br/>　　“苏莫，那我呢？”司空炎开口了，目光直直的盯着苏莫，满脸惊疑之色，别人都有安排，唯独他没有安排。
<br/>　　“你的任务最重！”苏莫看向司空炎，嘴角勾起了一丝笑意。
<br/>　　“什么任务？”司空炎好奇的问道。
<br/>　　“我要靠你来对付紫霄。”苏莫眸中精光一闪。
<br/>　　“啊！”司空炎闻言，顿时满脸愕然。
<br/>　　随即，他立刻从座位上跳了起来，不满的叫道：“苏莫，你没搞错吧，让我去对付紫霄？人家可是神王，不是圣王！”
<br/>　　司空炎彻底无语了，就他这点实力，在神路能呼风唤雨，横行无忌；但在神域就是蝼蚁，还不够紫霄一个手指头碾压的。
<br/>　　“我是认真的。”苏莫面色郑重的说道。
<br/>　　“这……怎么对付？”司空炎怔怔的问道，他也知道苏莫不是爱开玩笑之人。
<br/>　　其他人亦是不解的望着苏莫，司空炎哪有本事对付紫霄？
<br/>　　“具体如何行事，等你加入了终极神王殿，再从长计议，不过你放心，我并不是让你杀紫霄！”苏莫沉声说道。
<br/>　　“好吧！”司空炎郁闷的点了点头，虽然苏莫没有说具体怎么行动，但是，他能感觉到自己必然肩负重担，切不可大意。
