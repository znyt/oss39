　　“当——”
<br/>　　一口巨大混沌神钟挡在身前，同时元始也催动这自己身上的无上法力，轰向那巨剑。钟声响彻空间，震碎虚空，巨大的爆炸声响起，那巨剑终于被阻挡住了，但它依旧没有毁去。
<br/>　　“杀！”
<br/>　　元始大叫一声，手持神钟，轰杀过去，经过一番大战，最终将那黑色利剑破灭。
<br/>　　元始目光凌厉，他终于发觉了，那人的状态不对，竟然只剩下了身体的执念，而他的元神早已不见，或者是死了，也或者是自己走了。
<br/>　　但即便如此，他们之间的差距也宛若鸿沟。
<br/>　　“有些道行，但还不够，与我为敌，就该做好身死的准备！”
<br/>　　那人好似没有恢复过来，目光幽幽，那漆黑的双眼之中透露着冷冽寒光，看向元始。
<br/>　　“你话太多了。”元始淡淡道，“原本还想过段时间突破，没想到会发生此事，但，现在突破好像也不迟。”
<br/>　　元始催动自己的无上道则，周边的无尽黑色迷雾如龙卷风般被倒吸过来，注入到他体内，让元始那本身就漆黑无比的身体更加黑暗了。他催动灵魂之火去燃烧那些黑暗物质，去掉其他杂质，如死气、诅咒、腐朽、破败、衰亡等气息，留下最纯净的黑暗本源，虽然不多，但了胜于无。
<br/>　　“以暗为心，黑为催化，三千道则为火，凝聚我的元始暗黑大道！”
<br/>　　元始心神一动，体内无尽道则催动，那原暗大道雏形，再加上黑暗本源的灌入，在迅速凝聚起来。
<br/>　　嘭！
<br/>　　元始体内无尽核心之处，燃烧起黑色火焰，火焰摇摇曳曳，如无尽光明之中的一点黑暗，之后在其他道则那近乎献祭的过程之中，那火焰在快速壮大，最后席卷诸天。
<br/>　　漫天黑色火焰爆发，黑暗降临人间，光明退避。
<br/>　　“死！”
<br/>　　与此同时，那大殿深处的尸骸再次出声，无量黑色物质在他体外翻滚，凝聚无尽魔影，有真龙，有神凤，有神魔，有恶鬼……他们向着元始杀来，冲破光幕阻拦，咆哮着无尽黑暗，每一个都堪比准圣大圆满，这还是被削弱的力量。
<br/>　　轰隆！
<br/>　　元始催动无上道法，力量加持，手持混沌神钟，与无尽魔影大战起来，战斗惊天动地，周围空间似乎承受不住这股强烈的冲击力，寸寸断裂开来。
<br/>　　黑色神龙翱啸九天，无尽黑色火焰落下，冲击过来，黑龙腾空，巨尾拍碎空间，以强大的力道撞击过来，神魔乱舞，无上神通，无尽黑影，恶鬼咆哮，截断生死……
<br/>　　“当当当——”
<br/>　　神钟连响成一片，面对同等级的高手，元始无惧，以无上法力催动开天至宝，无尽神威降临，无上音道连绵，音波混乱，动其神魄，星辰凝聚，开天神门，向着众多黑影镇压过去。
<br/>　　刚刚破碎的空间还来不及恢复过来，便又被冲散开来，空间演化黑洞，风暴不断，再加上此处乃是时空乱流，使之情况变得更加复杂。
<br/>　　混沌神钟宛若化成一道巨大的开天神门，占尽此方空间，以无上神威，镇压一切魔影。
<br/>　　与此同时，元始体内，熊熊黑火燃烧更旺了，好似要将一切都燃尽，化成元始的黑暗。
<br/>　　大战连天，那边的道祖好似也遭遇到了麻烦，因为他体内也有秘密，所以并没有被那人的意志所取代，但他毕竟是初步突破，却没有如元始这般的强大神通，多次被打得破散开来，但又不得不再次凝聚。他的脸色苍白无比，显然是消耗巨大，并不好受，再来几次说不定就得死了。
<br/>　　道祖手中造化玉牒在闪耀着造化青光，他在接引无上神秘存在，一股无形的强大意志降临，带着浓浓的天威宇无上的威严，淡漠而又强大。
<br/>　　那股神秘力量聚在道祖身上，欲以无上意志取代道祖。
<br/>　　“啊，怎会如此？这造化玉牒有问题，不——”
<br/>　　道祖大惊，怎么也想不到自己利用造化玉牒接引而来的神秘存在欲取而代之，他想反抗，却又反抗不了，那股神秘的力量实在是太强大了，这时候，他才想起自己得到的造化玉蝶似乎并不是意外，而是早有预谋，这一刻，他想大笑，却又笑不出来。
<br/>　　“道友，这东西给你，希望你能度过此劫！”
<br/>　　突然，一道声音在元始耳边响起，紧接着，一块神秘的光明棱晶闪耀出无尽光辉，照耀开此处大地，神秘的黑暗物质见到此物，竟然诡异的如春雪便消融退避，好似遇到克星一般。
<br/>　　棱晶如同一个太阳一般，向着元始飞来过来，无尽黑暗物质组成的魔影开始消散、退避，元始趁机催动神钟，镇压一切，几个回合，彻底覆灭这些黑影。
<br/>　　同时，元始接过那神秘的光明棱晶，以他的目光，竟然看不出这东西的来历，只是知道这是一件光明至宝，而且品质极高，超出先天至宝的范畴，也不知道这是什么东西。
<br/>　　元始将目光看向道祖，却见他的意志已经消散了，取而代之的是一股带着无尽威严的陌生气息，他身上带着浓浓的天威，那双眼睛之中，充斥的是无尽的淡漠，冰冷，枯寂，不带半点情感。
<br/>　　看着这一幕，元始感觉有点熟悉，像是在什么地方见过一般。突然，他心神一颤，紧接着目光看向“道祖”，他眼中有种难以置信的表情，此刻的“道祖”，好似当初合道之后的鸿钧道祖，难怪如此熟悉。这股淡漠的气息，是天道的气息。
<br/>　　再次看向“道祖”，元始的目光之中，却又多了几分复杂，他知道这位“道祖”终究是与洪荒道祖不同，洪荒的鸿钧道祖是想以自身去合天道，虽然最后变得淡漠无比，但却还有自己的意识，而这位道祖，他的意识却是已经消散了，取而代之的是天道的意识，天道已经完全占据了此身。
<br/>　　他与道祖只是见过几面，甚至双方都不算是朋友，但对方却在临死之前却能够将一件神秘至宝送给自己，虽然那有他不想死后便宜敌人的嫌疑，但也难能可贵了，在死前，都让元始欠他一份因果人情，道祖果然不简单。
<br/>　　元始明白他的想法，就是元始若是能够躲过此劫，需替他保护此界。
<br/>　　“也罢，既然承你一份人情，也自当担你一份因果，此界我替你保护了。”元始苦笑一声，心中暗自应答道。
<br/>　　“你来了。”此刻，那殿宇深处的尸骸却并没有去看元始，而是将目光看向道祖，淡淡说道。
<br/>　　“我来了。”道祖看着大殿中满身黑暗的尸骸，皱了皱眉，淡漠道，“不过，你终究是堕落了。”
<br/>　　“你也不一样。”那残骸冷笑一声，平淡道。
<br/>　　元始在一旁听得疑惑不已，这什么意思，这两人认识？这个世界的天道与这神秘的尸骸到底是什么关系？
