　　“大祭司，我们不是抓了一位苏莫的朋友吗？为何不以她来引出苏莫？”苏莫缓缓平复心境，再次问道。
<br/>　　“此法并不可行，再说了，那女子已经不在族中了！”
<br/>　　大祭司摇了摇头，随即不愿多谈此事，挥了挥袖袍道：“好了，你先退下吧！”
<br/>　　言罢，他喊了一位下人，带苏莫去新的住所。
<br/>　　苏莫见此暗叹，既然对方不愿意多谈，他也只能作罢，不敢再行询问。
<br/>　　不过，对方说青璇不在族中了，是什么意思？那青璇会在哪里？
<br/>　　苏莫心中疑惑，但丝毫没有表现出来，向大祭司行了一礼，便跟随下人离开了偏殿。
<br/>　　望着苏莫离去的背影，大祭司面无表情，目光深邃不可测。
<br/>　　……
<br/>　　引领苏莫前往新住所的下人，是一名瘦弱的巫族中年人，修为只有武王境二重。
<br/>　　很快，中年人便将苏莫带到了新的住处。
<br/>　　这是一处独立的院落，非常的大，仿若是一座小型府邸一般。
<br/>　　“少司大人，这就是你的府苑，我以后也住在这里，随时听候你的差遣！”中年下人恭敬的向苏莫说道。
<br/>　　“那劳烦你了！”苏莫淡淡的点了点头。
<br/>　　随即，他便在这座府苑中闲逛了片刻，府苑中没有半个人影，宫殿、阁楼、凉亭水榭应有尽有。
<br/>　　最后，苏莫找了一个宽敞的房间，暂时住了下来。
<br/>　　“苏莫，汝之身份，已经引起怀疑了！”
<br/>　　苏莫刚刚在房间中坐下，阎魔鬼圣的声音便在他的脑海中响起。
<br/>　　“不会吧？”苏莫闻言微惊，难道自己的询问，引起了大祭司的怀疑？
<br/>　　“汝的那枚少司令牌有猫腻，其内有巫族大祭司的一缕意念！”阎魔鬼圣说道。
<br/>　　“什么？”闻听此言，苏莫顿时心中一震，真的暴露了吗？
<br/>　　他没有怀疑阎魔鬼圣的话，对方是老古董了，虽然现在只剩一道魔魂，但手段也不是他能比的。
<br/>　　大祭司为何如此？他哪里露出了破绽？
<br/>　　苏莫眉头紧皱了起来，低头沉思。
<br/>　　不对！
<br/>　　他并没有露出破绽，因为他询问大祭司有没有查到他的行踪之时，对方早已将少司令牌给了他。
<br/>　　那大祭司为何在令牌上留下一丝意念呢？
<br/>　　这定然是不信任他，毕竟他表现的天赋太过逆天，又是突然崛起，而大祭司稍一调查，便能发现其中有很多不同寻常之处。
<br/>　　所以，大祭司在少司令牌上留下意念，便是为了监视他，查探的的底细。
<br/>　　换言之，大祭司估计并没有想到他的真实身份，只是觉得他的一切比较古怪，所以才这么做。
<br/>　　“鬼圣前辈，少司令牌已经被我收进了储物戒，对方应该探查不到任何事情吧？”苏莫用意念向阎魔鬼圣问道。
<br/>　　储物空间和外界，完全是独立的空间，他只要将令牌一直放在储物戒中，应该能屏弃对方的查探。
<br/>　　此刻，苏莫不禁有些后怕，辛亏没将令牌丢进玉镯空间，不然的话对方立刻就能发现其内的巫娴三人。
<br/>　　一旦如此，他必死无疑了！
<br/>　　“天真！”
<br/>　　阎魔鬼圣闻言嗤笑一声，道：“武圣境的武者，手段何等强大，就算只是一缕微不可查的意念，也不是小小的储物戒可以束缚的！”
<br/>　　阎魔鬼圣以前就是武圣，自然了解武圣的手段，眼界远不是苏莫可比。
<br/>　　“那我怎么办？”苏莫急忙问道。
<br/>　　“立刻离开巫族，并且丢弃少司令牌！”阎魔鬼圣告诫道。
<br/>　　苏莫闻言沉默了，离开？现在就离开？
<br/>　　他此行的目的还没有完成呢？
<br/>　　现在好不容易混进了宗府，接近了大祭司，不查探到青璇的情况，他如何能甘心离开！
<br/>　　再者，他一旦丢弃少司令牌，怕是大祭司立刻就能发现，到时候能不能逃离天冥星都是两说了。
<br/>　　苏莫缓缓压下心中纷乱的情绪，心中急思破解此局之策。
<br/>　　现在他丝毫不敢大意，搞不好的话，说不定小命就彻底交代在巫族了。
<br/>　　“苏莫，别考虑了，尽快离开吧！”阎魔鬼圣的声音再次在苏莫脑海中响起，他必须催促苏莫离开，不然的话苏莫死在这里，他也要死在这里。
<br/>　　“离开了又能怎样？令牌在我身上，我如何能摆脱？”苏莫询问道。
<br/>　　他自然是可以离开巫族，但是只要丢弃少司令牌，大祭司立刻就发现，以对方的速度，他根本逃不了对方的追踪。
<br/>　　阎魔鬼圣闻言沉默了片刻，随后凝声道：“吾有办法屏弃对方的查探，但最多只能屏弃两三个呼吸的时间！”
<br/>　　“两三个呼吸的时间？”苏莫闻言眼眸一亮，但随即又暗了下来，时间太短了！
<br/>　　除非，他利用这两三个呼吸的时间，让本体逃走。
<br/>　　而分身依旧留在巫族，继续查探青璇的下落。
<br/>　　“两三个呼吸的时间，足以让你本体逃离了！”阎魔鬼圣和苏莫是同样的想法，显然也没指望苏莫的分身和本体一起离开。
<br/>　　“嗯！”苏莫心中点头，为了安全起见，本体必须离开了。
<br/>　　只要本体离开了，分身留在此地，就算暴露了也无妨，对他没有丝毫的伤害。
<br/>　　哎！
<br/>　　他不禁心中暗叹，之前还是计划的不够周详，才会导致现在这番局面。
<br/>　　随即，苏莫仔细思考了一番，并有了详细的逃离计划，本体要逃离，临走之前一定要大干一场，赚够足够的修炼资源。
<br/>　　最起码，也要让未来十年之内，他和苍穹神宫无需再为资源担忧。
<br/>　　想做到这一步，需要数之不尽的资源。
<br/>　　但天冥星这么大，巫族族人数以百亿计，他可以肆意的掠劫，甚至是屠城！
<br/>　　他的本体想要逃离天冥圣城，目前有两个办法，第一个办法比较简单，就是直接离开。
<br/>　　然后，离开天冥圣城之后，让阎魔鬼圣屏蔽储物戒内少司令牌中大祭司的意念，他本体趁机逃走，分身再返回。
<br/>　　第二个办法就稍微有点隐秘了，分身不离开宗府，让本体夺舍别人离开。
<br/>　　这两个办法各有利弊，最后苏莫仔细斟酌了一番，决定使用第二种方法。
<br/>　　因为第二种方法更安全，第一种方法的话，他直接离开天冥圣城，绝对会引起了大祭司的注意，到时候连本体都逃不掉。
