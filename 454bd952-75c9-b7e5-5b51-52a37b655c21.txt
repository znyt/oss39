　　“阁下，我们都是狼神帮的人，我们帮主的实力可不比你差！”
<br/>　　“没错，你虽然厉害，但是我们也不是好惹的，大不了这个女娃娃让给你，我们不玩他罢了！”
<br/>　　那小女孩哭着穿起了自己的小裙子，指着那群汉子说，“这群人杀了我的妈妈，他们想让我和我妈妈舔他的那个玩意，我妈妈不同意，他们就把我妈妈的脑袋硬生生的塞在丧尸嘴巴里去了，他们都是坏人，呜呜呜呜呜呜。”
<br/>　　林峰瞬间面色阴沉，“你们还挺会玩的，是要玩岛国那一套母女同床是吗？”
<br/>　　几个汉子面色难看，“小子，我们已经做出了让步，可不要得寸进尺！我们老大可是个七阶的高手！”
<br/>　　七阶的？
<br/>　　林峰目光微微眯起，这种级别的高手可不多见，而且，强者榜上，林峰也没看到过有一个叫做狼神帮的老大是七阶高手的。
<br/>　　七阶高手整个基地就那么大概二三十人罢了。大部分都是被军部和几个大组织给瓜分了。
<br/>　　“狼神帮，七阶，你们和狼窝是什么关系？”
<br/>　　林峰忽然想到了什么。
<br/>　　“呦呵，你还知道我们之前叫做狼窝？”
<br/>　　几个汉子都意外。
<br/>　　然后一个个都哈哈大笑：“看来我们老大很牛逼嘛，这么多人都知道他，小子，现在是不是后悔了，不该多管闲事。”
<br/>　　林峰果然猜中了，一来这七阶高手的确不多，数来数去就那么些，毕竟七阶不是大白菜，随便来个都是七阶，二来这狼神帮和狼窝名字都有狼。
<br/>　　“小子，看在你和我们老大可能认识的面子上，速速滚蛋，今天这件事情，就当没发生。”
<br/>　　几个汉子得意洋洋道，有个七阶老大在，他们也可以狐假虎威一次。
<br/>　　“既然你们是狼窝的人，那就去死吧。”
<br/>　　林峰冷笑直接冲了过去。狼窝这群人之前强女干了林峰一个女同学，后来又要打劫林峰他们，再然后林峰又抢了他们一个八阶怪物的晶石，因此遭到了战门之人的追杀，可以说林峰和这个狼窝算是不共戴天之仇了。
<br/>　　刷刷。
<br/>　　两刀下去又是两个大人头腾空而起。
<br/>　　“什么？”
<br/>　　几个汉子明显没有反应过来，都知道他们老大的事情了，还敢对他们动手？
<br/>　　回答他们的只是林峰的无情屠杀。
<br/>　　几刀过去后，这些人全被被斩杀殆尽。
<br/>　　林峰目光落在那个小女娃身上，对方抱着膝盖呆呆地坐在那里两眼无神，看起来还是蛮可怜的，毕竟亲眼目睹了许多不好的事情。
<br/>　　“没事了，这群人以后不会再来骚扰你了，我带你去曙光基地吧。”
<br/>　　林峰给了她一个棒棒糖。
<br/>　　小女孩眨巴着眼睛看着林峰，然后点了点头，撕开棒棒糖就是含在嘴巴里，搞的腮帮子鼓鼓的，看上去还挺萌的。
<br/>　　林峰牵着这个小女孩自然是送回基地内了，至于这个小女孩以后怎么办，那也不是林峰能够预料的了，毕竟林峰能够救下她，也算是仁至义尽了。
<br/>　　半路之中。
<br/>　　林峰遇到了军部的车队。
<br/>　　林峰立即把他们拦住了，还凑巧的是，这个车队上载的人林峰还认识，就是那个猎豹小队的吴刚和他的队员们。
<br/>　　“林峰？”
<br/>　　看到林峰吴刚也是大感意外。
<br/>　　林峰把小女孩和吴刚的人说了一下，吴刚略一犹豫，还是让个人亲自去把小女孩送回基地去了。
<br/>　　若是其余人，他是不可能这样做的，毕竟他们是特警，负责特别任务行动，不是负责救助幸存者。
<br/>　　不过既然是林峰开口，他肯定会同意。
<br/>　　“谢谢大哥哥。”
<br/>　　小女孩坐上车后对林峰摇了摇手然后车子就开远了。
<br/>　　“到了基地要乖哦，遇到什么危险，找这些穿军装的大哥哥就行了。”
<br/>　　林峰笑道。
<br/>　　“嗯，我知道了。”
<br/>　　小女孩甜甜一笑然后就和车子回基地去了。
<br/>　　“怎么，出任务啊？”
<br/>　　林峰看着吴刚道。
<br/>　　“是啊，没办法，军部就是这样，天天任务，你是战士会的吧？我在强者榜上看到你名字了。战士会好，自由。没那么多规矩。”
<br/>　　吴刚道。
<br/>　　“什么任务？”林峰问道。
<br/>　　“这一次任务有点邪门，据说是一个村子里的人都变成人干了。”
<br/>　　吴刚郑重道。
<br/>　　“人干？”林峰疑惑道。
<br/>　　“就是浑身的鲜血被人给抽干了，目前还不知道是丧尸做的还是怪物做的。”吴刚摇头道。
<br/>　　“所以就派我们几个去实地调查一下。”
<br/>　　林峰点了点头，“既然如此，那我和你们一起去吧，你说的那个地方应该是叫做黄龙村吧。”
<br/>　　“你怎么知道？”吴刚疑惑道。
<br/>　　“我接了个任务，也是黄龙村的，任务介绍里面有几句简单的描写，也提到了什么人干，我一开始没在意，听你一说，就想起来了。正好顺路，还可以蹭一蹭你们军部的顺风车。”林峰道。
<br/>　　“原来如此，那正好，有你这个大高手加入，我自然是无比欢迎。”吴刚笑道。
<br/>　　上了运输车，林峰居然看到了许飞那小子。
<br/>　　“林峰兄弟又见面了，还真是够巧的。”许飞表哥道。
<br/>　　“大哥好。”许飞十分乖巧的主动喊了声大哥。
<br/>　　这俗话说得好，伸手不打笑脸人。
<br/>　　林峰看对方老实巴交的样子，也就没有再找对方麻烦，坐在运输车上，一行人朝黄龙村驶去。
<br/>　　黄龙村距离江城有点远，是个小山村，人不多，也就那么二三十户人，都是种地为生。
<br/>　　到了黄龙村后。
<br/>　　林峰才发现这个地方居然有几百只丧尸。
<br/>　　察觉到车子的声音，瞬间每个丧尸都冲了过来。
<br/>　　“射击！”
<br/>　　吴刚等人立即下了车开始进行射击。
<br/>　　林峰就拿着斩灵刀跳下车直接冲入了丧尸大群之中，一刀一个，杀死丧尸的速度远比吴刚他们这些拿枪的要快上许多，毕竟丧尸必须要爆头，然而爆头不是那么容易的事情。
<br/>　　吼吼吼
<br/>　　只听到一道道咆哮，就看到远处几个房屋内冲出来了几个强大的三阶丧尸，三阶丧尸立即怒吼着杀来。
<br/>　　“两个三阶的？”
<br/>　　林峰一脚把正面的一个丧尸给踹飞，然后拿着斩灵刀就是朝这三阶丧尸冲了过去。
<br/>　　青莲剑歌！
<br/>　　一刀下去，两个三阶的丧尸直接被斩杀。
