　　离开苍穹神殿之后，苏莫并没有回到自己的婚房，而是先去找了一趟海明江。
<br/>　　让其想办法，将天涯海阁所有的分阁之人，全部收拢，召集到苍穹神宫之中。
<br/>　　这样做，则可以最大程度的保留天涯海阁的有生力量，不然的话，怕是全部都会被黄泉魔宗所灭。
<br/>　　不过，想将所有的分阁之人全部召集回来，根本没有可能，因为天涯海阁的分阁实在太多了，遍布大陆五大洲，甚至在四方海域都有很多分阁。
<br/>　　若想将所有分阁之人全部召回，没有两年时间，根本不可能完成。
<br/>　　不过，偏远的四洲分阁可以暂时不管，但中洲一些较大的分阁，却是要赶紧召回。
<br/>　　海明江没有犹豫，立刻派遣了十数名强者，让他们离开了苍穹神宫，前去召回各大分阁。
<br/>　　海明江这次来苍穹神宫，带了二、三十名强者，这是天涯海阁最强的一批人，修为全部都在武皇境七、八重左右。
<br/>　　不过，为了防止遭到黄泉魔宗的截杀，海明江也仔细的向众人交代了一番，一切隐蔽行事。
<br/>　　苏莫和海明江商谈了良久，最后在天色渐晚之时才离开，随后他便回到了自己的宫殿中。
<br/>　　婚房之中，夕儿静静的坐在床沿上，在等待着爱人归来。
<br/>　　“夕儿！”苏莫走进了婚房，坐在了夕儿的身边。
<br/>　　“苏莫哥哥，是不是出了什么事了？”夕儿柔声问道，之前他也注意到了外界的异常，不过他并没有查看。
<br/>　　“没事！”
<br/>　　苏莫伸手轻轻锊了捋夕儿的秀发，笑道：“今天是我们大婚的日子，任何的烦心事，都让他们见鬼去吧！”
<br/>　　“呵呵！”夕儿被苏莫的话逗笑了。
<br/>　　随即，苏莫搂着夕儿，两人温言软语，诉说着情话。
<br/>　　这一.夜，两人结合了，像所有新婚夫妻一样圆了房。
<br/>　　夕儿的身体洁白如玉，完美无瑕，细腻而顺滑，仿若最好的丝绸，让苏莫着迷。
<br/>　　两人整整奋战了一整夜，直到第二日天明才结束。
<br/>　　夕儿虽然是第一次经历房.事，但她和苏莫均是武王境以上的武者，身体素质非常强大，根本不是常人能比。
<br/>　　所以，两人的持.久力均是惊人无比。
<br/>　　就算是折腾了一.夜，第二日两人不仅没有丝毫的疲惫，反而精神熠熠，荣光焕发。
<br/>　　不过，苏莫显然无法多享受婚后的幸福时光，第二****便离开了苍穹神宫。
<br/>　　和苏莫一起离开的还有海明江，两人离开苍穹神宫之后，便径直向帝玄宫的方向而去。
<br/>　　现在黄泉魔宗发难，苏莫自然不会坐以待毙，准备先去帝玄宫看看情况。
<br/>　　其次，他还要想办法弄些资源，为不久之后冲击武皇境四重做准备。
<br/>　　至于苍穹神宫，有水幕大阵在，安全根本不用担心。
<br/>　　而且，在苍穹神宫之中，有九位强大的傀儡人，足以让苏莫没有后顾之忧。
<br/>　　“海阁主，我们先去看看帝玄宫的情况，然后再伺机而动！”一边飞行，苏莫一边向海明江说道。
<br/>　　“嗯！”海明江点了点头，他本来是想先去天海城，不过还是尊重苏莫的意见。
<br/>　　毕竟，若是对付黄泉魔宗和阴阳神宫，还需要苏莫和帝玄宫的力量。
<br/>　　两人一路无话，爆发出了最快的速度，在四日之后便抵达了帝玄宫。
<br/>　　抵达帝玄宫之后，映入眼帘的情景，让得苏莫心中大震。
<br/>　　只见帝玄宫的山门，已经不复存在，变成了一片废墟。
<br/>　　残碎的尸体堆积如山，腥臭的血水汇聚成河，多达数十万的帝玄宫弟子惨死。
<br/>　　“这……！”海明江也是震惊了，他活了九百年，也从未见过如此场景。
<br/>　　“去内宫！”毫不犹豫，苏莫立刻前往帝玄宫的内宫入口处。
<br/>　　唰唰唰！！
<br/>　　一个个复杂的印诀在苏莫手中结起，而后印在了虚空之中，这正是打开内宫的印诀。
<br/>　　然而，片刻之后，虚空中的内宫入口，没有丝毫变化。
<br/>　　“怎么回事？”苏莫的脸色沉了下来，心中涌现出不好的预感。
<br/>　　难道帝玄宫被灭了？
<br/>　　苏莫继续尝试开启内宫入口，但无论如何，都无法打开了。
<br/>　　“苏莫，不用再试了，此地已经没有入口了！”海明江脸色凝重，摇头说道。
<br/>　　“没有入口？”苏莫闻言疑惑，不解的看向对方。
<br/>　　“不错！”
<br/>　　海明江点了点头，道：“此地没有任何的空间波动，显然不存在小世界，就算以前有，现在也消失了！”
<br/>　　苏莫闻言脸色凝重无比，难道帝玄宫真的被灭了！
<br/>　　嗖！
<br/>　　随即，苏莫立刻身形一闪，急速向不远处的帝灵城飞了过去。
<br/>　　此刻的帝灵城已经变得有些萧条了，没有了往日那般繁华，人口减少了一大半。
<br/>　　唰！
<br/>　　苏莫身形落在帝灵城的街道上，二话不说，直接一把抓住了一名年轻的武者。
<br/>　　“快说，帝玄宫是怎么回事？”苏莫大声喝问道。
<br/>　　“找死……！”
<br/>　　这名年轻的武者突然被人抓住，心中大怒，正要发火，但一见到苏莫的面容，顿时心中大惊，颤声道：“苏……苏莫！”
<br/>　　这名年轻武者以前观看过苏莫和上官昊的战斗，所以一眼便认出了苏莫。
<br/>　　“少废话，快说！”苏莫不耐烦的道。
<br/>　　“帝玄宫遭到黄泉魔宗和北堂世家、殷家的联合攻击，已经彻底被灭了！”年轻武者急忙说道。
<br/>　　“什么？”心中的猜测被确认，苏莫顿时心中一沉，眸中杀机爆射。
<br/>　　帝玄宫被灭，对于他来说也是一个天大的损失，毕竟他还想将帝玄宫纳入苍穹神宫呢！
<br/>　　“黄泉魔宗是如何攻进帝玄宫的内宫？”苏莫冷声问道，这是他疑惑的地方。
<br/>　　内宫的入口，只有帝玄宫的核心之人才能打开，就算黄泉魔宗实力再强，也不可能轻易攻进去吧！
<br/>　　“是上官昊……上官昊回来了，帮助黄泉魔宗打开了帝玄宫的内宫入口！”这名年轻的武者，感受到苏莫身上的杀机，身体瑟瑟发抖。
<br/>　　“上官昊！”苏莫闻言神情一怔，面现不可置信之色，上官昊居然背叛了帝玄宫？背叛了他的师尊？
<br/>　　这时，跟随苏莫而来的海明江，听到年轻武者的话，亦是神情一呆，完全没想到是这个情况。
