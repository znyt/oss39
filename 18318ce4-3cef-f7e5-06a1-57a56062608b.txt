        &lt;h3&gt;第四卷第四章  等待时机的谷强&lt;/h3&gt;
<br/>
<br/>        闫欢以一个投资者的身份正式出现在甸城。
<br/>
<br/>        闫欢比陆战君早到了一个礼拜，而这也给了闫欢机会，在秦少卿的家里，谷强和谷正娟都见到了闫欢，谷正娟认为闫欢和谷强都不会知道之前的事情，在看到闫欢带来投资机会的时候，谷正娟接受了闫欢，并且想要入股闫欢的投资项目。
<br/>
<br/>        谷强听闻闫欢和陈子悦、陈子昂认识，心里紧绷着的防线坍塌了，陈子昂过一段时间回来，他们是约定好了吗？
<br/>
<br/>        秦少卿因为愧对闫芈，心里极想促成闫欢在甸城投资成功，这样可以挽留闫芈，最起码他还有改错的机会。
<br/>
<br/>        他和陆玉没有未来的，第一陆玉有家室，第二，他对陆玉没有爱。
<br/>
<br/>        陆战君此来也不知何事，这几天陆玉是要陪着她父亲的，无暇顾及秦少卿，秦少卿才有了脱身的机会，秦少卿一直在说服陆玉不能要了那个孩子，可是陆玉铁了心似的，弄得秦少卿精疲力竭。
<br/>
<br/>        当然他们的事情也是被他的父母亲知道了，秦寿让他的母亲佟寒昕将陆玉接到了他们的别墅里照顾，这让他更不知如何是好，闫芈回来了，他就直接将闫芈接回了他们自己的小家，到现在还没有回去看望秦寿他们。
<br/>
<br/>        陆战君住在皇后大酒店里，陆玉请了几天假去了皇后大酒店，陪着她的父亲，在这几天里，秦少卿利用手中的关系，迅速的将闫欢投资的项目落实了。
<br/>
<br/>        谷正娟也和闫欢谈妥了，以投资的形式将皇后大酒店旁边的大楼交给了闫欢用。
<br/>
<br/>        从那天吃完饭之后闫欢就没有见到谷强，当闫欢见到谷强的时候，她有一种想哭的感觉，可是她一直在笑，时隔多年，谷强还是那样，一点都没有改变，只是这么多年，他是怎么过来的？
<br/>
<br/>        她很想知道，可是她知道她不能和谷强有太多的牵扯，她一头扎进了工作中，在新的地方进行项目是需要耗费一番精力的，况且她还有别的目的，为了另外的一个目的她必须得将手中的事情办好，能够站稳脚跟。
<br/>
<br/>        她在甸城各个角角落落的骑着自行车转悠着，去看当地的小吃，看当地的风土人情，了解当地的风俗习惯，早出晚归。
<br/>
<br/>        谷正娟以投资者的身份参入到了闫欢的项目中，当谷正娟告诉谷强，她想和闫欢一起做项目的时候，谷强并没有反对。
<br/>
<br/>        “这个项目从闫欢的策划上来看是没有什么问题的，甸城也是一个大的城市，如果做得好的话，一年的收入比房租要高，你愿意入股就入股吧。”谷强对谷正娟说。
<br/>
<br/>        “闫欢不太熟悉这里的情况，而我刚好熟悉，我负责人员招聘和管理，她负责策划营销及招商。”谷正娟和谷强商量。
<br/>
<br/>        “具体分工你们自己协商，记得走好正规的手续，另外，陆玉那边你得想好怎么应对。”
<br/>
<br/>        “陆玉那边推给秦少卿就行了，我们就说是秦少卿让我们给腾的房子，闫欢是闫芈的姐姐，秦少卿肯定会帮着闫欢的。”
<br/>
<br/>        “我的意思是，陆玉如果为难闫欢的话，能不能做下去？”
<br/>
<br/>        “让闫欢给她分点红利呗，她是不会和钱过不去的，要不然她会到甸城来盯着我们？”
<br/>
<br/>        “如果是红利倒好说，陆玉和秦少卿的关系，你觉得陆玉还能容忍闫欢姐妹多久？”
<br/>
<br/>        “她自己都结婚了，还和秦少卿纠缠着，她凭什么容忍不了闫欢姐妹啊？”谷正娟反驳着。
<br/>
<br/>        “反正陆玉的心思和行为我理解不了。”谷强看了一眼谷正娟。
<br/>
<br/>        “我也理解不了，不过每个人都有自己的生活方式，我也不好多说什么，那种事情都是两厢情愿的，只是苦了闫芈，闫芈人其实挺好的。”
<br/>
<br/>        “我只是提醒你，你应该怎么去劝解陆玉接受闫欢投资的事情，别弄到后面不好收场。”
<br/>
<br/>        “这个你放心，我会去说服她的，这几天她爸爸来了，她和秦叔叔一直陪着她爸爸呢。”
<br/>
<br/>        “她爸爸来是什么事情你知道吗？”
<br/>
<br/>        “不知道。”
<br/>
<br/>        “我们家和陆家、秦家也算是一根藤上的人，你随时注意着他们的情况，别我们玩着命，连什么情况我们都不知道。”谷强叮咛谷正娟。
<br/>
<br/>        “我知道了，我又不傻。强子，你觉得闫欢好不好？”
<br/>
<br/>        “不知道，没有深交过，不过我觉得她有点像一个人。”
<br/>
<br/>        “像谁？”
<br/>
<br/>        “我曾经在甸村呆过一段时间，那个时候我身上有伤，他们给我安排了一个叫公孙楠的女人照顾我，感觉有些像。”
<br/>
<br/>        “公孙楠？”谷正娟是知道闫欢的，但是不知道陆玉将闫欢的名字改成了公孙楠。
<br/>
<br/>        “是的。”谷强观察着谷正娟的反应，看来谷正娟不知道公孙楠，那么闫欢在哪个小山村的事情谷正娟是不知道的。
<br/>
<br/>        “你那次和人打架我都不知道，以后别那么鲁莽了。”
<br/>
<br/>        “你看我这几年不是安安分分的嘛，好了，我还有别的事情。”谷强要去基地看看。
<br/>
<br/>        这一段时间他没有管那些事情，但是他要知道一些情况，顺便将消息想办法送出去，陈子昂安排过来的人是信息上的高手，谷强利用他们将谷强的代码发给有关部门，至于有关部门能不能注意，或者是能不能破解也只能是听天由命的，好的是他们命中率各占一半。
<br/>
<br/>        正与邪的较量此消彼长！
<br/>
<br/>        谷强现在只能背着谷强的身份，这么多年了，他感觉他已经被组织遗忘了，他是真的找不到回去的路了，没有人来和他联系，而他也不能去联系别人。
<br/>
<br/>        他和祁峥一直在寻找机会，也在等待机会，现在情况越来越明了，知道了谷家身后的靠山，他们更不能轻举妄动，拖一天算一天，如果真到了最后，他们也只好自己出手。
<br/>
<br/>        他看到了陈子昂手下的人员，他多么希望能有一个人将这些信息发送到有关部门，可是他不敢说，也不能说他必须有一个可靠的人。
<br/>
<br/>        他迫切的希望陈子昂的到来，又害怕陈子昂的到来，祁峥他们说，陈子昂是一个非常厉害的人物，可以自己写编程，如果她愿意，她对有些地方如无人之地。
<br/>
<br/>        他现在对生命有一种迫切的渴望，也对结束这份工作有迫切的渴望，他渴望回到父母亲身边，渴望过正常人的生活。
<br/>
<br/>        每当无人之时，每当他知道有些货物又被运出去的时候，他充满了焦虑和愤怒，他深深的自责，感觉到了恐惧和无能为力，原来离开组织，他什么都不是。
