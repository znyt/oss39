　　苏莫和绫裳两人，会到了绫裳的塔楼顶层。
<br/>　　两人的脸色，比之前回来之时，难看百倍。
<br/>　　“此人，是在找死！”苏莫眸中杀机涌动，他若是实力足够，第一个斩杀此人。
<br/>　　绫裳沉默不语，她心中同样愤怒，她们已经被紫霄吃的死死的了。
<br/>　　“走一步看一步吧！”良久之后，绫裳叹了口气道。
<br/>　　目前，没有任何的办法，只希望苏莫能快速强大起来。
<br/>　　只要苏莫比她先一步成为神王，那就还有希望。
<br/>　　苏莫无言，暂时也只能这样了，他必须尽快成为神王，到时候，就由不得紫霄不救芊寻月了。
<br/>　　“你回去吧，在终极神王殿，你有什么需求，随时可以来找我！平常我都会在这里闭关！”
<br/>　　绫裳凝视着苏莫说道，随即她意念一动，将芊寻月的棺椁取了出来。
<br/>　　“嗯！”苏莫无奈的点了点头，随即收起了棺椁，便准备离开。
<br/>　　“苏莫，在神剑山你可能也待不长，尽量提升你的剑道造诣吧！”绫裳叮嘱道。
<br/>　　她心中明白，帝一魂肯定在找苏莫，终有一天能找到。
<br/>　　所以，苏莫在终极神王殿，也不可能一直停留。
<br/>　　只是不清楚，轮回神朝地王宫的那位，是否也在找寻苏莫。
<br/>　　“明白！”苏莫点了点头，随即转身离开了塔楼。
<br/>　　他的心中充满了无奈，只能深深的吸口气，将无奈压下，振作心神。
<br/>　　苏莫并没有回居住的阁楼，而是再次进入了剑神山内部，来到了之前所在的无人之地。
<br/>　　随即，他再次闭上了眼眸，神念弥漫而出，开始参悟剑道。
<br/>　　修炼是枯燥的，但是，当心神完全沉浸其中，时间确实过的飞快。
<br/>　　苏莫一边感受此地无边的剑道意志，一边思忖自己的剑道绝学。
<br/>　　他的身躯之上，渐渐的浮现出了剑道之势，并且缓慢的增强。
<br/>　　这与元神无关，完全是他自己对于剑道的理解，越来越深刻。
<br/>　　不知道过去了多久，苏莫突然出手，又是一记剑指，向前猛然刺出。
<br/>　　刹那之间，犀利的指芒，如同利箭洞穿长空，奇快无比。
<br/>　　这一剑，更是蕴含吞噬之力，而且颇为强生，剑气所过吞噬一切。
<br/>　　这种吞噬之力，不仅能增强剑气的威力，更有削弱敌人攻击的效用。
<br/>　　随即，苏莫继续参悟，如何才能让吞噬之力，与剑道更完美的融合。
<br/>　　随后的时间里，几乎是每过十数日，苏莫便出一次手，吞噬的威力一次比一次强大。
<br/>　　这一日，苏莫突然起身，手中长剑出现，一剑狠狠的斩向前方。
<br/>　　一剑出，风云倒卷，混沌剑气井然化为了一道吞噬剑气，吞噬虚空、吞噬源气，吞噬一切。
<br/>　　直到消失在苏莫的视线之中，良久之后，才传出一声轰响。
<br/>　　“不错！”
<br/>　　苏莫面带笑意，心中很是满意，此剑之中，吞噬之力远没有他全力催动吞噬元神来的强，但是，吞噬之力与剑道融合，却是有非凡的妙处。
<br/>　　这虽然只是他剑道的雏形，但是却也无比强大，比他单纯的蛮力攻击，也好很多。
<br/>　　“这一招就叫吞天噬地！”苏莫低声自喃，虽然这一招与吞天噬地相差甚远，但是，以后一定能达到吞天噬地的作用。
<br/>　　少倾，苏莫收起长剑，再次盘膝而坐，继续参悟。
<br/>　　这一次，他要将不败战气，也融入剑道之中。
<br/>　　这一步就会比较困难了，首先，苏莫虽然能自如的运用不败战气，但是，对于不败战气的了解，远不如吞噬元神。
<br/>　　他开始静静的参悟了起来，抛弃一切杂念，将不败战气与剑道相融。
<br/>　　……
<br/>　　在苏莫参悟剑道之时，下界，神路的格局，早已改变。
<br/>　　神路，第五山，苍穹神宫之中。
<br/>　　此时此刻，神路的苍穹神宫，可谓是强者如云，威震神路九山，已经成为了神路最强的实力。
<br/>　　大殿之中，帝释高坐于主座之上，下方端坐着十几名强者。
<br/>　　司空炎、闻人天都、夕儿、姜枫然、彩芸、龙腾、翼晓晓、战乱、清雅，九人尽皆在坐。
<br/>　　除了九人之外，还有李峰，海明江，龙族雄主龙鏖，以及原三神宗的姜神夜。
<br/>　　姜神夜本是万界山的弟子，但是万界山已经不存在了，便被清雅邀请加入了苍穹神宫。
<br/>　　姜神夜和清雅的关系，虽然不是道侣，但也非常亲近。
<br/>　　大殿中的众人，除了龙鏖和姜神夜、海明江、夕儿、翼晓晓、李峰之外，修为尽皆是虚神境四变。
<br/>　　龙鏖作为龙族雄主，修为早已超越了虚神境四变，至于李峰，依旧没有修为，但是肉身已经堪比神兵利器。
<br/>　　“你们八人，已经决定了吗？”帝释目光环视司空炎和李峰等人，沉声问道。
<br/>　　因为，除了夕儿和翼晓晓之外，司空炎等七位九宫之主和李峰准备今日前去神域。
<br/>　　“决定了，苏莫都走了二十多年，我们也不能再耗下去了！”司空炎大声说道。
<br/>　　“宫主，苏莫如今在神域，不知道是什么情况，我必须尽快过去！”龙腾沉声说道，他在两日之前，才刚刚突破到虚神四变，算是七人之中，突破速度最慢的了。
<br/>　　没办法，他虽然是五爪金龙，天赋绝世，但是到了如今的修为，在天赋方面，已经不及司空炎等人了。
<br/>　　毕竟，司空炎等人，已经被苏莫全部调教成了妖孽。
<br/>　　而翼晓晓，天赋还不如他，自然没有达到虚神境四变。
<br/>　　至于夕儿，由于失踪了多年，虽然后来天赋也被苏莫提升了，但终究是比众人慢了一步。
<br/>　　“可是，神域的情况，我们完全不了解，你们贸然过去，会不会有危险？”帝释面带沉吟之色，他们都知道，苏莫在神域有帝一魂这个大敌。
<br/>　　“宫主，正因为苏莫在神域有危险，我更要前去，龙族在神域必有一席之地！”龙腾沉声道，他相信，龙族在神域，同样是最强大的种族之一。
<br/>　　“龙腾说的对，老夫若是去了神域，也会想尽办法，让龙族给予苏莫庇护！”龙族雄主龙鏖开口了。
<br/>　　他也是准备和龙腾等人一起，进入神域的，因为，他一人无法破开界障。
<br/>　　反正，如今龙道天的实力，已经不弱于他，有龙道天坐镇龙族，他完全可以放心离开了。
<br/>　　“宫主，不必多说了，我们立刻出发了！”体型高大魁梧的战乱，声如闷雷，他们决定的事情，帝释是左右不了的。
<br/>　　“好吧！”帝释叹了口气，几人的实力，都已经站在了神路的巅峰，他也没有太大的担心。
<br/>　　不过，他自己目前不能离开，还要坐镇神路，最起码要等到苏小莫来到神路。
<br/>　　“我们走吧！”姜枫然站了起来，不由分说，便向外飞去。
<br/>　　随即，司空炎等人，便也纷纷离开。
<br/>　　夕儿和翼晓晓两人，望着众人离去的背影，均是微微叹息，她们两人自然也想去，但是奈何修为不足。
<br/>　　“苏莫哥哥，希望你在神域平安！”夕儿心中长叹。
<br/>　　包括李峰和龙鏖在内，一行九人，离开了苍穹神宫，向界障而去。
<br/>　　龙腾等人，并不担心离开之后，苍穹神宫的安全。
<br/>　　因为，苍穹神宫不仅有帝释坐镇，还有整个霸天族的辅佐。
<br/>　　控制霸天族奴印的主印，已经留了下来。
<br/>　　一行九人，没过多久，便来到了界障之地，纷纷停了下来。
<br/>　　临近界障，除了李峰之外，另外八人的身上，全部玄力汹涌，均是承受了巨大的压力。
<br/>　　至于李峰，强大的肉身，都微微变形，全身骨骼咯咯作响。
<br/>　　“诸位小友，以我们九人的实力，击破界障应该问题不大，但是，进入神域之后，会不会还在一起，就不得而知了！”龙鏖环视众人，朗声说道。
<br/>　　他活了一大把年纪，自然考虑的长远，要事先想到一切可能。
<br/>　　“这样吧，进入神域以后，我等若是不在一起，都向龙族的聚集地汇合。”龙腾略一沉吟，道。
<br/>　　“好！”
<br/>　　“可以！”
<br/>　　“没问题。”
<br/>　　众人纷纷应允，这样的话，若是就让分开了，那就能更快的汇合。
<br/>　　“李峰，你能不能撑得住？”彩芸看向面色古板的李峰，略显担心的问道。
<br/>　　李峰因为不修玄力，虽然肉身练到了极致，但实力依旧是众人之人最弱的存在。
<br/>　　“没事！”李峰摇了摇头，他的肉身强度，已经堪比最顶尖的上品圣器，不朽不灭，不会这么轻易死的。
<br/>　　他之所以去神域，倒不是为了帮助苏莫，因为他有自知之明，自己根本没有实力帮助苏莫。
<br/>　　他去神域，是因为，他的肉身已经修炼到了尽头，无法再进一步了。
<br/>　　虽然因为肉身超强的原因，他看起来很年轻，但是，实则寿元已经不多了。
<br/>　　“那好，我们一起出手吧！”闻人天都点了点头，朗声大喝。
<br/>　　随即，九人身上强大的气息升腾，纷纷开始凝聚最强一击。
<br/>　　轰轰轰！！
<br/>　　而后，九人齐齐出手，恐怖的攻击，瞬间全部轰击在了界障之上。
<br/>　　轰隆隆！
<br/>　　一阵惊天动地的爆响，九人的联手攻击，威力之大，无语伦比，比之当初苏莫的一击，还要略强几分。
<br/>　　只是一轮攻击，界障瞬间便被破开，直接炸出了一个方圆两丈左右的大动。
<br/>　　“走！”
<br/>　　龙鏖暴喝一声，身形如电，急速冲向界障之内，其他人仅随其后，几乎是在一个眨眼的时间之内，便纷纷进入了界障之内。
<br/>　　少倾，界障恢复原状，九人消失无踪。
