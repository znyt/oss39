　　不过李二的郁闷并没有就此结束，李泰虽然被揍了，但却偿到了被群臣夸耀的甜头，一个多月之后，又开始撺掇着老头子花钱赎回当年东征高句丽那些将士的遗骨。
<br/>　　只是这一次老李并没有全听李泰的，而是简单的写了一封国书递送给高句丽。
<br/>　　前朝将士遗骨，说重要也重要，说不重要也不重要，但如果说拿钱买回来，怎么听这事儿都不靠谱。
<br/>　　老李现在虽然有些钱，但有钱也不是这样的花法，李泰可以崽卖爷田不心疼，但李二不能。
<br/>　　他知道这钱是自家大小子一点点从富商和异族手里扣出来的，如果就这么花出去，不说大小子怎么想，只怕朝臣都会有些想法。
<br/>　　不过好面子的老李这下可真是失算了，写出去的国书很快就有了回复，只是答案只有两个字：免谈！
<br/>　　老李看到这份国书的时候，气的当时就炸毛了，砸着桌子吼着要发兵高句丽，这点面子都不给，让李二这位‘天可汗’颜面何存。
<br/>　　不过这些都和李承乾没关系，他现在等于被变像的流放，没人来管他在干什么，也没人通知他朝堂上发生的事情。
<br/>　　当然，从其他渠道得知这一情况属于例外。
<br/>　　长孙冲已经带着五百‘獠牙’出发了，同行的还有‘獠牙’副大队长尉迟宝林，程处默、李崇义、唐善识等亦在同行之列，美其名曰出去见见世面。
<br/>　　这些事情的联系人是长孙无忌，为了家族这老货也算是挺拼的，约了不少当初秦王府旧部，一起探讨了大概有七八天，最后才达成一致，也就有了少年纨绔们的薛延陀之行。
<br/>　　李二同样知道这件事，但是李承乾没说，他也就装成不知道，反正这里有五成的份子是交到国库的，这就已经足够了。
<br/>　　对李承乾，李二现在是又爱又恨，觉着如果这个小子能少气自己几回的话，完全可以评为有史以来的十佳太子之首。
<br/>　　可是事与愿违，这小子太能折腾，哪怕被排斥在权力圈子之外，也不能阻止他的得瑟，不能用权利，那就用利益，一趟薛延陀之行，将天策府的老班底几乎一网打尽，偏偏还让人挑不到他的错处。
<br/>　　或许有人会说李承乾这样的行为是在勾结外臣，但同样的事情在不同的皇帝看来，严重性是不同的。
<br/>　　以李二这种可以和自己的旧部共享天下的皇帝，对李承乾这样的行为并不怎么在乎。
<br/>　　况且李承乾也没有背着李二搞东搞西，所有的一切都是作在明面上，而且做的光明正大，这更让老头子没什么话说。
<br/>　　钱啊，谁跟钱有仇？如果现在国库里有几千万的银钱，李二早就挥军东进，去搞那个敢不给他面子的高句丽了。所以李二这次问都没问，反正生意作成了，这里面有他一半的钱，这就足够了。
<br/>　　长安，大唐医学院一片繁忙的景像，无数人正在挖坑、挑水，李承乾带着孔雯两人也在不停的忙碌。
<br/>　　只是相比于其他人的进度，他们两个的坑才将将能把两只脚装进去，如果往里填土，只怕连脚面都盖不上。
<br/>　　“小雯，要不咱们换个地方挖怎么样？这里的土太硬，就算把坑挖出来，估计那树也种不了。”李承乾抱着一根锄头，吐着舌头，呼呼带喘的说道。
<br/>　　“什么嘛！明明都是一样的地，是你太缺乏锻炼了好不好。”孔雯娇俏的皱了一下鼻子，有些俏皮的数落着。
<br/>　　“谁说的？我体力好着呢，就是没有动力罢了。”李承乾边反驳，边把锄头丢掉。
<br/>　　后世见多了春天时候的植树造林工程，所以李承乾这一次突发奇想，打算在医学院里也搞一个植树活动，并且还打算亲历亲为，带上小情人一起种树。
<br/>　　结果没想到这该死的坑竟然这么难挖，别人树都栽了好几棵，自己却只刨了一个能容纳两只脚的小土洼。
<br/>　　“动力？什么动力啊？”孔雯好奇的问道。
<br/>　　“亲一下我就告诉你。”李承乾故技重施，只是对像换成了孔雯。
<br/>　　结果孔雯根本就没上当，白了他一眼之后哼了一声：“哼，少来，上一次你就是这样欺负小四妹妹的，对不对？”
<br/>　　李承乾尴尬的咂咂嘴，正想说话，就看到苏猛正站在一边，顿时有些恼羞成怒：“看什么看，滚一边挖坑去。”
<br/>　　等到苏猛灰溜溜的跑去一边挖坑之后，李承乾又恢复了正人君子的模样，从杨雨馨手中抓过自己的羽扇，向着远处的建筑群一指：“这里，将会培养出大唐未来最顶尖的医生、护士，百年之内这里将会成为全世界所有医学工作者心目中的圣地……”
<br/>　　后世太祖爷爷的诗词中提到过：指点江山，激扬文字，粪土当年万户候……。
<br/>　　此时的李承乾终于找到了一丝感觉，羽扇轻点间侃侃而谈，一颗脑袋却在不断向孔雯那边靠了过去。
<br/>　　终于位置差不多了，李承乾大头一摆，果断在孔雯的面颊啄了一口，然后就飞也似的就向远处逃去。
<br/>　　“哎呀！”孔雯骤然被李承乾‘袭击’先是吓了一跳，接着立刻就反应过来发生了什么，小脸刷的一下变得通红，扭身就像着李承乾的方向直追。
<br/>　　“夜魅姐，我，我们不跟过去么？”眼看李承乾和孔雯越追越远，杨雨馨问道。
<br/>　　“蝙蝠在那边，不用我们过去。”夜魅看着李承乾离开的方向，淡淡说着，心底却发出一声无奈的叹息，脑中不由自主的又想起那天晚上的一幕。
<br/>　　夜魅依旧是一身党卫军军服，不是熟人绝不会知道这一身制服下面是一个女人的躯体。
<br/>　　但中性的外表不等于中性的人，夜魅不管怎么说也是一个女人，那天晚上李承乾的亲密接触对她来说还是第一次，所以尽管时间已经过去很久，但对于她来说却如同昨日刚刚发生一般。
<br/>　　杨雨馨那天晚上因为位置的关系，并不知道李承乾和夜魅之间到底发生了什么，但是她却可以清楚的感觉到夜魅的变化，随然那种变化有些说不清道不明。
