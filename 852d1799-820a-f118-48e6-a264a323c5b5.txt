　　苏莫离开那座城池之后，连续飞行了几日，终于离开了终极剑域。
<br/>　　期间，他也连续数次改变路线，并数次释放出噬冥虫。
<br/>　　几日时间，均是一路顺利，没有遇到任何的风险。
<br/>　　“应该没有追来了吧！”
<br/>　　一边飞行，苏莫一边心中暗道，若是有神王级别的强者追来，应该早已经追到他了。
<br/>　　苏莫面无表情，终极剑域他是不能再回去了，但是他心中，却也有些担忧。
<br/>　　他倒不是担忧绫裳，而是担心紫霄会怎么对付他。
<br/>　　他得到了终极剑道和终极神剑，必然已经被终极神王殿知晓，紫霄肯定不会放过他。
<br/>　　芊寻月在紫霄的手上，此人一定会运用这个筹码，逼他就范。
<br/>　　这让苏莫心中没有底，面对神王级别的紫霄，他目前完全没有底气。
<br/>　　叹了口气，苏莫心中无奈，目前没有能力对付紫霄，他只能离去，日后有实力再说了。
<br/>　　好在，芊寻月不仅是紫霄要挟他的筹码，也是要挟绫裳的筹码，紫霄轻易不会伤害芊寻月。
<br/>　　“去哪里呢？”
<br/>　　飞驰在长空之上，苏莫面带沉吟之色，是回轮回神朝还是去找龙腾等人。
<br/>　　回轮回神朝的话，风险很大，还是去天龙神朝比较稳妥一些。
<br/>　　不过，在去天龙神朝之前，他还要先将擒拿的几人处理一下。
<br/>　　片刻之后，苏莫发现了一片荒林，闪身钻了进去。
<br/>　　荒林之中，苏莫盘膝坐在一株古树之下，双眸微闭，意识进入了吞噬元神之中。
<br/>　　吞噬旋涡之中，拓跋流明和云沐两人，以及古剑山的元神俱在。
<br/>　　三人均是没有死，因为苏莫并没有施以炼化。
<br/>　　拓跋流明和云沐两人，伤势恢复了不少，在吞噬漩涡之中不断的攻击、挣扎，让苏莫的元神承受了很强的压力，必须尽快的解决。
<br/>　　苏莫意念一动，吞噬元神便开始炼化古剑山万化神剑元神，抹杀其中蕴含的古剑山神智。
<br/>　　啊！
<br/>　　只听得一声惨叫声响起，古剑山的万化神剑元神，便成为了无主之物。
<br/>　　只剩下元神的古剑山，并且是遭受大创的元神，古剑山根本没有反抗的实力。
<br/>　　“这……！”
<br/>　　这一幕，自然也看在了云沐和拓跋流明的眼中，两人顿时心中大骇。
<br/>　　“秦陨，我们无冤无仇，你放我离开！”云沐娇声大喝。此时此刻，她心中惧怕不已，早已没有了当初的傲气。
<br/>　　“秦陨，你到底是什么人，你那是什么元神？”拓跋流明同样暴喝，他没有让苏莫放了他，因为这显然不太可能。
<br/>　　“云沐，我们的确是无冤无仇，但你想抢夺终极剑道，这就是仇怨！”苏莫冷漠的意念之音响起，他不是嗜杀之人，此女他也在考虑要不要杀了。
<br/>　　“抢夺终极剑道，的确是我的错，我愿意赔偿！”云沐沉声说道。
<br/>　　苏莫闻言，心中沉吟，她不能放过此女，因为一旦放过对方，很可能会遭到报复。
<br/>　　他的敌人已经够多了，不能再增加新的敌人了。
<br/>　　况且，此女有天罚神瞳元神，还有天衍之体，这都是至宝。
<br/>　　为了自己的命运，为了不断的强大，他只能辣手摧花了。
<br/>　　“不好意思，当你出手抢夺终极剑道之时，你的命运便注定了！”
<br/>　　苏莫冷漠的声音响起，随即立刻催动吞噬元神，开始炼化云沐。
<br/>　　在吞噬漩涡之中，吞噬元神的炼化之力，是非常恐怖的。
<br/>　　只要不是实力超越苏莫很多，基本上都不可能抵挡的住。
<br/>　　“秦陨，你敢杀我，定然会后悔的！”
<br/>　　在吞噬元神的炼化之下，云沐全身的血肉、源力都开始消融，并且神智遭到了强力的磨灭，顿时惊恐的大吼了起来。
<br/>　　苏莫没有理会云沐的吼叫，极力炼化，仅仅是十几个呼吸的时间，云沐便彻底的消失了。
<br/>　　身躯消融，元神被剥夺，神智被抹杀，血脉之力和一身精气流入了苏莫的体内。
<br/>　　这一幕，被拓跋流明看在眼力，顿时惊骇欲绝。
<br/>　　不过，他没有什么动作，在极力的疗伤，只有伤势恢复，他才有逃出去的可能。
<br/>　　苏莫暂时也没有对付拓跋流明，体内源力涌动，功法运转，在炼化吸收云沐的精气。
<br/>　　云沐的精气，包含了其一身的修为和气血之力，非常的庞大，只不过不是五行之力，对于苏莫来说，用处不大。
<br/>　　云沐的天衍之体血脉，被苏莫融入了自身的血脉之中，他也没有去查看这种血脉之力有什么能力。
<br/>　　毕竟，他的血脉之力太多了，也不在乎这种血脉的强弱了。
<br/>　　除非是太古异体榜前四的血脉，其它的血脉之力，对他的作用并不是很大。
<br/>　　少倾之后，苏莫将云沐的一切吸收了之后，意念再次沉入吞噬元神之中。
<br/>　　“拓跋流明，你和拓跋青云是什么关系？”苏莫意念之中，传出不容拒绝的声音，询问拓跋流明。
<br/>　　他知道对方在疗伤，但他并不在意，毕竟，以对方的伤势，没有一个月的时间，很难重回巅峰。
<br/>　　就算对方恢复到了巅峰，也别想从吞噬元神之中逃脱。
<br/>　　“拓跋青云是我二叔。”拓跋流明沉声说道，他并未隐瞒，因为此事并不是什么秘密，随便一打听就能知晓。
<br/>　　他暂时也不想激怒苏莫，而是尽量拖延时间，恢复实力。
<br/>　　“二叔？”苏莫闻言了然，果然是有关系，而且关系还不浅。
<br/>　　“我需要知晓你二叔的详细情况，包括元神、血脉等等！”苏莫冷然的说道。
<br/>　　“可以！”
<br/>　　拓跋流明点了点头，出奇的配合，随即略一沉吟，道：“二叔的修为是中级神王，元神和你的这种吞噬旋涡类似，但是效果完全不同，作用是封印，封天封地封人，足以封困一切存在……！”
<br/>　　拓跋流明徐徐开口，将其二叔的情况，详细的向苏莫讲述了一番。
<br/>　　片刻之后，拓跋流明讲述完毕，道：“大概就是这种情况！”
<br/>　　苏莫闻言，沉默不语，在对方的讲述之中，拓跋青云的元神能力是封印，并且并无其它元神和血脉之力。
<br/>　　对此，他并不相信，若是没有其它元神和血脉之力，那么拓跋青云的天赋，也太一般了。
<br/>　　“你确定拓跋青云只是单一元神，并且没有血脉之力？若是被我发现你说谎，你将承受难以想象的折磨。”苏莫冷声说道，他并不会轻易相信对方的话。
<br/>　　“我句句属实，至于二叔有没有其它元神和血脉，没有人知晓，因为，从来没有谁，能让二叔全力出手！”拓跋流明沉声说道。
<br/>　　苏莫闻言，心中思忖，就算拓跋青云有其它元神和血脉之力，应该也不会过早的暴露。
<br/>　　毕竟，他敌人的是帝一魂，肯定会隐藏自己的实力。
<br/>　　总之，拓跋青云绝对不可能就这点天赋，就算没有其它元神，也该有强大的血脉之力。
<br/>　　“拓跋青云如今在何地？”苏莫再次问道。
<br/>　　“就在地王宫，一直在闭关，千年都未出关了！”拓跋流明说道。
<br/>　　“看来也在争取早日突破到高级！”苏莫心中了然，帝一魂和拓跋青云在竞争，争取率先成为高级神王。
<br/>　　因为，一旦率先成为高级神王，基本上就胜券在握了。
<br/>　　毕竟，在目前的神域，至强者不见踪影，以两人的天赋，一旦成为高级神王，基本上在神域就能无敌了，就算不能无敌，也没有人能伤到他们了。
<br/>　　可以说，帝一魂若是先一步成为高级神王，再加上天王宫的帮助，拓跋青云必死；反之亦然，帝一魂必死。
