　　　　“剩下的这一处关押人质的地方，是影子部队的一处营地。”蛰龙指了指红色砖块西侧位置的水泥块，说道：“这里原本是当地政府军用来关押要犯的军事监狱，反政府武装势力在西伦科特的帮助下发动政变后，这里就被影子部队所控制，成为了影子部队的营地，而且因为这里原本就是军事监狱，所以更加方便关押被抓来的特种部队。这个军事监狱中，除了有影子部队的精锐驻守外，还有两架AH-1Z‘蝮蛇’武装直升机，也就是你们刚才看到的那两架，以及三辆BMP-3步兵战车。”
<br/>　　“确定这三个地方关押的是哪个国家的特种部队吗？”罗昊对蛰龙问道。
<br/>　　“暂时还不清楚。”蛰龙摇头说道：“但是在你们要去的那个军事监狱中，不过如果有一个人能提供线索。”
<br/>　　蛰龙从口袋中拿出一张照片递给罗昊，说道：“照片上的人叫Parker，代号‘八爪鱼’，是西伦科特的作战参谋，很受西伦科特的信任，如果你们能抓住他，并且撬开他的嘴，我想应该可以得到很多有价值的东西，不仅是特种部队关押的位置，还有西伦科特的下落。”
<br/>　　“等一下。”慕容飞打住蛰龙的话，说道：“你的意思是让我们四个人去搞掉这个军事监狱，就用我们手里这些轻武器对抗两架‘蝮蛇’武装直升机和三辆BMP-3步兵战车，你确定你没有开玩笑？而你的小队去解决那两个反政府武装势力的营地？”
<br/>　　“如果你不愿意的话，我们可以交换。”蛰龙说道：“我的小队去搞掉这个军事监狱，你们去端掉这两个反政府武装势力的营地。我的小队加上我自己一共就只有五个人，而这两个反政府武装势力营地中的兵员数量总数加起来接近400人，1:80的敌我人数比，而且这两个地方距离非常靠近，很容易相互之间支援，如果你愿意交换，我非常乐意。”
<br/>　　“你的小队一共就只有五个人！？”慕容飞瞪大着眼睛看着蛰龙。
<br/>　　“你如果不信，可以问他们两个。”蛰龙指了指罗昊和李然。
<br/>　　罗昊和李然摸了摸脑袋，然后点了点头，证明蛰龙所言的真实性。
<br/>　　“我去！”慕容飞对着蛰龙竖起大拇指，说道：“兄弟，我连墙都不扶就服你！”
<br/>　　“既然没意见，我们就分头行动。”蛰龙把摆在地面上的砖块和水泥块打散掉，说道：“你们可以向联合行动司令部申请情报支援，获得‘八爪鱼’的详细情报，以便于你们后面的行动。”
<br/>　　“等一下。”罗昊叫住蛰龙，问道：“你救出那些特种部队队员以后，打算怎么办？”
<br/>　　“我会带着他们进入营地北边的树林中，有战斗力的就跟着我们继续救人，没有战斗力的，就把他们留在那片树林子里面自生自灭。”蛰龙冷漠的回答道：“他们的命是我救的，就得服从我的安排，想活命，他们就只能听我的。”说完，蛰龙走下楼梯，从这栋大楼废墟中离开。
<br/>　　看着蛰龙消失的地方，李毅峰沉吟一声，说道：“这个人很冷，就像一块冰，不，比冰还要冷。”
<br/>　　“这跟他的出身和经历有关系。”罗昊叹了口气，说道：“如果不是这样冷，对敌人冷，对自己人也冷，他连红旗学校的大门都走不出来，或许当年就已经死在搏击台上了。走吧，我们该干活了。”
<br/>　　罗昊通过单兵PDA上的卫星地图，很容易就找到了那个军事监狱的位置。这个现在被影子部队所控制，并且成为了影子部队一处营地的军事监狱位于罗昊他们四个人现在位置西南方向，距离他们差不多有三公里的路程。
<br/>　　罗昊通过无线电跟云熙取得了联系。
<br/>　　“小熙，帮我找一个人的资料，越详细越好。”罗昊对云熙说道：“这个人叫Parker，代号‘八爪鱼’，是西伦科特的作战参谋。必要的时候，可以向詹姆斯寻求帮助，在西伦科特还是美军少将的时候，这个人应该就是西伦科特的手下，从詹姆斯那里应该可以得到一些资料。记住，我要最详细的资料，最好把这个人身上有几颗痣都给我弄清楚。”
<br/>　　“你这个癖好可有点奇怪。”云熙一边敲打着键盘，一边对罗昊说道：“而且除了我自己的男人，我可没有兴趣去弄清楚其他男人身上长了几颗痣。放心吧，我会在最短的时间里把你要的东西给找来。”
<br/>　　云熙没有问罗昊为什么突然要找“八爪鱼”Parker的资料，既然罗昊有这个要求，那必然是对营救行动有重要影响的。
<br/>　　罗昊中断了跟云熙的联系后，慕容飞用手捂着自己的小心脏，学着刚才云熙在无线电中说话的口气，说道：“除了我自己的男人，我可没有兴趣去弄清楚其他男人身上长了几颗痣。我说狂龙，我知道你们俩恩爱，可是能不能别没事就撒狗粮啊？”
<br/>　　“你现在都告别五姑娘了，我们撒狗粮你吃吗？”罗昊瞥了眼慕容飞，反问道。
<br/>　　“是哦，我都忘了自己也是有老婆的人了。”
<br/>　　慕容飞点了点头，恍然大悟。
<br/>　　“你小子现在把柄可是被我们抓住了。”李毅峰一把勾住慕容飞的脖子，说道：“下次见到佟医生，我一定去打小报告，让她晚上偷偷给你扎针。”
<br/>　　慕容飞刚才对自己呛话，让罗昊想起了徐旭东，只是现在还不知道他到底能不能度过这次难关，能不能继续在作战部队服役。
<br/>　　罗昊甩了甩头，抛开自己脑海中想的东西，把蛰龙留给自己的那张照片在李毅峰、慕容飞、李然三个人眼前展示了一遍，说道：“记住这个人，他叫Parker，代号‘八爪鱼’，是我们这次行动主要目标之一。记住，我要活口！全都听明白没有！？”
<br/>　　“明白！”
<br/>　　李然、李毅峰、慕容飞三个人沉声回答。
<br/>　　罗昊收起照片后，四个人离开了大楼废墟，朝着三公里外的军事监狱前行过去。
