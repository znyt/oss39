
<br/>　　<br/>　　    侍应生刚进了隔壁的房间，把茶盘放到桌子上，菜刀文手插在口袋里，斜了着眼睛随后推开门就闯了进去。<br/>　　<br/>　　    “唰，唰”屋内，胡胡，常子敬和齐峥坤全都望了过去，侍应生一回头看见胡胡，吓的大腿根都发紧了。<br/>　　<br/>　　    “找谁啊？”胡胡笑着问道。<br/>　　<br/>　　    “啪”菜刀文仿佛瞬间就地痞，流氓和无赖上了身，他棱着眼珠子掐着侍应生的领子说道：“你怎么着啊？我们说话没有力度系不系？把茶盘给我端回去，我们先点的茶没送，你给送到了这边来了，我看你这破茶楼不想开了系不系？”<br/>　　<br/>　　    侍应生脸色难看的说道：“先生，我跟您说过多少次了，是他们先来先点的茶，您要的茶水稍后我就给你送过去”<br/>　　<br/>　　    “别他么和我扯了，我来的时候这还没人呢……”<br/>　　<br/>　　    胡胡连忙站起来，伸手拦了下菜刀文，和颜悦色的说道：“朋友，是不是有什么误会，别动手，好好说行么，就是喝个茶而已，也没必要弄出流血事件来吧”<br/>　　<br/>　　    “呵呵，怎么的？你要扛事是么”菜刀文挡了下他的胳膊，指着胡胡说道：“给我老实的眯着，没你的事就不要扎刺，我现在是和他在讨论先来后到的问题，你别一不小心把火药星子喷到自己身上去”<br/>　　<br/>　　    胡胡皱了皱眉，也没吭声，他的性子向来比较稳当和成熟一点，看见菜刀文这么胡搅蛮缠的，他也就没心思跟对方掰扯了，但是坐在旁边的齐峥坤就略微年轻气盛了一些。<br/>　　<br/>　　    “就一壶茶，至于么？你要就给你，动手难为人家服务员算怎么回事啊？喝个茶，还给你喝出优越感来了啊？”<br/>　　<br/>　　    胡胡低头说了声：“阿坤，别说话”<br/>　　<br/>　　    “呵呵……”菜刀文棱着眼珠子瞅着坐在椅子上的齐峥坤，眼神里充满了小雀跃和挑衅的味道，坦白讲他还真怕对方不接腔呢。<br/>　　<br/>　　    齐峥坤则眯着眼睛看着他，一脸冷淡的笑容，似乎根本都没拿他当回事。<br/>　　<br/>　　    “你瞅啥啊？”菜刀文昂着脑袋说道。<br/>　　<br/>　　    在北方一带，有不少的争端都是出自两句对白里。<br/>　　<br/>　　    如果有人跟另外一个人说了一句：“你瞅啥啊？”<br/>　　<br/>　　    那这个人要是没有说话到还好办，但他要是回了一句：“瞅你咋的？”<br/>　　<br/>　　    这两句对白一旦接上，那妥妥的下面就会紧接着起争端了，特别是在喝酒多了以后，就因为这两句话而干起来的可能性是非常大的。<br/>　　<br/>　　    此时没有酒，不过对于故意找茬的菜刀文来讲，他现在就眼巴巴的等着对方接上他那句话呢。<br/>　　<br/>　　    “瞅你怎么的，你不能看啊？”齐峥坤毫不客气，硬邦邦的回了一句过来。<br/>　　<br/>　　    “咣当”一直在外面站着的林汶骐突然就一脚蹬开门，指着齐峥坤说道：“看多了我怕你承受不住，社会的水深着呢，你那点个头容易被淹死，脾气太大了对你没好处”<br/>　　<br/>　　    “你看把你们给厉害的，觉得自己有杀人许可证啊？”齐峥坤非常不爽的顶了一句，但他话刚说了一句，林汶骐就彪悍的把桌子上的茶盘给掀翻了。<br/>　　<br/>　　    “哗啦”林汶骐一掀茶盘，滚烫的茶水和盘子连带着全都飞了起来，直接就奔着齐峥坤那边砸了过去。<br/>　　<br/>　　    泡茶的水标准性大概八十度左右，这个温度的水落在人身上的话，足以给你烫出一层的水泡了。<br/>　　<br/>　　    齐峥坤和常子敬没想到对方一言不合就掀了桌子，作风彪悍还不讲道理，两人完全都没有这个意识，自然也没有机会躲开，顿时一壶茶水还有被子全都砸在了他俩的身上，瞬间裸露在外面的手臂还有脸上就喷上了茶水，模样狼狈至极，而在林汶骐动手的时候，菜刀文紧接着就跟上了，他一把扒拉开身前的侍应生，抬腿照着椅子上的齐峥坤就踹了过去。<br/>　　<br/>　　    “么的，见过捡钱的，没有见过捡麻烦的，今天我算是开了眼界了，干他！”<br/>　　<br/>　　    林汶骐和菜刀文先后动手，两人的动作完全都是街头流氓斗殴的架势，拳砸脚踢，骂人的时候唾沫星子都喷了出去。<br/>　　<br/>　　    齐峥坤，常子敬被茶壶给砸到的时候没有反应及时，但接下来再动手，他俩就有准备了，再加上旁边的胡胡，三个玄门中人就展现出了高手风范。<br/>　　<br/>　　    胡胡阴着脸，伸手架着菜刀文要砸过来的拳头，右手握紧中指突出，直接就捣向了对方的胸膛。<br/>　　<br/>　　    “嘭”菜刀文胸口被一指点中，瞬间一股酥麻的感觉就遍布了全身，他倒吸了一口冷气，疼的直咬牙，胡胡趁机左手反方向一握，就抓上了他的手腕，然后往自己这边一带，抬起膝盖照着他的小腹就撞了过去。<br/>　　<br/>　　    “咣”一膝盖顶上，菜刀文顿时就憋着一口气没吐出来，冷汗直冒的往后退了几步。<br/>　　<br/>　　    旁边，林汶骐似乎也不敌常子敬和齐峥坤，交手几下之后，就被逼的连连躲闪，躲闪不及之下身上也挨了好几次，两人刚才一顿郎朗的吹牛逼就像一道流星一样一闪即逝，被揍得差点人脑袋给打成狗脑袋了。<br/>　　<br/>　　    菜刀文跌跌撞撞的从房间里出来，林汶骐指着屋内三人说道：“敢还手，是么？行，你们等着，有种别跑，我摇人过来”<br/>　　<br/>　　    菜刀文咬牙切齿的说道：“别跑啊，别跑，你们等着的……”<br/>　　<br/>　　    两人说完撂下几句狠话之后狼狈的掉头就跑，速度嘎嘎快，几步就跑到了楼梯口这里。<br/>　　<br/>　　    “这真是我行走江湖，人生中最耻辱的一天了，以后我说什么都得把这一段记忆给选择性的忘过去”林汶骐边走边说道：“么的，明明能跟人拼一下的，还得装成个犊子样，窝囊不的？我就问你窝囊不窝囊？”<br/>　　<br/>　　    “哎呀，别说了，演戏那不得演全套么，咱俩不就得装着不敌人家，然后回过头来好再下手的么，你也不能在这就跟他们分出个胜负来啊，没事，一会好好报复他们就是了”<br/>　　<br/>　　    两人仓惶而逃，出了茶楼之后，就朝着车里的王惊蛰比划了个“ok”的手势，然后转头就走向了一旁。
