　　一时间，数十架战斗机，纷纷起飞。
<br/>　　巡洋舰、驱逐舰、护卫舰也全部出动，各种导弹纷纷弹射而出，犹如漫天的流星雨，朝叶尘爆射而来。
<br/>　　“八咫镜！！”
<br/>　　叶尘心念一动，祭出了防御法宝，同时挥舞着紫琼剑，冒着漫天的导弹，朝这只航母舰队的核心冲了过去。
<br/>　　轰！轰！轰！
<br/>　　漫天的导弹纷纷在空中爆炸开来，然而却根本无法伤到叶尘分毫，叶尘在炮弹之间穿梭前行，简直游刃有余。
<br/>　　“斩！”
<br/>　　叶尘的速度实在太快，在加上运用了空间穿梭的能力，几乎瞬间就跨越了那漫天的导弹，同时随手几剑，便将空中的几十架战斗机全部摧毁。
<br/>　　“再斩！”
<br/>　　叶尘轻松击毁了空中力量之后，忽然身体猛然一顿，俯冲到一艘巡洋舰的上空，然后再次狠狠一剑斩出！
<br/>　　轰隆！
<br/>　　整艘巡洋舰，瞬间便被叶尘站位两段！
<br/>　　“啊啊啊！！”
<br/>　　军舰被叶尘斩为两半之后，顿时便开始迅速下沉，一道道惨叫声，也在海面上此起彼伏。
<br/>　　轰！
<br/>　　叶尘却在那艘巡洋舰的残骸上一蹬，整个人已经扑向下一只军舰。
<br/>　　美国的海军实力虽然强悍，可是一旦让叶尘近了身，就立马丧失了战斗力，机动性和叶尘根本无法相提并论。
<br/>　　这就好比，用一根绣花针，去刺一张白纸，白纸虽然庞大，可是却根本拿这绣花针毫无办法，只能眼睁睁的看着自己，被对方戳成了稀巴烂！
<br/>　　“什么！”
<br/>　　舰队指挥官看到这一幕，顿时脸色惨白，虽然他得到前方的战报，心里早就有所准备，可还是没有想到，居然会败的如此迅速，如此的凄惨！
<br/>　　“撤退！全军撤退！”
<br/>　　最终，舰队指挥官当即下达了撤退的命令。
<br/>　　反正上头分派给他们的任务，也只是让他们拖延时间，好为后方发动核武争取准备的时间。
<br/>　　然而，他们想要撤退，叶尘又岂会答应？
<br/>　　轰轰轰！
<br/>　　一道道长达逾千米的紫色剑芒，在夜空中划过，而每当有一道剑芒划过，便会有一支军舰，被劈成两半，沉没在海洋之中。
<br/>　　前后不过才二十来分钟，整支航空母舰战斗群，便已经被打的支离破碎，只剩下几支补给舰，保护着三艘航母，分别朝不同的方向暴退而去。
<br/>　　叶尘的身影，在空中一阵闪烁之后，追上其中一只航空母舰，直接抬手就是数剑！
<br/>　　轰轰轰！
<br/>　　一艘近十万顿级的航空母舰，竟然被叶尘区区几剑，劈成了一堆废铁！
<br/>　　伴随着一道道哀嚎声，这艘航空母舰连同舰上数千米国海军，全部沉没在太平洋之中……
<br/>　　“太凶残了！”
<br/>　　“小叶将军简直就是超人！”
<br/>　　“即便是神灵，也不过如此吧？”
<br/>　　……
<br/>　　华夏军舰之上，众人一直在密切关注着叶尘的一举一动。
<br/>　　看到叶尘一人挑翻了整个航空母舰战斗群，顿时一个个激动不已，忍不住大喊大叫起来。
<br/>　　唯独叶天歌和林震南两人，眉头紧皱，脸上露出一抹担忧之色。
<br/>　　两人对视了一眼，林震南道：
<br/>　　“叶将军，你可是看出了什么？”
<br/>　　叶天歌深吸了一口气，缓缓点头道：
<br/>　　“经过之前珍宝港那一战，米国军方不可能不知道，寻常海军对小尘已经构不成威胁，可是他们竟然还派出一支航空母舰战斗群，前来阻拦，简直就是在故意送死！”
<br/>　　林震南也点了点头道：
<br/>　　“不错！他们这么做，很显然只是拖延时间，如果我没有猜错的话，接下来他们怕是要动用那个了……”
<br/>　　叶天歌自然知道，林震南所说的那个，指的是什么，脸色顿时越发的凝重，连身体都有些微微颤抖，不由得低声喃喃道：
<br/>　　“小尘，适可而止吧！你真的不能再往前走了！”
<br/>　　……
<br/>　　而早在航空母舰战斗群，去拦截叶尘之时，米国国会大厦，
<br/>　　“总统阁下，部长阁下，州的一百多枚核弹，已经全部准备就绪，目标也已经锁定，是否对其发动攻击？”
<br/>　　米国总统普朗克，和国防部长拉姆斯，两人对视了一眼之后，纷纷点了点头，
<br/>　　“攻击！”
<br/>　　随着米国总统一声令下。
<br/>　　很快，
<br/>　　无数米国民众看到，一道如流星般的火光，自得克萨斯州飞出，横跨上千里，直奔西面的太平洋上空飞去。
<br/>　　而与此同时，太平洋上的动静，也早已经惊动了世界各大国高层。
<br/>　　岛国离的最近，而且又和米国同气连枝，所以最想得知了情报。
<br/>　　而此刻，岛国首相龟田三郎等人，正死死的盯着前方卫星传来的战况，眼眸中透着浓浓的震惊之色，
<br/>　　“没想到才短短几个月，这家伙的实力，竟然比之前强大了这么多！”
<br/>　　“原本这叶狂仙失踪之后，我们制定了一系列对华夏进行反扑的计划，还好都没有付诸行动，否则后果还真是不堪设想！”
<br/>　　“我们筹划了这么久，难道就这么放弃了吗？”
<br/>　　“先别急！现在谁能笑到最后，可还不一定呢！”
<br/>　　……
<br/>　　就在众人正议论纷纷之时，忽然前方又传来了战报，
<br/>　　“报告首相大人！米国分部在得克萨斯州的核弹基地，似乎出现异动！”
<br/>　　岛国一众高层听到这个消息，顿时全都精神一震，纷纷面露喜色，
<br/>　　“我就说吧！米国肯定不会就此罢休的！”
<br/>　　“米国手中的核弹量，足以轻松将整个地球摧毁，区区一个叶狂仙，也敢挑衅世界第一大国的威严，简直就是不知死活！”
<br/>　　“他若是在珍宝港行凶之后，立即返回华夏，或许米国还有所顾忌，偏偏他不知死活，竟然想要踏足米国国土，米国肯定不惜一切代价，也要诛杀此人！”
<br/>　　……
<br/>　　岛国一众高层，全都喜形于色，似乎都已经看到，叶尘死在核弹之下的情景。
<br/>　　而与此同时，华夏军方也发现了这一情报，顿时个个脸色无比难看，
<br/>　　“米国终究还是要动用核武了吗？”
<br/>　　叶天歌双拳紧握，望着屏幕上，依旧在朝米国方向疾驰的那道紫色光芒，脸色无比惨白，低声喃喃道：
<br/>　　“小尘，停下！快停下！”
