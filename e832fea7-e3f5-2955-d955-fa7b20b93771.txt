　　第241章
<br/>　　长安城的城门打开，鼓乐相送，戎族谈判团将离开长安，踏上回程的路。一行三百余人，每个人都满载而归，身上带着长安购买的胭脂，丝巾，香料，以及长安顶级铁匠打制的兵器。
<br/>　　这些东西在草原可是抢手货，带着这些回到草原，他们每个都会如从藏宝洞归来的盗贼一样受欢迎。
<br/>　　李乐天带着戎族一行送到城外，与呼延力依依惜别。
<br/>　　“愿此去一路顺风，愿我们两族友谊如日月不移。”李乐天举起送别酒。
<br/>　　“长生天的子孙铭记帝国赠予的友谊，我们珍视这份友谊甚于珍视我们的弯刀。”
<br/>　　“大皇子，临别之前，我想请教一件事，关于程大雷帝国会如何处理？”金问道此刻过来。
<br/>　　“程大雷是帝国的子民，他触犯了帝国律法，会由帝国的律法处置他。”李天乐。
<br/>　　“原也应该。”金问道微微点头，然后叹口气：“可否请求大皇子一件事，程大雷死后，请讲他的遗体妥善安葬，为此，北蛮部愿献上我们的友谊。”
<br/>　　以程大雷犯的罪，即便凌迟后尸体也要得到惩罚，乱马踏成肉泥，烈日下鞭尸，或剥皮实草。可是，北蛮部的友谊对帝国来讲也是极其珍贵。
<br/>　　“据我所知，北蛮部应该很恨程大雷？”
<br/>　　“对于敌人，我希望他死，但同样身为战士，他值得被尊重。如果可能，请转告他，北蛮部每一个人都会用尽一切办法杀死他，可同时，他也拥有整个北蛮部的尊敬。”
<br/>　　“请放心，他的遗体会得到妥善的安置，我们也会珍视北蛮部的友谊。”
<br/>　　队伍踏上回程的路，临别前，呼延力向李天乐抛下一句话。
<br/>　　“我们将会在草原等待明玉公主，她将是我们两族友谊的象征。”
<br/>　　李乐天自然明白这句话的含义：没有明玉，也就没有友谊。
<br/>　　在处理掉余下事后，李乐天来到皇城朝华殿。
<br/>　　“出去，出去，我不吃。”
<br/>　　还未走进大殿，便听到李婉儿发怒的声音，紧跟着，一盘碗碟被丢了出来，珍馐美味差点掉在李乐天靴上。
<br/>　　李乐天走了进去，挥挥手让跪在地上的宫女太监先下去。再打眼瞧大殿内的景象，嚯，就连李乐天这种人物，也不得不震惊李婉儿的手笔。
<br/>　　整座大殿看不到一样完整之物，水洗碧的花瓶砸得稀巴烂，锦缎撕成一条条的，碧玉簪落在地上四分五裂。这样的挥霍，等于长安城一个富户几代的积累。
<br/>　　李婉儿坐在榻上鼓着腮生气。
<br/>　　李乐天想找个地方坐下，发现没有可做的，于是弯腰凑到李婉儿身边。
<br/>　　“婉儿，听说你绝食了，多久没吃东西了？”
<br/>　　“出去，这里不欢迎你。”
<br/>　　“御膳房的桂花糕。”李乐天竖起拇指，赞赏自己出色的嗅觉：“婉儿，我早就和你说过，不要吃甜的，容易胖。”
<br/>　　“哼。”李婉儿红了脸，将头扭头一边：“任你把死的说成活的，我也绝不会嫁到草原。”
<br/>　　“瞧瞧这嗓子，中气十足。”李乐天凑着身子靠近：“好妹妹，我是你大哥，我会害你嘛。到了那边，你可是万人之上，母仪天下，大哥以后见到你，都得行礼呢。”
<br/>　　“见我，你以后还见得到我么。”
<br/>　　“天高水远，不管你走到那里，都是我妹妹。”
<br/>　　“大哥，你说六哥在，会把我嫁到草原么？”
<br/>　　李乐天脸上的笑容忽然僵固：“老六……”
<br/>　　“六哥不会。”李婉儿站起来摇摇头：“他会拿起剑与戎族拼命，他死在和戎族厮杀的战场，也绝不会让我嫁到戎族。”
<br/>　　“老六……”
<br/>　　“大哥，现在你明白，为什么你兢兢业业，文武双全，可在父王的儿女中，六哥就算再不争气，再多人骂他，大家骨子里还是最喜欢他。大哥，你和六哥不一样。”
<br/>　　“老六他不在，就算在，他也做不了主。”李天乐心里却想：其实大哥也做不了主。
<br/>　　脚步声响起，明帝出现在李乐天身后，李乐天立刻行礼。李婉儿鼻头一酸，道：“父王，大哥他欺负我。”
<br/>　　明帝挥挥手，让李乐天退下，看着哭花脸的李婉儿，脸上却没有笑意。
<br/>　　“不是他欺负你，是我让他来的。”
<br/>　　“父王，您真忍心把我嫁到那么远的地方，女儿舍不得你。”
<br/>　　明帝叹口气，挥挥手好像要抓住什么，可随之哀容收起，整张脸像岩石般坚硬。
<br/>　　“这件事已写在和谈的条约中，半个月后，你将由鱼龙卫送，出夕照关，到戎族与草原王完婚。”
<br/>　　李婉儿一怔，忽地咬紧贝齿：“你这是要女儿死！”
<br/>　　看着她绝望的眼神，明帝轻叹口气，手伸过来想扶她脸颊，李婉儿却扭过头，明帝什么都没摸到。
<br/>　　“婉儿，生在帝王家，很多事都不得自主。你不仅是我的女儿，也是帝国的公主。”
<br/>　　“如今的帝国已没有和戎族正面作战的能力，千疮百孔，需要及时修补。牺牲你一人，却能为帝国争取五年安稳，万千百姓不必流离失所。有这五年休养生息，帝国就可以恢复元气，不必再受戎族的欺辱。”
<br/>　　“现在，当是你身为帝国公主牺牲的时候。”
<br/>　　李婉儿眼中的泪水不知何时停住，她看着父亲，发现他原来已满头华发。
<br/>　　“我可以嫁，当父王必须答应我一件事。”李婉儿道：“放了程大雷。”
<br/>　　明帝一顿，眼睛微微眯起：“假若我不答应呢？”
<br/>　　李婉儿道：“女儿以死相求。”
<br/>　　“这个人对你就真的这么重要？”明帝道：“程大雷必死，他不死，难安天下人心。”
<br/>　　“这就是父王的事了，与我无关。”
<br/>　　明帝看着自己女儿，像第一次见到她一般。
<br/>　　这不是撒娇，卖萌，想要父王赏赐一样东西，这是谈判。当撒娇，卖萌，耍赖都不能获得时，你必须学会谈判。用一样东西交换另一样东西。虽然两样东西你都不想失去，可有时候你必须牺牲其一。
<br/>　　半晌，明帝脸上露出一丝笑容，目光竟有些微赞许。
<br/>　　“婉儿，你长大了。”
<br/>　　“生在帝王家，不得不长大。”
<br/>　　
<br/>(本章完)
