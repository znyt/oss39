　　“吕罡，回来吧！”苏莫遥望高空中的吕罡，朗声大喝。
<br/>　　高空中，吕罡刚刚猎杀了几只魔物，由于魔物实力较强，所以他身上带了不少伤势。
<br/>　　听闻苏莫的喝声，吕罡身形缓缓的飞了下来，落在了苏莫的身边。
<br/>　　“你得到了多少骨刺？”苏莫沉声问道。
<br/>　　“有一万三千多根吧！”吕罡道，他也没有细算，大概就是这个数量。
<br/>　　言罢，吕罡一挥手，一大堆寒光闪烁，长短不一的骨刺出现在地面，堆积如山。
<br/>　　苏莫见此，暗暗点了点头，他虽然一直在修炼，他也分出了一缕心神注意着吕罡，对方获得的骨刺差不多就是这些。
<br/>　　随即他一挥袖袍，收取了其中九成的骨刺。
<br/>　　“好了，我们到此结束吧！”苏莫对吕罡说道。
<br/>　　“好，那我们回道宗吧！”吕罡闻言，暗暗松了口气，他早已等急了，若是再拖下去，怕是天虚宗的人就要来了。
<br/>　　旋即，吕罡也是急忙收取了剩下的一成骨刺。
<br/>　　“嗯，走吧！”
<br/>　　苏莫点了点头，随即身形一展，破空而去。
<br/>　　吕罡见此，便也冲天而起，紧跟着苏莫离去。
<br/>　　众人见苏莫离去，不少让你也是松了口气，终于走了，他们也能猎杀魔物了。
<br/>　　接下来，魔渊的控制权，再次开始了新的争夺。
<br/>　　……
<br/>　　苏莫和吕罡一起，不到半个月的时间，便回到了道宗。
<br/>　　道宗一片平静，天虚宗的人并没有找上门来，让苏莫也是微微送了口气。
<br/>　　随即，苏莫和吕罡前去内宗大殿，将所有骨刺全部上缴。
<br/>　　苏莫的一万余根魔物骨刺，按照品质的等级，共计价值三千二百块中品灵晶。
<br/>　　这个价值，已经算是颇丰了，足够苏莫冲击一重修为境界了，再加上他在道宗邀人挑战得到的赌注，暂时是不用愁了。
<br/>　　随后，苏莫告别了吕罡，离开了道宗，向太阴族而去。
<br/>　　这一次去太阴族，苏莫的目的不是去见青璇，而是要去九重浮屠塔。
<br/>　　九重浮屠塔内的空间，对于他来说，是修炼的绝佳之地，他要在哪里参悟五行属性的进一步融合。
<br/>　　当然，在太阴族中，若是能见到青璇，那就再好不过了。
<br/>　　太阴族和道宗同在太阴星，三日之后，苏莫便来到了天阴山脉，飞到了巨大的太阴城城门之前。
<br/>　　“来者何人？”
<br/>　　苏莫刚刚降落在太阴城的城门之前，护城的守卫便拦在了他的面前。
<br/>　　“我是道宗真传弟子！”苏莫见此，立刻取出了道宗真传弟子的身份玉牌，交由守卫查看。
<br/>　　“嗯，你可以在太阴城待一个月，若有延期，每日需补交一百中品灵晶！”守卫稍一检查，便将身份牌还给了苏莫。
<br/>　　其实，这些守卫也都知道苏莫，毕竟上次会武，苏莫的表现实在是抢眼。
<br/>　　只不过，出于规矩，还是要检查盘问一番。
<br/>　　“明白！”苏莫点了点头，心中却是暗暗惊讶，延期一日就要补交一百中品灵晶，这代价未免也太大了吧！
<br/>　　随后，守卫放行，苏莫大步的走进了太阴城之中。
<br/>　　走进太阴城，苏莫直奔九重浮屠塔，缴纳了相应的灵晶，便再次来到了第九层的空间之中。
<br/>　　浮屠塔第九层，只有寥寥几名太阴族之人，苏莫走到一旁，立刻盘膝坐了下来。
<br/>　　在这第九层空间之中，他的头脑清明无比，精神也特别活跃。
<br/>　　少倾之后，苏莫抛却了心中杂念，缓缓的闭上了眼眸。
<br/>　　“生中有制，制中有生……！”
<br/>　　苏莫的脑海中，回想着普阳神长老的话语，关于五行之力的奥妙。
<br/>　　其实，这也不算是五行的奥妙，只是关于五行相生相克的理论，但是苏莫有一种感觉，若是他能真正的明悟，五行之力彻底融合，将不是梦想。
<br/>　　苏莫仿佛是陷入了石化之中，完全沉浸在了参悟之中，脑海之中心思瞬息万变，偶尔还在丹田之内，调动五行玄力，缓缓的尝试。
<br/>　　不过，他每一次只是稍稍试探，免得属性相冲，在丹田内爆炸。
<br/>　　苏莫非常想将五行属性彻底融合，因为一旦彻底融合，他的战斗力会增加到极其夸张的地步。
<br/>　　具体能达到什么地步，苏莫也不清楚，但想必越个七、八重修为杀敌，甚至九重修为杀敌，完全不在话下。
<br/>　　所以，他没有急着冲击修为境界，而是率先来到了九重浮屠塔，就是想要早日将属性玄力融合成功。
<br/>　　哪怕不能彻底融合成功，只是四属性玄力融合，对于他的战力加成，也是非常恐怖的。
<br/>　　时间悄然流逝，一日又一日，苏莫静心的参悟。
<br/>　　就在苏莫静心参悟五行属性之时，一群强者来到了问道山，走进了道宗的宗门大殿。
<br/>　　这群强者约莫有十余人，为首的一人，是一名头发半白的矮壮老者。
<br/>　　余下的人全部都是武尊境修为，而且大多都是高阶武尊，修为非常强大。
<br/>　　而修为最低的一人，是一名面容俊朗的青年，此人正是扬天成。
<br/>　　大殿之中，一众强者落座，而在主座之上，端坐着一名面容威严的灰袍中年人。
<br/>　　“无事不登三宝殿，琥圣驾龄我道宗，不知所谓何事？”
<br/>　　灰袍中年人眼眸开阖之间，精光如电，打量了一番扬天成等人，微笑着向矮壮老者问道。
<br/>　　眼前的一群天虚宗之人，如此阵仗，还由大名鼎鼎的‘琥圣’率领，让得他非常疑惑对方的来意。
<br/>　　“全宗主，此次琥某来此，是特来问罪！”矮壮老者冷冷的说道。
<br/>　　“问罪？什么意思？”灰袍中年人眼眸微眯，略带诧异的问道。
<br/>　　这名灰袍中年人，不是别人，正是道宗的宗主，全铭。
<br/>　　“全宗主，你们道宗有一位名为苏莫的弟子，你应该知晓吧？”矮壮老者反问道。
<br/>　　“有所耳闻！”全宗主点了点头，前些日子他也听闻了，宗门内出现了一个妖孽天才，在会武中横扫六大势力的天才，所向披靡，大大的为道宗长了脸面。
<br/>　　“哼！这个苏莫十恶不赦，残杀我天虚宗两名真传弟子，你是否也知晓？”矮壮老者琥圣声音冷漠，厉声喝问道。
<br/>　　“还有这等事？”全宗主闻言顿时皱眉。
