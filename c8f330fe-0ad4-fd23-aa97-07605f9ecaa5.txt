　　众人闻言，顺着那名弟子的目光望去，顿时心中一震。
<br/>　　只见在极远之处，一片辽阔的平原之上，孤独的耸立着一支旗幡。
<br/>　　不错，就是旗幡。
<br/>　　这旗幡非常的夸张，旗杆高约十几万丈，直径足有三千丈，堪称巨大无比，仿佛是一根擎天神柱，通天彻地。
<br/>　　而旗杆顶上的旗幡，同样是巨大无比，仿佛是一片天幕，则天蔽日，随风飘荡。
<br/>　　无论是旗杆还是旗幡，均是闪烁着九彩之光，美轮美奂，霎是耀眼。
<br/>　　不少人都惊呆了，苏莫亦是一脸惊讶，不知这半废的星辰之上，为何会有一支如此巨大的旗幡？
<br/>　　这旗幡明显的不简单，即便隔着数万里的距离，苏莫也能感觉到，那旗幡上传出的让人心悸的气息。
<br/>　　“吴晋导师，这是什么旗幡？”有人向吴晋导师问道。
<br/>　　吴晋导师瞥了一眼远处的旗幡，眸中闪过一丝忌惮之色，随即叹道：“这是一个禁忌，一个未至的存在，存在了无数的岁月，没有人知道是什么？”
<br/>　　众人闻言更惊讶了，没有人知道是什么？这怎么可能？这么大一支旗幡，不可能是自然存在的吧！
<br/>　　“这旗幡如此不凡，难道没有人调查过吗？”有人问道。
<br/>　　吴晋导师叹了口气，道：“此种旗幡并不是此地仅有，古灵星河少说也有上百支，恐怖异常，任何生灵只要靠近其千里之内，皆会被瞬间灭杀！”
<br/>　　“什么？这么厉害？”众人闻言更为惊讶了。
<br/>　　“不错，就连武圣境强者也不例外，传闻中无数年前，曾有虚神境大能死于这种旗幡之下！”吴晋导师说道。
<br/>　　众人闻言心中剧震，虚神境大能，那可是宇宙中无敌的存在，怎么可能死在这种旗幡之下？
<br/>　　“吴晋导师，这旗幡真的这么恐怖？还有什么其它的诡异之处吗？”谢彬满脸震惊而又好奇的问道。
<br/>　　“的确恐怖！”
<br/>　　吴晋导师身为高阶武尊，亦是对此感到不可思议，随即，他又道：“只要是拥有这种旗幡的星辰，都有一个共同点，那就是全部是半废星辰！”
<br/>　　“半废星辰？”众人更加的疑惑了，这种旗幡为何只出现在半废星辰上？
<br/>　　在众人说话之间，虚行舰已经远去，那通天彻地的旗幡已经消失在了众人的视线中。
<br/>　　随即，众人也没有再问，虽然这种旗幡很是奇怪，但与他们也没有什么关系！
<br/>　　苏莫对此也没有多管，他现在考虑真圣界的事情，这才是他现在的当务之急。
<br/>　　虚行舰的速度虽然降了下来，但依旧是奇快无比。
<br/>　　一个时辰转瞬即过，众人来到了一片丘陵地带，一片荒凉。
<br/>　　而此时，众人的视线之中，出现了一座万丈巨峰，孤独的耸立在大地之上。
<br/>　　在巨峰的四周，拥有密密麻麻，数之不尽的人影。
<br/>　　天上地下，均是黑压压一大片，无边无际，漫山遍野全是人。
<br/>　　这些人不单单是人族，还有古灵星河众多的异族，人数多到爆炸，一眼望不到尽头。
<br/>　　人数众多，自然喧嚣不已，嘈杂的声音简直铺天盖地。
<br/>　　“好多人！”苏莫心惊不已，此处聚集的人，少数也有千万之众吧，各种强大的气息交织在一起，让人心神震颤。
<br/>　　神武学府两艘虚行舰的到来，顿时吸引了所有人的注意，几乎所有的人都转过了目光，望着两艘虚行舰。
<br/>　　这倒不是因为神武学府多么强大，值得众人关注，神武学府虽然很强，但在整个古灵星河，也就是一般的大势力而已，普普通通。
<br/>　　之所以引起了所有的人关注，那是因为真圣榜第一之人，就在神武学府。
<br/>　　“是神武学府的虚行舰！”
<br/>　　“苏莫应该来了吧？”
<br/>　　“肯定来了，作为真圣榜第一，怎么可能不来？”
<br/>　　“这一次终于能见到他本人了！”
<br/>　　“呵呵，我倒要看看所谓的真圣榜第一，有何等风采！”
<br/>　　密密麻麻的人海之中，声音此起彼伏，几乎所有人都是同一个目的，要看看所谓的真圣榜第一人，苏莫。
<br/>　　“大家下去！”
<br/>　　少倾之后，虚行舰来到了近前，停了下来，吴晋导师打开舰上的光幕气罩，朗声说道。
<br/>　　随即，苏莫等众人纷纷身形闪烁，飞下了虚行舰，落在了下方的一座矮丘上。
<br/>　　另一艘虚行舰上的人，也全部飞了下来，所有导师和弟子都聚集在了矮丘上。
<br/>　　随后，两艘虚行舰被吴晋和唐婉两位导师收了起来，消失不见。
<br/>　　一众神武学府的弟子，个个目光扫视四方，打量着前方无尽的人海，但他们几乎个个心中发悚。
<br/>　　因为此刻，几乎有上千万道灼灼的目光，都在望着他们，让他们倍感压力。
<br/>　　此时，这里上千万人都是在打量了神武学府的众人，想要一睹苏莫的风采。
<br/>　　只不过，基本上认识苏莫的人不多，只有在大荣城见过影像的人才知道苏莫的模样。
<br/>　　“哪个是苏莫？”
<br/>　　“没发现谁比较特别啊！”
<br/>　　“是那个穿灰白色长衫的青年，他就是苏莫！”
<br/>　　“他就是苏莫？没有什么出众的地方啊！”
<br/>　　“感觉很普通，完全不像传闻中那么夸张！”
<br/>　　无尽的人海之中，询问声、议论声此起彼伏。
<br/>　　经过认识苏莫的人提醒之下，所有人都知道了谁是苏莫，所有人的目光都落在了苏莫的身上。
<br/>　　但是，事实却是与众人想的完全不符。
<br/>　　在众人的猜想之中，能被排上真圣榜第一，苏莫定然非常出众，估计像姜神夜一般，俊逸非凡，气质独特，人中之龙，璀璨耀眼。
<br/>　　但是这一看之下，这苏莫貌似平平凡凡，一袭灰白色长衫，相貌青秀，气质平庸，就是一个普通的青年。
<br/>　　可以说，苏莫给众人的感觉，完全不符合真圣榜第一，万古妖孽的名头。
<br/>　　一瞬间，不少天才都大感失望，感觉所谓的真圣榜第一，怕是名不副实。
<br/>　　此时，在众人瞩目之下的苏莫，眉头微皱了起来，他也是感到极不自在。
<br/>　　任谁被上千万人注视打量，都会感觉到巨大的压力，况且这些人中还有很多势力的高层人物，目光如同刀锋般凌厉无比。
<br/>　　苏莫的身边，不少人都是微微移动脚步，立刻和他拉开了距离。
<br/>　　他们也知道所有人关注的都是苏莫，和苏莫站在一起，这种被所有人注视的感觉，实在是太不自在了。
<br/>　　转瞬之间，苏莫立身之地，方圆三丈之内便空无一人，他被彻底的孤立了出来。
<br/>　　“尼玛！”
<br/>　　苏莫心中无语，少倾，他深深的吸了口气，压下心中的不自在，迎着无数的目光，抬目打量着前方所有的人。
