        &lt;h3&gt;第三卷第三百三十六章  不小心听到了&lt;/h3&gt;
<br/>
<br/>        莲鱼的项目要开标了。
<br/>
<br/>        夏颜安排了孙毅组织这次招标活动，和前几个项目一样，他们都是计划好了的，这一次因为冯睿的离去，陈子霖和陈子昂无暇顾及公司的事情，易倾秋主持大局。
<br/>
<br/>        易倾秋和李秀、谭九儿商量如何能将这个项目拿下来：
<br/>
<br/>        “我私下探听了一下消息，这一次参加报名招标的人没有任何消息透露出来，谁也不知道谁参加了，但是我估计宋长生、顾恩赐和许弋澄他们会参加，有可能和前几次一样，如何能在他们的围堵中突围而出，我们需要仔细商讨一下，如今俩个陈总和江总都不在，我很想拿下这个标。”
<br/>
<br/>        “前几次都是他们轮着中标，这一次他们肯定也会如法炮制，我们讨论一下他们前几次的中标价，然后综合分析。将标书做好，参数写好，价格衡量一下，有百分之五十的把握。”李秀敲着桌子。
<br/>
<br/>        “我前几天还见了乔小东，乔小东问我前几次怎么回事，一次脱标次次都脱标，我说如果光明正大的竞争我们是会中标的。”谭九儿微微一笑。
<br/>
<br/>        “乔小东怎么说？”易倾秋问谭九儿。
<br/>
<br/>        “乔小东说后期市场会越来越规范化，让我们坚持一段时间，以后会好的。”
<br/>
<br/>        “乔小东是个正直的人，有他在，西城的市场也不会乱到哪里去，有些人还是多少得有些顾虑。”李秀看着窗外的阳光微微有些出神。
<br/>
<br/>        他们公司从夏颜上任之后一直被各种挤兑，在商场上到处都是刀光剑影，这一段时间能稳住以前的老客户就已经很不错了，想开展新业务真的很困难，也不知道夏颜和他们陈总有什么过节，以前认为是李凌菲，但是李凌菲已经到了北京，按说夏颜也该放下了，可是夏颜比原来还难对付，联合了好几家企业进行市场分割，在有些项目开展之前就内定了合作方。
<br/>
<br/>        原来他们引以为傲的业绩不复存在了，当然随着社会的发展，各个行业已经开始了多元化，需求点和出发点也完全不一样了，百家争鸣，市场份额在减少，唯一令他们骄傲的是，他们的研发团队一直走在前沿，在每一个和他们合作的单位中，后期都会有可以对接和二次开发的机会。
<br/>
<br/>        “这次莲鱼的项目必须拿下来！”易倾秋的语气不容置疑。
<br/>
<br/>        “明天早上10点开标!”谭九儿看着易倾秋。
<br/>
<br/>        “九儿我们俩一起去！”易倾秋决定亲自出马。
<br/>
<br/>        “好！”
<br/>
<br/>        三个人确定了最终报价，李秀亲自和部门员工一起去封装标书。
<br/>
<br/>        易倾秋想起陈天天，这两天忙得晕头转向一直都没有给陈天天打电话，陈天天的生母不在了，无论是作为公司的员工还是陈天天的朋友她应该去探望一下。
<br/>
<br/>        电话响了半天，才被陈天天接起来，陈天天实在是很困，躺在床上睡着了。
<br/>
<br/>        “倾秋，你在哪里？”
<br/>
<br/>        “我准备下班了去看看你，你把地址发给我。”易倾秋还记得前一段时间陈天天开心的说她妈妈让她带着自己去她们家里吃饭。
<br/>
<br/>        “你不忙了吗？”陈天天的嗓子沙哑。
<br/>
<br/>        “我下班就没有事情了，我去陪陪你。”
<br/>
<br/>        “好！”
<br/>
<br/>        易倾秋按照陈天天给的地址到达冯睿的家里时，才发现冯睿的家离她们公司并不远。
<br/>
<br/>        “倾秋。”陈天天见了易倾秋就抱着易倾秋哭。
<br/>
<br/>        “天天，你节哀，已经这样了，你要是在不好好的照顾你自己，你妈妈也不会安心的。”易倾秋安慰着陈天天。
<br/>
<br/>        “倾秋，你不知道，我原来对她其实一直有意见的，她生下我却不带我，后来和她在一起的那一段时间里，我感觉她好像在利用我，我爷爷生病了，她一直装着重感冒不去看我爷爷，我心里就对她反感了，我这几天一直陪着我爷爷，可是我没有想到再见她的时候，她已经躺在医院里，再也醒不过来了。倾秋，如果她能醒过来，我愿意被她利用一辈子。”
<br/>
<br/>        “说什么傻话呢，哪有妈妈想利用自己孩子的，她就是想和你多呆几天，你别想那么多了，你现在这样痛苦悲伤都已经不起作用了，你妈妈肯定希望你快快乐乐的生活着，等找到肇事者将他绳之以法你妈也就能安息了。”
<br/>
<br/>        “对了，我小姑说和我一起去警局，到现在都没有叫我，你陪我一起去看看。”陈天天拉着易倾秋的手去找陈子昂。
<br/>
<br/>        陈子昂正在和陈子建还有冯乔在说话，他们在分析夏颜和冯睿的关系。
<br/>
<br/>        “冯睿之前就认识夏颜，她们合伙调离了子昂的成绩，子昂才没有考上大学，你说我生气不生气？再怎么说，子昂也是她的妹妹吧？你说是我亲还是夏颜亲？”陈子建生气的对冯乔说。
<br/>
<br/>        “怎么会？你是给你找借口！”冯乔不相信冯睿会做这样的事情。
<br/>
<br/>        陈天天和易倾秋在他们身后嘴巴张得大大的。
<br/>
<br/>        “怎么不会，我父亲生病的时候我回来了，那个时候我还想着和冯睿能和好就和好，毕竟这么大的年龄了，我们在一起过了这么久，她除了小气贪婪一些也没有大过，对我也算是好的，你知道吗？她的好干女儿夏颜亲口告诉我的，亲口告诉我的！”陈子建浑身发抖。
<br/>
<br/>        “夏颜为什么要告诉你这么多？难道她不怕你和二嫂吵架吗？”陈子昂的声音很冷。
<br/>
<br/>        “你二嫂留我在家里吃饭，说吃完饭和我一起去医院看爸，可是不一会夏颜就来了，你二嫂在厨房做饭，她就和我说起了之前的事情，我当时很冲动，打了你二嫂我就走了，后来就是在医院见过她一面，之后就再也没有见过面。”陈子建此时心里那个悔啊，他怎么就听信了夏颜的一面之词。
<br/>
<br/>        “他们调换了我的成绩是真的，夏颜也亲口说过，被七哥听见了，但是怎么调换的和谁一起调换的我不知道，这件事情没有必要再提起来了，二哥，二嫂的离去不是你的过错，你也不要纠结。”陈子昂停顿了一下看着冯乔接着说：
<br/>
<br/>        “乔哥，二嫂和二哥之间的是是非非你大概也能知道一些，你是聪明人，我希望你擦亮眼睛，不要被蒙蔽了心智，如果说我们陈家人品不行，那我们大可不管这些事情，无论怎么样，我们还是亲戚还是一家人。”
<br/>
<br/>        “子昂妹子，你放心，我虽然是个孬种，但是大是大非面前我还是知道的。”冯乔紧了紧拳头。
<br/>
<br/>        无论冯睿怎么样，那都是和他冯乔打断筋骨连着血脉的亲人！
<br/>
<br/>        陈天天拉着易倾秋的手悄悄的退出去了。
