有，不过是需要相当硬的关系才行的。傅华心猜测宁慧能够拿到这笔贷款，一定是李凯在背后帮她运作的。<br/>　　<br/>　　    “诶，傅华，”宁慧说道，“我一会要跟我姐姐和子乔联系，告诉他们我回来了，晚你跟我们一起吃饭吧。”<br/>　　<br/>　　    傅华愣了一下，要他跟冷子乔一起吃饭倒是没什么问题了，只是宁慧还要邀请冷子乔的妈妈宁馨，这可能有点别扭了。到现在为止，宁馨对他和冷子乔的交往还是不那么接受的。<br/>　　<br/>　　    傅华笑了一下，说：“我是没什么问题了，不过您姐姐可不一定会高兴跟我一起吃饭的。”<br/>　　<br/>　　    宁慧笑了起来，说：“傅华，这个你不用担心了，我姐那个人实际面冷心软的，眼下她对你和子乔的交往虽然还是有些芥蒂的，但是只要你跟她多接触接触，让她多了解一下你，很快她会接纳你了。好了，我打电话了。”<br/>　　<br/>　　    宁慧打了电话给宁馨，说自己回来了，现在在傅华这里，晚准备叫傅华一起去吃饭。宁馨倒也没有说不愿意，不过傅华在一旁听电话里传出来宁馨说话的语气还是很冷淡的。<br/>　　<br/>　　    宁慧结束了跟宁慧的通话，然后笑了笑说：“我姐姐同意了，晚你可要表现得好一点啊，多逗她开开心。”<br/>　　<br/>　　    傅华笑了笑没说什么，他心里很清楚他是表现的再好，恐怕一时之间也是难以改变宁馨的态度的。<br/>　　<br/>　　    宁慧又跟傅华约定了晚吃饭的地点和时间，然后说她要先回去休息一下，跟傅华告辞离开了驻京办。<br/>　　<br/>　　    傅华有继续忙他的事情，过了一会儿他的手机响了起来，显示的是一个陌生的号码，他接通了，说：“你好，我是傅华，请问您是那位。”<br/>　　<br/>　　    一个嗓音有些细声细气男人在电话的那头说：“诶，你不要管我是哪位，我打电话给你主要是想要提醒你一下，不要随便去搀和别人的事情，管得太多会不得好死的。”<br/>　　<br/>　　    傅华有些愣住了，他搞不清楚这个男人为什么会莫名其妙的打这个电话给他，他有些纳闷的问道：“你究竟是什么人啊，我又管了你什么事情啊？”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
