　　第11章
<br/>　　“我这孩子最是崇拜名将，苦读兵法，也算用功，但这本事嘛……”韩虎踞哈哈笑道：“不过，蛤蟆岭被我们打了一次，哪里来得三百山贼，这黄三元怕死胡说而已。既然玄之有领兵的梦想，就让他过过瘾。”
<br/>　　“城主明察秋毫，半川也就放心了。”薛半川。
<br/>　　“薛军师不必多虑，出不了大事。”韩虎踞一拍桌子：“累了，回去睡觉，花儿草儿，扶我起来。”
<br/>　　身后一对玉瓷般的双生子将韩虎踞搀起，薛半川道：“公子准备出兵，有很多事情要安排，城主不过问一下嘛？”
<br/>　　“不看啦，我现在什么都不想，就想着多和年轻女孩亲近亲近。薛军师，我就不明白，你怎么不碰女人呢？要不我让花儿草儿陪你一夜，让你尝尝她们的滋味？”
<br/>　　薛半川脸上一红：“不敢，不敢。”
<br/>　　“哈哈，你敢我也舍不得呢。”韩虎踞拍拍他的肩膀：“谁不希望被年轻女孩丝绸般的肌肤贴着自己皱巴巴的身体，嗅着她们身上年轻的味道，可这福分，也不是谁都能有，若薛军师尝过这滋味，就发现世上没什么其他事情值得在意啦。”
<br/>　　薛半川将腰晚得很低，韩虎踞哼着小曲向后堂去，花儿草儿在左右搀着他，清纯的模样像刚淋过雨的油菜花。
<br/>　　……
<br/>　　“……你的意思，黑石城没有常备军？”
<br/>　　蛤蟆寨，土屋内，程大雷奇怪的皱起眉头。为了了解黑石城的情况，他自然要向苏樱询问。可不问不知道，一问才明白有些事情和自己想得还真不一样。
<br/>　　“常备军？黑石城有民两万多户，怎么养得起一只五千人的军队？”
<br/>　　“那打仗呢，士兵从哪里来，如果有人来攻城，没有军队怎么办？”程大雷更加困惑。
<br/>　　“用兵时可以从军户之中抽调，若是大规模攻城，城内的壮丁都要上阵。”
<br/>　　其实这个世界普遍采用的还是军户制，子孙后代世袭一个军职，用兵时临时抽调。至于这人会不会打仗，根本没有人会关心。比如黑石城，人口大概有十万左右，根本养不起一只五千人的常备军，城内只有几百卫兵负责城主府的安全和城内治安。
<br/>　　“还真是落后的制度呐。”程大雷发出一声长叹。
<br/>　　“落后？”苏樱微颦眉心：“言外之意，你知道更先进的制度？”
<br/>　　“没有啦，没有啦。”
<br/>　　程大雷挥挥手，至少程大雷也是一个文科生，知道生产关系是由生产力所决定，以如今每亩二三百斤的粮食产量，根本不可能供养一支常备军。二三百斤还要是丰年，天有不测风云，老天一直不下雨，老天一直下雨都会让粮田颗粒无收。
<br/>　　所以，他纵然知道很多制度更先进，也未必适合这个世界。
<br/>　　“其实黑石城的薛半川曾经打算训练一支五百人的常备军，然而消耗太大，这件事也就不了了之。”苏樱。
<br/>　　“薛半川？”程大雷问。
<br/>　　“他是韩虎踞的军师，韩虎踞沉迷享乐，城中大小事务都是由薛半川管理，他……才是一位真正了不起的人。”
<br/>　　程大雷点点头，赶紧把这句话默记下来。看着他认真颦起眉头的样子，苏樱微微出神，这样真的不像一个山贼。
<br/>　　“你们逃吧，黑石城来兵，至少一二百众，你们只有四个人。”
<br/>　　“不，是五个，林少羽已经醒了。”程大雷站起身：“我的计划是挡五百人。”
<br/>　　自己才对蛤蟆寨拥有归属感，决不允许任何人破坏。他这几天一直没有好好休息，都在思考着如何应对这次危机。这中间，徐神机给程大雷提供了不少帮助。
<br/>　　每绞尽脑汁想到一个防守计划，就向徐神机询问：军师觉得此计如何？
<br/>　　如果徐神机说此计有七成把握，那程大雷就知道只有三分数，如果徐神机说此计定能成功，那么程大雷就要硬着头皮推倒从来，后来，他对徐神机都有了心理阴影，费尽心血想到的主意都不敢再询问徐神机。
<br/>　　他离开土屋，看到林少羽站在院子里，脸上的表情阴晴变幻，似乎是想说什么又没琢磨好用词。
<br/>　　程大雷再次查看林少羽的信息，忽然发现经过这件事，林少羽竟然升级了。
<br/>　　姓名：林少羽（初出茅庐的优秀游侠）
<br/>　　年龄：20
<br/>　　技能：断浪枪
<br/>　　隐藏属性：立志成为大侠的少年
<br/>　　这还真是塞翁失马焉知非福，程大雷笑笑，迈步向对方走过去。看着越来越近的程大雷，林少羽脸上更加尴尬，嘴里结结巴巴：“大……大……”
<br/>　　程大雷冲他伸出自己的手：“为了我们的理想！”
<br/>　　林少羽瞳孔伸缩，激动得一抱拳：“为了我们的理想！”
<br/>　　程大雷尴尬得收回手，也只好同样抱拳。然后他对周围人道：“以后林少侠就在蛤蟆寨入伙，大家以兄弟相称，徐军师，把地图拿来，大家再商议商议。”
<br/>　　徐神机抱着那份得来地图，在院内的木桌上摊开，笑道：“大当家又有什么计划，想让老朽替你把把关？”
<br/>　　程大雷无奈的瞥他一眼，目光落在那份青牛山地图上。要说这次收获的东西，除火药外就是这份青牛山地形图。有了它不至于再对青牛山周围两眼一抹黑，不过对于这个世界的地图精度，程大雷根本没什么自信。有些重要地点，他还是要亲眼看过才能确定。
<br/>　　每次摊开地图，程大雷都难免震惊。青牛山方圆五十里的山林，竟大大小小盘踞着五六十股山贼，大的有三百余人，小的……好吧，就蛤蟆寨这五个人的山寨，也没有谁比他们更小。
<br/>　　还真是一个土匪横生的时代呐。
<br/>　　程大雷感慨一声，冲徐神机道：“军师，目前有个任务搁在你身上，这件事关系重大，你一己之身牵扯着咱们整个山寨的安危。”
<br/>　　“这么重要……”徐神机眨眨眼睛：“要不你换个人？”
<br/>　　“此时非军师不可，你去一趟飞虎寨。”程大雷手指落在地图上某处：“找高飞虎。”
<br/>　　“飞虎寨！”徐神机两眼一黑，差点昏迷过去。
<br/>　　
<br/>(本章完)
