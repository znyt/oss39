        “那你觉得，我又是哪样的女人？”童筱颖心里在苦笑，脸上表现出来的却是冷笑，她反过来质问他。
<br/>
<br/>        筱颍是哪样的女人？
<br/>
<br/>        在这一刻，他竟然迷茫了。
<br/>
<br/>        傅昼景眉头紧锁，黯然神伤地垂下了眼帘。
<br/>
<br/>        他的那个天真无邪、温柔娴淑的筱颍去哪里呢？
<br/>
<br/>        为什么筱颍会变成现在这个样子？
<br/>
<br/>        是他害的吗？
<br/>
<br/>        童筱颖冷傲地扬起下巴，手下爱怜地抚摸着怀里的小爱，却故意对傅昼景嗤之以鼻道：“你能变，为什么我就不能变？你对我的了解又有多少？我爱你有多深，当你背叛我的时候，我恨你就有多深！在我眼里，你和大哥真的好好骗！我本想在我和大哥举行婚礼的那天给你这个特大的惊喜，而我之所以不希望你死在我前头，是因为我还想听你天天叫我‘大嫂’，还想天天看着你们兄弟两自相残杀、明争暗斗！就为了我这么一个一文不值的女人！兄弟见面分外眼红，那该是多有趣啊！你说，是不是呢？这样的报复，对你够不够深？足不足以让你痛彻心扉？”
<br/>
<br/>        忽然间，她还得谢谢落以柔。
<br/>
<br/>        若不是落以柔那日辱骂她的这些话，让她不经意间地想起，她还真不知道要怎么去“演”一个坏女人。
<br/>
<br/>        这样一来，昼景只会将所有的恨，都转移到她童筱颖一个人身上。
<br/>
<br/>        事已至此，她打算破罐子破摔。
<br/>
<br/>        “呵！”傅昼景忽然间冷冷一笑，紧锁的眉心舒展，眉梢轻挑，满是不屑，“你就不怕，你说的这些话，被我大哥听到吗？”
<br/>
<br/>        “……”童筱颖身心一愣，一种不祥的预感漫上心头。
<br/>
<br/>        当她本能地再次侧过身去时，傅夜沉那高大的身影已经湮没了她的头顶。
<br/>
<br/>        从他身上散发出来的一股强大如雪崩般令人寒颤的气场，瞬间压得她喘不过气来。
<br/>
<br/>        她刚刚以为他不会走过来的！
<br/>
<br/>        但他什么时候走过来的？
<br/>
<br/>        他听到了多少？
<br/>
<br/>        刚刚她所说的那些话，他会不会信以为真？
<br/>
<br/>        此时，傅夜沉整个一张俊脸阴沉得令人难以靠近。
<br/>
<br/>        童筱颖柳眉微蹙，直觉告诉她，傅夜沉似乎是信了她刚刚后面所说的那些胡邹乱编的话。
<br/>
<br/>        现在，她该怎么做？
<br/>
<br/>        继续自导自演，还是全盘否认？
<br/>
<br/>        两个男人同时等着她的下文。
<br/>
<br/>        童筱颖垂下眸子，不敢直视傅夜沉的眼睛。
<br/>
<br/>        “怎么不说话呢？刚刚你不还理直气壮，滔滔不绝的吗？”傅昼景心里莫名其妙地暗自得意起来。
<br/>
<br/>        童筱颖闭了闭眼睛，释然一笑，瞥了傅昼景一眼，又鼓足勇气，仰面凝望着傅夜沉。
<br/>
<br/>        “大哥，现在你……”
<br/>
<br/>        “给我闭嘴！”
<br/>
<br/>        童筱颖的话还未说完，傅夜沉剑眉紧锁，一声呵斥，堵住了她的话。
<br/>
<br/>        这就是她突然转性，对他忽冷忽热的真正原因是吗？
<br/>
<br/>        他确实不指望她会爱上自己，但是她的那句对阿景的爱有多深，恨就有多深的话，深深地刺痛了他的心。
<br/>
<br/>        这个女人，到底是有多狠的心，才会如此轻易地在他伤口上撒盐？
<br/>
<br/>        她一边说着想跟昼景撇清关系，愿意当好他太太，哄他开心的话，一边却又可以在昼景面前，放出毫不在乎他任何感受，甚至可以随便利用他，将他凌迟的狠话。
<br/>
<br/>        “童筱颖，既然你是在利用我来报复阿景，好！我如你所愿！你这一辈子都是阿景的‘大嫂’！当然，我也不会如你所愿，让你离间我和阿景之间的亲情。”傅夜沉伸手将童筱颖直接拽入自己怀中，铿锵有力地对傅昼景和童筱颖宣示自己的主导地位。
<br/>
<br/>        童筱颖只觉自己的手臂刚脱离傅昼景带来的痛楚，手腕又似乎快要被傅夜沉给捏得像粉碎了般的痛。
<br/>
<br/>        “大哥，你不能娶筱颍！”傅昼景暗哑着嗓音，低沉道。
<br/>
<br/>        傅夜沉却不以为然地冷哼：“阿景，你不是很有野心的吗？那你又何必为了这个一文不值的女人，跟我把关系闹僵？你现在要做的，是学会如何当一名好父亲，然后娶了季思妍，去继承她父亲的季氏集团。毕竟，季思妍的哥哥就是个扶不起的刘阿斗！你娶了季思妍，哄好了岳父，比在这跟她童筱颖纠缠不清要来得更有价值，不是吗？”

<br/>        傅昼景黯然神伤，随之看向了童筱颖。
<br/>
<br/>        童筱颖抿了抿唇，默不吭声，就当是默认了大哥所说的话。
