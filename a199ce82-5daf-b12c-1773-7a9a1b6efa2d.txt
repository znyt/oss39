　　“我不反对！”贺兰殇沉声道，他和公孙长绝关系尚可，所以不反对。
<br/>　　再者，就算第三城之中，还有其它强者也无所谓，先联合起来击败其它城池才是重点。
<br/>　　当然，贺兰殇不傻，他现在也在猜测，苏莫会不会在第三城？
<br/>　　毕竟，苏莫是龙族的人，现如今龙双双出面结盟，他难免猜测背后有没有苏莫？
<br/>　　“我也不反对！”阎魔鬼圣淡淡的道，他的想法和贺兰殇差不多，不管第三城是什么情况，结盟不会有错。
<br/>　　反正，结盟又不意味着就要同仇敌忾，一切都要按照局势而定。
<br/>　　“既然你们三人都没有意见，我们也没有意见！”
<br/>　　“我同意！”
<br/>　　“我也同意和第三城联合！”
<br/>　　随后，不少人纷纷开口，在龙羽三人同意结盟之后，他们也没有什么意见。
<br/>　　“好，既然大家都没有意见，那么，我们商量一下接下来的行动吧！”龙羽朗声说道。
<br/>　　众人闻言点头，贺兰殇略一沉吟，道：“现在，其它城池的情况，我们完全不明，还如何行动呢？”
<br/>　　“一旦我们主动出击，必然会成为众矢之的！”阎魔鬼圣声音幽幽的提醒道。
<br/>　　在他看来，联盟虽然可以，但是，做出头鸟却是并不是好事。
<br/>　　“但是，一直等下去也不是办法，我准备去试一试第四城的虚实！”龙羽道。
<br/>　　他当然知道，静观其变是最好的办法，但是，总要有个开头，不然所有人都精光其变，那千选战要到什么时候才能结束！
<br/>　　“龙羽哥，你准备怎么做？”龙双双问道。
<br/>　　“我准备去试试第四城的底细，如果第四城不强，就一举击败！”
<br/>　　龙羽眸中闪过犀利的精光，他们第二城足够的强大，不仅有三位公子级人物，还有差不多二十位厉害的强者，都是本身修为高达九品的存在。
<br/>　　“贺兰兄，阎兄，你们觉得如何？”龙羽问道。
<br/>　　“可以试试，不过，此事不能操之过急，要留意其它几座城池的动态！”阎魔鬼圣提醒道。
<br/>　　“嗯，先搞清楚第四城的情况！”
<br/>　　贺兰殇点头道，随即，他略一沉吟，看向龙双双道：“你们第三城，可知晓其它几座城池的虚实？”
<br/>　　他们第二城刚刚稳定局势，还没来得及查看，各个城池就已经开启了护城大阵，至于第九城虽然没有开启护城大阵，但是他们也没有去查看。
<br/>　　“古魔太子在第八城，其它几座城池情况不明！”龙双双道。
<br/>　　此言一出，人群中的两位魔族之人，眼眸微亮，原来太子在第八城。
<br/>　　“好，你先回去吧，我们会先试探第四城的虚实，一旦我们大举进攻，你们要立刻支援！”贺兰殇道。
<br/>　　“嗯！”龙双双点了点头，随即看向龙羽和另外两位龙族之人，道：“我就回去了！”
<br/>　　“去吧！”龙羽点了点头。
<br/>　　随即，龙双双身形一闪，身形腾空而起，穿过护城大阵，向第三城返回而去。
<br/>　　……
<br/>　　第三城之中，苏莫身形伫立在半空中，一直在凝视着第二城的方向。
<br/>　　同时，他也在观察着其他几座城池，他们能想到联盟，别人自然也会想到联盟。
<br/>　　他的神念不停的在外扫荡，他能感觉到，外面到处是各种神念，全部是九座城池之中的人在查探。
<br/>　　不过，苏莫没有发现有其它人在外，当然，就算是有人的话，也是在暗中行动，不会轻易的让他发觉。
<br/>　　嗖！
<br/>　　这时，苏莫看到龙双双出来了，从第二城之中飞了出来，向第三城而来。
<br/>　　“打开阵法！”苏莫立刻吩咐人打开阵法，很快，阵法裂开了一道缝隙，龙双双顺利的返回，并未再受到攻击。
<br/>　　“情况怎么样？”苏莫沉声问道。
<br/>　　“已经谈妥了……！”随即，龙双双将此行的情况，以及第二城的情况详细的告知了苏莫等人。
<br/>　　“既然如此，那我们就静等第二城行动吧！”公孙长绝沉声问道。
<br/>　　“嗯！”苏莫点了点头，想要获胜需要耐心，他也不会急于求成。
<br/>　　“这护城大阵，你们查探的如何了？
<br/>　　随后，苏莫环视众人，沉声问道，他是想要知道这护城大阵的威力，所以之前明人查探。
<br/>　　“苏莫，这护城大阵无法加持，威力并不强，若是有百人攻击，可能承受不住十个呼吸！”有人道。
<br/>　　“十个呼吸吗？”苏莫心中沉吟，他并没有奢望护城大阵太强，不然这千选战也没有意义了。
<br/>　　十个呼吸的时间，这大阵的威力，也算是不错了。
<br/>　　“嗯？”就在此刻，苏莫眼眸一凝，因为此刻，第二城之中，有一大群人浩浩荡荡的飞了出来。
<br/>　　这群人以龙羽为首，人数不下六十，离开第二城之后，便浩浩荡荡的向第四城飞去。
<br/>　　“这么快就开始了！”苏莫微眯着眼眸，这龙羽的动作未免也太快了吧。
<br/>　　不过，这样也好，早点行动，其它城池说不定还没有结盟。
<br/>　　六十余名第二城之人中，并没有贺兰殇和阎魔鬼圣，看来那两人坐镇在第二城之中。
<br/>　　“他们的实力够强，应该能试探出第四城的底细！”公孙长绝轻声道。
<br/>　　“所有人准备，若是有其它城池插手，我们也要出手！”苏莫朗声吩咐道。
<br/>　　众人闻言，均是微微的点了点头。
<br/>　　很快，龙羽等六十余人，飞到了第四城的上空，停在了护城大阵之前。
<br/>　　“破阵！”龙羽振臂一呼，率先开始了攻击，一抓狠狠的抓了出去。
<br/>　　霎时之间，幻化出一只巨大的金色龙爪，重重的抓在了第四城的护城大阵之上。
<br/>　　轰隆隆！！
<br/>　　剧震震天，大阵波动，却是丝毫无损。
<br/>　　轰轰轰！！
<br/>　　其他六十余人，也纷纷出手了，普天盖地的攻击，疯狂向第四城的护城大阵之上袭去。
<br/>　　轰隆隆~~~
<br/>　　大阵遭到狂风暴雨般猛烈的攻击，顿时剧烈的震动了起来，好似随时都有可能破碎。
<br/>　　轰隆隆！！
<br/>　　龙羽等人毫不停歇，不断的攻击，疯狂的攻击大阵，他们看得出来，这大阵的威力极为有限，只要十几个呼吸的时间，差不多就能破开了。
<br/>　　不过，他们也都尽量与第四城拉开了一点距离，因为，若是大阵破开之后，第四城的强者太多的话，他们就要后退回第二城之中。
