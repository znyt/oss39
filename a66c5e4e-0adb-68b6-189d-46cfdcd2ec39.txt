　　第727章
<br/>　　行到汝南城前，崔白玉特意撩开车帘向外望了一眼，随之轻轻摇摇头，又退回车厢中。
<br/>　　汝南崔氏，天下一族。
<br/>　　这是帝国大多数人都知道的八个字，是汝南崔氏还是崔氏汝南并不好确认，但在曾经，崔家就是汝南，汝南就是崔家。
<br/>　　崔家发迹于汝南城，往上数几百年崔家便是当地的豪族。如今帝国皇族是李氏，但当年李氏刚出来打江山时，崔家已经是封疆大吏，朝中重臣。
<br/>　　数百年来，这座城内留下太多崔家的痕迹，比如，某位崔家家主喜欢牡丹，城中便遍植牡丹，有商人将天下的奇种送到汝南城，城中以此为贵。每到花开，芬芳全城，竟然成了城中一景。
<br/>　　至于这座城的城主、官吏更是崔家的嫡系，崔家老太爷过寿的时候，可以说满城同庆。
<br/>　　当然，这是以前的事情了。崔家这样的大家族本不该倒的，可以历数朝而不败，天下风雨我自巍然不动。然而，崔家倒了，谁也没想到高高在上的帝国天子做了一次亡命徒。
<br/>　　事成之后，自然会对崔家进行一轮大清洗，崔家全族只有四个字满门抄斩。
<br/>　　此刻程大雷入城，还能嗅到一股若有若无的血腥气，城中家家户户紧闭房门，没多少生气。
<br/>　　程大雷找了个茶楼歇脚，在二楼坐了，几人要了一壶香茶，两屉包子。
<br/>　　“小二，我这一路过来，见街上没几家开张的店铺，这是怎么会事？”程大雷问道。
<br/>　　小二蔫蔫一张脸，道：“现在城里都没多少人，生意都做不下去，不关门又能如何。宁肯少挣钱，也免得惹来杀身之祸。”
<br/>　　“喔，这话怎讲？”
<br/>　　小二左右看看，楼上楼下都没有几个客人，掌柜正在楼下算账。
<br/>　　程大雷赏给他一个银锞子，道：“吞吞吐吐做什么，老爷看你模样喜庆，喜欢和你说话。掌柜的上来又能怎样，我有话与他说。”
<br/>　　小二见到银子立刻来了精神，他笑笑将银子揣进怀里。
<br/>　　“客官是远路来得，这我一眼就看出来了。您别嫌我多嘴，听我一句，最好不要在汝南城留宿，我也是好心。”
<br/>　　“喔，这是为何？”
<br/>　　“汝南城换城主了。”
<br/>　　“倒是听说过，听说上一任城主姓崔，不过我有耳闻，崔家密谋造反，被圣上陛下铲除了。这难道不是好事？”
<br/>　　“好事坏事我们可不知道，不过原先崔城主在的时候，大家的日子倒还都过得去，像小店这样的地方，每天都能赚到一些银子。但这新城主上任以来，日子可就有些过不下去了。”
<br/>　　“莫非这城主是个贪官？”刘发财道。
<br/>　　“贪不贪的有谁知道，现在是捉拿崔氏一案的同犯。许多大户人家都遭了秧，下面做事的有些不地道，谁家不上够银子，便是崔家的同犯。本来不关我们这样小店的事，但那帮如狼似虎的官吏在茶楼里大吃大喝，也是不给钱，小店可一句话不敢多说，生怕对方一句话，我们也成了造反的同犯。”
<br/>　　“这样嘛，还真是，还真是……”程大雷忽然问：“新任城主叫什么名字？”
<br/>　　“似乎是叫魏牧冲，具体小人也不清楚。”
<br/>　　程大雷摆摆手让其退下，小二答应一声：“客官，有什么事您再吩咐？”
<br/>　　随之，程大雷目光落在崔白玉身上：“魏牧冲，什么来历？”
<br/>　　“寒门出身，祖籍扬州，当初在御史台做事，上折参过崔家，所以十几年来一直被雪藏，做了一个小小的刀笔吏。”
<br/>　　崔白玉的声音很平静，但程大雷还是听见她极力压制的颤抖。
<br/>　　“呦，听上去像个好人呐。”徐神机道：“虽千万人吾往矣。”
<br/>　　程大雷没有搭理他，侧头看着窗外。好人坏人，那是一句话说得清的。崔家只手遮天，说不得断了多少人的前程，在大家眼里自然不算好人。可在这店小二眼里，崔家毕竟为他们遮了一片风雨，能让他们在乱世中有太平日子过。
<br/>　　这时候，长街上一队官兵押着一伙犯人招摇而过，犯人衣衫褴褛，只要敢走慢半步，就有官兵在他们身上抽一鞭子。
<br/>　　崔白玉也看着这一幕，忽然皱起眉头，愣了愣，目光转向程大雷。
<br/>　　“大当家，我能否求你一件事。”
<br/>　　“喔，求这个字未免太严重了，从姑娘口中说出来更是令人不敢相信。”
<br/>　　崔白玉没有理睬程大雷的打趣，道：“恳请大当家帮我救一个人。”
<br/>　　“谁？”
<br/>　　崔白玉一指长街：“我表妹乔弄云。”
<br/>　　程大雷顺着她手指的方向看过去，的确见一清秀姑娘，只不过如今她身上伤痕累累，也是狼狈得很。
<br/>　　这伙犯人自然是因为崔家的事牵扯进来。那些牵扯比较深的，自然就是个斩字，有些瓜葛的，男的流放到边关，女的就要变卖为官妓。当然，其中也不乏一些姿色靓丽者，被人捎带手顺到自己府中，只要势力够大，也不算什么事。
<br/>　　程大雷看着崔白玉：“我为何要帮你？”
<br/>　　崔白玉狠狠咬了咬牙：“崔白玉愿给大当家当牛做马。”
<br/>　　“你现在就是给我当牛做马呀？”
<br/>　　崔白玉无言以对。她并非有太多感情的人，与家里的亲戚姊妹基本没什么牵绊。但终究是兔死狐悲，现在崔家遭到这灭门之祸，就连祖坟都被人扒开挫骨扬灰。崔白玉知道自己什么都做不了，所有心酸只能咽下。但乔弄云与她关系当真还算不错，如果自己不管，她必然卖入娼窑，一辈子生不如死。
<br/>　　程大雷不是很乐意管这种闲事，你崔白玉愿意救一个乔弄云，万一乔弄云也有父母兄弟，也开口要自己救呢，自己是救还是不救。
<br/>　　终究是一件很麻烦的事情，而且你崔白玉完全不肯下本么。开口求人连个笑容都不肯给，似乎向自己求情就是天大的侮辱。什么做牛做马，完全是对自己职业的鄙视，你明明可以说以身相许嘛。
<br/>　　崔白玉垂下眼帘，轻轻摇了摇头。
<br/>　　“是我多口了。”
<br/>(本章完)
