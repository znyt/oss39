　　正当慕清霄思绪万千之际，脑海中的系统提示声，将他拉回现实。
<br/>　　“叮，宿主所在位面《仙剑奇侠传3》，开始发布任务……”
<br/>　　提示声刚落，慕清霄微微一愣。
<br/>　　果真与他猜的八九不离十，仙剑奇侠传位面的修炼体系虽不如古剑奇谭，但也算是高级仙侠位面一列。
<br/>　　不过，此位面也有些特殊。
<br/>　　仙剑奇侠传中共有六界，虽分强弱，但六界却是平等，不像古剑奇谭中，人界与天界根本不是一个层次。
<br/>　　仙剑六界：神界、魔界、仙界、妖界、鬼界、人界。
<br/>　　神界：因盘古身体崩解而形成的神界，无肉体无欲求，拥有永恒的生命。
<br/>　　永生不老，无人间温暖，神之间冷漠无情。却羡慕人间的情，故总有痴男怨女冲破禁锢，为情而抛却一切，潇洒人间。
<br/>　　魔界：它和神界完全对立的魔界，存在于天上与神界平行的反世界，空间与时间错乱，有“神魔之井”和神界相通。
<br/>　　无思无想无欲求，无生无死无秩序。
<br/>　　任性而为的魔，偶尔会由于长久的相处而渐悟，由于意外的触动而顿悟，也可能会自此脱离魔道返正道。
<br/>　　妖界：禽兽草木修炼可成妖，它是无秩序分散的存在，有弱肉强食的传统。其实真正的妖界，似乎隐藏在蜀山！
<br/>　　仙界：徘徊在人与神之间的族群，仙为人修炼而成，其是成神的必经之途，寿命长而有尽，肉身有而无涯。
<br/>　　鬼界：轮回的终站和起点，亡人为鬼，鬼入轮回而投身五界。
<br/>　　人界：生存于大地，为女娲所造，由女娲后人世代所守护，寿命短，有国家、社会制度、道德等约束。
<br/>　　六界中，神界实力最强，魔界次之，魔界是六界中唯一不入轮回的，永生不灭，死后仍可聚气重生，其余五界生灵死后均要进入鬼界轮回转世。
<br/>　　六界都有相应的强者，而这些强者地位都差不多，只是实力各不相同而已。
<br/>　　当然，也有六界外的生命，比如神树果实所造的唐雪见，没有实体的邪剑仙，皆是属于六界之外。
<br/>　　根据天地间的灵气看来，仙剑奇侠传位面，绝对不可能高于古剑奇谭，可能还会差上一两个层次。
<br/>　　“主线任务1：诛仙，斩杀邪剑仙，成功奖励：1000000掠夺积分，失败惩罚：境界掉落至化神一重！”
<br/>　　“掠夺任务1：掠夺神界神树，成功奖励：600000掠夺积分，失败惩罚：境界掉落至练虚一重！”
<br/>　　“掠夺任务2：掠夺魔剑，成功奖励：400000掠夺积分，失败惩罚：境界掉落至练虚五重！”
<br/>　　“掠夺任务3：掠夺土灵珠，成功奖励：200000掠夺积分，失败惩罚：境界掉落一重！”
<br/>　　“掠夺任务4：掠夺火灵珠，成功奖励：200000掠夺积分，失败惩罚：境界掉落一重！”
<br/>　　“掠夺任务5：掠夺雷灵珠，成功奖励：200000掠夺积分，失败惩罚：境界掉落一重！”
<br/>　　“掠夺任务6：掠夺风灵珠，成功奖励：200000掠夺积分，失败惩罚：境界掉落一重！”
<br/>　　“掠夺任务7：掠夺水灵珠，成功奖励：200000掠夺积分，失败惩罚：境界掉落一重！”
<br/>　　……
<br/>　　……
<br/>　　系统提升声刚落，慕清霄脑子有些懵，一次性接受如此多的任务，真的是第一次。
<br/>　　回过神来，在街道上随意找了个摊位坐下，取出一锭银子放在桌上，道：“老板，来壶好酒，在来几盘下酒菜。”
<br/>　　“好嘞客观，您请稍等。”
<br/>　　几分钟后，老板就提着一壶好酒，端着下酒菜摆在桌上。
<br/>　　斟上一杯酒，嗅着酒香，慕清霄开始思考下一步该怎么走。
<br/>　　仙剑奇侠传位面修炼体系虽不如古剑奇谭位面，但想必也不会差上太多的层次。
<br/>　　主线任务1中，斩杀邪剑仙，后者可是本位面中的大BoSS，连魔界至尊重楼都拿他没有一点办法。
<br/>　　而他自己，现在不过合体二重的实力，若是对上邪剑仙，估计不够给对方塞牙缝的。
<br/>　　别说是邪剑仙，他现在连重楼都对付不了。
<br/>　　至于掠夺任务1，掠夺神界的神树，慕清霄现在实力也绝对不足。
<br/>　　神树由神界女神夕瑶掌管，在夕瑶的上面，还有一个天帝，想要从神界手中抢夺神树，最起码也要有相应的实力。
<br/>　　系统既然发布了任务，自然就有应对的方法。
<br/>　　想到此处，慕清霄暗道：“系统，神界天帝的实力有多强。”
<br/>　　“叮，回答此问题，需支付100000掠夺积分，是/否支付！”
<br/>　　“支付！”
<br/>　　区区十万掠夺积分，能够查探出天帝的信息，绝对是物超所值，有了天帝的信息，他就能大致推断出其他人的信息。
<br/>　　“叮，支付成功，天帝乃神界主宰，六界至尊，拥有至高无上的权利，地仙二重境！”
<br/>　　系统提示声刚落，慕清霄剑眉紧皱在一起。
<br/>　　果然如他所料，仙剑奇侠传位面比较特殊，体系比古剑奇谭位面，最起码低了两个层次。
<br/>　　六界至尊的天帝，居然只有地仙二重的实力，真是耐人寻味。
<br/>　　当然，六界至尊只是一个称号，并不是说天帝就是神界最强，如果记得不错，神界最强，乃是景天的前世，飞蓬将军。
<br/>　　天帝是地仙二重境，那么飞蓬最起码也是地仙五重，甚至更高一些。
<br/>　　魔尊重楼能与飞蓬打成平时，并且没有分出胜负，实力恐怕也差不多。
<br/>　　如此看来，飞蓬与重楼都拿邪剑仙没有办法，后者的实力，恐怕不是慕清霄现在能够接触的。
<br/>　　对于系统发布的任务，真是任道而重远啊。
<br/>　　好在，现在剧情似乎并没有开始，他有的是时间，可以先从简单的任务下手。
<br/>　　况且，慕清霄也有自己的底气，若是将阴阳玉中的焚寂剑完全炼化，他的实力，绝对会飞跃式的成长。
<br/>　　到时候有黑凤神通，未必不能将邪剑仙斩杀。
<br/>　　仅仅一壶酒的功夫，慕清霄就将未来的计划决定的差不多，并且将接下来，可能会要遇到的情况也分析的七七八八。
