　　　　不得不说火焰这项属性能力很好用，杀人还能毁尸灭迹。
<br/>　　不过十几个呼吸，鹰钩鼻男子就彻底的从这个世界上消失。
<br/>　　这时，涂小安看到地上一堆骨灰中有一块亮晶晶的东西。
<br/>　　玉笛？
<br/>　　涂小安一眼看过去，这玉笛自动的悬浮了起来。
<br/>　　“果然是好东西，被火焰焚烧，还能安然无恙的保存下来”
<br/>　　涂小安伸出手将玉笛握住，顿时间冰清透亮，此笛通体晶莹无暇，内有浩光萦绕，品相极佳，一看就不是凡物。
<br/>　　光拿去买卖，恐怕都是无价之宝。
<br/>　　涂小安试着放在嘴边轻轻一吹，倏然一道音律飘荡，空气微微涟漪起来。
<br/>　　他眼睛猛然一亮，继续吹奏，笛声悠扬，浩浩荡荡，周围的空气开始剧烈的混乱起来，有种颠倒乾坤的错觉。
<br/>　　悄然间，大青石附近无数隐藏的生物骤然死去。
<br/>　　这...
<br/>　　涂小安停下，陷入思忖，这笛子吹起来，能形成一股无形的杀伤力，不过既然非常的消耗意识力。
<br/>　　看似用嘴吹，其实是用意识力，也就是精神力吹奏。
<br/>　　普通人根本吹不起来，一吹恐怕会立刻头昏脑胀。
<br/>　　鹰钩鼻男子用笛声控制蚊虫，这算是一种控虫之术，笛声随到之处，蚊虫就立刻被控制，所奴役。
<br/>　　涂小安并不会吹笛，他刚才瞎吹的几下，却已经照成了空间的震荡，但也严重的消耗意识力。
<br/>　　这玉笛蕴含可怕的杀伤力，运用得到，笛声便可杀人，鹰钩鼻男子仅仅用它控虫，有点大材小用。
<br/>　　但想要大材大用，并不是那么简单，必须有强大的意识力支撑。
<br/>　　就好比你要有浑厚的内力一样，没有足够的内力，还会有反噬的危险。
<br/>　　怪不得刚才鹰钩鼻男子吹了一会儿，脸色就骤然苍白起来。
<br/>　　不过涂小安的意识力现在已经足够强大，可以做到外控，这玉笛在他的手中到如虎添翼，相得益彰。
<br/>　　运用意识力吹笛，彻底杀人于无形。
<br/>　　好东西，好宝贝。
<br/>　　涂小安心安理得的收下了，到时候在好好研究一下这玉笛。
<br/>　　瞎吹出来的音律显然是不合格的，无法做到控制自如。
<br/>　　今天的蛇王节也算告一段落了，出现了中毒者恐慌事件，谁也没心情庆祝了。
<br/>　　好在没有人员死亡，也好在涂小安快刀斩乱麻的解决到了祸根，也不用担心还会有类似的事情出现。
<br/>　　蛇王庙他也懒得去了，反正麻烦解决了，涂小安转身离开山脚下，朝着自家的方向走去。
<br/>　　最令涂小安高兴的是，街道蚊子伤人，自己熟悉的人都不在其中，老妈跟老姐不知道现在回家了没有。
<br/>　　涂家。
<br/>　　晚十点时许。
<br/>　　涂家灯火通明，今天有一位客人大晚上登门拜访，温秋郑重的亲自招待，涂小月都回避了。
<br/>　　“秋姐，没想到那么多年了，你还能把我认出来”
<br/>　　一位轮椅上的男子温和的看着对面的妇人开口。
<br/>　　温秋礼貌性的笑了笑：“陈医生当年跟我丈夫可是交情颇深，我又怎会忘记，只是后面忽然没了你的消息，没想到今日还能见到你”
<br/>　　“我离开白镇十几年了，今天回来，自然要拜访一下故人，只是没想到匆匆岁月，我跟涂兄就已经阴阳两隔了”
<br/>　　做在轮椅上的人，不是别人，正是蛇博士陈默。
<br/>　　蛇王庙的中毒者一一苏醒，事情解决，他就来到了涂家。
<br/>　　“你能来，已经有心了”
<br/>　　温秋最后将陈默领到了一间摆放灵位的小房间，默默的退了出去。
<br/>　　陈默看着墙壁上一张挂着的黑白照片微微发呆，照片上是一位英气男子正在对着他笑。
<br/>　　“涂兄，我来看你了”
<br/>　　陈默的声音萧条沙哑，不轻不重，却带着复杂的情感：“十几年过去了，你已不在，而我也成为了一个废人”
<br/>　　当年白镇最具天赋的捕蛇人，一个已经死了，一个已经残了。
<br/>　　如见再见，早就物是人非。
<br/>　　陈默其实一直都想来涂家，只是心里一直记得当年的一个赌约。
<br/>　　他是想带着自己创造的绝世蛇种来看望这位故人。
<br/>　　可惜想法很美好，现实很骨感。
<br/>　　最终都是折在了蛇的手中，终日玩蛇，终有被蛇伤的一天。
<br/>　　陈默看着灵碑，看着照片嘴巴轻声的呢喃自语，好像在诉说当年的一幕幕场景。
<br/>　　眼睛充满了浮华跟伤感，人也一下子老了十几岁一样。
<br/>　　所谓的绝世蛇种，他们都没有成功。
<br/>　　门外，温秋安静的守在外面，涂小月鬼鬼祟祟的来，问道：“妈，这人是谁，瞧你一脸慎重对待的样子”
<br/>　　温秋唏嘘的开口：“你老爸曾经的故友，他们已蛇会友而认识”
<br/>　　武侠小说中已剑会友，搁在白镇，都是已蛇会友，很多人相互斗蛇，都能斗出感情来。

<br/>　　温秋将自己的女儿赶走。
