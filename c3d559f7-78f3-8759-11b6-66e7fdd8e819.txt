　　叶少阳瞠目结舌，这种惨状，光是想象一下就觉得触目惊心，不敢想象受到这种酷刑的人，当时那种疼痛和绝望……简直无法想象。
<br/>　　他转头看了一下，翠云好像是在厨房收拾，不在身边，不然让她听到这种事情，估计要吓趴了。
<br/>　　陈三接着说道：“还不止这样，为了让人不死，长毛在点天灯的时候，有时候会给人喂参汤之类的续命，好让刑罚能够持续更长时间……”
<br/>　　叶少阳嘴角抽搐了一下，“行了别说这个了，后来怎么样了？”
<br/>　　“掌教死了之后，那些长毛就逼几个监院交钱，监院也拿不出钱啊，但那些长毛不信，就一个一个点天灯，其中有个监院悲愤不已，就暗中给自己下了一种死咒，在自己被点天灯的时候，借助刑罚的怨气，死后成为厉鬼，附身在一个长毛身上，把那个下令点天灯的长毛首领砍死了，又屠杀了好些个长毛，之后自己因为死咒的缘故，也魂飞魄散了。”
<br/>　　说到这里，陈三叹息不已，端起酒杯猛灌了几口，神情复杂地望着窗外远处，显然，对于这些往事，他至今还是有些耿耿于怀。
<br/>　　叶少阳听了这个真实的故事，心中也很是感慨。之后陈三说起他自己的故事，在那场混乱之中，他逃离了道观，南下来到中原地区，因为会一点法术，开始是给人捉鬼降妖，不过他本事实在是不怎么样，只是道童牌位，有几次都差点在捉鬼的时候挂了，后来这边有了义庄，青冥道长到处找人赶尸，他就进了义庄，当了尸工，总算能过上稳定的日子，在这个乱世之中，他对此已经很满足了，现在的心愿就是攒点钱娶个老婆过日子。
<br/>　　说到这里的时候，他突然望着厨房方向，笑了一下，低声说道：“那个，你姐姐嫁人了没有？”
<br/>　　叶少阳本来听他说的挺可怜的，没想到突然蹦出来这么一句，惊了一下，看着他笑起来一脸的褶子，“卧槽，你居然打我姐姐的主意啊，不是我打击你啊，你们年纪差距太大，根本不可能的。”
<br/>　　陈三尴尬地笑了笑，“我也就是随便说说，想想而已。”
<br/>　　“不可能的，你想都不要想！”
<br/>　　吃完饭，叶少阳和陈三各自回房间躺了一会，翠云去洗碗，将东西收拾起来。叶少阳醒来时候发现，翠云坐在房间里在纳鞋，看他醒了，立刻放下手里的活计，让叶少阳陪她一起到镇上去转转，买点东西。
<br/>　　“咱们不是有这么多行李了，还买？”叶少阳有些纳闷。
<br/>　　“这些都是吃的啊。我得去买点针线布头什么的，路上好用啊，再买点草纸牙粉什么的。我一个人不敢去，正好天还没黑，你陪我一起吧。”
<br/>　　叶少阳反正也没什么事，于是陪她一起出去。
<br/>　　这是一个山里的小镇，叶少阳之前从陈三口中得知，这附近是有几个村子的，都会到这个镇上来买东西，所以卖东西的店铺是有一些的。
<br/>　　两人进入镇子之后，果然找到了一条小街，沿街有些商铺，不过卖的东西以农具之类的居多，还有买卖牲口的，逛的人不多。
<br/>　　两人在街上走了一圈，找到一家买布的铺子，翠云拉叶少阳进去，挑了几种布，让叶少阳自己选一个。
<br/>　　“我不是有衣服了吗。”叶少阳低头看着自己身上的衣服，这还是之前翠云帮他从成衣店买的缎子长衫。
<br/>　　翠云道：“你这是好衣服，我们要在翻山越岭的，你这衣服好看不经穿，磨坏了多可惜，我给你做两套耐穿的衣服。”
<br/>　　店家听见他们的对话，笑着夸叶少阳这媳妇好，会持家。
<br/>　　“我是他姐！”翠云红着脸说道。
<br/>　　之前张道长给了几十块大洋，他们也算是有些钱在身上，最后翠云挑了几块布料，又买了一些做鞋用的布面和鞋底，针线剪刀……走的时候，表情十分满意和满足。
<br/>　　逛街快到天黑，两人打算买点东西吃，找到一个包子铺，寻思进去吃一顿，再多买一些路上吃。
<br/>　　这家店铺只有肉包子，个头很大，两人要了几个。叶少阳拿起一个，吃了一口，突然整个人都僵住了。
<br/>　　“怎么了，不好吃吗，我觉得挺好吃的啊。”
<br/>　　叶少阳半天才把包子咽下去，转头寻找老板，这时候店里没几个人，老板也不忙，站在案板前悠闲地和着面。
<br/>　　“老板！”叶少阳叫了一声，“你这包子——”
<br/>　　“怎么了？”老板在围裙上擦了擦手，走过来。
<br/>　　叶少阳一时间不知道怎么说，怔怔发呆。
<br/>　　老板有些纳闷。
<br/>　　翠云又问了叶少阳一遍：“不好吃吗？”
<br/>　　不等叶少阳开口，老板先说道：“这不可能，不是我夸口，我家包子在整个江西那是一绝，肉馅材料用的都是祖传配方，绝对好吃。”
<br/>　　叶少阳望着老板，说道：“您是本地人？”
<br/>　　“对啊，土生土长，祖上多少代都在这卖包子，以前在县城，早年为了躲长毛，才到了这山里来。”
<br/>　　叶少阳一听，心中有些失落，那应该是自己想多了，虽然……这包子的味道，跟石城那家大肉包子完全一样，叶少阳还以为自己吃到了同一家，听老板说本地人，觉得这里离南京那么远，不可能是同一家店。而且时隔了九十年，手艺应该不会传到那时候，虽然石城那家包子铺号称百年老店，不过多半应该是吹嘘的。
<br/>　　翠云道：“你家包子真的很好吃，在这小地方开店，真的有点委屈了呢。”
<br/>　　老板叹口气说道：“是啊，这镇上没多少生意的，县城又不太平，我想攒两个钱，找个太平的大城市去开店。”
<br/>　　叶少阳一听，立刻警觉起来，问道：“你想去哪个大城市？”
<br/>　　“暂时还没打算，看看时局，店不是随便开的呀，万一打仗，很容易就砸了。”
<br/>　　叶少阳再度放松下来，抓着一个吃起来，但是老板又补充了一句让他血脉贲张的话：“我有个亲戚在南京，一直想让我过去开店，等两年我看看时局，没准会过去。”
