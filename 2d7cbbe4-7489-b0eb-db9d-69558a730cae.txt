　　苏莫只修炼了两个时辰，便睁开了眼眸，走出了练功房，因为古天意来到了雅翠居。
<br/>　　“古兄，怎么样？”快步走出宫殿，苏莫来到了古天意的面前。
<br/>　　“哈哈！上官兄，古某不辱使命，已经成功的说服了父皇，他会向学府中打招呼，释放你的两位朋友！“古天意满面笑意的说道。
<br/>　　“那就好！”苏莫闻言松了口气，如此的话，他就不用担心了。
<br/>　　“上官兄，我们去殿内一叙如何？”古天意笑问道。
<br/>　　“请！”苏莫一摆手，示意对方进入宫殿就坐。
<br/>　　随后，苏莫和古天意，便在宫殿内坐了下来。
<br/>　　“上官兄，既然你加入了我皇祖圣朝，自然要给予你职务，这几****先在雅翠居暂住，接下来我会给你安排一个适合的职务！”古天意说道。
<br/>　　“嗯，有劳古兄了！”苏莫点了点头，其实他丝毫不在意什么职务，但他也不会拒绝。
<br/>　　反正暂时他也不可能离开皇祖圣朝，就先在此地逗留一段时间吧！
<br/>　　等待聂乾和韩盈被放出来，他再做接下来的决定。
<br/>　　“上官兄不必客气，我准备将你安排在赤龙营！”古天意说道。
<br/>　　“赤龙营是什么地方？”苏莫疑惑的问道。
<br/>　　“赤龙营是一支小军团，由天才武者组成，成员均是在圣朝各地选拔而出，以及圣朝下辖的一些势力的武者，其中也有一部分神武学府退下来的弟子！”古天意说道。
<br/>　　“原来如此！”苏莫恍然，看来古天意是要把他安排在军中了。
<br/>　　“以上官兄的实力，进入赤龙营之后，必定能大放异彩！”古天意笑道。
<br/>　　“古兄安排吧！”苏莫点了点头，面色不喜不忧。
<br/>　　“好，既然上官兄没有意见，那我便去为你安排！”古天意满面微笑的说道，赤龙营可是皇祖圣朝军中的一柄尖刀，一直是他的势力无法触及到的地方。
<br/>　　他相信，以苏莫的实力和天赋，将来在赤龙营，必定有所作为，甚至为他掌控赤龙营。
<br/>　　少倾之后，古天意便离开了雅翠居，前去为苏莫安排。
<br/>　　望着对方离去的背影，苏莫面露沉思之色，他是不会被绑在古天意的战车上的。
<br/>　　等到聂乾和韩盈脱困之后，他随时可以离去。
<br/>　　目前，他唯一比较担心的事情，就是万界山的任务。
<br/>　　离开真圣界时，他倒是忘了问真贤道圣，如果完不成任务会有什么后果！
<br/>　　这个任务，苏莫虽然不想去做，但如果可以的话，他还是会尽量去完成。
<br/>　　因为他知道，一旦完成任务，好处肯定不小。
<br/>　　“小妍，你过来！”
<br/>　　片刻之后，苏莫呼唤一声，叫来了雅翠居的那名侍女。
<br/>　　“公子，请吩咐！”一袭绿衣的小妍来到了宫殿中，向苏莫躬身行礼。
<br/>　　“小妍，你在皇宫中待了过久的时间了？”苏莫微笑着问道。
<br/>　　“禀公子，小妍在圣皇宫四年了，一直在殿下的意宫中！”小妍恭敬的说道。
<br/>　　“四年，那也不短了，你对于皇宫中的局势，应该都了解吧？”苏莫轻声问道，既然要在皇祖圣朝逗留一段时间，他自然要弄清一些局势。
<br/>　　不然的话，很容易被古天意给坑了，到时候有麻烦就不好了。
<br/>　　而眼前的侍女，在皇宫中待了数年，不可能不了解一些。
<br/>　　“这……！”小妍闻言面带迟疑之色，他只是一名小小的侍女，哪敢妄谈皇宫中的局势啊！
<br/>　　“无妨，你知道多少，都如实的告诉我，无论对与错，我都不会责怪你的！”苏莫笑道。
<br/>　　“是！”小妍闻言轻点螓首，苏莫有言她不敢拒绝。
<br/>　　“好，那我问你，听说皇祖圣朝，天赋最好的皇子有三人是吧？“苏莫问道。
<br/>　　“是的，分别是大皇子、三皇子和八皇子殿下，每一个人都曾经在真圣碑留名，将来有很大的希望成为武圣！”小妍点头说道。
<br/>　　“那三位皇子之间的关系如何？”苏莫再次问道，他知道三位皇子之间关系不好，但具体是什么程度，还是要了解一番。
<br/>　　“听说并不怎么好，大皇子和三皇子势力比较大！”小妍怯生生的说道，这种事情她本不想回答，但又怕惹得苏莫生气。
<br/>　　“哦，那三位皇子之间，是否时常有冲突呢？”苏莫继续询问。
<br/>　　“冲突倒是没有，不过，好像三位皇子之间时常会有争斗！”小妍轻声道。
<br/>　　苏莫微微点头，三位皇子的争斗，无非就是皇位之争，他稍微有些担心的也正是这点，不想日后被古天意束缚。
<br/>　　“那赤龙营你知道吧？”苏莫再次问道。
<br/>　　“赤龙营？”
<br/>　　小妍闻言一怔，随即笑道：“当然知道，赤龙营可是我们皇祖圣朝天才的集中营，每一个赤龙营的将士，都是天才武者！”
<br/>　　“是吗？那这个赤龙营是八皇子的势力？”苏莫问道，八皇子要将他安排在赤龙营，想必这可能八皇子的势力了。
<br/>　　“并不是，赤龙营是直属圣皇陛下的军团！”小妍摇头说道。
<br/>　　“哦！”
<br/>　　苏莫微微点了点头，面带沉吟之色，这个赤龙营居然直属于圣皇统领，那古天意将他安排进去，用意不言而喻了。
<br/>　　直属于圣皇的军团，何等的重要，对于古天意，以及大皇子和三皇子而言，肯定是要想办法掌控的。
<br/>　　看来古天意将他安排进赤龙营，是想要他将来能控制赤龙营，日后为其所用。
<br/>　　苏莫轻叹一声，现在古天意对他客气有佳，如朋友般相处，但他心中却是清楚，自己不过是古天意夺嫡的棋子罢了。
<br/>　　自古以来，最冷莫过帝王家，夺嫡之争一切都可以牺牲。
<br/>　　又何小妍闲聊了片刻，苏莫对于赤龙营已经有了大概的了解。
<br/>　　随即，他便挥退了小妍，独自坐的宫殿中沉思。
<br/>　　古天意想利用他，他说不定也能利用整个皇祖圣朝，来完成万界山交代的任务，夺取血瞳族的至宝血轮眼。
<br/>　　不过，此事事关重大，还需要从长计议。
<br/>　　因为想要夺取血轮眼，估计要灭了整个血瞳族了。
<br/>　　血瞳族在古灵星河，虽然只是弹丸小族，但在赤粱星可是超级势力，若是没有皇祖圣朝之助，他断然没有完成任务的可能。
