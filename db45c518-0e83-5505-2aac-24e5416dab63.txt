　　第675章
<br/>　　蛤蟆城诸多人，许多人都到过凉州城。但对程大雷来说，这委实是第一次，以他的身份，若敢擅闯凉州城，宋伯康一定不惜一切代价干掉他。
<br/>　　不过事先程大雷已着人绘过一份城主府的布局图，带着人马闯进来后，对周围建筑并不陌生，也不会有迷路的担忧。
<br/>　　这城主府如今应该说是宋伯康的王府，不过宋伯康也并非骄奢淫逸之人，并没有因为自己封了王便大兴土木。程大雷闯进正门后，第一感觉就是好大，好气派。自己如今已经控制两座城，包括蛤蟆城和朔方城，论气派都不如凉州城。
<br/>　　现在不是看风景的时候，程大雷也没有仔细端详，他挥舞着大斧子，口中哇呀呀爆叫：“宋伯康在哪里，宋伯康在哪里……”
<br/>　　身后的兵丁震天齐吼：“降者免死，降者免死……”
<br/>　　大厅被占领，花厅被占领，玉燕阁被占领，听风院被占领……
<br/>　　城主府虽然有守卫，但并没有驻军，程大雷长驱直入，并没有遇到怎样的抵抗。但直到现在，程大雷还没有逢到宋伯康，宋伯康不除，这场攻占城主府的战斗就算不得胜利。
<br/>　　如果让宋伯康逃走，逃到兵营组织起军队，那么这场战斗便会变得艰难起来。
<br/>　　他哇呀呀大叫，拎着大斧头寻找宋伯康，刚开始的确是虚张声势，但现在是真的必须找到宋伯康。
<br/>　　喊声震天，火光四起，在外人看来，程大雷可能引了千军万马入城。然而事实上，这场战斗的规模并不大，只是搞得声势很大。
<br/>　　程大雷的确是在刀锋上起舞，看似华丽异常，实则铤而走险，稍有不慎，便是满盘皆输。
<br/>　　目前寻不到宋伯康，程大雷也无暇想太多，首先要做的是逐步占领城主府，再做其他打算。
<br/>　　估计宋伯康现在已经撒丫子逃了，这也并非很难想到。第一，以程大雷的兵力，根本没办法对城主府形成严丝合缝的包围。第二，将城主府团团围住，也并不是他的目的。
<br/>　　围城必阙，一旦让这些人陷入死地，就会爆发出难以想象的战斗力。程大雷是故意放他们逃的，不仅对城主府如此，对整座凉州城也是一样。
<br/>　　赵子龙占领其他城门，但却唯独放开一座东门，就是给城中守军一条生路。
<br/>　　程大雷逐步推进，前厅已完全被其占领，那些家丁仆人，并没有谁咬着牙与程大雷反抗。便是有的，也被程大雷手下兵丁一刀杀了。程大雷再往前进攻，已深入城主府的后院。
<br/>　　“宋伯康在哪里，宋伯康在哪里……”
<br/>　　程大雷拎着大斧头，一脚踹开一座紧闭的房门。忽然间，刀光扑面，这等偷袭，自然伤不了程大雷。他轻描淡写便接住对方刀，定下神时却愣了愣。面前是个十几岁的女孩，往里面看，见大屋内躲了十几个女人，此刻都在瑟瑟发抖。
<br/>　　后院便是府中女眷居住的地方，宋伯康的老婆孩子都住在这里。虽然宋伯康并不十分好色，但家里养着七八房小妾，几十名歌舞乐班还是理所应当的事情。
<br/>　　这些女人逃无可逃便躲在此地，此刻聚在一起，神情惶恐，脸色苍白。
<br/>　　程大雷将敢袭击自己的女人推开，挥手传令手下人。
<br/>　　“传我军令，胆敢侵犯女眷者斩立决。”
<br/>　　“胆敢劫掠他人财物者斩立决。”
<br/>　　“胆敢滥杀无辜者斩立决。”
<br/>　　程大雷不需要大声说话，自然有人将他的话传遍全城。瞬间，这三道军令在夜里传开，人人大呼小叫，便传遍三军。
<br/>　　戎族入侵，往往屠城，一座城池一旦被戎族攻破，便是人间地狱。但会屠城的不仅仅是戎族，各路诸侯间斗争，也往往以屠城激励手下士兵。一旦破城，便是几日几日不封刀。女人，金银财宝能抢多少要看你各人本事。
<br/>　　当初林少羽攻破长安，达官贵人能逃的就逃了，那些不能逃走的，往往生造横祸。便是大内皇宫的嫔妃公主也多有人被糟蹋。
<br/>　　所以，林少羽虽然冠以正义之名，但他手下兵丁的所作所为远远谈不上正义。如此，也不怪李行哉铁了心要杀他。死后尸首分离，也算是一报还一报。
<br/>　　不过，屠城这种事在程大雷治下从不曾发生过。他很早就注意到士兵素质的问题，也不仅仅是他们的战斗力，还包括士兵的组织性纪律性。程大雷一声令下，便无人敢违抗。
<br/>　　崔白玉被歌舒离抱在马上，纵马向东门逃去。也是事急从权，若是从前，哥舒离绝对不敢如此唐突。崔白玉也懂得这一点，所以并没有表示什么。只是手中紧握着一柄匕首。
<br/>　　她虽然才智无双，但本身并没有任何武艺，这柄匕首不是留给敌人的，而是留给自己的。一旦破城之后逃不掉，自己这无双美人可和普通人没有任何区别。甚至要比普通人遭遇更多不堪。
<br/>　　逃出去，一定要逃出去。
<br/>　　程大雷传下的三道令也传到她的耳中，崔白玉第一次觉察到：这个程大雷并不简单。
<br/>　　能以如此军令约束手下士兵，收买人心，其人所图肯定不小。怪只怪，自己还是太小看他了。
<br/>　　若有再来一次的机会，希望与你势均力敌的战一场。
<br/>　　崔白玉在亲卫队护送下逃生，能做她的侍卫自然是千挑万选出来的，尤其是哥舒离也是天下超一流高手。他们长街横行，无人可挡。如此表现，却又格外显眼些。
<br/>　　“呔，留步吧。”
<br/>　　一匹黑马忽然奔来，拦住了他们的去路，马上人使一口大刀，正是蛤蟆城的高飞豹。
<br/>　　高飞豹没有具体任务，就是在城中搞事情，把声势造得大一些。好巧不巧，正与歌舒离撞个对面。
<br/>　　歌舒离忽然勒住马，手中用的恰好也是一口大刀。
<br/>　　“小蟊贼，也敢挡爷的路。”
<br/>　　高飞豹气得是哇呀呀大叫，他是什么身份，至少也是虎豹军的副指挥。你怎么还可以当我是小蟊贼的，我是有身份的人好不好。
<br/>　　“贼人，去死。”
<br/>　　手持大刀，拍马攻来。
<br/>(本章完)
