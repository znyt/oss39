　　“青云，怎么样？”
<br/>　　远离了天君河和帝一魂等人之后，高空之中，一边飞行，地渊神王一边询问拓跋青云。
<br/>　　“没有什么大碍，数十年之内就能恢复！”拓跋青云摇了摇头，他的确是受了伤，适才帝一魂的那一击，无比强大，让得他九龙神鼎的九条龙灵纷纷受伤。
<br/>　　若是仅凭九龙神鼎元神，他自然是挡不住那一招，不过，他还有封印之门。
<br/>　　对方的攻击，最后时刻已经被他封印在了元神之中。
<br/>　　当然，他为了示敌以弱，故意让得九条龙灵气势暴降，显得受到重创的模样。
<br/>　　实则，九条龙灵虽然受创，也伤到了他的神魂，但是，并不严重，数十年之内就能恢复。
<br/>　　“看来帝一魂的实力，比你还要强大两分！”地渊神王并没有松口气，而是面色更加的凝重了。
<br/>　　这可不是一个好兆头，而且，拓跋青云受伤，这又要耽搁数十年的时间。
<br/>　　数十年的时间，虽然不算长，但是，对于现今的局势来说，也很不妙。
<br/>　　毕竟，现如今就是在争分夺秒，那帝一魂天赋可怕，不知道什么时候，会晋升高级神王。
<br/>　　一旦帝一魂晋升高级神王，他们都性命难保。
<br/>　　“青云，那秦陨的身份，你确定了吗？”另一位地王宫的高级神王问道。
<br/>　　此人，乃是地王宫的二长老，也是一位老者，头发略显花白，脸庞狭长。
<br/>　　“秦陨的身份基本能够确定，应该是分魂！”拓跋青云沉声说道，虽然他没有得到确切的答案，但是，在他看来，八九不离十了。
<br/>　　因为，在他看来，无论是天尘宇的戏耍，还是帝一魂的淡然，都更像是一种隐瞒。
<br/>　　若秦陨不是分魂，两人应该不至于如此。
<br/>　　所以，他基本能确定，秦陨就是分魂。
<br/>　　“确定了秦陨是分魂，也无济于事，这个小子太不简单了，想要找到他不容易！”天君河叹了口气道。
<br/>　　天王宫一直在找秦陨，他们地王宫也在找，但是，目前并没有丝毫的发现。
<br/>　　“无论如何，一定要比天王宫先一步，找到秦陨！”拓跋青云深深的吸了口气，在他看来，帝一魂之所以能超越他，是因为帝一魂融合的分魂比他多。
<br/>　　所以，只要他能融合了秦陨，天赋和战力，将能超越帝一魂，更能更快的达到高级神王的地步。
<br/>　　“此事，为父会想办法，并亲自着手调查，就算是将神域掘地三尺，也要将秦陨找出来。”地渊神王点了点头说道，他自然明白那秦陨的重要性。
<br/>　　若是秦陨被帝一魂找到了，最坏的情况，就是帝一魂融合秦陨之后，立刻晋升高级神王。
<br/>　　这不是没有可能，而是可能性很大。
<br/>　　想要找到秦陨，也并不是多困难，困难的是抢在帝一魂和天王宫的前面。
<br/>　　“听说终极神王殿的数十万弟子，已经将天池神朝八十一个界域，详细的查探了一遍，还没有秦陨的任何踪迹，接下来应该是开始探查其它神朝了！”拓跋青云说道。
<br/>　　“嗯！”地渊神王点了点头，他会和终极神王殿交流，联合起来，查探秦陨的下落。
<br/>　　只不过，其它几大神朝，加起来地域极广，需要消耗不少的时间。
<br/>　　随后，三人不再多言，加快赶路，向地王宫而去。
<br/>　　……
<br/>　　高空之中，帝一魂和天君河、罡雷大长老三人，并没有急着返回。
<br/>　　因为，他们知道，此地，除了他们之外，肯定还有其他人。
<br/>　　天君河转头环视四方，朗声道：“人德老儿，老夫知晓你来了，还不速速现身？”
<br/>　　朗朗的声音，传遍八方，浩荡方圆十万里。
<br/>　　“呵呵，天君河，老夫可是看了一处好戏啊！”
<br/>　　一声轻笑随之响起，便见到极远之处，虚空一阵扭曲，一道身影显现而出，一步十里，大步而来。
<br/>　　此人不是别人，正是人德神王。
<br/>　　“的确是一出好戏，不过，不知你有何感想？”天君河微眯着眼眸，凝视着大步走来的人德神宫。
<br/>　　“感想？倒的确是有！”
<br/>　　人德神宫闻言，停下了脚步，遥望着帝一魂，道：“都言帝一魂为六魂之主魂，今日一见，果真不凡！”
<br/>　　“前辈妙赞了！”帝一魂面色平静，非常的客气，向人德宫主抱了抱拳。
<br/>　　“你当的起！”人德宫主摇头说道。
<br/>　　“前辈，你觉得苏莫与我相比，谁更甚一筹？”帝一魂沉声问道。
<br/>　　“你们都是天尊大人转世之身，同样不凡！”人德宫主道。
<br/>　　“既然如此，前辈又何必帮衬苏莫呢？”
<br/>　　帝一魂面色郑重，继续道：“苏莫或许得你看重，但是，他落后的太远太远了，只要前辈助我，融合苏莫，镇压地王宫，一切都能结束了！”
<br/>　　人德宫主闻言，沉默了下去，目前来说，的确是这样。
<br/>　　他若是帮助帝一魂，能轻易的找到苏莫，一旦找到了苏莫，帝一魂将更上一层楼。
<br/>　　然后，在他人王宫的帮助之下，地王宫和拓跋青云，将不足为惧。
<br/>　　不过，他知道苏莫的天赋，比帝一魂更加的可怕，而且，苏莫的品性极佳，对于他极为信任，他实在不想走这一步。
<br/>　　再者，他答应过苏莫，若是出尔反尔，也违反自己的道心。
<br/>　　“前辈，您所做的一切，都毫无意义，只会让局势更加的混乱，你认为晚辈之言，是否在理？”帝一魂徐徐的说道，言语非常的随和，仿佛是在和人德神王商议一般。
<br/>　　因为他知道，人德神王的态度，能决定最终的结局。
<br/>　　见人德神王依旧沉默，帝一魂心中大动，他知道自己的话语，对方听进了心中。
<br/>　　“前辈，晚辈也不需要你立刻做出决定，这样吧，过段时日，我亲去人王宫，我们再详谈，您看如何？”帝一魂微笑着说道，他知道自己不能逼的太紧，对方不可能立刻同意。
<br/>　　毕竟，能达到这个地步的强者，心志之坚，稳如磐石，不会被轻易说动。
<br/>　　所以，他给予对方一段时间考虑，过段时日，再去大力游说一番。
<br/>　　天君河和罡雷大长老两人，并未开口，但两人对视一眼，均看到了对方的意思。
<br/>　　此事，或许真的有转机。
<br/>　　“帝一魂，老夫告诉你，你……真的不如苏莫！”
<br/>　　人德宫主凝视着帝一魂，淡淡的开口了，言罢，他转身离去，一道话音随之飘来：“老夫在人王宫等你！”
<br/>　　听闻此言，帝一魂微微一怔。
<br/>　　对方这是什么意思？
<br/>　　对方明确表明，苏莫比他更加的优秀，但为何还要在人王宫接见他？
<br/>　　天君河和罡雷大长老，亦是微微一怔，眉头微皱了起来。
<br/>　　“晚辈定会前去叨扰！”
<br/>　　眼见人德神王远去，帝一魂朗声说道，不管如何，他不会放过这次机会，定会前去人王宫。
<br/>　　“这人德老儿，不会是有什么阴谋吧？”
<br/>　　待人德神王远去之后，罡雷大长老皱眉说道，这倒是很有可能。
<br/>　　帝一魂若是大意，去了人王宫很可能有去无回。
<br/>　　“应该不会！”天君河摇了摇头，因为目前，以对方的立场，不可能对付帝一魂。
<br/>　　不然的话，若是帝一魂死了，拓跋青云将完全得势，会有大量的顶级势力，愿意帮助拓跋青云，踏平人王宫轻而易举。
<br/>　　再者，帝一魂前去人王宫，不可能单独前去，他们肯定会做好完全的准备。
<br/>　　“具体情况，到时候再说吧！”帝一魂叹了口气，他也猜不到人德神王的目的，只能到时候再看情况了。
<br/>　　言罢，帝一魂转身离去，向天王宫的方向而去。
<br/>　　天君河和罡雷大长老，对视一眼，无奈的叹了口气，便也立即返回。
