　　　　不过接下来，海棠将涂小安心中的疑问给解开了。
<br/>　　“因为我在不突破，我就要死了，如果我死了，高阳哥哥就没有人照顾他了，他的眼睛又看不到，孤苦伶仃，我不忍心，我答应过他，要当他的眼睛，他的手，他的脚，一辈子照顾他”
<br/>　　恢复理智的猫精，它的眼睛漂亮极了，亮晶晶，就宛如世间最纯洁的宝石，忽闪忽闪光芒。
<br/>　　但此时，这双眼中却充满了悲伤，又犹如绝望，那深陷的眼窝里出现了一滴亮晶晶的泪水，泪水顺着毛茸茸的脸颊，无声无息的流淌下来。
<br/>　　其实灵智大开的生物，可以口吐人言，跟人真的没什么区别了。
<br/>　　因为它们也是高等生物了，具备七情六欲。
<br/>　　“不突破就会死？”
<br/>　　涂小安咋的有点不明白，可转念认真一想，却有了答案，然后怔怔的看着眼前哭泣的猫精。
<br/>　　海棠伤心的说：“没错，因为我的寿命到了，我在不突破三阶，就会死”
<br/>　　“我不怕死，对于一只猫来说，我活的太久太久了，我只是放心不下，舍得不下我的高阳哥哥，他真的好可怜”
<br/>　　不过区区一只猫，趴在地上，全身搐动，一声声压抑的，痛苦的唏嘘，仿佛是从它灵魂的深处艰难地一丝丝地抽出来，散布在这昏暗的山洞中，织出一幅暗蓝的悲哀。
<br/>　　恍然间，涂小安都差点忘记了，这是一只猫。
<br/>　　如果闭上眼睛，他甚至以为眼前是一位娇柔的姑娘在哭泣，哭的让人怜爱。
<br/>　　海棠如今的寿命已经是一百六十三岁了，它要寿终正寝了。
<br/>　　除非突破三阶生灵，才能寿命大涨，继续活在这个繁华的人世间。
<br/>　　普通猫的寿命只有七到十五岁，能活到十五岁，基本上等于人类的百年了。
<br/>　　也正是因为海棠开了灵智，成为二阶，才会一直活到一百六十二岁。
<br/>　　可二阶生物的寿命也就不到二百岁，海棠知道自己的寿命要到了，要死了，所以才不得已孤注一掷的拼一把。
<br/>　　这就是赌徒的心理了，想成为三阶生灵是非常的难的，不但要一脚先迈过去，等你迈过去了，还要接受上天天雷天火的洗礼。
<br/>　　抗住了，你才能成为三阶生灵，凌驾普通生物之上，可化为人形，寿命大增。
<br/>　　海棠知道凭借自己的修炼，甚至一脚都迈不过去，只能剑走偏锋，从白家的人手中抢来了蛇王石，希望靠着蛇王石里面的力量，助自己迈过去，然后承受承受天雷天火的洗礼。
<br/>　　天雷天火都承受住了，那么它这只猫精也就可以登基坐殿，成为一代猫王。
<br/>　　可这真的是非常非常的难，就好比你要飞升成神，渡劫的死亡率高的可怕。
<br/>　　也不知道是不是几句话说下来，让海棠开始信任涂小安这条蛇精，它接下来开始诉说自己的猫人生。
<br/>　　涂小安也开始充当聆听的观众，同时也很好奇海棠如此能成为二十九级的猫精。
<br/>　　这也是迄今为止，他遇到觉醒生物等级最高的存在。
<br/>　　一百六十年前，那是应该属于清朝的年代，那时候的陈家可是温城的大户人家，人丁兴旺。
<br/>　　可那也是一个没落的朝代，民不聊生，百姓都快吃不上饭，更别说一些猫猫狗狗。
<br/>　　那时候街道上有很多的流浪猫跟流浪狗，很多都被饿死，或许被打死，但有一只出生刚满三个月的小猫，它是幸运的。
<br/>　　它流浪到了陈家大宅前，陈宅的女主人，心地善良，信佛，见此猫生的品相极佳，可爱非常，就不忍它饿死街头，将它给收养了。
<br/>　　自此，这只猫在陈宅过上了幸福的小生活，它不用受冻受饿，不用受动物或人的欺凌，每餐都有食物可吃。
<br/>　　日复一日，小猫长的越发的“水灵”俊俏，整个宅院的人都很喜欢这只小猫。
<br/>　　它几乎成了所有人的宠物猫。
<br/>　　有一日夜晚，此猫蹲踞屋上，望着浩然星空，似有所感下，伸口对月，莫名有一股能量涌现，它误打误撞的吸其天地精华，久而成精，开通灵智。
<br/>　　这便是海棠，有了灵智的海棠没有做出什么对陈家人不利的事情，反而一心一意的报恩，看宅护院，从此别说是老鼠，就算是汪洋大盗，都别想半夜翻陈家的院墙。
<br/>　　三十年之后，那位收留海棠的女主人去世了，在后二十年，陈家这一代的人基本上都离世。
<br/>　　可大家奇怪的是，这只猫，还是这是猫，时间不曾在它的身上留下一点痕迹。
<br/>　　反而这只猫的品相变的越发的不凡，简直久活成精，这五十年，海棠成为了二阶生物。
<br/>　　老一辈的陈家人都死了，海棠开始守护新一代的陈家人，兢兢业业，不让这家人受到任何不良因素的影响。
<br/>　　紧接着，又是一甲子六十年过去。
<br/>　　流水的陈家人，铁打的猫。
<br/>　　陈家人知道这只猫的不凡，视为吉祥之宝，每一代人都非常的善待海棠。
<br/>　　不知不觉到了建国，陈家没落，人才凋零，最后只剩下陈高阳的奶奶，跟出生就眼瞎的陈高阳。
<br/>　　海棠开始守护这对祖孙。
<br/>　　它也是从小看着陈高阳长大，由于陈高阳眼睛看不到，海棠对他格外的关注跟照顾。
<br/>　　可那时候的海棠虽然早就是猫精，灵智大开，但却情窦未开，活的虽然久，但是心智如小姑娘一样。
<br/>　　直到陈高阳的奶奶临终，她对着这只守护了陈家上百年的猫精说：“你早就报了我们陈家对你的收养之恩，你也早就可以离开，但是我现在恳求你在守护我们陈家最后一代好吗”
<br/>　　几乎每一代，海棠都会认一个主人，也唯有它的主人知道它会口吐人言。
<br/>　　奶奶知道这是只猫精，但却是善良而只属于陈家的猫精。
<br/>　　它不会害人，更不会吃人。
<br/>　　陈高阳眼不能明，奶奶临终就对这个孙子放心不下，他也是陈家唯一的独苗了，只能寄托给这只猫精。
<br/>　　海棠答应了这位将死奶奶的要求，陈家大院最后只剩下这只猫精跟一位天生眼瞎的陈高阳。
