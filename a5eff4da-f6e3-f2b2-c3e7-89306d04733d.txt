　　猎妖殿闭关室内。
<br/>　　萧逸盘膝而坐。
<br/>　　之前特意在血雾谷谷主面前说，要用他的大量天材地宝修炼。
<br/>　　实则，是要吞服妖兽精血。
<br/>　　以及炼制无杂质丹药。
<br/>　　先是取出一瓶瓶妖兽精血，大概在1200瓶左右。
<br/>　　全是七级妖兽精血，蕴含的妖元，极其庞大。
<br/>　　萧逸一一将其吞服。
<br/>　　他如今的肉体力量，只在地元五重。
<br/>　　而七级妖兽，可是等同人类武者天元境的存在。
<br/>　　故，它们的精血，对现在萧逸的效果极大。
<br/>　　萧逸大概每服下200瓶左右，肉体力量便增加一重。
<br/>　　当然，越是修炼到后面，需要的妖兽精血，也会逐步增加。
<br/>　　待得900瓶七级妖兽精血服下。
<br/>　　肉体力量，从地元五重，达到地元九重。
<br/>　　下一步，便是要修炼下一层了。
<br/>　　萧逸取出修罗战体后三层的功法。
<br/>　　功法本身难度也极大，但对萧逸而言，不成问题。
<br/>　　数个时辰后，萧逸参悟成功，再次开始了修炼。
<br/>　　剩下的200瓶精血，悉数灌入，却并没有突破。
<br/>　　如萧逸所料，修罗战体，每每到达突破下一层的临界点时，都要凝聚‘修罗之力’。
<br/>　　200瓶七级妖兽精血，只凝聚出了10滴修罗之力。
<br/>　　结合之前在破玄九重到地元境那个临界点时凝聚的10滴。
<br/>　　便是20滴。
<br/>　　顺带一提，这次虽然同样只凝聚了10滴修罗之力。
<br/>　　但，这次的10滴，比之上一个突破临界点突破的10滴，要强大得多。
<br/>　　简单来说，‘修罗之力’本身，也在升华。
<br/>　　通过大量的妖兽精血力量，在让自身变得更强。
<br/>　　现在的‘修罗之力’，怕是几滴，就比以前10滴要强大得多。
<br/>　　所有妖兽精血已服用完毕，萧逸停下了修炼。
<br/>　　‘咔咔’，握了握拳头，那骇人的肉体力量，让萧逸一阵心惊。
<br/>　　“这就是地元九重的肉体力量。”萧逸面带喜意。
<br/>　　他甚至有把握，若现在瞬间消耗20滴修罗之力，施展出崩界拳。
<br/>　　怕是一拳，就能把火长老这个层次的武者秒杀。
<br/>　　结束了肉体力量的修行。
<br/>　　萧逸将修炼放到真正的武道修为上。
<br/>　　如今自己的武道修为是地元三重。
<br/>　　虽说如今冰鸾剑已不‘抢夺’力量了。
<br/>　　但由于自身气泉庞大得骇人，故所需的无杂质丹药，仍旧是一个天文数字。
<br/>　　看了眼血谷主乾坤戒内的七级内丹，同样只有一千多颗。
<br/>　　八龙焚火炉拿出，不多时，300粒无杂质天元丹被炼制而出。
<br/>　　悉数服下后，修为增加了两重。
<br/>　　从地元三重，达到地元五重。
<br/>　　如今，丹药入腹，转化的力量，乃是元力，而并非真气。
<br/>　　作为比真气更高层次的元力，其蕴含的力量更大，更强，但转化所需的药力，也更多。
<br/>　　幸好这些是天元丹。
<br/>　　否则，若是地元丹，这个数字，绝对要翻几倍以上。
<br/>　　“呼。”萧逸深呼吸一口气，彻底停下了修炼。
<br/>　　这次的收获，不得不说，确实巨大。
<br/>　　肉体力量已达地元九重，修罗之力，达到了20滴。
<br/>　　武道修为，则是地元五重。
<br/>　　实力，无疑又上升了一个高度。
<br/>　　最重要的是，得到了修罗战体后三层。
<br/>　　日后自己的肉体力量，可一直修炼到极境都不成问题。
<br/>　　当然，前提是有足够的妖兽精血，而这个数量，也必将相当惊人，甚至是遥不可及。
<br/>　　……
<br/>　　做完一切，待得萧逸离开闭关室，发现天色已黑。
<br/>　　自己竟是已修炼了一天。
<br/>　　看了眼外头，血雾谷谷主，仍旧在等待着。
<br/>　　那森然的血腥眼眸，让人不寒而栗。
<br/>　　那眼神，仿佛要将萧逸整个吞下，杀之而后快。
<br/>　　萧逸笑笑，并不管。
<br/>　　只要自己一日在这猎妖殿中，血雾谷谷主便不敢强行对付自己。
<br/>　　这时，一个猎妖殿人员缓缓走来，对着萧逸行了一礼。
<br/>　　而后说道，“易霄总执事，今日之事，已由本殿的总执事，上报分殿主。”
<br/>　　“另外，后堂已为您准备好房间。”
<br/>　　“嗯，有劳。”萧逸点点头。
<br/>　　今日被血谷主重重震了一掌，伤势极其严重。
<br/>　　虽然自己稳下了伤势，并差不多痊愈了，但适当的休息，还是好的。
<br/>　　……
<br/>　　一夜无话。
<br/>　　美美睡了一晚的萧逸，起来后，走到猎妖殿外，看了眼。
<br/>　　果然，血谷主仍在。
<br/>　　“血谷主，一夜未睡，怕是疲惫不堪吧。”萧逸淡笑着说道。
<br/>　　血雾谷谷主冷笑一声，“小子，你不必得意。”
<br/>　　“我倒要看看，你能在里面躲多久。”
<br/>　　“大不了，本谷主在此常住。”
<br/>　　他已经打算将猎妖殿周旁的建筑买下来，以供自己常住。
<br/>　　萧逸冷笑一声，没有再说话。
<br/>　　刚欲转过身离去，打算去闭关室修炼时。
<br/>　　忽然，远方，一道澎湃气息，疾速而来。
<br/>　　气息之强，给人一股极致压迫感。
<br/>　　不，甚至说，萧逸单单远远地感知了一下，已觉浑身无力。
<br/>　　他甚至觉得，自己在这股气势面前，简直渺小得如同一只蚂蚁。
<br/>　　不过数秒，两道人影，从天而降。
<br/>　　来人，是两个老者。
<br/>　　其中一人，萧逸认得，正是安云郡的分殿主。
<br/>　　上次安云郡兽潮时，萧逸出手帮过，自是认得。
<br/>　　不过安云郡武者力量不强，故这位分殿主，实力也不太强。
<br/>　　倒是另一位老者，萧逸不认识。
<br/>　　刚才那股恐怖气势，便是由他身上散发而出。
<br/>　　如今近距离面对，那股感觉更为明显，甚至是可怕。
<br/>　　这到底是怎样一个武者。
<br/>　　表面看起来，平平无奇。
<br/>　　内里，却深不可测。
<br/>　　对，这是面前老者给萧逸的第一感觉，深不可测。
<br/>　　那苍老的双眸中，带着深邃，以及智慧的光芒。
<br/>　　“敢问这位前辈是？”萧逸拱了拱手。
<br/>　　他没发现，在老者到来后，血雾谷谷主的表情，早已变了。
<br/>　　从之前的霸道，变为如今的忌惮。
<br/>　　分殿主笑笑，而后对着老者行了一礼，道，“参见殿主。”
<br/>　　“什么？”萧逸闻言，先是一愣，随后一惊，甚至脸色大变。
<br/>　　三十六郡，各郡殿主，皆被称为分殿主。
<br/>　　整个炎武王国，能直接称呼为‘殿主’的，只有一人。
<br/>　　正是王都主殿的那位殿主。
<br/>　　“参见殿主。”萧逸连忙行了一礼。
<br/>　　老者，也就是王都殿主，笑了笑，打量了萧逸几眼，满意地笑笑。
<br/>　　“你便是易霄吧，果真是少年天才，后生可畏。”殿主说道。
<br/>　　“殿主过奖了。”萧逸回答道。
<br/>　　“不必谦虚，走吧，我带你离开。”殿主说着，转身出了猎妖殿。
<br/>　　随后，深邃的目光，看向血雾谷谷主。
<br/>　　“血谷主，老夫带他走，你可有意见？”殿主直视血谷主。
<br/>　　语气，很是平淡。
<br/>　　但，血谷主，已然身躯一震。
<br/>　　“不敢。”血谷主拱了拱手，道，“只是…”
<br/>　　“易霄此子，与我血雾谷有大仇，且是私人恩怨。”
<br/>　　“殿主您，无权干涉吧。”
<br/>　　殿主脸色淡然，道，“私人恩怨？”
<br/>　　“大半年前，你二子血无伤在北山郡霸道行事。”
<br/>　　“更当众折磨了一位分殿主。”
<br/>　　“三十六分殿主，皆由我任命。”
<br/>　　“我是否也要将其当作私人恩怨，为北山郡分殿主出头，与你计较个清楚？”
<br/>　　殿主的语气，一直很平淡。
<br/>　　但血雾谷谷主，却已脸色一变，额间，甚至出现了几滴汗水。
