　　百年时间匆匆而过，带走的不仅仅是那逝去的青春，还有那无尽生灵的生命。
<br/>　　三族大战虽然暂时结束了，但仍然会爆发一些不小的战斗。三族都在蛰伏，等待着哪一天一出就震惊天下。
<br/>　　大战之后无边的血液渗进大地，仿佛冥冥中得到指引一般，全部慢慢地向着血海汇聚而去。一点一滴地壮大着无边血海。
<br/>　　在血海的最深处，一个身穿血袍，满头血色长发，脸色苍白，有几分英俊的脸庞的青年男子盘坐在一火红色的巨大莲台之上，莲台上散发着黑色火焰，将四周带有阴冷污浊血气的灵气过滤，慢慢渗透到莲座之***那血袍青年吸收炼化。
<br/>　　血袍青年脚下是十二品业火红莲，他头顶一枚蓝色小旗在缓缓旋转，散发出滴滴幽蓝色水汽，在血光之下耀耀生光。
<br/>　　他的怀中抱着两柄杀道宝剑，一柄名为阿鼻，另一柄名为元屠。
<br/>　　两柄杀剑散发出无边杀气，杀人、杀仙、杀天、杀地、杀神，无物不可杀，杀气惊天动地。
<br/>　　突然，血袍青年睁开眼睛，眼中闪过血色光芒，厉声道：
<br/>　　“怎么停了，怎么不打了？！混蛋，你们打得越欢，血液流进幽冥血海的就越多，我修炼就越快。但现在怎么不打了。哼！你们早晚会再次打起来的。”
<br/>　　……
<br/>　　不周山之大，无法形容，横跨数千亿里，即使麒麟一族以不周山为根基，也只是占据灵气相对浓郁的一部分山脉罢了。
<br/>　　不周山上不知有多少大神通者在此潜修，麒麟族虽然现在势大，但明显不想将众多强者得罪太惨，况且也没那个必要，洪荒地大物博，三族瓜分也绰绰有余了。
<br/>　　此时，不周山的一个巨大无比的神秘山洞之中，
<br/>　　砰！砰！砰！
<br/>　　十二颗巨大的心脏声不断地响起，并且有越来越快的趋势，要不是洞府口有一道淡淡的金色禁制屏蔽了这声音，恐怕洞府方圆百里都能听到这巨大的响声。
<br/>　　洞府之中，十二个万丈巨大的血茧以一种奇怪的阵势排列着，一阵阵苍茫荒古的气息散发出来，砰！砰！砰！的心跳声传出，似乎孕育着十二个远古凶兽，从其散发出来的气息来看，每一道都有着金仙的修为。
<br/>　　无边的煞气从四周蜂拥而来，注入到这十二个巨大血茧之中，使得他们的气息更加狂暴深厚，想来，要不了多久，他们就会化形而出。
<br/>　　……
<br/>　　东海之滨，永恒仙岛，玉虚宫中，演道场上。
<br/>　　元始三人具皆从深层领悟之中醒来，元始本已经有了灵魂道的基础，自从当年觉醒灵魂之火，再到后来参悟元始印中的灵魂法则，他对灵魂一道已是很熟悉了，现如今再次有了灵魂一道的神纹道则碎片，可以说这只是锦上添花而已。
<br/>　　但，就算如此，他对灵魂一道的领悟也增强了不少，可以明显感受得到，元始的灵魂之火燃烧更旺了，灵魂意志也更为强大了。
<br/>　　灵魂可不是元神，灵魂是灵魂，元神是元神，这是两种不同的概念。灵魂人一生下来就有，而元神却是要随着修道者修为高深才会出现。有的人灵魂一生变化不大，但有的人却是灵魂天生强大。
<br/>　　灵魂的强大与否虽然与修为的高强也有关系，因为随着修为的增强，灵魂也会相应地随着增加，但这种增加的程度很微小，几乎没有。而元神却是会随着修为的增强而增大，修为越高，元神就会越强。
<br/>　　灵魂一道修的是意志，修的是内心，修的是不屈，只要是有灵的东西，他就有灵魂。灵魂修炼的是自身。而元神修的是天地，修的是自然，修的是道，参悟天地大道，壮大己身元神。
<br/>　　灵魂强大之人，意志也必然强大。就如同十二祖巫，他们就没有元神，不修天地大道，但，他们有灵魂，他们修炼的是自身，以自身之力，破除万法。
<br/>　　元始当然意志也十分惊人，毕竟经历了这么多，想不强大都不行。
<br/>　　“如何？你们领悟的如何？”元始对着老子通天问道。
<br/>　　“大兄，我领悟的是火之一道，还算可以。”老子面无表情道，火之一道，他本身就有所领悟，现在不过是锦上添花罢了。
<br/>　　“我的是金之一道，我已经领悟到法则初级阶段了。”通天道。
<br/>　　“好，都不错，金火都是五行之道，五行乃基础，都十分不错。既然你们都有领悟，那么我们就开始交流道法好了。”元始对着两人说道。
<br/>　　“先从我开始好了。”
<br/>　　元始盘腿坐下，只见他背后氤氲升腾，色泽幽暗，玄妙异常，紧接着泥丸大开，气冲霄汉，与氤氲交织成一朵混混沌沌的庆云，丈许大小，却有笼罩万方之势。
<br/>　　庆云似虚似实，似大似小，带有无极混沌之意。
<br/>　　只见元始周身升腾起了黑白赤青黄五色，黑暗幽深，白色锋利，赤色热烈、青色生机、黄色厚重。五色轮转，甲乙化青莲，丙丁化红莲，戊己化黄莲，庚辛化金莲，壬葵化水莲，五行相生相克，生生不息，造化不休。
<br/>　　轰隆！
<br/>　　五色坍塌，出现黄蓝红青四色，黄色包容，蓝色咆哮，青色狂暴，红色炙热，地水火风肆掠，天地倒塌，三界动荡。
<br/>　　四色之后再次化为黑白两色，黑色幽暗、冰冷、寂静、至阴，蕴藏着深深的死亡之意，白色光明、温暖、蓬勃、纯阳，散发着浓浓的生之气息，两者盘绕交杂，汇成了一条首尾相连的阴阳鱼，死意中孕育出一线生机，生气里演绎出最终的死亡。
<br/>　　轰隆！
<br/>　　阴阳太极图成形，引来无穷高处莫名鼓荡，像是诸天万界的阴阳生死之道有了某些方面的共鸣。
<br/>　　气机牵引，大道互感！
<br/>　　阴阳鱼环绕盘旋，飞快落入了太上无极元始庆云之中，刹那之间，混混沌沌的庆云像是有了微妙的分化，散发出最初和最终两种意味，可又似乎依旧浑然一体，难以分割。
<br/>　　最初与最终统一的无极之意刚现，元始周身便再生变化，地水火风肆掠，继而形成一颗古老之心，膨胀收缩，化为金莲，金芒迸发，天地倾倒，三界覆灭。
<br/>　　紧跟着周身放光，毕方腾飞，五德成凤，七十二般变化一一展现，伴随着雷霆汪洋，传出道印，于玉虚宫方寸之间簇拥着元始盘坐的身影，让他仿佛诸天至尊。
<br/>　　又是渺小，又是宏大！
<br/>　　最后，虚空浮现，一个个黑点凸显，汇聚成线条，交织为面，重叠成天地，充斥着时光，演绎出玄奇奥妙的层次结构。
<br/>　　因果之线，笼罩天地！
