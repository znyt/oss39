<br/>　　这每一滴的远古战龙精血，都是呈现出了一种耀眼的金色，每一滴都蕴含有着极为庞大的力量。

<br/>　　对抗生死境巅峰的强者，都不成问题。
<br/>　　同时他的剑术，更是得到了极大的升华，那五十道无上剑气的力量，已经是全部融入到了楚风眠的九域剑术之中去。
<br/>　　无上剑气，是真正的至宝，曾经远古宗门上剑宗的立宗之本。
<br/>　　可以说当时的上剑宗，正是以这无上剑气，才成为了远古时代的剑道宗门。
<br/>　　这种宝物，都被楚风眠全部得到，融入到了他的身躯之中去，楚风眠这一次所得到的好处，只怕是不亚于在远古龙帝洞府之中的收获。

<br/>　　楚风眠的身上，一股凌厉的剑意，爆发而出，同时在他的身边，也是一道道的剑光，围绕在楚风眠的身躯周围。

<br/>　　“这归墟天中，既然是存在有着无上剑气，便是不止是只有这么少的一些，不过大部分的无上剑气，依然是留在了归墟大陆之中，看来是曾经被一些远古宗门得到！在那些远古宗门的遗迹之中去！”
<br/>　　楚风眠一双目光，看向就在这远方，归墟天中心的巨大无比大陆。
<br/>　　归墟大陆，归墟天的真正中心。
<br/>　　曾经无数远古宗门林立的地方，那些远古宗门的遗迹，全部都在其中。
<br/>　　楚风眠已经是感觉得到，在这归墟天中，还是有着无上剑气的存在，不过这些无上剑气，都已经不在这周围了。
<br/>　　这周围应该一共就只有这五十道无上剑气，现在已经是全部落在到了楚风眠的手中。

<br/>　　归墟天，是一处真正的拥有无数宝藏之地，但是九域之中的任何武者，都是只能够进入其中一次。
<br/>　　虽然说归墟天，其实并没有这个规则，但是实际上现在归墟天，可是由整个九域中的无数宗门一起掌管。
<br/>　　尤其是七大宗门，他们制定下来的这个规则。
<br/>　　以现在的楚风眠，绝对是无法挑战的，将来除非是楚风眠真的是足以做到跟七大宗门，平起平坐的地步，自然不用在乎这一切，随意进入其中。
<br/>　　但是现在的楚风眠，显然还是远远做不到这一点，所以这次归墟天之行，其他宝藏，楚风眠可以少得到一些。
<br/>　　这无上剑气，楚风眠必须是要尽可能多的收取。

<br/>　　想到这里，楚风眠便是毫不犹豫，他的身形，顿时便是化为了一道遁光，从这沙漠之中直接飞了出去。

<br/>　　楚风眠的身形，连连是从虚空之中闪动，很快便是离开了这一方区域，直接便是一道遁光，向着归墟大陆之中飞了出去。

<br/>　　楚风眠心神一动，便是不再去想。
<br/>　　归墟天中，一共只有一个月的时间，楚风眠为了度过生死大劫，提升境界，都是花费了四五天的时间。
<br/>　　如今所剩的时间可已经不多，除非是楚风眠打算要在这归墟天之中，一直待上三年时间，不然的话在这所剩的二十五天之内，楚风眠必须要尽可能多的收集无上剑气。
