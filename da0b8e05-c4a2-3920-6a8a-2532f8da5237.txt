　　第666章
<br/>　　柳芷好不容易从书房里撤出来，看外面炉火上的热汤。骨与肉已经熬得酥烂，掀开锅盖的时候冒出一阵白烟，熏得柳芷眼睛都睁不开。
<br/>　　如此，倒是掩饰了她的心慌意乱，面红耳赤。她用勺子将香料筛出，将热汤盛出来。心中却是不停的责怪自己：原本就想着找个机会要个名份，不然日子一天天过下去，自己没有个正经名份，就只能随便找个仆人嫁了。
<br/>　　刚才多好的机会，怎地不知顺水推舟，投怀送抱。
<br/>　　柳芷有自己的小心思，程大雷窝在书房内，感觉也不是如何好受。这时候，柳芷已经端着汤进来，又分到细瓷碗中。
<br/>　　汤熬得极香，已经泛起白色。不过程大雷此刻的心思不在这些肉食上，早已飘到了别处，却是可惜了这一锅好汤。
<br/>　　“坐下一起吃些。”
<br/>　　程大雷坐在桌前，柳芷仍立在一旁，当然，从前许多次都是这样。
<br/>　　“大当家坐着，那有我坐的份，我在这里伺候着就好。”
<br/>　　“嗳，哪有那么多规矩，从前在青牛山不也是很随便的么。”
<br/>　　“现在不是从前，这里也不是青牛山了。”
<br/>　　柳芷轻轻道了一声，心里也有些酸楚。现在的确不是从前，程大雷的手下人越来越多，蛤蟆城的势力也越来越强，可还是感觉自己和程大雷的距离越来越远。以前还当自己是身边的人，如今就只是个端茶递水的丫鬟了。
<br/>　　程大雷同样长叹一口气，倒是没有柳芷如此多的感慨。大敌当前，他本该全神贯注，但这一锅肉香还是分走了自己的注意力。也许和珅这混蛋说得不错，自己做人不要太拘着。但可恼的是，和珅并没有告诉自己怎么不拘着。
<br/>　　他吃得很慢，中间与柳芷也没多少话说，二人同处一室，但却又各自默不作声。
<br/>　　心里翻来覆去，程大雷想着怎么打破僵局。他从青牛山走到蛤蟆城，过程中经历许多事，如今论武艺已称得上当世无双，谈到打仗程大雷也懂一些，有统领一支队伍的能力。然而，在男女之事上，程大雷终究没多少经验，如果按系统评级的话，至多属于糟糕，连普通都谈不上。
<br/>　　碗中的汤尽了，柳芷再不动声色与他续上，程大雷虽然再三让她坐下，但她坚持拒绝，自己摆明身份就是个伺候的丫鬟。
<br/>　　程大雷也是无奈，这样熬着时间终究不是个办法。他将碗在桌子上一搁，道：“不吃了。”
<br/>　　二人都有些冒汗，或者是因为热，或者是因为其他缘故。听到程大雷这话，柳芷也立刻松口气，将桌上的碗筷收拾起来。
<br/>　　“大当家您早歇着，我就在外面，您有事唤我就好。”
<br/>　　程大雷一颗心七上八下，百爪挠心，当真是不知如何是好。最后还是想起和珅的话，这种事本不需要考虑太多，也不值得自己考虑太多。开口直说便可。
<br/>　　“嗯。”程大雷冷着脸：“今天你就歇在这里。”
<br/>　　柳芷捧着碗筷正要离去，听到这话忽然顿住，程大雷看着她的背影，心中咯噔一下，这姑娘该不会当场翻脸吧。
<br/>　　然后，就听到她用细弱蚊蝇般的声音轻轻『嗯』了一声。
<br/>　　柳芷慌张无比，但假若她回头看一眼，就会发现程大雷比她更加慌张，也不过是一头纸老虎而已。
<br/>　　等柳芷去而复返，再次进来时，程大雷躺在炕上假寐，眯缝着眼睛偷看柳芷。见柳芷面色潮红，羞得抬不起头，耳边听见悉悉索索的声音，她将外面的襦裙解了，木钗拔下，流出一卷长发，之后垫着脚尖吹灭油灯，摸黑爬上了炕。
<br/>　　黑暗中，感觉有一支只粗糙的手握住自己的手腕，柳芷全身绷紧，她并不缺少和程大雷同处一室的经验，但今时不同往日，程大雷毕竟已不打算做个君子。
<br/>　　朦胧中感觉程大雷将自己拉近怀里，柳芷浑身燥意，觉得胸口有团火焰，呼之欲出。
<br/>　　她嘤咛一声，口中糯糯的声音道：“大当家……”
<br/>　　程大雷堵住了她的嘴唇，令她说不出话来。
<br/>　　……
<br/>　　次日天明，响晴薄日。
<br/>　　程大雷睡得较晚，醒来的时间也不会走。当他从书房内醒来，看到自己身侧的床铺收拾得干干净净，已不见了柳芷的踪影。
<br/>　　刚打了个哈欠，从炕上坐起，柳芷便从外面进来。从她脸上倒是看不出如何，只是两晒见红。手里托着铜盆，胳膊上搭着毛巾。
<br/>　　“我伺候大当家洗漱。”
<br/>　　程大雷答应一声，却也不知该如何面对今日的关系。只好虚伪的保持正经的面目，柳芷在一旁伺候，等程大雷洗脸漱口之后，再伺候他更衣。
<br/>　　裹上皂色长袍，登上牛皮快靴，整个人也显得精神几分。过程中，柳芷只是细心服侍，却也没有多说一句话，只是给人的感觉轻快几分，不像几日前那样幽怨。
<br/>　　“大当家，昨晚你说要将夫人接回来的。”柳芷细心提醒。
<br/>　　“嗯嗯。”程大雷点点头，打算将这件事交给龙字军的人办。
<br/>　　如今的局势的确有些紧张，还是让苏樱和樊梨花住在蛤蟆城安心。就算金丝雀要踏出牢笼，也不是现在这个时候。
<br/>　　程大雷的确有事要做，今日要去看看军策府讨论得如何，的确没有太多时间耽搁在柳芷身上，考虑二人此后的关系。
<br/>　　而柳芷的反应就像是什么事情都没有发生过，宛若他就像是程大雷的私人财物，程大雷予取予夺。
<br/>　　程大雷不仅有些吃惊，心里下意识冒出个念头：仅仅是如此么。
<br/>　　紧跟着又冒出下一个念头：原来如此么。
<br/>　　第一次，程大雷感受到权力的甘甜。从青牛山走到蛤蟆城，他此身并不干净，江湖上流传的恶名，也不仅仅是谣传。他杀过很多人，所做的事也并不是完全正确，但有一点程大雷从没变。
<br/>　　他从不曾觉得自己高人一等，一直当自己是个普普通通的小人物。
<br/>　　然而，事实早已不是如此。
<br/>　　如今的他，已经一言定人祸福，一言定人荣辱。
<br/>(本章完)
