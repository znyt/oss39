　　“嗯，新晋弟子！”苏莫点了点头，随即将自己的身份牌，递给了郑长老。
<br/>　　郑长老接过身份牌，稍一查探，便颌首道：“武学、宝器、灵晶，三种奖励，你可以任选一样！”
<br/>　　“我选灵晶！”苏莫沉声说道，他也没有仔细询问，是什么级别的武学或者宝器，直接选择了灵晶。
<br/>　　因为他不需要武学和宝器，只需要灵晶。
<br/>　　“这是一万下品灵晶！”郑长老取出一枚储物戒，交给了苏莫。
<br/>　　苏莫闻言讶然，一万下品灵晶，虽然对于他用处不大，但这个奖励已经颇为不小了。
<br/>　　“多谢长老！”收起储物戒，苏莫向对方抱了抱拳，便转身向外走去。
<br/>　　刚走了几步，苏莫又突然停了下来，转头望向吕罡，冷声道：“若是再让我听到你胡言乱语，我割了你的舌头。”
<br/>　　言罢，苏莫大步离开了宗务殿。
<br/>　　“找死！”听到苏莫的话，吕罡面上怒火汹涌，脾气本就暴躁的他，差点就忍不住出手了。
<br/>　　割他的舌头？
<br/>　　这真是天底下最大的笑话！
<br/>　　一个武帝境八重的武者，居然敢在他面前如此猖狂？这简直是对他的藐视！
<br/>　　若不是在宗务殿，若不是有长老在此，他一定要让苏莫遭受惨痛的教训。
<br/>　　吕罡眸中寒芒涌动，改日再找机会，收拾这个苏莫，不然难消心头怒火。
<br/>　　宗务殿中的其他弟子，亦是暗暗咂舌，这位刚加入宗门的苏莫，真是胆大包天啊！
<br/>　　难道此人不知道吕罡的名头，不然哪里敢如此狂妄！
<br/>　　吕罡的实力，在道宗核心弟子之中，可是顶尖的存在，整个道宗的核心弟子之中，实力能超过吕罡的人，不会超过一手之数。
<br/>　　不少人暗暗摇头，这个苏莫得罪了吕罡，怕是以后要吃苦头了。
<br/>　　苏莫离开宗务殿之后，并未回自己的院落，而是向冯光长老的住所而去。
<br/>　　他要去找冯光，问一下关于会武的事情，之前听吕罡等人交谈，说是要去太阴族参加会武。
<br/>　　对于任何能去太阴族的机会，他都不会放过。
<br/>　　冯光的住处，苏莫不清楚，但随便拦住一名弟子询问，他便知道了。
<br/>　　一座雅致的小型宫殿之前，苏莫走到大门前，敲了敲殿门，朗声喊道：“冯长老，苏莫求见！”
<br/>　　宫殿中寂静无声，约莫过去了十几个呼吸之后，殿门打开了，一身道袍，大腹便便的冯光面色有些难看。
<br/>　　在冯光的身边，还有一名女子，年约二十岁许，容貌娇美。
<br/>　　苏莫见此神情一怔，他发现这名女子有些衣裳不整，耳根绯红，呼吸微微急促，面色极不自然。
<br/>　　“卧槽！”苏莫无语了，他心中顿时有十万只***奔腾而过，这冯光长老居然与这名女弟子有.染。
<br/>　　这让苏莫大开了眼界，堂堂的道宗长老，穿着一声道袍，居然白日宣yin，和门中女弟子在宫殿中行苟且之事。
<br/>　　“苏莫啊！你怎么来了？”冯光轻咳一声，故作威严的问道。
<br/>　　“冯长老，我有些事想向你请教！”苏莫沉声说道，他面色丝毫不变，仿佛没有看到那名女弟子一般。
<br/>　　对于冯光的事，他不想多问，一切都与他无关。
<br/>　　“嗯！”
<br/>　　冯光闻言点了点头，随即转头看向身边的女弟子，道：“本长老传授与你的武道知识，你都记住了吗？”
<br/>　　“都记住了！”女子声如黄雀，轻灵悦耳。
<br/>　　“好了，你回去自行参悟吧！”冯光摆了摆手，随后那名女子便告辞离去。
<br/>　　苏莫冷眼看着冯光的表演，心中暗笑，传授武道知识？他可不会相信，当然，他也不会拆穿对方。
<br/>　　随后，冯光将苏莫请进了宫殿中，两人相对而坐。
<br/>　　“苏莫，你有何事，尽管问吧！只要本长老知道的，绝对不会隐瞒！”冯光轻笑着说道，虽然他心中有些恼火苏莫坏了他的好事，但他知道苏莫有真贤道圣这个后台，也不敢得罪。
<br/>　　“冯长老，我听说道宗要举行会武，不知是什么会武？”苏莫沉声问道。
<br/>　　“哦，你说会武啊，你想要参加？”冯光无奈的问道，他还以为是什么重要的事呢！原来只是询问会武之事。
<br/>　　“我只是问一下，先了解一番。”苏莫道。
<br/>　　冯光闻言，略一沉吟，道：“这会武是七大势力核心弟子之间的较技，若是能进入前十名，太阴族会给予丰厚的奖励！”
<br/>　　“会武的地点是在太阴族吗？”苏莫急忙问道，奖励他不在意，他在意的是在哪里举行！
<br/>　　“嗯，这是由太阴族举办的会武，在太阴族之内进行。”冯光点了点头。
<br/>　　苏莫闻言暗喜，这是一个好机会，进入太阴族的好机会。
<br/>　　虽说他若是进入太阴族，众目睽睽之下，不太可能做什么，但最起码有很大的机会和青璇见面。
<br/>　　“会武的时间在什么时候？”苏莫再次问道。
<br/>　　“下月初十！”冯光道。
<br/>　　苏莫闻言，算了一下日子，距现在还有一个月出头的时间。
<br/>　　他心中已经决定了，去参加这个会武，去太阴族走一趟。

<br/>　　“嗯，这需要报名吗？”苏莫点了点头，又开口问道。
<br/>　　“你就算了吧！”
<br/>　　冯光闻言却是摆了摆手，告诫道：“这种会武是小规模的较技，七大势力之中，都只会挑选武帝境九重的核心弟子参加，你才武帝境八重，没有资格去参加的！”
<br/>　　冯光也不怕打击苏莫，直言他没有资格参加，他知道苏莫刚进入宗门，可能是急于表现自己。
<br/>　　“这不是还有一个多月的时间吗？说不定我就突破了呢！”苏莫微笑着说道。
<br/>　　虽然他没有把握在这一个多月的时间中突破到武帝境九重境界，但以他目前的实力，就算不突破修为也有资格参加。

<br/>　　真贤道圣知道他的实力，想必不会拒绝，而且以对方的身份和地位，安排这种事只是一句话的事情。
<br/>　　“突破？哪有那么容易！”冯光摇了摇头，肥胖的脸皮抖了抖，修为到了武帝境高阶，哪里还是说突破就能突破的！
<br/>　　就算是最绝顶的天才，到了武帝境高阶的地步，都要经过长时间的沉淀，才有可能进阶。
