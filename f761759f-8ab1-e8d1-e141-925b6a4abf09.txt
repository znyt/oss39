　　一瞬间，苏莫陷入了绝境之中。
<br/>　　眼看着巨大的刀影急速斩来，他咬了咬牙，不动用一切底牌，看来是不行了。
<br/>　　嗡~~
<br/>　　突然之间，苏莫的身上发出刺目的白光，耀眼无比，将整个战界都照的一片通透。
<br/>　　他整个人，犹如一团光辉倾洒万界的烈日，神威无比。
<br/>　　不仅如此，他的背后，同样升起了一轮更大的烈日，与他的身体相辅相成，彼此呼应。
<br/>　　这一刻，苏莫不仅催动了大日神体，更是催动了太阳武魂。
<br/>　　他的大日神体，在当初斩杀了上官昊，将对方吞噬干净之后，已经继承了对方所有的神体血脉，威力大增。
<br/>　　只不过，遗憾的是，苏莫并没有修炼契合这种神体的武学，无法发挥出大日神体的真正威力。
<br/>　　不过，即便如此，大日神体也是非常强大的一种体质，并且随着修为的提升，越来越强大。
<br/>　　随着苏莫动用大日神体和太阳武魂，光芒照耀八方，伴随而来的便是恐怖的高温。
<br/>　　光芒照耀之处，融化天地万物。
<br/>　　强大的大日领域，笼罩苏莫周身方圆千里。
<br/>　　千里之内的空间，瞬间焚化成了虚无，下方的海水，瞬间蒸发成了雾气，露出了海底。
<br/>　　苏莫身上的束缚之力，也因为这恐怖的高温，而变得非常孱弱。
<br/>　　巨大的刀影，遭到大日领域的影响，都冒出阵阵的白烟。
<br/>　　不过，这刀影着实强大，威力并没有减弱太多，依旧向苏莫当头斩来。
<br/>　　“给我碎！”
<br/>　　苏莫厉喝一声，不退反进，身形冲天而起，同样双手握剑，狠狠的斩向巨大的刀影。
<br/>　　体内玄力如长江大河，配合上大日神体和太阳武魂，苏莫的气势一时无两。
<br/>　　咻！
<br/>　　剑光逆斩而上，瞬间和刀影碰撞在了一起。
<br/>　　轰！
<br/>　　霎时之间，一声爆响整动天地，巨大的刀影直接被苏莫一剑斩暴，四散崩灭。
<br/>　　嗖！
<br/>　　苏莫受到了巨大的反震之力，顿时向下方坠.落，身形坠.落数万丈，才堪堪止住了身形。
<br/>　　呼！
<br/>　　苏莫轻轻呼了口气，他全身的血液沸腾，筋骨酸麻，险些被震出了内伤。
<br/>　　不过，他的眼眸却也是闪亮无比，武帝境三重，也不是不强嘛？他完全有胜利的希望，因为他还有很多底牌。
<br/>　　“这……这怎么可能？”
<br/>　　高空之上的冷江，有些傻眼了，他双眼瞪的老大，仿佛不敢相信自己看到的一切。
<br/>　　苏莫的太阳武魂，他没有在意，这种武魂虽然特殊，但也要看等级。
<br/>　　他震惊的是，苏莫居然拥有特殊体质。
<br/>　　这完全出乎了他的预料。
<br/>　　这一刻，整个战界都安静了下来，只剩下了极远之处海浪拍击的声音。
<br/>　　少倾之后，十几座观战的小岛之上，顿时惊呼声四起。
<br/>　　“天啊！这个苏莫居然拥有特殊体质！”
<br/>　　“不可思议，居然是拥有特殊体质的妖孽！”
<br/>　　“这是什么体质？是烈日圣体？还是大日神体？亦或是九阳之体？”
<br/>　　“厉害啊！难怪如此修为敢和冷江一战，原来有如此底牌！”
<br/>　　所以人都震惊了，特殊体质，这可是上天的宠儿，天之骄子。
<br/>　　整个神武学府的弟子中，拥有特殊体质的人，也不足两手之数，可见特殊体质的给予众人的冲击是多么的强烈。
<br/>　　聂乾和韩盈两人，张大了嘴.巴，简直是目瞪口呆。
<br/>　　两人思维几乎停歇，他们做梦都没有想到，苏莫居然是拥有特殊体质的天才。
<br/>　　同样震惊的还有木璃和太史轩，尤其是木璃，她心中的震惊，完全不比韩盈两人弱。
<br/>　　苏莫有超强的精神力，不仅炼器天赋超强，居然还是特殊体质的武道妖孽，这未免太夸张了吧！
<br/>　　难道苏莫，真的是上天的宠儿！
<br/>　　嗖！
<br/>　　苏莫身形冲天而起，飞上了高空，和冷江相对而立。
<br/>　　“冷江，还有什么绝招，都使出来吧！“苏莫淡淡的说道。
<br/>　　冷江闻言，脸色沉了下来，几乎是阴沉如水，因为他知道，今日很难杀死苏莫了。
<br/>　　因为苏莫如此的天赋，肯定有人不希望苏莫死。
<br/>　　他是赤霞洞天派系的人，其他派系的人，肯定会拉拢苏莫。
<br/>　　“你这是什么体质？”冷江沉声问道。
<br/>　　“大日神体！”苏莫淡淡的说道，对于这神体他了解的也不多，只是上官昊称这种体质为大日神体，那就是大日神体。
<br/>　　“大日神体？”冷江闻言，顿时眼眸一凝。
<br/>　　在太古异体榜上，上榜的一千种特殊体质之中，大日神体排名七百六十二名！
<br/>　　其他观战的人闻言，均是心中恍然，原来是大日神体！
<br/>　　这种体质虽然很强，但比九阳之体和烈日圣体，还是有很大的差距。
<br/>　　“既然你不出手，那我便出手了！”苏莫见对方迟迟不出手，顿时冷笑一声。
<br/>　　嗡~~~
<br/>　　下一刻，他手中的长剑顿时嗡嗡震颤了起来，其内灌注的玄力，浑厚到了难以想象的地步。
<br/>　　“大苍穹剑法——日月转！”
<br/>　　厉喝一声，苏莫手中长剑一挥，一道万丈剑气，便向冷江袭杀而去。
<br/>　　这一剑，苏莫暗暗催动了体内几种强大的剑武魂，让得这一剑威力大增，锋芒无尽。
<br/>　　咻！
<br/>　　万丈剑气撕裂长空，剑气霸道森然又玄妙无比，仿佛乾坤颠覆，日月倒转。
<br/>　　“即便你有神体，也一样要败！”冷江见此，顿时怒喝一声，心中彻底发了狠。
<br/>　　他堂堂武帝境三重武者，若是打不败苏莫，那真的是无脸见人了！
<br/>　　他的背后金光耀眼，升腾而起，出现了一柄三丈长的金色战刀。
<br/>　　这是他的战魂，金刀战魂！
<br/>　　战魂一出，冷江身上的刀威暴冲而起，冲破九霄，浩荡八荒。
<br/>　　虚空直接被凌厉的锋芒撕开，出现了一个长达数万丈的恐怖裂缝。
<br/>　　“给我破！”
<br/>　　双手握刀，冷江沟通了战魂之力，一刀横斩而出。
<br/>　　这一刀，为囚天二式！
<br/>　　巨大的刀影再次出现，如天刀现世，横贯长空，威势无敌。
<br/>　　由于动用了战魂，冷江这一刀比囚天一式，强大了足有五成左右。
<br/>　　瞬息之间，横贯长空的刀影，便和万丈剑气相击在了一起。
<br/>　　轰！
<br/>　　一声爆响，万丈剑气立刻崩碎，化为万千剑雨，激.射八方。
<br/>　　刀影着实强大异常，犹如真正的神刀，斩碎万丈剑气之后，威势依旧强大，瞬息向苏莫袭杀而去。
