　　写字楼，指挥中心。
<br/>　　通过网络直播，龙隐部队众人收看了这次新闻发布会的全程直播，当看到K国国防部长给出这样的解释时，慕容飞顿时就被气乐了，“嚯，这么不要脸的借口他们也说得出来？真是脸皮比城墙还厚，还有你看他拿着手帕擦眼泪，整的跟真的似得，那手帕上涂过洋葱汁了吧！这货不去当演员实在是好莱坞的损失，他要是去当演员，妥妥拿着小金人回家，继续让莱昂纳多陪跑奥斯卡啊！”
<br/>　　“K国这次要做的不仅是对地铁站爆炸案的解释，还有要维护国家形象，军队形象。”
<br/>　　李然靠在椅子上，看着还在进行的新闻发布会，说道：“如果让民众知道，昨天晚上的爆炸案是因为他们军队狙击手自负造成的，会让军队形象一落千丈，导致民众对这个国家的军队不再信任，这对于K国的国防力量将会是一个致命打击。政客都是天生的演员，脸皮不厚怎么做政客？”
<br/>　　“不过，如果只是这样的话还不够。”李然窝在椅子里，动了下身体，换了个比较舒服的姿势说道：“如果要让民众继续对这个国家保持信任，他们还需要有人站出来承担责任。”
<br/>　　李然的推测很快就得到了印证。
<br/>　　仅仅只是隔了一天，K国最高法院就对地铁爆炸案中的几个主要责任人进行庭审，而这几个主要责任人除了直接导致爆炸案发生的狙击手外，还有狙击手的观察员，以及现场监视小组组长。
<br/>　　事实上，真正的责任人只有一个，但显然这一个人是很难抚平民众愤怒的。
<br/>　　法庭上，现场监视小组组长被认定为指挥不当，被判处5年监禁；观察员作为最靠近狙击手的人，被认定为在行动过程中工作疏忽，被判处10年监禁；至于肇事狙击手这个直接导致地铁爆炸案发生的责任人，被判处105年监禁，最后应狙击手要求改判死刑，立即执行。
<br/>　　现场监视小组组长和观察手无疑是“陪跑”的，他们在法庭上的说辞也应该是在那天晚上被调查小组带走以后，就已经准备好的。毫无疑问，他们为了国家军队的形象做出牺牲，他们的家人将会得到特别的照顾，享受特殊福利，监视小组组长和观察手在狱中的生活也不会难过，更甚至于，他们或许根本不会进监狱，而是带着家人直接消失，去一个谁都不认识他们的地方生活。
<br/>　　K国国防部长也是参加这次庭审，在庭审结束后，这位国防部长直接表示，他们不会向恐怖主义的暴行妥协他们会立即采取行动，向恐怖分子表明他们强硬的态度！
<br/>　　K国快速反应部队也的确是做出了反应，他们对全市所有地铁线路进行排查，并且在几条地铁线路中又发现了经过改装的反坦克地雷。这让K国总统悬着的心总算是放下了一点，这两天的努力总算是有了成效，也不算白费。
<br/>　　目标写字楼，那天晚上被子弹击中碎落在地上的玻璃碎片还静静地躺在地面上，并没有人来清扫。
<br/>　　写字楼内，智天使和影天使（SHADOW）正盯着桌子上的笔记本电脑，看着上面的庭审视频。智天使冷笑一声，说道：“真是一群好演员啊，说起谎话来一套接一套，就连我差点都相信了呢。”
<br/>　　影天使只是一笑，并没有答话。他当然知道，智天使最后那句话是对K国方面的嘲讽，这么低劣的谎话如果能让智天使相信的话，他就不是智天使了。
<br/>　　“既然你们演得如此认真，我不给你们捧捧场，又怎么说得过去呢？”智天使抬起头对影天使问道：“我要的东西都准备好了吗？”
<br/>　　影天使把一个U盘放在智天使面前，说道：“你要的东西都在这里面。”
<br/>　　“我们也差不多可以召开一个发布会了，顺便宣布一下我们的游戏规则。”智天使拿起U盘，拿在手里把玩着，说道：“只是我们的发布会没有这么记者捧场，会有点冷清啊。”
<br/>　　“只要有劲爆的内容，就算是再冷清的现场也会远远超过他们的作秀。”影天使盯着视频画面上，侃侃而谈的K国国防部长，冷声说道：“这些人的心都肮脏。”
<br/>　　就在这个时候，吴云推开门，脚步匆匆走了进来，对房间里的智天使和影天使说道：“K国的快速反应部队出动了，我们埋设的炸药被发现了好几处。”
<br/>　　“他们能全都找出来吗？”智天使抬眼扫了吴云一眼，说道：“我们故意埋了这么多东西，不就是为了给他们找的吗？再说了，他们找不出我们埋设的最重要的那个位置都是白费功夫，就算是能找出来，他们也拆不掉！撤退的工具准备好了吗？”
<br/>　　“全都准备好了。”吴云点头道。
<br/>　　智天使停止了把玩手中U盘的动作，竖起食指对吴云说道：“我们准备的那么多东西会让K国快速反应部队疲于奔命，人员分散，方便我们撤退，所以不需要大惊小怪，我从一开始就没打算引爆这些东西。记住，你现在是暗天使，是‘十二主天使’之一，遇事要冷静。你是我跟权天使带出来的，权天使虽然死了，但是你不能丢他的脸。你最开始因为你父亲的关系才加入我们，可是你的表现出乎我们的意料，上面对你很看重，明白吗？”
<br/>　　“是！”
<br/>　　吴云“啪”的一声，站直身体。
<br/>　　智天使满意的点了点头，从椅子上站起来，对影天使说道：“走吧，我们也该召开我们的发布会了，这潭水太平静了，我不喜欢这种平静，比起风和日丽的平静，我更喜欢乌云密布，狂风暴雨电闪雷鸣下的惊涛骇浪！”
<br/>　　“好戏要上演了。”影天使冷冷的一笑。
<br/>　　“智天使！”吴云从背后叫住智天使，犹豫了一下，最后还是开口问道：“如果，我说如果，那天晚上没有那个狙击手的一颗子弹，你还会引爆地铁站里那颗反坦克地雷吗？”
<br/>(本章完)
