　　一瞬间的心慌环绕在了江凌轩心头，恍然大悟中又无比后悔，原来如此，难怪怎么都找不到，原来王之暴君的目标是何培源！
<br/>　　何培源是猎王请来坐镇猎者联盟总部的长老，虽说一把年纪了也战力很低，但偏偏能让神秘家族的人都节节败退，这其中必然有着江凌轩不知道的辛秘，但却是猎者联盟得意中立存在的重要一环，他也没想到王之暴君会直接将目标放在了何培源身上。
<br/>　　“神秘家族指使的？”江凌轩瞬间想到了这一环，他瞳孔一缩后又自言自语道：“那么将猎王引走的那个任务发布者，也是神秘家族的人？”
<br/>　　如此一想，这件事中的种种疑惑就解开，只不过却知道的太晚了！
<br/>　　猎者联盟总部的大批高战力猎者一下子聚集，朝着禁区内何培源的休息室狂奔而去，江凌轩很清楚这些人加起来恐怕都不敌王之暴君一个人，但无论如何都要一试。
<br/>　　就在整个猎者联盟总部都陷入无比紧张的时刻之际，安静的禁区某个休息室房门被轻轻敲响。
<br/>　　咚咚！
<br/>　　叩门声很规律，显示出了敲门者的不慌不忙，亦或是势在必得？
<br/>　　“进来，门没锁。”何培源的声音响起，依旧宛如一个普通年长者般的慈祥。
<br/>　　吱呀——
<br/>　　门应声而开，穿着厚重斗篷的王之暴君踏步而入。
<br/>　　何培源就坐在正对着门的沙发上，他的面上带着一丝笑意，一只手抓着烟斗有一口没一口的抽着，桌上有两杯热腾腾的茶，散发着徐徐白气。
<br/>　　王之暴君进来后先是关上门，这才转身，当他看到何培源的这种状态时，忍不住发出一声带有轻蔑感情的笑声。
<br/>　　他望着桌上的两杯茶，道：“你这是知道我要来，还专门给我沏了一杯茶？”
<br/>　　何培源笑呵呵的摇头：“我不知道来的人是你。”
<br/>　　王之暴君的动作一顿：“你知道我是谁？”
<br/>　　何培源继续摇头：“不知道。”
<br/>　　王之暴君继续笑道：“那你胆子很大啊，猜不到我是来杀你的吗？”
<br/>　　这回何培源点了点头，面上的神情却不变：“这个我能猜到，从你进来开始，就有杀气环绕。”
<br/>　　王之暴君闻言不禁对眼前的人好奇起来，便坐在了何培源对面，道：“让我来杀你的人告诉我，你的家族是这世上唯一能克制神秘家族的存在？你是神秘家族成员最畏惧的人？”
<br/>　　何培源笑着摇头：“当然不是。”
<br/>　　王之暴君眉头一皱：“什么不是？你和你的家族无法克制神秘家族？”
<br/>　　何培源忍不住哈哈大笑：“我是说，不是唯一。”
<br/>　　王之暴君闻言一惊，而后愣住：“什么意思？”
<br/>　　何培源看着他，眼神有些深邃：“能克制神秘家族成员的一共有四，我和我的家族只是其中之一。”
<br/>　　王之暴君心跳加速：“这么多！？另外三家在哪里？他们很强？”
<br/>　　何培源笑道：“这我就不知道了，不过你不是来杀我的么？再不动手的话，可能就没有机会了。”
<br/>　　王之暴君好笑的看着何培源：“哪怕猎者联盟内的那些人猜到我的意图也没用，猎王走后这里我想杀的人，没人能阻止。”
<br/>　　就在王之暴君的话音还未落下之时，忽的‘吱呀’一声，这间屋子的房门再次被打开。
<br/>　　屋内顿时一静，背对着房门的王之暴君没有回头，而是在遮住容貌的斗篷内，露出了一个嘲讽的笑容：“我说了，就算有人来也没用……”
<br/>　　“让开。”只是还不等他的话说话，就被一个清冷的声音强势打断：“那是我的位置。”
<br/>　　王之暴君瞳孔剧烈收缩，猛然回头望去，紧接着下一秒，便整个人的心神都好似陷入深渊。
<br/>　　进来的人有两个，一男一女，女的浑身带着冰冷气息，约莫二十岁不到，男的则是一个只有十一二岁的少年。
<br/>　　只是这个少年，恰好就是王之暴君见过，并且被与他无关的一次打斗余波影响，导致重伤！
<br/>　　神秘家族白家的白羽！
<br/>　　哗！
<br/>　　冷汗一下子在王之暴君的后背冒出，浸·湿了衣衫，他腥红的瞳孔里满是恐惧。
<br/>　　倾巢大战时在银市的那条小巷里，白羽随手爆发而出的恐怖战斗力，他当今还历历在目！
<br/>　　“喂，让开，谁允许你坐我小姐姐的位置？”此时的白羽毫不客气的出声，冲着王之暴君呵斥。
<br/>　　王之暴君猛地回神，惊吓的瞬间站起，这时候他才注意到白羽旁边的这个年轻女孩，一开始打断他话让他让开的，也就是这个女孩。
<br/>　　惊恐之中，王之暴君忽的感觉有些眼熟，紧接着又猛然看向桌上的那两杯茶。
<br/>　　这第二杯茶，分明就是给这个年轻女孩准备的！
<br/>　　这人是谁？！
<br/>　　白家的白羽叫她小姐姐，还一副以她为尊的样子。
<br/>　　王之暴君忽然陷入思维混沌中，他好像见过这个女孩，但却怎么也想不起来她的身份。
<br/>　　这时候的何培源却出声打破沉默：“白允儿啊，好多年没见了吧？”
<br/>　　轰！
<br/>　　王之暴君这个身躯都开始剧烈颤抖，满脸的不可置信中，更是冷汗直冒。
<br/>　　白允儿！
<br/>　　她就是那个白允儿？！
<br/>　　白家的嫡系大小姐，传说中杀人不眨眼比之楚涵还过之而不及，最重要的是他似乎跟楚涵有着不清不楚的关系！
<br/>　　她怎么会出现在这里？
<br/>　　不是说神秘家族的人，根本进不来这猎者联盟总部么？
<br/>　　就在王之暴君的震惊之下，白允儿朝着何培源点点头，道：“当时不过末世爆发初期，我也没认出您就是制裁者。”
<br/>　　何培源笑着摆手：“我已经退任了，现在是个闲散老人。”
<br/>　　白允儿轻笑：“虽然已经不是当代制裁者，但您的事迹，恐怕是历代制裁者中最令人闻风丧胆的一位。”
<br/>　　“哈哈！”何培源哈哈大笑：“不过就是抓了肖家一整个家族，导致他们灭族而已，不值一提。”
<br/>　　闲谈中，跟着白允儿而来的白羽，从对话开始便整个人都震惊的差点晕过去，同时更是瑟瑟发抖的下意识躲在屋内角落。
<br/>　　制，制裁者？！
