　　弓，是比较稀缺的兵器，毕竟使用弓的武者比例较少。
<br/>　　不过，拍卖会聚集了近十万人，以弓为兵器的人还是有一些的，这些人怎么可能错过尊级中品的宝弓？
<br/>　　龚忠的话音一落，激烈的竞拍便开始了，金裂弓的价格急速攀升。
<br/>　　“好！”
<br/>　　二十九号包厢中，苏莫面色大喜，以这种势头下去，这件大弓拍卖出二千多中品灵石的价格，完全不成问题了。
<br/>　　“苏莫，其实，你若是想让价格被拍的更高，你可以从中抬价！”吕罡笑着说道。
<br/>　　“抬价？”苏莫闻言一怔，随即立刻恍然，他只顾着高兴了，居然把这一茬给忘了。
<br/>　　他可以从中作梗，哄抬价格，将最终的竞拍价抬的更高。
<br/>　　而且，这种做法，拍卖会的举办方，不仅不会动怒，还会喜闻乐见。
<br/>　　这也是很多拍卖会，常用的手段。
<br/>　　“待会看情况而定！”苏莫点了点头，现在竞拍才刚刚开始，他还无需出价。
<br/>　　此时，苏莫开心不已，但在另一座包厢中，扬天成、杨天威，以及绿衣女子三人，却是全部都站了起来。
<br/>　　“这不是武仁师兄的金裂弓吗？”绿衣女子满脸惊讶的说道，这金裂弓，她再熟悉不过了，正是武仁的兵器。
<br/>　　武仁作为天虚宗的真传弟子，其所用宝弓，在天虚宗内，几乎没有人不认识。
<br/>　　因为所有真传弟子之中，也就武仁以弓为兵器。
<br/>　　“就是武仁的金裂弓！”扬天成的脸色阴沉了下来。
<br/>　　“混账，武仁已死，谁把他的金裂弓拿来拍卖的？”杨天威口中厉喝一声，眸中寒光闪耀。
<br/>　　他和武仁的关系不错，如今武仁被杀，兵器被人拿来拍卖，他如何不怒。
<br/>　　“难道是苏莫拿来拍卖的？”扬天成的眸中闪过惊疑之色，当初在魔渊，武仁是死于苏莫之手，所以很有这个可能。
<br/>　　当然，他并没有看到苏莫收取了金裂弓，当时魔渊人数众多，武仁陨落之后，金裂弓被其他人拾取。也是很有可能的事情。
<br/>　　“你确定？”杨天威闻言，眸中杀机一闪，若是苏莫再次，他一定要出手擒拿。
<br/>　　因为宗内已经下了暗令，谁若是擒拿了苏莫，奖励一万中品灵晶，这几乎让所有弟子都心动无比。
<br/>　　只不过，苏莫一直在道宗内不出，他们也没有任何办法。
<br/>　　“不确定，也有可能是那个吕罡！”
<br/>　　扬天成摇了摇头，随即，他又想到了和吕罡一起的银面人，惊疑的道：“那个银面人……不会就是苏莫吧？”
<br/>　　扬天成不傻，心念及此，眸中顿时精光闪烁，越想越有可能。
<br/>　　首先银面人和吕罡一起，说明很可能是道宗的弟子，其次苏莫现在不敢露面，而一旦露面的话，肯定不想让别人发现，所以会带着一张面具。
<br/>　　再者，那位银面人的体型，与苏莫非常的相似。
<br/>　　之前扬天成还没有在意，现在，想到此点，他有九成九的把握断定，银面人就是苏莫。
<br/>　　“如果真是他，那他就不用回去了！”扬天成闻言，眸中闪过一道森冷的笑意。
<br/>　　“天成师兄，你的仇也能报了！”绿衣女子轻笑道。
<br/>　　“哼！天堂有路他不走，地狱无门他自来投！”扬天成冷笑了起来，他已经迫不及待的想要去找苏莫了。
<br/>　　“天成师兄，上次听厉恨天师兄说，那苏莫已经突破到了武尊之境，想必实力更强了，不然也不可能从厉恨天师兄的手中逃走！”绿衣女子提醒道。
<br/>　　“的确，苏莫此人太过妖孽，不能大意！”扬天成点了点头，他并没有被仇恨冲昏头脑。
<br/>　　“放心，有我在，纵然他有三头六臂，也逃不出我的手掌心！”
<br/>　　杨天威自信的说道，上次苏莫之所以能从厉师兄手中逃脱，完全是厉师兄大意了，不知道苏莫突破到了武尊之境，不然怎么会失手！
<br/>　　“嗯，有族兄在此，谅苏莫也翻不起什么大浪！”扬天成点了点头，他对于杨天威的实力，有着绝对的自信。
<br/>　　在天虚宗，除了厉恨天之外，任何人也无法企及杨天威。
<br/>　　苏莫虽然战力逆天，扬天成可不相信，苏莫能有抗衡杨天威的实力。
<br/>　　“我们等拍卖会结束之后再动手吧！”绿衣女子说道，现在正值拍卖之中，他们若是直接去找苏莫的话，也是不妥。
<br/>　　虽然他们都是天虚宗的弟子，但此城的大势力，都是归属于啸天狼族管辖，他们若是扰乱拍卖会，六壬商会肯定不会轻饶他们。
<br/>　　“嗯，等拍卖会结束之后，立刻动手！”扬天成和杨天威均是点了点头。
<br/>　　此时，拍卖场中，金裂弓的价格，已经竞争到了后期，被抬到了二千中品灵石以上。
<br/>　　高昂的价格，已经让九成九的人放弃了，只剩下两个人在竞价。
<br/>　　“二千二百中品灵石！”一号包厢中传出低沉的声音。
<br/>　　“二千三百中品灵石！”七号包厢中传出淡淡的女声。
<br/>　　二十九号包厢中，苏莫眸中精光熠熠，竞拍到了这个地步，他也该动手了。
<br/>　　“苏莫，你还是别抬价了吧！”吕罡突然出手，面带无奈之色的望着苏莫。
<br/>　　“怎么了？”苏莫闻言疑惑，对方之前让自己抬价，现在又不让自己抬价，是什么意思？
<br/>　　“一号包厢中，不用想也是啸天狼族的人，只有啸天狼族的人，才有资格坐在一号包厢！”
<br/>　　吕罡面色凝重，继续道：“而七号包厢中的人，听声音就是乙薇师姐！”
<br/>　　“是吗？”苏莫闻言不置可否，啸天狼族的人和乙薇在竞价，难道他就不能出价了？
<br/>　　这是什么道理？他又不违反规矩！
<br/>　　“二千七百中品灵晶！”苏莫毫不犹豫的出价了，并且将价格直接提到了二千七。
<br/>　　当然，他也不是盲目的出价，不然别人都不跟价了，他还玩个屁。
<br/>　　他也是仔细斟酌之下，才出价两千七的，因为乙薇和一号包厢的啸天狼族之人，出价都没有任何的犹豫，显得干净利落。
<br/>　　所以，苏莫断定，现在的价格，还远远没有到两人的底线。
<br/>　　还有就是，他的声音没有丝毫的改变，也不怕被扬天成认出来了，因为在他看来，金裂弓和庚金破虚箭一拍卖，扬天成就是再傻，也能猜到是他。
