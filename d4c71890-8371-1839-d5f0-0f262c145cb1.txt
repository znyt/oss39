　　天王宫。
<br/>　　密室之中，帝一魂、天君河、天尘宇三人，围坐在一起。
<br/>　　“一魂，你能肯定，龙族的祖龙大典，与苏莫有关系？”天君河沉声问道。
<br/>　　“嗯！”
<br/>　　帝一魂面无表情的点了点头，沉声道：“祖龙塔的祖龙精血，普天之下，除了我、拓跋青云还有苏莫我们三人的能力有可能得到之外，其他人没有可能得到祖龙精血，而祖龙塔，拓跋青云是无法进入的，所以，是苏莫的可能性极大。”
<br/>　　“苏莫也不太可能进入祖龙塔吧，那祖龙塔不是龙族之人，没有机会进入的！”天君河皱眉说道。
<br/>　　“这并不绝对，至少龙族的奴仆和一些龙族的外族族人，都是能够进入祖龙塔的！”
<br/>　　帝一魂声音低沉，继续道：“虽然我无法肯定，但是，此事的可能性极大！”
<br/>　　对于此事，他心中猜测很可能与苏莫有关，因为，按照当年下界万界山分身传递的消息，在下界之时，苏莫和龙族关系莫逆。
<br/>　　所以，在神域之中，苏莫很可能也与龙族取得了关系。
<br/>　　“那这个祖龙大典，新晋祖龙不会就是苏莫吧？”天尘宇面带诧异的问道，他感觉有点匪夷所思，这苏莫的手段简直多到无法揣测。
<br/>　　“这倒是不会，但取得祖龙精血之人，极有可能是苏莫！”
<br/>　　帝一魂摇了摇头，继续道：“不过，苏莫为人谨慎，估计现在已经离开了龙族，但是你们还是要去探查一番。”
<br/>　　“这样吧，这个祖龙大典，就有尘宇前去观礼吧，老夫暗中前去查看，若是苏莫真的在龙都城，老夫定然将他擒拿。”天君河沉声道。
<br/>　　此事，帝一魂不宜出面，免得地王宫那边又出什么幺蛾子。
<br/>　　“嗯！”帝一魂点了点头，虽然苏莫现在可能已经不在龙族了，但是，还是要仔细查探。
<br/>　　上次，他和人德宫主单独见了一面，本来想要拉拢到人德，但是失败了。
<br/>　　所以，想要靠人德来找到苏莫，已经不太可能了。
<br/>　　……
<br/>　　地王宫，大殿之中，地渊神王、地藏神王、等地王宫的十位核心高层，唯缺拓跋青云一人。
<br/>　　上次与帝一魂一战，拓跋青云受伤，目前一直在闭关之中。
<br/>　　“对于龙族的祖龙大典，你们怎么看？”地渊神王环视众人，朗声问道。
<br/>　　“真是奇怪，龙族之中，怎么可能有人得到祖龙精血，进化为祖龙呢？”地藏神王面带疑惑之色，出声问道。
<br/>　　他倒是对龙族的祖龙塔颇为了解，按照祖龙塔的情况，龙族之中，根本不可能有人得到祖龙精血。
<br/>　　“具体情况不得而知，但是，此事非同小可，你们谁前去龙都城。”地渊神王朗声问道。
<br/>　　此去一为观礼，二为看看那所谓的新的祖龙，到底是何许人也。
<br/>　　“父亲，还是我去吧！青云在闭关，您还是坐镇地王宫为好。”地藏神王沉声说道。
<br/>　　“嗯，这样也好！”
<br/>　　地渊神王微微颌首，随即又问道：“最近，有没有查到秦陨的消息？”
<br/>　　“禀父亲，目前，终极神王殿的人手，已经查到了天星神朝，我们的人手都在混沌神朝，暂时还没有消息！”地藏神王道。
<br/>　　“继续查，就算掘地三尺，也要把秦陨找出来！”地渊神王郑重的道。
<br/>　　“嗯！”
<br/>　　地藏神王点头，继续道：“不过，最近天王宫又不老实了，天王宫很多强者，频繁和其它四大王宫的强者接触，而且，上次青云败给了帝一魂之事，也被另外四大王宫知晓了。”
<br/>　　地渊神王闻言，面带沉吟之色，这可不是好兆头，若是其它四大王宫倒向天王宫，那就麻烦了。
<br/>　　这倒不是没有可能，毕竟，帝一魂略胜于拓跋青云，其它四宫难免会倒向帝一魂。
<br/>　　“你们有什么想法？”地渊神王环视众位地王宫的长老。
<br/>　　“宫主，我觉得，是不是应该将秦陨的身份，公布于众！”一位身材魁梧的老者，沉声开口道。
<br/>　　此人体态雄壮，面容刚毅，乃是地王宫的大长老，磐华。
<br/>　　“公布于众？”
<br/>　　众人闻言，均是纷纷沉默了下来，思索此事的利弊。
<br/>　　若是秦陨的身份公布于众，那么，局势就会更加的复杂了，不过，好处就是，另外四大王宫暂时不会倒向帝一魂。
<br/>　　毕竟，又出现了一具分魂，肯定所有势力都会静观其变的。
<br/>　　但是，这也有些不妥，因为秦陨的身份公布于众的话，有太多的不确定性。
<br/>　　“你们觉得，此法是否可行？”地渊神王环视众人，征求意见。
<br/>　　众人的面色，均是凝重了下来，此事重大，需要仔细的斟酌。
<br/>　　“宫主，若是将秦陨的身份公布于众，秦陨定然难以藏身，会比较容易被人找到，不过，我们不一定能第一时间擒拿秦陨！”一位长老沉声说道，现如今不只是他们在找，天王宫也一直在找。
<br/>　　“我倒是觉得，此事利大于弊，那秦陨修为较低，除了人王宫之外，没有哪个势力会帮助他！”
<br/>　　“而且，秦陨仇敌很多，不仅有终极神王殿，还有雷焱神王殿、万化神殿等势力，想要制其于死地之人很多。”
<br/>　　“我觉得可以将秦陨的身份，公布于众！”
<br/>　　“但是，我们要多多派出人手，抢在天王宫之前擒住秦陨。”
<br/>　　一众地王宫的长老，纷纷开口，有些意见比较保守，但是大多数赞同将秦陨的身份公布于众。
<br/>　　地渊神王闻言，沉默不语，他面无表情，仔细的思索了良久，终于有了决心。
<br/>　　“那就这么办吧，将秦陨的身份和各方面特征，昭告整个神域，并且，多多派人前往各大神朝，密切注意任何的风吹草动。”地渊神王朗声说道。
<br/>　　“是！”众人纷纷点头。
<br/>　　……
<br/>　　接下来的时日之中，地王宫将秦陨的身份，毫无保留的宣布了出去。
<br/>　　其宣布的内容便是，秦陨的真实身份，乃是轮回天尊的六具转世之身之一，拥有万化神剑元神、不败战气元神，修炼混沌源力。
<br/>　　这个消息一经传出，便以地王宫为中心，以无与伦比的速度，横扫整个轮回神朝，向其它八大神朝飞速传播。
<br/>　　而神域各大神朝，本来因为龙族的祖龙大典而一片哗然，但是祖龙大典的热度还没有散去，秦陨是轮回分魂的火爆消息，便再次轰动了神域。
<br/>　　一时之间，整个神域再次为之沸腾，无数的大势力为之震动，苏莫彻底的陷入了风波的中心。
