　　“没问题。”
<br/>　　到了下午。
<br/>　　林峰带着陈勾去了一次矿山，至于李广，一来，这基地要有人看着，二来也算是个小子创造和寡妇一起的机会。这人和人得接触，不接触怎么谈情说爱，这也是为什么现在光棍这么多，初中高中禁止早恋，到了大学，LOL吃鸡这类游戏诱惑太多，出了社会，同事都是名花有主名草有主，怎么找对象。
<br/>　　废话不多说，林峰带着陈勾两人来到这矿山，这个地方之前已经开采了一半。
<br/>　　林峰来到这里，发现了几个旷工带着安全帽在附近漫无目的的闲逛了，显然这几个家伙变成了丧尸。在旁边还有警示标语。
<br/>　　生产安全，效益一流。生态优美，矿风文明。
<br/>　　林峰扫了眼现场，点了点头，“的确生态优美，也很和谐，大家都不吵不闹的，自己一个人默默的思考人生。”
<br/>　　“林峰，一共十三个！三阶丧尸就一只！”陈勾道。
<br/>　　“那还废话什么，直接给他抹杀了。”
<br/>　　林峰拔出刀就是冲了下去，十三只丧尸根本不够看的，三秒钟，全部斩杀殆尽。
<br/>　　林峰来到工人曾经住的活动板房，板房里面还有三个矿工，这些矿工末世之前也算是苦力了，虽然工资比较高，但是很危险，毕竟一些矿老板实在是太黑心了，该有的安全措施全部没有。
<br/>　　林峰杀掉这些丧尸也算是让他们解脱了，早死早投胎，不用再这么人不人鬼不鬼的在世界上游荡了。
<br/>　　两人带着矿工的安全帽，打开了探灯然后就进入了矿洞之中，他要去看看，这个地方有没有塌陷。
<br/>　　矿洞内黑黢黢的，什么都没有，不过两人都是实力不低的黑铁战士，这点问题倒也不算什么，可以清晰看到四周的景象，他们脚下有一个铁轨，这个铁轨自然不是火车铁轨了，而是给运矿车铺的。
<br/>　　哪怕是现在看起来人类高度文明了，天上地下海里的，都有人类的足迹，但实际上，还是很落后的，这挖矿还得煤矿工人来挖，需要大量人力的投入。
<br/>　　林峰和陈勾两人一路朝下走，越下面空气越浑浊，已经出现了一些不舒服的感觉。
<br/>　　唧唧。
<br/>　　也就在这个时候。
<br/>　　前面忽然出现了非常刺耳的声音。
<br/>　　然后就是一只只变异老鼠冲了出来。
<br/>　　林峰一开始还砍了那么十几只。
<br/>　　但是越砍越不对劲，然后脸色就变了。
<br/>　　“握草，跑！”
<br/>　　林峰二话不说就拉着陈勾朝外面跑。
<br/>　　本来他是想着要丢汽油弹来封路的，但是一想到这个地方，还是没丢，这要是搞不好，整个矿山就这么爆炸了然后塌了，那他林峰就算是九阶的黑铁战士，那也得被活活的压死。
<br/>　　“怎么了林峰。”
<br/>　　陈勾还不知道发生什么了。
<br/>　　“不就几十只变异老鼠吗？虽然这玩意恶心了点，但是最强的也不过才四阶罢了，我们实力应付起来还不是绰绰有余吗？”
<br/>　　“几十只？那是几千只！后面还有一大堆变异老鼠！你小子没看到，我看到了！跑就对了！！”
<br/>　　林峰大喊道。
<br/>　　“啥玩意？几千只？这尼玛鼠灾啊。”
<br/>　　陈勾也跟着跑了。
<br/>　　好在林峰发现得早，也好在这个矿道就这么大个范围，老鼠虽然多，但是他不好一下子全部冲出来。
<br/>　　林峰和陈勾两人来到矿道外，顿时就是朝旁边跑。
<br/>　　几乎在同时。
<br/>　　轰！
<br/>　　一堆黑色的东西是像炸弹一样的从矿洞内炸出来的，一群群滚着出现的，看的陈勾和林峰都面色发白。
<br/>　　太恶心了，这么多老鼠，虽然这群老鼠哪怕变异了体积也没有变得很夸张，但是太多了，老鼠和蟑螂是林峰最讨厌的两个位于地球上的东西，尤其是蟑螂，老鼠还肥嘟嘟的，某些时候看起来还有些萌感，但是这么多，那简直就是恶心爆棚了，屁的萌感。
<br/>　　叽叽叽叽
<br/>　　数不尽的老鼠从矿洞内冲了出来，就好像是开闸泄洪一样，井喷式的老鼠飞了出来。
<br/>　　“林峰，还好你发现得早，不然我们此时估计要被啃的渣滓都没了。”陈勾害怕道。
<br/>　　“嗯。”林峰也点了点头，他不认为，自己在矿洞那个狭小的地方，可以一人之力干掉这几千只变异老鼠。
<br/>　　数千只老鼠从矿洞内不断的冲出，这一过程持续了足足好几分钟，随后才是消失了，只有大概几十只老鼠可能是迷路了，可能也是被同伴压的有些喘不过气来留在了矿山。
<br/>　　这些老鼠那自然林峰没有放过，直接全部灭杀。
<br/>　　“林峰，你说这些老鼠都去哪里了。”陈勾问道。
<br/>　　林峰闻言面色微微一变，“不会去我们老龙口基地做客了吧？”
<br/>　　陈勾也是愣了，然后两人迅速的又回了一次老龙口据点。不过好在是他们想多了，以老龙口据点现在的安全力量，这要是真的被这群老鼠来光顾一下，哪怕有坦克坐镇，也根本没辙，这坦克估计可以杀了几百个甚至上千只老鼠，但也无济于事，这群老鼠又多速度又快，几发炮弹的时间，早就可以冲过去了，然后估计就是坦克的那个炮管都可以啃断。
<br/>　　“峰哥，胖哥，不是去矿山了吗，怎么又回来了。”
<br/>　　李广正在那里撩寡妇。
<br/>　　看的林峰和陈勾来了楞了一下。
<br/>　　“没什么，看好你的据点。”
<br/>　　林峰道了句又是和陈勾杀了回去。
<br/>　　再一次进入矿山，这一次，没有遇到多么大的危险。
<br/>　　林峰来到了采矿层，这里留下了许多工具，什么钻机什么铲子之类的，不过钻机之类的肯定用不了了。
<br/>　　“看来要一大群人力来挖了。”
<br/>　　林峰道。
<br/>　　“那我们怎么找人，难道去58同城上发个招聘信息，老龙口矿业公司现招聘旷工若干，包吃包和保住一周七天不休息，日工资多少晶石这样吗？”陈勾道。
