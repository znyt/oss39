　　　　几日之后，苏莫和东海龙王、帝释、海明江三人，一起回到了苍穹神宫。
<br/>　　如今的苍穹神宫，没有多大的变化，所有的弟子，每日依旧是拼命的修炼，努力的提升修为实力。
<br/>　　让苏莫比较意外的是，蛟龙太子龙腾，居然也在苍穹神宫之中。
<br/>　　原来这一次，龙腾是和其父东海龙王一起离开了东海，前来苍穹神宫。
<br/>　　只不过，东海龙王和帝释半路改道，前去了天海城。
<br/>　　而龙腾则没有去，直接来了苍穹神宫。
<br/>　　苍穹神殿。
<br/>　　苏莫端坐于帝座之上，下方海明江和帝释坐于左侧，东海龙王和龙腾父子二人坐于右侧。
<br/>　　“苏兄，几月不见，你的变化真是翻天覆地，简直让我不敢相信啊！”龙腾面带微笑，目露惊叹之色。
<br/>　　苏莫仔细的打量了龙腾一番，同样面带微笑，道：“龙兄，你的修为实力提升的也不慢啊！”
<br/>　　现在的龙腾，赫然是武皇境四重巅峰的修为，而以对方的战斗力，估计已经不下于一般的武皇境六、七重武者。
<br/>　　“与你相比，不值一提！”龙腾摇了摇头，神情有些感慨。
<br/>　　苏莫微微一笑，随即转头看向东海龙王，道：“龙王前辈，不知你有何事要与我商议？”
<br/>　　东海龙王闻言，微微叹了口气，古板的脸上充满了凝重之色。
<br/>　　心中组织了一番语言，东海龙王目视苏莫，沉声道：“小友，我们要以最快的速度，剿灭黄泉魔宗！”
<br/>　　“这个自然，我与黄泉魔宗仇深似海，自然不会放过他们！”苏莫点头说道。
<br/>　　“这并不是仇怨的问题！”
<br/>　　东海龙王摇了摇头，道：“黄泉魔宗的宗主，乃是巫族的后代，会对苍穹世界造成不可估量的后果！”
<br/>　　“巫族的后代？”苏莫闻言一怔，随即心中恍然，难怪都说黄泉魔宗和试炼者有关系，原来其宗主是巫族后代。
<br/>　　巫山！
<br/>　　苏莫想到了巫山，对方正是姓巫。
<br/>　　“不错，若是不剿灭黄泉魔宗，将来试炼者降临，会将我苍穹世界的强者，一网打尽。”
<br/>　　东海龙王声音沉重，继续道：“黄泉魔宗对于苍穹世界各大势力非常的了解，到时候无人能逃脱！”
<br/>　　“原来如此！”
<br/>　　苏莫心中了然，问道：“那我们如何动手？”
<br/>　　“苏莫，现在有一个不确定的因素！”这时，帝释开口了，满脸凝重的道：“黄泉魔宗的宗主，已经十数年没有现身了，这是一个很大的变数。”
<br/>　　“此人实力如何？”苏莫问道。
<br/>　　“黄泉魔宗宗主名为巫森，实力自然是准帝武者中最顶尖的存在，最主要的是，我们探寻到其正在冲击武帝之境。”帝释道。
<br/>　　“武帝之境！”苏莫闻言心中一惊，若是黄泉魔宗宗主达到了武帝之境，那就真的麻烦了。
<br/>　　不过，想突破到武帝之境，可不会那么容易！
<br/>　　“所以，我们必须尽快诛灭黄泉魔宗！晚了就来不及了！”帝释告诫道。
<br/>　　苏莫闻言面露沉吟之色，目前来说的话，他苍穹神宫加上东海蛟龙一族，再加上天涯海阁和帝玄宫的残余势力，就算全部倾巢而出，也不太可能灭了黄泉魔宗。
<br/>　　最起码，大魔老、二魔老等准帝强者，就很难杀死他们。
<br/>　　就算将黄泉魔宗的那些小喽啰全部斩杀殆尽，也不能解决根本问题。
<br/>　　而且，到时候黄泉魔宗只剩下几名准帝的话，反而更为难杀。
<br/>　　“诸位，给我两个月的时间可否？”沉默了片刻之后，苏莫目光凝视几人，沉声说道。
<br/>　　“两个月？”
<br/>　　几人闻言一怔，随即海明江问道：“苏宫主，你是想在两个月之内，将修为突破到武皇境四重境界？”
<br/>　　“不错！”
<br/>　　苏莫重重的点了点头，道：“我只要突破到武皇境四重境界，不敢说天下无敌，想必实力不会比龙王前辈差，到时候剿灭黄泉魔宗，才会有极大的把握。”
<br/>　　苏莫很有自信，虽然东海龙王的实力颇为恐怖，但他的修为只要突破到了武皇境四重，绝对能超越东海龙王。
<br/>　　他说不会比对方差，也只是保守而言。
<br/>　　“两个月突破修为！”
<br/>　　东海龙王和帝释、海明江三人，闻言均是沉默了。
<br/>　　三人个个脸色凝重，若是别人说两个月时间从武皇境三重突破到武皇境四重，三人是断然不会相信的，但苏莫的话，他们却是深信不疑。
<br/>　　因为这一点，在苏莫这个充满奇迹之人的身上，完全不成问题。
<br/>　　他们在乎的是，现在时间紧迫，每多拖延一天就多一天危险。
<br/>　　毕竟，若是黄泉魔宗宗主巫森，真的突破到了武帝之境，立刻便能横扫天下。
<br/>　　苍穹世界将再也没有任何人、任何势力，能阻挡其分毫。
<br/>　　到时候，就算是苍穹神宫的水幕大阵，怕是都有可能挡不住了。
<br/>　　“苏莫小友，本王就给你两个月的时间！”
<br/>　　沉默了良久之后，东海龙王率先开口了，因为他知道，不给苏莫时间也不行了。
<br/>　　因为以他们现在的实力，最多打败黄泉魔宗，断然没有诛灭的可能。
<br/>　　“苏莫，两个月之内，你尽量突破，时间越早越好！”帝释叹道。
<br/>　　“嗯！”苏莫点了点头，他从殷家获得了这么多资源，两个月时间，完全足够了。
<br/>　　“那个……苏兄，我来你苍穹神宫也有几日了，最近在神宫中逛了逛！”这时，龙腾突然开口了，面上带着无法言喻的笑意。
<br/>　　“哦？龙兄感觉我苍穹神宫怎么样？”苏莫笑着问道。
<br/>　　因为知道东海蛟龙一族和苏莫的关系，所以龙腾来到苍穹神宫，金一等傀儡人就打开了水幕大阵，让其在神宫中住了几日。
<br/>　　“哈哈！苏兄，苍穹神宫的确不错，不愧是远古霸主级势力！”
<br/>　　龙腾哈哈一笑，随即话锋一转，笑问道：“听说苍穹神宫有九大宫，每一宫都有一位传人，此事是否为真？”
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“呵呵！听说到目前为止，九宫传人还未凑齐，你看在下如何？”龙腾胸膛一挺，满脸笑意的说道。
<br/>　　呃~~
<br/>　　苏莫闻言顿时神情一怔，他倒是没想到对方居然要自荐，想做九宫之一的传人！
