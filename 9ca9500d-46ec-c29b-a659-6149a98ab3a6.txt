　　嘶…嘶…嘶…
<br/>　　一丝丝气血力量，从血色圆球中被牵引而出。
<br/>　　不过，萧逸的眉头，却忽然皱了起来。
<br/>　　“怎么回事，难度大了这么多？”萧逸皱起了眉头。
<br/>　　此刻，顾怜星已盘膝闭目。
<br/>　　萧逸将一丝丝气血力量打入她体内。
<br/>　　一边引导着气血力量在血脉上的游走，一边暗暗皱眉。
<br/>　　他此刻牵引这些气血力量，竟是感到异常吃力。
<br/>　　抽调而出的气血力量，也不过是一丝丝。
<br/>　　前些天，在幻光洞府内，抽调那魂力圆球时，可没有如此吃力。
<br/>　　抽调而出的魂力，一股一股，延绵不断，异常轻松。
<br/>　　可到了血光洞府这里，抽调气血力量，却只能一丝丝的抽取，还异常吃力。
<br/>　　这颗血色圆球，与那颗魂力圆球，二者的力量还有浑厚度，应该在同一个层次。
<br/>　　明明是同一个层次的力量。
<br/>　　自己在抽调魂力时，轻松无比。
<br/>　　抽调这些气血力量，却感到吃力？
<br/>　　萧逸不禁皱眉轻咦。
<br/>　　主府之外，那八位天骄，所牵引的气血力量，同样只是一丝丝。
<br/>　　不过，他们身外的阵法显然有辅助之效。
<br/>　　且阵法亦绝非等闲阵法。
<br/>　　故他们牵引起来，反倒看起来比萧逸要快上不少，也轻松不少。
<br/>　　萧逸暗暗皱眉。
<br/>　　这般缓慢速度，要将顾怜星的天星血脉彻底修复，起码需要数天时间，甚至更久。
<br/>　　“算了。”萧逸心头低语一声，也只能暂时摒弃杂念，专心替顾怜星修补血脉。
<br/>　　但，恰在此时。
<br/>　　萧逸另一只手上，手指一阵异动。
<br/>　　手指上，他的乾坤戒震颤了数分。
<br/>　　“嗯？”萧逸心头疑惑下，放出感知。
<br/>　　当然，事实上，他早有猜测。
<br/>　　果然，乾坤戒内，是冷焱剑在异动。
<br/>　　早在他进入主府，见到这颗血色圆球时，他就知晓冷焱剑肯定会有异动。
<br/>　　每每有气血力量时，冷焱剑都会有异动。
<br/>　　更别说面前这颗巨大圆球内，气血力量浑厚惊人到极点。
<br/>　　“知道了。”萧逸传音一句。
<br/>　　“等我帮人修补完血脉，就轮到你吸收。”
<br/>　　萧逸知晓，冷焱剑内有剑灵。
<br/>　　它听得懂自己的传音的。
<br/>　　不过，他现在正为抽调气血力量缓慢的事而烦闷着，便也只随便传音了一句。
<br/>　　孰料，乾坤戒内，冷焱剑抖动异常。
<br/>　　震动，亦愈发剧烈。
<br/>　　冷焱剑的震动，让得乾坤戒亦抖动不停。
<br/>　　萧逸的手指，一阵摇晃。
<br/>　　“小贼？”一旁公孙火舞看着萧逸不断摇晃的手指，疑惑地问了一声。
<br/>　　“没事。”萧逸淡漠地摇了摇头。
<br/>　　心头，则传音则乾坤戒内。
<br/>　　“做什么？再胡闹我可不客气了。”
<br/>　　传音落下。
<br/>　　但乾坤戒内的异动，丝毫没有停下，反而愈演愈烈。
<br/>　　“混账，可是真要我发怒，毁你剑身，焚你剑识？”
<br/>　　萧逸只是威胁一句，可不会真的毁了自己的冷焱剑。
<br/>　　孰料，冷焱剑内，却是一抹气急败坏的苍老声音发出。
<br/>　　“臭小子，你敢？”
<br/>　　“老夫忍你许久了，这也不让，那也不让。”
<br/>　　“若是放到千万年前，你这等毛头小子敢对老夫如此不敬，早便被老夫一念抿去心神，送入黄泉。”
<br/>　　“嗯？”萧逸闻言，心头一惊。
<br/>　　说起来，他还是头一次见剑灵说那么多话。
<br/>　　他虽早知冷焱剑内有剑灵。
<br/>　　但剑灵说话的次数，却屈指可数。
<br/>　　而且，这还是他头一次准确听出剑灵的声音。
<br/>　　“呵。”萧逸冷笑一声，“听你此话，便是千万年后的今日，奈何不了我了？”
<br/>　　“你…”那抹苍老的声音，显然一滞。
<br/>　　“乖乖给我待着。”萧逸冷声道，“我不知你这剑灵算是什么东西，或许千万年前了不起，可现在，你不过是一抹灵识。”
<br/>　　“我要毁去，不过弹指之间。”
<br/>　　这剑灵的话，事实上让得萧逸惊讶的同时，又带着早有所料。
<br/>　　早有所料的是，这剑灵存在的岁月，必然不短。
<br/>　　毕竟冷焱剑数百年前便被锻造而出。
<br/>　　而理所当然的，让萧逸惊讶的，便是剑灵的存在，似乎远比他认为的要长。
<br/>　　听这口气，似乎还非寻常剑灵。
<br/>　　不过，剑灵便是剑灵。
<br/>　　萧逸更不可能怕了这威胁。
<br/>　　冷焱剑内的异动，已然停下。
<br/>　　不过剑灵的声音，愈发气急败坏。
<br/>　　“臭小子，你这般缓慢的抽调速度，何时才能轮到老夫吸收？”
<br/>　　萧逸本就因为速度缓慢而略微感到烦闷，现今闻言，更是脸色发黑。
<br/>　　“我已尽全力抽调，你既等不了，那便自己吸收。”
<br/>　　传音落下，萧逸手中光芒一闪，取出了冷焱剑。
<br/>　　几乎是剑身出现的瞬间，血色圆球上，已一缕缕血气力量被抽调而出，随后进入剑身。
<br/>　　冷焱剑，毕竟是中品圣器。
<br/>　　这个阶品的圣器，起码需要圣皇境巅峰修为，方能发挥全部威力。
<br/>　　自然的，剑内剑灵，依此吸收气血力量的速度，甚至还在萧逸之上。
<br/>　　数分钟后。
<br/>　　又是一阵传音传来。
<br/>　　“小子，你就不能搭把手帮帮老夫？”
<br/>　　“你乃剑修，剑不在你手，你不激发剑力，剑身威力如何发挥到极致？”
<br/>　　剑灵气急败坏的声音，再次传来。
<br/>　　“与我无关。”萧逸淡漠传音。
<br/>　　许久前，他就默认了冷焱剑吸收气血力量的事。
<br/>　　故冷焱剑此刻要吸收这颗血色圆球内的力量，萧逸也不阻止。
<br/>　　但他现在正全力替顾怜星修补血脉。
<br/>　　抽调血气力量的速度本就慢，若还分出一半速度操控冷焱剑，那顾怜星的血脉修补速度将更慢。
<br/>　　剑灵的传音，戛然而止。
<br/>　　足足大半个时辰后。
<br/>　　“这颗血丹内的力量，足够精纯浑厚，小子，算你走运。”
<br/>　　“老夫便给你些好处。”
<br/>　　萧逸未理会。
<br/>　　但半晌，一缕缕武道文字，凭空出现于萧逸脑海。
<br/>　　“小子，这是血刹魔经，乃是气血一道的功法，能多久参悟，便看你自己本事了。”
<br/>　　剑灵的声音，苍老中，带着一抹戏谑。
<br/>　　“血刹魔经？”萧逸眉头一皱。
<br/>　　萧逸脸庞抽了抽，这功法，一听就不知道是什么好东西。
<br/>　　不过…
<br/>　　萧逸看了眼一旁的顾怜星，又看了眼冷焱剑。
<br/>　　半晌，还是带着好奇，感知着脑海中的文字。
