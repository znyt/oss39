        而这个秋容，她也是最近才新进来的，只是令唐倩兮没想到的是，秋容竟然也认识婴宁。
<br/>
<br/>        听那位十爷的口气，好像是说，婴宁的死与秋容有关。但是，秋容似乎是受了什么天大的委屈，不承认这件事情。
<br/>
<br/>        虽然脑海里满是疑惑，但是这都是五年前的事情了，唐倩兮并不打算将这事情放心上，更何况，当年婴宁又不是她手下的姑娘。
<br/>
<br/>        不过，十爷似乎是在替婴宁出恶气才针对秋容。
<br/>
<br/>        “小谢，你给嫂嫂记住了！在这‘夜惑’里，一定不要多管闲事！”唐倩兮再次跟童筱颖强调。
<br/>
<br/>        童筱颖再次犹如小鸡啄米般猛点头。
<br/>
<br/>        走着走着，唐倩兮又突然停下脚步，深沉地接着说道：“不要让十爷看到你的真容。”
<br/>
<br/>        “噢，好！”童筱颖继续点头。
<br/>
<br/>        秋容、婴宁、十爷……
<br/>
<br/>        还有，嫂嫂为什么叮嘱她，不要让十爷看到她的真容？
<br/>
<br/>        其实，童筱颖的脑海里满是疑惑，但唐倩兮似乎不肯多透漏这其中的一些事情，她也不大好直接问。
<br/>
<br/>        还是算了……
<br/>
<br/>        反正跟她没关系！
<br/>
<br/>        童筱颖想开后，跟着唐倩兮的步伐去了后台。
<br/>
<br/>        今晚虽然也有打赏的客人，但是因为少了“连少”和“季少”两位金主，她此刻驻唱的收入也一下子从上万跌到了两三千。
<br/>
<br/>        不过，没有了“连少”和“季少”的“骚扰”，童筱颖心里倒是比之前要踏实多了。
<br/>
<br/>        唱完五首歌后，童筱颖很快就结束了自己的舞台驻唱，一如既往地匆匆离去，任何男顾客主动上前搭讪，她都是微笑着礼貌性地委婉拒绝。
<br/>
<br/>        就在童筱颖返回自己的休息室途中，端茶送酒的小莹突然拦住了她的去路，只见她一手托着酒盘，一手捂着肚子。
<br/>
<br/>        “你……这是怎么呢？”童筱颖关问道。
<br/>
<br/>        上次小莹在季运恒的指使下给她下药的事情，她完全不知情，所以她对小莹并没有什么防备心理。
<br/>
<br/>        小莹瘪着嘴唇，愁眉苦脸，看起来很痛苦的样子：“我……我肚子好痛……但是现在必须给KTV楼202VIP包厢把这瓶酒送过去……不能怠慢了客人。”
<br/>
<br/>        “我去替你送吧！你赶紧去看一下医生。”童筱颖二话不说便接过了小莹手里的酒盘。
<br/>
<br/>        小莹连连欠身颔首地道谢：“谢谢你，小谢姐，太谢谢你了。”
<br/>
<br/>        “没事！那我先送过去了，是多少号包厢？”童筱颖摆了摆手，接着问。
<br/>
<br/>        “KTV楼202VIP包厢。”小莹故作吃痛地回答道。
<br/>
<br/>        童筱颖点了下头，便端着酒盘过去了。
<br/>
<br/>        小莹看着童筱颖离去的背影，嘴角不经意间地微微扬起一抹阴森森的弧度。
<br/>
<br/>        这趟浑水，有人替她去蹚，滋味还真不错！
<br/>
<br/>        童筱颖端着酒盘，来到KTV楼的202VIP包厢门前，按响了门上的门铃，听到门栓自动解锁的声音，她便推门走了进去。
<br/>
<br/>        她刚走出玄关口，看到眼前的景象，顿时惊愕得瞪大了眼睛。

<br/>        包厢里的气氛十分暧昧，暖色调的灯光，模糊了室内所有人的脸，但那边长长的真皮沙发上男女间隔坐着的就共有七八个人的样子。
<br/>
<br/>        “脱脱脱！脱脱脱！”
<br/>
<br/>        男男女女，都在起哄。

<br/>        童筱颖不知道这里面的前因后果，只知道自己一进来，就是这样一幅场景了。

<br/>        “秋容，赶紧脱呀！我们都等着看咧！”沙发上不知道是哪个女人突然高调地扯着嗓音大喊。
<br/>
<br/>        “是呀！赶紧脱！我们的几位老总们都等着了！”又有一个女人起哄。

<br/>        此时，秋容的双手握成了拳头。

<br/>        很明显，她不愿意。
<br/>
<br/>        “只要你脱光了，桌上那一万块钱就是你的了！当然，脱光了又给我们跳一支舞，我再给你加两万！”一个男人笑淫淫地说。

<br/>        “是啊！赶紧脱！”
<br/>
<br/>        “脱吧！”
<br/>
<br/>        “你不脱也得脱！”
<br/>
<br/>        又有几个女人的声音彼此起伏。

<br/>        他们完全沉浸在催秋容脱衣的亢奋与挑逗中，都没注意到站在玄关口端着酒盘不知进退的童筱颖。

<br/>        童筱颖聋拉着脑袋，双手握着酒盘放在双腿前，微微欠身，刚准备转身离开的时候，耳畔响起了秋容的求助声。

<br/>        “小谢，不要走！求你……”秋容带着哭腔，突然喊了一声，将童筱颖吓得神经紧绷。
