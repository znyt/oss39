　　　　二万多条毒蛇在大庭广众之下如此放出来，基本等于谋杀。
<br/>　　密密麻麻，规模庞大数量的毒蛇如毒气弹投放现场，顷刻间弥漫出去，随之而来的就是民众的恐慌。
<br/>　　现场一下子彻底的混乱了，惊慌声，尖叫声层层迭起，这几十万民众的现场要是乱了，那是非常严重的后果，恐怕不小心踩都能踩死不少人。
<br/>　　不少人都怒火中烧的质问道袍男子李顺，但李顺却不言不语，不辩解，不答复，只是开筐之后安安静静的退到了一边。
<br/>　　任由大家将他成了千古罪人。
<br/>　　吟...
<br/>　　现场因为万蛇涌动，乱成一锅粥，但霎时间，一道九霄蛇吟冲天而起，民众蓦然回首，那蛇王雕像既然发光发亮，如同活过来了似的。
<br/>　　猛然，众人醍醐灌顶般的冷静了下来，一个个露出了惭愧之色。
<br/>　　蛇王大人在此，他们怕什么？
<br/>　　有什么好怕的。
<br/>　　蛇王号令天下群蛇，莫敢不从，这些毒蛇怎么敢在蛇王的仪式上放肆呢？
<br/>　　果不其然，接下来发生的事情将几十万民众大吃一惊，呆若木鸡。
<br/>　　万蛇涌动，从蛇筐宣泄出来，但随着蛇吟声冲天而起，这些通体鳞片在阳光映照下闪闪发亮的毒蛇诚惶诚恐了起来，一条条朝着蛇王雕像不由自主的匍匐下来，尾巴弯曲轻翘，蛇身笔直，纷纷俯首顶礼膜拜。
<br/>　　将近二万四千条毒蛇同时朝着蛇王雕像膜拜，举行庄重而古老的仪式，一丝不苟。
<br/>　　这种场景简直给人无比强烈的视觉冲击力。
<br/>　　众人不在惊慌失措，而且纷纷的拿出手机拍摄，此情此景，百年难得一遇。
<br/>　　也只有在白镇的地界上，才看到如此不可思议的一幕。
<br/>　　这些毒蛇是如此的有灵性，跟信徒一样的想蛇王表达自己的敬意。
<br/>　　吟...
<br/>　　蛇王雕像再次传出一声威严不凡的蛇吟声，二万多条顶礼膜拜的毒蛇仿佛是收到了什么指令，纷纷掉头，形成了一条长长的蛇形队伍，有条不紊的朝着蛇王庙的后山游去。
<br/>　　很多毒蛇经过人的脚边，没有一丝的停留跟放肆，紧随大部队，赫然，这便是一副无比美观的蛇大军拔寨行军的神奇画面，蛇的行走千姿百态，或直线行走或弯蜒曲折而前进，二万多条毒蛇同时游走，既然让人看了赏心悦目，不凶煞，不乖张，曼妙婀娜。
<br/>　　上空望下，仿佛是一条无比庞大的蟒蛇拖着长长的身躯朝着佘山游曳。
<br/>　　几十万民众在一片目瞪口呆下，看着一条一条毒蛇游走到了后山，归依佘山，直至一条不剩。
<br/>　　现场久久不见声响，所有人都还沉寂在这场万蛇涌动的情景中无法自拔。
<br/>　　半晌....
<br/>　　忽然掌声雷动，山呼海啸声此起彼伏。
<br/>　　“蛇王大人万岁...”
<br/>　　“蛇王大人万岁...”
<br/>　　“蛇王大人万岁...”
<br/>　　蛇王为世人献上了一幅万蛇上佘山的奇妙场景，看的人惊心动魄，震撼无比。
<br/>　　这场蛇王仪式可谓高潮迭起，让人热血澎湃。
<br/>　　晚18时许...
<br/>　　一切事宜都落下帷幕，只瞧李老爷子高呼一声，声如洪钟：“时辰已到，恭请蛇王入庙堂”
<br/>　　四个大力士向前而来，重新抬起蛇王雕像，在震耳欲聋的鞭炮声中，在几十万民众的欢呼呐喊声中，蛇王雕像正式被抬入庙堂，上正位正坐。
<br/>　　雕像入庙堂中，才算是仪式正常的结束，紧接着，一个个手持香火之中，络绎不绝的将手中的香插入庙中容器内。
<br/>　　仪式结束后，并不代表蛇王节的完了，相反才仅仅是一个开始。
<br/>　　蛇王庙外遍地摊俗活动也随着开始，各种活动拉开的序幕。
<br/>　　渐渐的，天开始黑了，白镇家家户户放鞭炮烟火，蛇王入庙正堂，一阵庄重的钟声再次响起，敲在了白镇人民的心间。
<br/>　　钟声落下，挨家挨户的蛇灯在自家门前，徐徐飘荡，顷刻间将这刚黑下的天给灯亮，无数蛇灯挂在上空，宛如无数耀眼的星星，蔚为壮观。
<br/>　　蛇王庙已蛇王仪式结束，反而越来的热闹非常起来，庙中源源不绝的上香人，庙外，摊俗活动，斗蛇比赛应接不暇。
<br/>　　毫无疑问，白镇的蛇王节真的比过年还要隆重跟喜庆，尤其是今年的蛇王节，不知道了打破往年多少次记录。
<br/>　　这是一场人与蛇的节日，街道上遍地可见的人与蛇在互动，他们脸上都是喜气洋洋。
<br/>　　一群小朋友围着一位足足四米长的大蟒蛇在拍照，一丝不见惧怕，拍照结束，好像还不尽兴，一群小朋友兴高采烈的一起将大蟒蛇抱了起来，像是自己上山得来的的猎物。
<br/>　　庙外，门庭若市，熙熙攘攘，一位清秀而透着邪魅的年轻男子缓缓的从香火鼎盛的庙堂中走了出来，精致的五官上，挂着淡淡的笑容。
<br/>　　“这样真好”
<br/>　　涂小安梦呓似的开口，放眼望去，街道上何其热闹，抬头仰望，白镇的天更是绚丽多彩，美不胜收。
<br/>　　“既然都那么美了，那就让这片天在美丽点吧”涂小安身出一只手，白皙的手掌摊开，既然诡异的出现了一股火焰。
<br/>　　自从吞噬了吸血蛇，掌控火焰之力，涂小安还没好好的展示过这项能力。
<br/>　　此时应景，那就让自己的火焰之力彻底点亮这片白镇的天。

<br/>　　只闻一声震耳欲聋的爆炸声，火蛇如火树烂漫，千树万树梨花开的散布开来，演化成无数条小火蛇在空中游曳起来。
<br/>　　紧接着，一条又一条的火蛇相互缭绕，在白镇上空争奇斗艳，火蛇的口一张，一粒粒“金砂”喷射而出，在空中傲然绽放。
<br/>　　这一下子，就将上面的天给染红了，红光似火，绚丽多彩。
<br/>　　几十万人的眼睛慢慢的睁大，被这场视觉盛宴所征服，甚至让他们连呼吸都快遗忘了。
<br/>　　“蛇王大人...”
<br/>　　呢喃自语的声音，在无数人的嘴中吐出。
