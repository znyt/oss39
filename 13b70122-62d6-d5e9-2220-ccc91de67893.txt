　　苏莫相信，古天意听了紫衣青年的传话，一定会来。
<br/>　　所以，他只需要在客栈中静等对方就行了。
<br/>　　在苏莫的修炼之中，时间悄然流逝，一天时间转瞬即过。
<br/>　　第二日中午时分，古天意来了。
<br/>　　古天意穿着一身蟒袍，孤身一人，大步的走进了天黄客栈的大堂。
<br/>　　静修之中的苏莫睁开了眼眸，他的灵识一直在留意着大堂，对方乍一到来，他便发现了。
<br/>　　“来了！”苏莫深吸一口气，随即意念穿过房间，进入了大堂之中。
<br/>　　“古兄，来306房间一叙！”
<br/>　　一身蟒袍，气质独特的古天意，刚刚进入客栈的大堂，耳中便响起了苏莫的声音。
<br/>　　“苏莫！”古天意闻言，顿时眸光一亮，果然，他的猜测没有错，果然是苏莫，苏莫果然没有死。
<br/>　　事实上，他之前也以为苏莫死了，但是，在听到紫衣青年的传话之后，他就猜到了，苏莫没有死。
<br/>　　这要见他的人，十有八九就是苏莫。
<br/>　　古天意微微点了点头，便大步的穿过大堂，向客栈楼上的客房走去。
<br/>　　不多时，他便来到了三楼的306房间，轻轻的敲了敲门。
<br/>　　“请进！”
<br/>　　房间内响起低沉的声音，古天意推开房门，便看到了一名面容坚毅的青年。
<br/>　　虽然房间内的这名青年面容与苏莫完全不一样，但古天意还是一眼便认出了这就是苏莫。
<br/>　　他乃是武尊境的武者，就算苏莫身上的气息收敛的再干净，他也能感知的到。
<br/>　　“苏兄，好久不见！”古天意微微一笑，面上露出和煦的笑容，随即大步的走进了房间内。
<br/>　　“古兄请坐！”苏莫同样微微一笑，立刻招呼古天意就坐。
<br/>　　少倾之后，苏莫关上房门，一挥手，将房间内用玄力隔绝了起来，以免被有心人察觉。
<br/>　　房间中，苏莫和古天意在桌前相对而坐。
<br/>　　“苏莫，你可是瞒过了天下人啊！”古天意微笑着说道，言语中充满了惊喜。
<br/>　　“古兄，我这也是侥幸逃得性命！”苏莫苦笑着摇了摇头。
<br/>　　“你能瞒过无数人，甚至是武圣境强者，这便是你的本事！”古天意道，同时他眸光打量着苏莫，发现苏莫的修为是武帝境八重，不免有些震惊。
<br/>　　这个提升修为的速度，实在是够惊人！
<br/>　　苏莫不想与对方闲扯，面色沉默了少倾，沉声道：“古兄，上次你说，可以把我两位朋友救出来，但是现在……！”
<br/>　　他话没有说完，想必对方能明白他的意思。
<br/>　　“呵呵，苏莫，你说聂乾和韩盈是吧？”
<br/>　　古天意轻笑一声，继续道：“之前都以为你死了，我也没有办法再去请动父皇，所以也救不出他们两人！”
<br/>　　苏莫闻言，目光笔直的望着对方，没有办法吗？
<br/>　　他可不相信对方没有办法，不过是不想而已，当然，他现在还要靠对方，也不好点明了。
<br/>　　“古兄，那现在呢？你能否请动圣皇前辈？”苏莫沉声问道。
<br/>　　“当然可以，只要我父皇出面，一句话就能搞定此事！”
<br/>　　古天意笑着说道，言罢，他面带迟疑之色，又道：“不过，若是不相干的人，父皇不可能出面，需要你真正的加入我们皇祖圣朝才行。”
<br/>　　“没有问题，我本来就是皇祖圣朝派系的人！”苏莫点了点头，心中却是冷笑不已，对方就是想让他加入皇祖圣朝而已。
<br/>　　而且是真正的加入皇祖圣朝，并不只是神武学府中的弟子派系。
<br/>　　“哈哈！那就好！”
<br/>　　古天意爽朗一笑，略作沉吟，道：“这样吧，现在我们一起去赤粱星，我立刻去请动父皇出面，你觉得怎么样？”
<br/>　　“可以！”苏莫点了点头，能尽快救出聂乾和韩盈，他当然不会拒绝。
<br/>　　“那好，我们现在即刻动身！”古天意站了起来，就准备前去赤粱星皇祖圣朝。
<br/>　　“古兄且慢！”
<br/>　　苏莫见此，抬手喊住了对方，沉声道：“现在，我在所有人的眼中，已经死了，所以并不想暴露身份。”
<br/>　　古天意闻言一怔，随即心中恍然，他也听说了苏莫和巫族有血海深仇，肯定是不想暴露出身份，而再遭到巫族之人的加害。
<br/>　　而苏莫之所以没有回神武学府，而是找人传话，肯定也是这个目的。
<br/>　　“明白，这倒是小事，你以后就改头换面吧！”古天意微微点了点头，心中却是暗想，这或许算是苏莫的把柄吧！
<br/>　　“嗯，从今以后，我叫上官昊！”苏莫道。
<br/>　　“好，苏莫已死，现在只有上官兄！”
<br/>　　古天意面带轻笑，随即招呼道：“上官兄，我们走吧！”
<br/>　　随即，两人联袂离开了天黄客栈，前往传送之地，向赤粱星而去。
<br/>　　赤粱星距离古周星不远，所以，没有耗费太多的时间，两人便来到了赤粱星的圣皇城。
<br/>　　圣皇城，自然便是皇祖圣朝的都城，雄伟壮阔的万年古城，人口数以亿计，一眼望不到尽头。
<br/>　　“上官兄，随我一起进圣皇宫！”古天意对苏莫说道。
<br/>　　“嗯！”苏莫点了点头，沉默不语，他的心中一直在思考，他不会在皇祖圣朝待太久的时间。
<br/>　　等到聂乾和韩盈被救出来，他就要考虑离开古灵星河，前去明虚星河了。

<br/>　　“好强的军队！”
<br/>　　苏莫见到这些驻守的将士，面带惊讶之色，这些将士之中，修为最低的人，都有武帝境六重的修为，而且其中居然有多达十数人是武尊境的修为。
<br/>　　“拜见八皇子殿下！”
<br/>　　“拜见八皇子殿下！”
<br/>　　当古天意带着苏莫走近宫门之时，一众金甲士兵，齐齐单膝跪地，向古天意行礼。
