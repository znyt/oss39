　　尸潮的战斗一展开就持续进行了两天两夜，这期间所有的参战者都严守战略的每一环，所有分工合作的各小队将配合进行到极致，用了最大可能将这场持久战展现出最完美。
<br/>　　两墙之间的战斗队伍已经替换过好几轮，这是除却三战队及空军之外压力最大的战区，所有进入基地的丧尸都要先经过这里，哪些丧尸要立即击毙，哪些丧尸要远离等待射击队解决，又有哪些丧尸需要放入内墙的基地迷宫·内，所有的细节都需要这些人来把关。
<br/>　　秦渊作为这批队伍的总队长，还需要时刻注意什么时候换人，三万人分为两批轮番战斗已经是一项保守战略，但哪怕如此依然有了不小的伤亡。
<br/>　　这恐怕也是除却倾巢大战之外，狼牙战团又一次大创。
<br/>　　射击队的情况最为良好，虽然时刻警惕让他们精神疲惫不堪，但最强的地理优势让他们四百人都没有出现伤亡，击杀率也高的可怕。
<br/>　　至于基地内最大战区的那片迷宫，情况则是排在了射击队之下，五万人的数量占了最大优势，分散在各处最佳近战地点的小队们，也在一次又一次的战斗中越来越默契。
<br/>　　他们是警戒队带出来的队伍，并没有像另外的那三万人一样战斗力爆表，甚至这里也是他们真正意义上的第一次实战。
<br/>　　但两天的时间，也足够他们适应以及成熟起来，后期的发展越来越好，也让所有进入迷宫的丧尸在最快时间内解决。
<br/>　　指挥台处的参谋部成员时刻注意着全战区动向，侦查队的成员不断奔跑于战区的各个角落，将所有情况一一带回。
<br/>　　何峰与大多数的士兵一样，也已经两天两夜没有合眼，他还在不断调整迷宫战区的细微之处，将那五万人的能力发挥到最大。
<br/>　　空军在两天内已经有多次降落补充物资的情况，他们时刻战斗在高空，无论是精神还是体力都到了极限，同时伤亡情况也让人倍感压力。
<br/>　　飞行丧尸的确是所有丧尸之中最难对付的一类，哪怕有高少辉的援助，也依然有三十架战斗直升机坠毁！
<br/>　　三战队虽然还没有出现牺牲，但也有很多成员受了轻伤，暂时还没出现精疲力竭而不得不下阵的情况，但楚涵知道这一刻早晚会来。
<br/>　　毕竟在这两天之中，光是三战队对付的高阶丧尸数量，就已经高达好八十万！
<br/>　　两天四十八小时内的每一分每一秒，三战队的每一名成员都在战斗，都在与丧尸拼杀，外面空地上体型巨大的那些丧尸，全部都是死在了三战队手中。
<br/>　　更别说这其中还夹杂着超级丧尸，并且他们要提防着超级丧尸中的特殊能力丧尸。
<br/>　　总计杀了多少数量的丧尸？
<br/>　　没有人有闲功夫去统计，他们只知道外面的尸海在不断缩小，而基地内的地面则是铺满了丧尸的尸体！
<br/>　　“总数有两千万吗？”徐锋结束了一头八阶丧尸之后，来到了楚涵身旁出声。
<br/>　　楚涵眯起眼睛，望着基地之外依旧不断涌来的丧尸群，道：“一千五百万，已经杀了一半。”
<br/>　　徐锋点头：“看来还是多亏了科技部，大部分的高阶丧尸都死在外面了。”
<br/>　　如果这次的尸潮一共有在一千五百万丧尸，那么高阶丧尸的数量绝对不止这么点，恐怕会有五百万之多，而他们的战斗已经进行到了一半，三战队所杀的高阶丧尸不过八十万。
<br/>　　可想而知从比例来看，还有一百到两百万之间的高阶丧尸，都死在了防守队控制的那些新武器之下。
<br/>　　“楚涵哥，空战传来消息，飞行丧尸的数量在骤减。”此时蒋天庆忽然跑来汇报。
<br/>　　虽说指挥台有着参谋部与何峰的把关，但谁都明白楚涵才是总指挥，他的战略覆盖了整个战区，而一旦遇到情况，他才是能给出最佳方案的人。
<br/>　　听到蒋天庆汇报的楚涵瞳孔一缩，立即下令道：“告知何峰负责迷宫战区，外圈交给我，空军解决完剩下的飞行丧尸后立即撤回，陈少爷结束最后一轮丧尸群分流后加入射击队，两墙之间的三万人最后一波猛攻后退入基地迷宫，侦查队不用游走了来一线战区，杨天的警戒队来一线战区来一线战区，所有狼牙老兵来一线战区！”
<br/>　　第一战区，就是两墙之间三十米宽的外圈，也是承受尸潮的第二轮防线。
<br/>　　蒋天庆心脏猛地一顿，立即回头将楚涵的命令传递而开！
<br/>　　侦查队的能力也在这一紧急命令出现的时候一下子暴露，短短五分钟的时间，整个南署基地的参战者都全体收到了命令，开始依照指挥官的安排转移。
<br/>　　警戒队的五千人第一个到达第一战区，他们同样依照了小队模式分散在各处地方。
<br/>　　侦查队第二个到达，这是他们极难得的全体聚集在同一个地方，有些紧张也有些激动。
<br/>　　来到第一战区的意义谁都明白，这是要打正面战！
<br/>　　何峰也开始将注意力逐渐全部转移到迷宫·内，此时总计八万人不到的新兵都重新安排了站位，但凡能够近战的最佳攻击地点都站着好几批小队，力保丧尸一来连个反应时间都没有就立即头破血流。
<br/>　　但是同样，这八万人从现在开始，不再有老兵带领，所有的一切都要靠他们自己解决，每一个十人小队中，也由离开的队长选拔而出的副队长，担任起小队长的职务。
<br/>　　八万人的任务没有变，依然是团队合作！
<br/>　　一名一直待在迷宫·内的新兵，望着新冒出来的很多面孔，紧张又迷茫的开口问道：“忽然改变了战略，是有什么事情发生吗？”
<br/>　　转移战区的一名新上任的小队长眼神复杂：“两个月一路走来，我大约能猜出上面的想法。”
<br/>　　另一名队员沉默了下，道：“这是在保我们，接下来出现的情况我们恐怕不敌，所以所有老兵都去第一战区了，我们这八万新兵都退居二线。”
<br/>　　很多人都在当下沉默了下来，神色复杂又感动。
<br/>　　“空军也开始撤回了。”一人指着上方，声音发抖。
