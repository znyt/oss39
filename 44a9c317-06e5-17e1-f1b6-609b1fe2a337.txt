        &lt;h3&gt;第三卷第二百零九章  加班&lt;/h3&gt;
<br/>
<br/>        江俞轩没有像往常一样去北京，他既不见陈子昂也没有去见张倩楠，当了那么多年的“卧底”生涯，让他自己都感觉到可笑。
<br/>
<br/>        陈子昂的一切活动都在他的眼皮子底下，工作上的，生活上的，包括她什么时候生理期他都能知道，她在他的眼里就是一个小透明。
<br/>
<br/>        张家二爷的态度一棍子打醒了他，张倩楠的无辜生病，让他感觉到了他们之间也许还存在着别的问题，这几年粉饰出来的美好的爱情也许就是一场泡沫，所以他想捋一捋这中间的厉害关系。
<br/>
<br/>        他们几个纠结的最主要的原因是陈子昂，一切都是围绕着陈子昂在转着，包括他江俞轩、包括张倩楠和张函，陈子昂这一两年和张函的纠葛也是因为他极力的撮合，现在他在怀疑，他们能复合吗？张函能给陈子昂幸福吗？想到这里他莫名其妙的笑了，如果张函能给陈子昂幸福何必要等到今天？
<br/>
<br/>        他极力的撮合只是把陈子昂推到了一个难堪的地步，如果陈子昂和张函旧情复燃，势必陈子昂就成了一个被人唾骂的第三者，如果陈子昂不和张函复合，介于孩子的问题，陈子昂不得不面对张函，会一次又一次的揭开内心的伤疤，在痛苦的边缘徘徊。爱和不爱都已经是把陈子昂放在火上烧烤着，他觉得这几年他错了很多。
<br/>
<br/>        时间容不得江俞轩有太多的想法，西城的项目已经步入收尾工作，整个新区呈现了一片欣欣向荣的景象，原来低矮的小楼已经变成了高楼大厦，每个小区里都是郁郁葱葱的绿植，现在就是三期的项目还有一些尾工，大概还需要一年就能完工。
<br/>
<br/>        杨总知道他回来了，直接就开车赶到他的办公室，这几年两个人俨然是忘年交，在工作上配合得很默契，谁说甲方和乙方的关系不平等？
<br/>
<br/>        甲方和乙方其实就是相互配合，相互协作，他们应该就是一个共同体，为了一个共同的目标把事情最好。
<br/>
<br/>        “俞轩，我最近的身体感觉不太好，总是感觉很累。”杨总看着神采奕奕的江俞轩，心里真的是很羡慕现在的年轻人，体力好，精力充沛。
<br/>
<br/>        “你啊，就是劳碌的命，总是不让自己闲着，你看看人家还在过年呢，你就把我喊回来，陪着你这个糟老头子苦逼的加班，不行，我还没有玩够。”江俞轩气呼呼的说。
<br/>
<br/>        “糟老头子的年纪大了，感觉是在和时间赛跑，我不跑快点就跑不过时间，领导信任我，让我到这里来搞开发，西城是十三朝古都，有其重要的意义，新旧结合方能体现我们的特色。不过你小子也应该感到自豪，西城的发展也会有你的功劳。”杨总笑着说，这小子最喜欢唠叨，但是工作起来一丝不苟，有时候觉得他是一个长不大的孩子，有时间又觉得他呼风唤雨无所不能。
<br/>
<br/>        在西城开发这个项目上，杨总觉得这么多年他和乙方的配合是最默契的一次，两个人的想法往往一拍即合，他都想不到他会和这么年轻的人在一起合作得这么顺畅。
<br/>
<br/>        “有我什么功劳，人们只会说那个臭商人，你看他们是为了赚钱的。”江俞轩委屈的说。
<br/>
<br/>        “哈哈哈哈，你这个臭商人，快点把工作干好，这里、这里需要你。”杨总指点着地图，整个沙盘南郊新区上都是他们开发的项目。
<br/>
<br/>        “自豪吗？骄傲吗？等有一天你翻看历史，你会发现这个地区有你的足迹和脚印，有你奋斗过的痕迹。无论你是一个商人还是一个工作人员，社会的发展离不开我们这样的人！”杨总看着沙盘笑着说。
<br/>
<br/>        “你这样一说还蛮有道理，好了，糟老头子，我们来谈谈今后的规划和工作情况吧。”江俞轩为杨总续上一杯新茶。
<br/>
<br/>        茶香袅袅，温暖如春，别人还在度假的时候，一老一少俩个人已经开始他们的工作规划，你来我往，时而皱眉，时而欢笑。
<br/>
<br/>        杨总敲着桌子，面前是他们刚刚摊开的地图：“轩轩，你看，这个是高新新区，也是南郊的地盘，如果可以，我们爷俩再合作一把。将这个区域打造成一个新型的社区，我们现在所处的位置因其独特的地理环境和人文环境，综合古建筑和历史遗迹，是一个新旧综合体，但是这一片你看，完全可以抛去任何桎梏，打造一个赞新的具有现代化的新区，当然了，这么大的项目不可能是你一个人来做，你得联合大家一起共同完成。”
<br/>
<br/>        “你这雄心壮志不小啊。西城最近这几年的发展势头无两，继南郊之后，东西北都已经在开始行动了，而且地铁开通之后会贯穿整个城市，拉动整个地区的发展，你呀就偷着乐吧。”杨总对待工作严谨而认真，快六十多岁的人经常和他们一起加班加点，研究方案，讨论规划，像他自己说的，他是和时间在赛跑，这样的一个老人是值得让人尊敬的。
<br/>
<br/>        俩个人在一起的时候，就是“臭小子，糟老头子。”他们有时候太忙了，中午就在工地上吃一顿工地灶上的大锅饭，等到忙完想起来时候，饭基本都已经凉了，晚上也经常是八点以后才吃。
<br/>
<br/>        在别人的眼中，他们一个是地产界大佬，一个是西城开发领航者，俩个人肯定是风光无限，谁知道他们有时候饿急了，大半夜的会吃泡面？
<br/>
<br/>        “明天我随你去工地再看看，这一上班估计要来视察的人很多，哎，我最是怕那些形式主义，但是不去还不行。”杨总叹口气，站起来伸了一下胳膊。
<br/>
<br/>        江俞轩走过去替杨总捶了捶肩膀，杨总微微一笑，臭小子为人善良，他们俩个人在一起工作的时候，他从来不会特意逢迎他，也从来不刻意远离他，对他这个老头子总是处于一种发自内心的关心。
<br/>
<br/>        “改天我陪你去做个检查吧，你这每天忙忙碌碌的，身体还是重要，年龄大了也要经常做做体检。”江俞轩自然而然的对杨总说。
<br/>
<br/>        “也好，等有时间了就去，对了，你的那个什么信息化公司你还管着没有？”杨总问江俞轩。
<br/>
<br/>        “管着啊，怎么了？”
<br/>
<br/>        “你阿姨公司的电脑有问题了，昨天让我帮她找个人给她收拾一下，你找人给她看看去。”
<br/>
<br/>        “小事一桩，上班的时候我让人去找阿姨。”
<br/>
<br/>        “行，我走了，你别送了。”
<br/>
<br/>        “我还是送送你吧。”
<br/>
<br/>        “送我干什么，还不回去谈你甜甜的恋爱？”
<br/>
<br/>        “你这个糟老头子坏的很，天天鼓捣小朋友谈甜甜的恋爱，就是不给小朋友谈恋爱的时间。”
<br/>
<br/>        杨总开着车离开，江俞轩拿起手机，在公司群里发了一条信息：“上班之后，去某某公司联系某人，做一个信息维护。切记！”
<br/>
<br/>        陈子昂：“切记！”
<br/>
<br/>        江俞轩看着陈子昂的回复私信了陈美人：“你有毒！”
<br/>
<br/>        陈美人：“你有毒，不回北京？”
<br/>
<br/>        江俞轩：“我已经在西城开始了苦逼的加班，北京一时半会儿回不去，你保重！”
<br/>
<br/>        陈美人：“保重!”
