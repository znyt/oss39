　　“还有谁知道？”屋内顾凉晨望着紧逼着眼的楚涵出声。
<br/>　　陆玟丞摇着头道：“不超过十人。”
<br/>　　“阴阳谷的探索要延后？”顾凉晨又问道。
<br/>　　陆玟丞踌躇了一下，点头：“怕是如此，但如果老大一直这样……”
<br/>　　“我们需要帮他完成这个理想。”顾凉晨接话道。
<br/>　　紧接着这间不大的小屋就又是一阵长久的沉默，两人一站一坐的待在楚涵床边，看着床·上这个熟睡的人，无论是紧闭的双目还是安详的呼吸，都给他们一种很陌生的感觉，也极少有人见过这样的楚涵，毫无防备的就这样睡着。
<br/>　　没有往日里的霸道，也没有黑色眼眸里的深邃和城府，更没有那标志性斜斜勾起的奸诈笑容。
<br/>　　什么都没有，毫无生气，好像再也醒不过来了一样……
<br/>　　也不知过了多久，顾凉晨看向了床边，那柄巨大又漆黑的修罗战斧：“我想，让叶永阳看看这把斧头。”
<br/>　　待在屋内角落里的旺财一惊，原本熟睡的它猛然睁开眼，而后忽然跳出一把将修罗战斧死死抱在怀里，更冲着顾凉晨和陆玟丞龇牙咧嘴的乱叫。
<br/>　　在楚涵昏迷的这段时间，旺财也不知道为何渐渐精神不振，每日熟睡的时间比清醒时要多，甚至很多时候屋里来了人它都没能察觉。
<br/>　　但是旺财有一种强烈的感觉，这柄斧头不能离开楚涵半步！
<br/>　　看到忽然出现的旺财，顾凉晨和陆玟丞都吓了一跳，尤其是顾凉晨好半响都没能反应过来。
<br/>　　陆玟丞在惊吓过后反应过来，连忙解释：“这个，这是老大的宠物，哦不，战宠，叫旺财。”
<br/>　　顾凉晨惊讶异常，他还真不知道这只兔子的来历，而战宠二字，不一向是于路冰泽专属么？
<br/>　　“它这是什么意思，不让我们带走修罗战斧？”顾凉晨看着旺财激烈的反应问道。
<br/>　　陆玟丞想了想，道：“去看看叶永阳醒过来没有，带来吧。”
<br/>　　半个小时后，知道楚涵当下情况的人便又多了一个，叶永阳的内伤很重，是坐在轮椅人被人一路推着过来的。
<br/>　　他看到这间防守森严的小屋后第一反应就是警惕，在进入屋内之后便是愕然，因为没有想到顾凉晨让他来见的人，竟然躺在床·上昏迷不醒。
<br/>　　在这之后，叶永阳便陷入了长久的震惊之中。
<br/>　　因为他看到了修罗战斧！
<br/>　　顾凉晨和陆玟丞都没有说话，就这么静静的看着叶永阳盯着那把斧头一动不动。
<br/>　　也不知过了多久，这种状态实在无法在持续下去时，陆玟丞首先按耐不住，走上前就想将修罗战斧拿起。
<br/>　　“别动！”哪知叶永阳却忽然大声喊道。
<br/>　　陆玟丞的动作一顿，手举在半空拿也不是放也不是。
<br/>　　顾凉晨也吓了一跳，没明白叶永阳在紧张什么。
<br/>　　或许是察觉到自己过分紧张，叶永阳老脸一红，解释道：“你们想拿也拿不了，说不定还会激起这病斧头的凶性。”
<br/>　　陆玟丞忙将手收回，他的确从未碰过楚涵的修罗战斧，但却听说过这柄斧头曾将一名鬼种吓出神经病，而修罗战斧又是叶默所制，当下又叶永阳出声制止，可见那个传闻说不定是真的。
<br/>　　这时候叶永阳将轮椅推上前，随手拿起旁边的一条毛巾裹住手，而后轻轻碰了碰修罗战斧，在见到并没有出现异常后，这才轻轻抚摸着斧刃。
<br/>　　“这柄黑斧，是叶默制造的吧？”他开口，带着一丝缅怀。
<br/>　　“是。”顾凉晨点头。
<br/>　　他与陆玟丞对望一眼，彼此眼中都带着惊异，果然亲兄弟之间有着某种说不清道不明的联系么？
<br/>　　“我一看就知道。”叶永阳轻叹了一口气，道：“这柄斧头的材料，是我和叶永默一起寻得的，它的设计叶永默也只跟我一个人说过，那时候的设计还不是全部，只是他的一个设想，没想到有一天，我竟然能见到成品。”
<br/>　　叶永阳看着修罗战斧，就像是在看自己的孩子，眼中的泪花闪烁，有种无尽的沧桑感。
<br/>　　“听我们老大说，这黑斧的材料很特别？”陆玟丞有些试探性的问道。
<br/>　　“当然。”叶永阳收回手，眼中的缅怀也转变为了一丝狠厉：“它可以杀神秘家族的成员，我是说正统的成员，包括嫡系。”
<br/>　　此话一出，陆玟丞当即震惊不已，眼珠子瞪到老大，顾凉晨则是满眼的不解和疑惑。
<br/>　　“我们的星河有法则，普通人无法伤到那几门人员，你们可知道？”叶永阳又问。
<br/>　　陆玟丞连忙指着旁边的顾凉晨道：“我知道，他不知道。”
<br/>　　顾凉晨呆愣原地，表情呆傻：“你们刚刚是说，我们这些人，是永远无法杀死神秘家族的成员对吗？无论有多强，战力有多高，都杀不死，更伤不到？或者说，碰不到他们？因为法则，星河法则？”
<br/>　　有太多的东西，长期在外的顾凉晨都一无所知。
<br/>　　“是这样没错，我们也是刚知道不久，这也是高级机密。”陆玟丞解释着，又看向叶永阳：“你刚刚说这柄斧头，可以杀死神秘家族的成员，也就是能够打破那个法则？”
<br/>　　“对。”叶永阳肯定道：“重点就是制造这柄斧头的材料，不过这斧头有灵性，我一看到它知道，它只认楚涵一个主人，所以哪怕是别的人能拿起它，也无法发挥出它的全部能力。”
<br/>　　“所以咱老大杀掉的那个罗家人，最重要的原因是这柄斧头？”顾凉晨终于找到了重点。
<br/>　　叶永阳笑道：“第一次听你说的时候，我也很惊讶，还以为楚涵的来历有异常，现在一想如果是叶永默将这柄斧头赠给了他，加之他又气运极强，那么这一切也都说得通了。”
<br/>　　说罢，叶永阳又看向了旁边的那张床：“这就是楚涵么？他给这把绝世之斧起了一个什么名字？”
<br/>　　顾凉晨和陆玟丞双双对望一眼，而后异口同声的回答：“修罗。”
<br/>　　“修罗。”叶永阳念叨着，而后满是皱纹的脸笑开了花：“好名字啊，对于神秘家族的人来说，修罗斧就是给他们制造的屠宰场。”
