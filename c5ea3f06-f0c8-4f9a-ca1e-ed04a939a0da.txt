　　这根本不是一个级别的战斗，盘古颇有种“一幡在手，天下我有”的气概，只手轻轻一划，数道凌厉的混沌剑气就劈在众魔神身上，连一个近身盘古的都没有，更别说破开混沌青莲防御了。
<br/>　　一场屠杀，都是修为在混元中期境界的魔神，被劫煞之气迷了心志之人，剩下的就只有排名前十的魔神了，几人都有混元后期的修为了，元释更是混元圆满。元释一挥手，将周围的魔神尸体收走，已经到了最后一刻了，没什么可掖着藏着的。
<br/>　　周围几个还没有被劫煞影响的魔神虽然有些奇怪元释的行为，但并没有多说什么？最多只是心里留下一个作死的印象。人都要死了，这人怎么还在收魔神尸体？真是要钱不要命。要是在平时，众魔神定会嘲笑一番，可如今，决战就在眼前，那还会管其他。
<br/>　　对此，元释也不予理会，只管收集魔神尸体，反正他已留有后手，就算魔神之躯身死，但他也不会正真的死亡。这些魔神尸体，在后世，可是想要都没地方去找！
<br/>　　十大魔神对盘古大神，谁输谁赢？
<br/>　　……
<br/>　　混沌之中，声响不断，天地还在开，在盘古开天斧的强大力量之下，不断的扩张，越来越广，四周的混沌气流不断地被这新生的宇宙吸收着，并演化为本世界的地水火风，化作本世界的混沌灵气，他只是刚刚出生，未来如何，一切都是未知的。
<br/>　　而，在这样一个新生的世界内，十大混沌魔神正在围攻一个大汉，大汉头顶混沌钟，脚踏混沌青莲，手持盘古幡，面对中魔神的围攻，显得应付自如。
<br/>　　“日月如梭，岁月如箭！杀！吾乃时间魔神时辰！”一个银发男子，岁月凝箭，直指盘古射去，这是混沌排名前五的时间魔神时辰。
<br/>　　“上下四方，空间切割！杀！吾乃空间魔神扬眉！”一个白发摇曳的男子，一手空间切割之术直击盘古。
<br/>　　“无量圣魔，魔道屠天！吾乃魔祖！”一千手千眼，浑身黑雾包围的魔神也一剑划破虚空混沌青莲的防御，斩向其上的盘古。
<br/>　　“生死有限，轮回无边！吾乃轮回之主！”一个霸道的魔神，一拳轰出，六道模模糊糊的生死大门出现在一起，缓缓旋转，一个不小心就会被陷入其中，生死不知，六道融合，轰向盘古！
<br/>　　“无双造化，造化剥夺！吾为造化之主！”一个极为漂亮的女子魔神出声，声音极为悦耳动听，漂亮的脸蛋上充满了怒气，一个造化大大的造化池出现在她上方，对准盘古，轰然砸去。池口传来一个强劲的造化法则，好似剥夺一切造化。
<br/>　　“恨天恨地，毁天灭地！吾为毁灭之主！杀！”一股强烈的毁灭法则从一直长枪枪尖传出，毁灭一切。
<br/>　　“何因？何果？何果？何因？因果轮回，因果业杀！吾为因果之主元释！”人首蛇身背生双翼的元释一步踏出，混元大罗金仙十一重天的境界暴露无遗，强悍的修为只有盘古比他高，到了此时，众人中也只有他还没有被劫煞之气所影响，其余几人都已杀红了眼。
<br/>　　“怎么可能，他的修为竟然比吾等还高，只比盘古低？！”
<br/>　　“嘶！怎么可能？！”
<br/>　　“嘶！这——这是因果魔神？！”
<br/>　　“混蛋！他比我后出世，怎么可能修为比我还高？！”
<br/>　　……
<br/>　　几大魔神脸上都露出了难以掩饰的惊讶，或嫉妒，或羡慕，或冷笑……
<br/>　　对此，元释不予理会，直接向盘古发起挑战：
<br/>　　“盘古开天证道，破灭混沌是为因，灭杀三千魔神是为因，有因就有果，有果必有因，既有因，何不出果？此时不出，更待何时？吾为因果之主，听吾号令，以三千魔神对盘古氏之执念为业，强结因果术，因果业杀！杀——！”
<br/>　　“因果不灭，业力不尽！”
<br/>　　元释一出手就不再保留，直接以三千魔神对盘古的滔天恨意为因果业力，强行与盘古结因，一股股粗大无比的黑白因果线于虚空之中延伸而出，没入盘古体中。
<br/>　　顿时，盘古周围的虚空之上，出现了一朵朵黑色的火焰，极致的黑，给人一种极度森寒的气息。近距离观察，给人一种难以形容的寒气从头传到脚，像是灵魂被冻结。
<br/>　　“嗞嗞嗞嗞！”
<br/>　　漆黑晶莹的黑色火焰，黑曜石一般的颜色，又透露着一种淡淡的红光，一朵又一朵，如同黑色妖艳的花朵在虚空中盛开，仅仅片刻，虚空中到处就都是黑色火焰，每一朵都不大，只有脑袋般大小，却让人全身寒毛为之倒立。
<br/>　　而且，在这些火焰之中，还有一股愿力，只是一种众生的意志仿佛就像信仰之力。不过，它又与信仰之力有着本质的区别。因为信仰之力，给人一种纯洁神圣的感觉，而之中黑色原力，给人的却是极度阴冷仇恨，向所有生灵无穷无尽的仇恨与诅咒汇聚而成，这是来自中混沌魔神的仇恨！
<br/>　　“因果业力！”
<br/>　　……
<br/>　　“哈哈哈！好！来得好，还有你这样的高手，因果之道么？看我力之极致，粉碎苍穹！因果业力又如何？看我如何粉碎它！”盘古不惊反笑，一如既往的霸道，没有将任何人放在眼中，这些人也算是混沌魔神中排名前十的几位了，自己正想跟他们较量一番。但凡阻我道者，不管是谁，只有一个结果，那就是——死！
<br/>　　轰隆——！
<br/>　　虚空震荡，混沌暴动，而虚空中的黑色火焰也没能将盘古怎么样，盘古的魔神真身到底是达到了不朽阶段了。而且随着战斗的持续，虚空之中的黑色火焰也逐渐在减少。显然，盘古也在强行断去一些因果。
<br/>　　一般来说，实力越强，他们之间的因果联系也就越紧密，但，也有强大的修士强行斩断因果，当然，这是损人损己的行为，作为修士，更看重因果，因果不清，大道难成！死人虽然也有因果，但已经很淡了，除非是彻底死亡，有关他的一切痕迹都不留下，不然，他们之间必有因果。
<br/>　　因果未了，又如何成道？
<br/>　　因而，利用死去的混沌魔神来施展因果业杀，还是存在弊端的，盘古实力强大，活着的魔神尚且不怕，死了的就更不怕了。所以，想要挣断这种关系的因果，盘古还是可以做到的。只不过，盘古还没有强大到强行断去因果而不受反噬的地步。
<br/>　　元释利用的就是这一点，你盘古是厉害，能够强行断去因果，但是只要你还没有证道成功，你再厉害还是没有逃脱因果轮回，虽然反噬是很小，但如此多的因果加在一起，就不信你没有受到影响。
<br/>　　……
