　　“哦，三生元神？哪三种元神？”龙帝面带惊疑的问道，三生元神不算多逆天，关键要看是哪三种元神。
<br/>　　嗡嗡嗡~~
<br/>　　苏莫起身，意念一动，头顶虚影升腾，分别是一团璀璨的绿光、一柄犀利的银色长剑、一股不断变幻的气流。
<br/>　　这三种元神，分别是天梦元神、万化神剑元神，以及千幻元神。
<br/>　　“龙帝请看，我有天梦元神、万化神剑元神、千幻元神，这三种元神分别排在太古武魂榜第二十二、第十四、第十二，均是强大无比的元神！”苏莫沉声说道。
<br/>　　三种元神展露而出，吸引了大殿之中所有人的目光，这可全部都是超级元神，普通的武者，只要任有一种，便可成为顶级妖孽，有两种便是顶级的万古天骄。
<br/>　　“果然厉害！”
<br/>　　“居然有三种这么强大的元神！”
<br/>　　“真的亘古未见的天才！”
<br/>　　很多龙族的长老，面露惊叹之色，纷纷被苏莫的天赋震惊，在神域虽然拥有三生元神之人不算很少，但是，大多数都是三种普通的元神，能有三种如此强大元神之人，见所未见闻所未闻。
<br/>　　除了龙帝之外，只有龙腾面色平淡，他知晓这只是苏莫的冰山一角而已。
<br/>　　事实上，苏莫有多少元神？有那些元神？他都不是很清楚了。
<br/>　　“屠霄，你果然逆天，不过，仅凭这些远远不足以击败同阶的祖龙！”龙帝面无表情的说道，这屠霄的天赋的确是万古罕见，但是，以这种天赋想要击败龙灵幻化的同阶的祖龙，无异于痴人说梦。
<br/>　　“屠霄，你还有和厉害的天赋，全部展现出来吧！”大长老朗声道，他也知道仅凭这种天赋，根本不足以打败同阶的祖龙，因为他清楚祖龙的战力是多么可怕。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即立刻收回了三大元神，丝毫不敢过多逗留在外。
<br/>　　因为，他这三种元神之中，千幻元神并不是本命元神，他只是在其内混杂了混沌源力，才显得和本命元神一般无二。
<br/>　　好在，这些人并没有细查，毕竟，所有武者身上的元神都是本命元神，没有人会想到他的不是本命元神。
<br/>　　收回三大元神之后，苏莫体外陡然迸射出夺目的金光，这金光并不纯粹，其内好似极为斑驳，有绿光、白光等等光芒掺杂其中，只不过都融合在了一起，并无法仔细分辨。
<br/>　　这是苏莫催动了血脉之力，爆发出的血脉体质的异像。
<br/>　　“龙帝、大长老，我还有血脉体质，这种血脉之力见所未见，连我自己都不清楚是什么血脉，但是，可以肯定的是，此种血脉之力威力远超至尊战体。”苏莫郑重的说道。
<br/>　　“哦？”
<br/>　　苏莫此言一出，全场皆惊，不知名的血脉之力，比至尊战体还要强大？
<br/>　　众人并没有怀疑，因为他们能清楚感知到苏莫的血脉之力，如同咆哮的太古巨兽，不断的在体内冲荡，好似随时就会冲出体外。
<br/>　　这让众人震惊，此种血脉之力，不是至尊战体，但是极度强大。
<br/>　　“龙帝，这血脉之力，感觉比之永恒神宫弟子公孙长绝的永恒之体，也丝毫不弱啊！”大长老面带惊叹的说道。
<br/>　　“的确！”龙帝微微颌首，眼眸之中压不免露出了一抹惊色，三大超强元神，逆天的血脉体质，这屠霄堪称妖孽中的妖孽，天骄中的天骄。
<br/>　　不过，略一沉吟，龙帝又道：“屠霄，你的天赋让本帝震惊，但是，以你这种天赋，战力不会超过同阶的祖龙，你具体是如何战而胜之的？”
<br/>　　“禀龙帝，我乃是太古武修，当初修炼了八八六十四枚神丹，最终融而为一，又修习混沌源力，源力相当浑厚，才能战而胜之！”
<br/>　　苏莫郑重的说道，言罢，他又补充道：“当然，能战胜龙灵所化祖龙，最主要的是，祖龙力量有限，禁不起消耗，三招之后实力便大大下降。”
<br/>　　苏莫信口胡诌了起来，首先，他当初是练就了四十九枚神丹，根本就不是六十四枚神丹。
<br/>　　其次，没有人进入过第十层，所以，也没有人知道第十层的情况，他完全可以按自己的想法来说。
<br/>　　“原来如此！”
<br/>　　大长老面带惊叹之色，语重心长的道：“屠霄，你的天赋可谓是冠绝神域，日后为我龙族效力，龙族不愧亏待你！”
<br/>　　“晚辈定然竭尽全力，回报龙族！”苏莫恭敬的说道。
<br/>　　“嗯，如此甚好！”大长老微微颌首，心中非常满意，这屠霄得到祖龙精血之后，能将精血给予龙腾，便可见其与龙腾关系非凡。
<br/>　　所以，他倒是不担心，屠霄会离龙族而去，或者说背叛龙族。
<br/>　　龙帝面无表情，他也没有去深究，屠霄的天赋有没有什么隐藏，因为这没有意义，仅拼现在的展露的天赋，已经差不多可以冠绝神域了。
<br/>　　他在意的是，这屠霄和龙腾关系莫逆，将来必然是龙腾的左膀右臂。
<br/>　　“屠霄，你如此天赋，若是无人教导，修炼之路很容易走入歧途，本帝愿意收你为弟子，你可否愿意？”龙帝面带威严的说道。
<br/>　　苏莫闻听此言，顿时心中一惊，对方想收他为弟子，绝对不是真心看重他的天赋。
<br/>　　“禀龙帝，您的好意，晚辈怕是无法接受！”苏莫心中虽惊，但依旧面色平静，躬身行礼道。
<br/>　　“怎么？你认为本帝不够资格做你师尊吗？”龙帝闻言，顿时脸色一沉，一股无形的冷意，从他身上散发而出。
<br/>　　若是屠霄不知好歹，他正好可以借此机会，重罚屠霄，打压龙腾。
<br/>　　大殿之中寂静无声，龙帝主动开口，破例收徒，屠霄居然拒绝了，这实在是有损龙帝的颜面。
<br/>　　若是屠霄没有合适的理由，估计难免要遭受惩罚了。
<br/>　　“禀龙帝，屠霄已经有了师尊！”龙腾急忙开口，为苏莫解围。
<br/>　　“哦？何人是谁，能教导出屠霄这种弟子？”龙帝疑惑的问道。
<br/>　　“呵呵，屠霄的师尊，就在龙族，只是修为很低，早已被屠霄超越了！”
<br/>　　龙腾无奈一笑，继续道：“此人，便是当年和我一起来龙都城的龙鏖！”
<br/>　　“是的，晚辈曾经向师尊发誓，此生唯拜入他一人门下，不会改换门庭，所以，龙帝的好意，恕晚辈难以从命了！”苏莫恭敬的说道。
<br/>　　龙帝闻言，沉默不语，眼眸深处，有淡淡的冷意弥漫。
<br/>　　不过，他并未发怒，因为屠霄的理由合情合理，他也不好强行定罪。
<br/>　　“既然如此，那真是可惜了！”
<br/>　　龙帝轻叹一声，随即环视众人，朗声道：“今日之事，到此为止了，大家都散了吧！”
<br/>　　“我等告退！”
<br/>　　众人闻言，纷纷起身行礼，随即鱼贯而出，离开了天龙大殿。
<br/>　　苏莫也没有久留，和龙腾一起，大步离开了。
<br/>　　大殿之中，除了龙帝之外，唯有一人没有离去，此人是一位面色略显黝黑的高手老者，正是二长老。
<br/>　　“龙帝，此事难道就这样了？”
<br/>　　待所有人离去之后，二长老面色郑重，继续道：“龙腾将来必定会威胁您的地位，还有那屠霄，如此天赋之前却甘愿为奴，来路不明！”
<br/>　　龙帝闻言，沉默不语，深邃的眼眸，凝视着大殿之外，不知在想些什么。
<br/>　　二长老见此，微微皱眉，不过，他也不好在追问，就这么静静的望着龙帝。
<br/>　　“龙柏，你最大的渴望是什么？”
<br/>　　良久之后，龙帝淡淡的问道，他的目光依旧望着大殿之外，仿似思绪早已离开。
<br/>　　龙柏，便是二长老的真实名讳。
<br/>　　“最大的渴望？”二长老闻言一怔，随即叹了口气，道：“自然是突破桎梏，成为至强者。”
<br/>　　他的修为，停留在高级神王，早已千万年之久，对于他来说，最大的渴望就是突破。
<br/>　　但是，他心中清楚，自己这一生，是不可能突破了。
<br/>　　他会随着纪元的终结，而陨落在荒界的天地之间，彻底的消失无踪。
<br/>　　“这是你的渴望，又何尝不是本帝的渴望！”
<br/>　　龙帝声音悠悠，继续道：“祖龙精血是我们唯一的机会，而现在，龙腾将是我们唯一的机会。”
<br/>　　二长老闻言，眼眸微亮，他明白了龙帝的意思，只是这样未免显得有些狠辣了，将来洪荒祖龙若是回来，该如何交代？
<br/>　　“龙腾以后的修炼之路，会一帆风顺，我们静等便是！”
<br/>　　言罢，龙帝略一沉吟，道：“至于那个屠霄，你去调查一番他的底细，此人诡异。”
<br/>　　“明白！”二长老点了点头。
<br/>　　“去吧！”龙帝挥了挥手，随即，他自己便也起身离开。
