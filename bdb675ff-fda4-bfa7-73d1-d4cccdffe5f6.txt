　　就这样，叶尘母子在苏岚家住了下来。
<br/>　　一晃眼，一个多月过去了，
<br/>　　叶尘在这一个多月的时间里，几乎每时每刻都在修炼，但是因为身体太小的缘故，无法承载太多真元，所以只能稳扎稳打，不过一个多月下来，也已经勉强达到了炼气三重境界。
<br/>　　加上他已经领悟的四十八条法则之力，即便是面对炼气境后期的修士，他也已经有抵抗之力。
<br/>　　要知道，在地球这种地方，哪怕是炼气境后期的修士，在普通人的眼中，就已经是神仙般的大人物了。
<br/>　　若是让人知道，一个只有三个月不到的婴儿，已经具备如此恐怖的能力，恐怕非得把人惊的下巴都掉下来不可！
<br/>　　叶尘也是怕吓到母亲和苏岚，自然不敢展露自己惊人的实力，依旧老老实实呆在婴儿床上。
<br/>　　不过即便是如此，叶尘一直不哭也不闹，表现的实在过于冷静，而且个头也窜的飞快，不到三个月的身体，就和一岁左右的幼儿相当，着实让林婉容和苏岚两人心惊不已，直到将叶尘带到医院，经过一番检查，确认他的身体指标一切正常，这才算勉强接受了这一事实。
<br/>　　“容容，我有一种感觉，小尘将来恐怕很不简单呢！”
<br/>　　苏岚望着婴儿车里的叶尘，一边轻轻拍着怀里熟睡的苏蔓，悠悠开口道。
<br/>　　林婉容轻轻抚摸着叶尘的脑袋，脸色露出一抹欣慰之色，旋即又轻轻叹了一口气，
<br/>　　“其实我倒是更希望，他将来可以做个普普通通的平凡人！”
<br/>　　苏岚秀眉微微一蹙，道：
<br/>　　“容容，虽然有些话你可能不爱听，但是我还是要提醒你，秦月娥那个女人，可不是什么善茬！”
<br/>　　林婉容听到这话，脸色顿时微微一变，
<br/>　　“我都已经主动退出，不再和他争夺天歌，她还想做什么？”
<br/>　　苏岚叹了一口气，再次望向叶尘，道：
<br/>　　“可是小尘，毕竟是叶天歌的儿子啊！他可是叶家的嫡系子孙！现在还小，或许没有什么，可是等到他长大了，可是有继承叶家的资格的！”
<br/>　　林婉容的脸色愈发难看，
<br/>　　“可是我们母子又没有打算去争什么……”
<br/>　　苏岚道：
<br/>　　“你说你不争，秦月娥就一定会相信？豪门世家的争夺有多残酷，你应该比我更清楚吧？”
<br/>　　林婉容默然无语，美眸中顿时浮现出一抹浓浓的担忧之色。
<br/>　　叶尘听到两人的对话，看到母亲脸上的担忧之色，真的忍不住想说，只要给我三年时间，即便是那秦家老祖秦踏天，我也能杀给你们看！
<br/>　　一转眼，又过去了一月有余，已经到了第二年的春天。
<br/>　　此时叶尘虽然还不足半年，可是个头却已经赶上近两岁的孩童，而且修为也提升到了炼气四重境界，其实早已经可以健步如飞，只不过考虑到林婉容和苏岚的接受能力，叶尘佯装才刚学会走路的样子。
<br/>　　即便如此，就已经让两人惊愕不已，感觉匪夷所思。
<br/>　　倒是苏蔓，还是一个孩童，并不懂得其中的道理，高兴的拉着叶尘四处玩耍，以至于让叶尘修炼的时间也变少了不少，让他郁闷不已。
<br/>　　这一天，傍晚时分，
<br/>　　叶尘正陪着苏蔓在院子里玩耍，更准确的说，是叶尘看着苏蔓在玩，他则一直神游在外，思索着如何将四十八法则与时空法则相融合。
<br/>　　突然，
<br/>　　“砰！”
<br/>　　大门忽然别人一脚踹开！
<br/>　　紧接着，
<br/>　　一群凶神恶煞之人，从外面冲了进来。
<br/>　　苏岚和林婉容听到动静，也顿时纷纷冲了出来，
<br/>　　“你们是什么人？想要干什么？”
<br/>　　那群人看到苏岚和林婉容，顿时一个个两眼放光，
<br/>　　“啧啧！早就听说朱成功这老小子，有一个如花似玉的老婆，没想到今天一见就见到了两个，这老小子当真是艳福不浅呐！”
<br/>　　为首的一名壮汉，脸上挂着淫笑，呵呵笑道。
<br/>　　苏岚顿时眉头一皱，将林婉容挡在身后，道：
<br/>　　“我是朱成功的妻子，你们要是想找朱成功的话，他不在，请回吧！”
<br/>　　那壮汉嘿嘿一笑，又往前凑近了几步，道：
<br/>　　“我们的确是来找朱成功的，不过他如果不在的话，那找他的老婆也是一样的！”
<br/>　　说到这里，壮汉微微一顿，从怀里取出了一张有些枯皱的纸，冷然道：
<br/>　　“你老公前些日子，在赌场输了个精光，向我大哥借了五十万，而且还将你们的这个房子，抵押给了我大哥，这个是他立下的字据，如今已经到了还款的日期，他却玩消失，我们找不到他人，也只好来收房子了！”
<br/>　　苏岚听到这话，又望了一眼那张字据，顿时脸色大变。
<br/>　　那字据，的确是朱成功所立，而且苏岚很清楚自己丈夫的德行，自从一年前染上了赌瘾，已经输掉了一大半的家产，做出这种事情，也一点不奇怪。
<br/>　　“这个混蛋！他怎么可以这样！这让我和女儿怎么活？”
<br/>　　苏岚脸色惨白，痛心不已。
<br/>　　那壮汉顿时桀桀一笑，道：
<br/>　　“美女，你这丈夫，的确不是什么好东西！不如你以后跟我，房子的事情，我可以和我大哥商量商量，如何？”
<br/>　　说完这话，壮汉又往前靠近了一些，脸上的淫色顿时越发浓郁。
<br/>　　苏岚和林婉容两人见状，顿时全都脸色一变，不由得连连倒退，
<br/>　　“你想干嘛？别过来！”
<br/>　　壮汉再次桀桀一笑，
<br/>　　“我想干嘛？看到这么漂亮的女人，我当然是想……”
<br/>　　嗖！
<br/>　　壮汉的话，才直说到一半，突然一道破空声响起。
<br/>　　砰！
<br/>　　一颗小石子飞射而至，瞬间击中了壮汉的脸颊。
<br/>　　这颗不起眼的小石子，竟然威力无穷，丝毫不亚于子弹！从壮汉的左脸颊击入，从右脸颊飞出，竟然将他整个脸庞洞穿！
<br/>　　“啊！！！”
<br/>　　那壮汉先是愣了片刻，继而捂着自己的脸，发出杀猪般的惨叫。
<br/>　　而他身后的那几名小弟见状，顿时一个个全都吓傻了，纷纷朝四周望去，可是整个院子里，除了苏岚和林婉容之外，便剩下角落里的两个两三岁的小孩，哪里还有其他人？
<br/>　　“跪下道歉！否则，死！”
<br/>　　就在众人惊疑不定之时，一道仿佛从九天之上传来的威压声音，陡然在院子里响起，顿时让众人更加心惊胆寒。
<br/>　　但是众人没有注意到的是，那个看上去只有约莫两岁左右的孩童，却无比的淡定，而且眼眸深处，透着一抹和他年龄完全不符合的威严！
