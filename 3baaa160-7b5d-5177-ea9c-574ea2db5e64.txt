        &lt;h3&gt;第三卷第二百九十六章  凌辰到西城2&lt;/h3&gt;
<br/>
<br/>        凌辰在下午五点半到了陈子昂指定的包间，也就是他们住宿的十二楼，陈子昂和江俞轩已经在包间等他们了。
<br/>
<br/>        董亚子见到陈子昂欢快的迎上去了，凌辰发现了一个问题，陈子昂很受人欢迎，不但是男孩子喜欢，女孩子更喜欢，这让他有了那么一丝丝危机感，董亚子才见陈子昂两面，就如此亲热，那么宁雅呢？
<br/>
<br/>        凌辰的心里立马拉起了警铃，他需要加快步伐，赢得宁雅的心，将宁雅的牢牢固定在他的身边。
<br/>
<br/>        此时的凌辰哪里还有那傲娇冰冷的感觉了，爱情真的能让一个人改变。
<br/>
<br/>        他们都见过面，也不会尴尬，四个人就算是小范围的聚会，董亚子是第一次见江俞轩，陈子昂为董亚子做完介绍，温柔的说：“俞轩是我们公司的智囊，有他在我就是天马行空，天塌下来都是他顶着的。”
<br/>
<br/>        董亚子和凌辰对视一眼，董亚子似乎在说：“毫不犹豫的被塞了一把狗粮。”
<br/>
<br/>        凌辰心里一松，还好陈子昂情有所钟，要不然陈子昂就是他潜在的威胁。
<br/>
<br/>        “子昂姐姐，好羡慕你有江总替你担着一切，人家说成功的男人身后都有一个默默付出的女人，同理，成功的女人背后一定有一个默默付出的男人，江总这是有多爱你啊，甘愿当你的绿叶。”董亚子口无遮拦，她并不了解陈子昂和江俞轩的关系，只因为听了陈子昂的那句话便觉得俩个人肯定是一对。
<br/>
<br/>        陈子昂微微有些尴尬，江俞轩却听得满心欢喜。
<br/>
<br/>        凌辰冷眼相看，不发一言，他也看出来俩个人关系不一般，但是俩个人从来都没有说出他们是什么关系。
<br/>
<br/>        “董总，你知道江总是什么身份吗？”陈子昂笑着问。
<br/>
<br/>        “管他是什么身份，只要他爱你就好了，身份什么的重要吗？有几个男人愿意默默的为女人付出？”董亚子竟然叹了口气羡慕的说，不知道刘诚之会不会和江俞轩一样？
<br/>
<br/>        江俞轩已经掩饰不住脸上的笑意了，陈子昂，连一个刚刚才见面的人都知道我爱你，我不相信你的心感受不到。
<br/>
<br/>        陈子昂张了张嘴巴，她竟然觉得董亚子的话很在理，而且她还觉得如果她再解释下去，江俞轩那张脸上肯定会开出花来。
<br/>
<br/>        江俞轩不去甸城是最正确的决定，没有未来的事情就不要却奢望了。
<br/>
<br/>        “凌总，你们这次来是视察工作？”陈子昂转变了话题。
<br/>
<br/>        “是的，西城市场很大，我们来看看，再加上快三一五，我们配合厂家和有关部门打假。”凌辰淡淡的应到。
<br/>
<br/>        “西城市场确实很大，但是也是良莠不齐，你们确实也应该视察一下。对了，我记得上一次我哥说他在有一家商店里拿了一些商品，有一半是假的，拿要去换货，商家竟然不承认，当时他很生气，主要是请客丢人。”陈子昂突然想起来陈子霖说有一次请客的事情。
<br/>
<br/>        尴尬的是，一箱商品一半真一半假。
<br/>
<br/>        “能告诉我们是哪一家公司或者是地址吗？”凌辰抓住了重点。
<br/>
<br/>        “我记不清楚了，好像他说在红砖路上，商铺还很大。要不我打电话问一下？”陈子昂征求凌辰的意见。
<br/>
<br/>        “也好，那就麻烦陈总了，我们先期调查一下。”凌辰听说在红砖路上，他在脑子里过了一下西城有几家代理商几家经销商他是清楚的，红砖路上有一家，好像是很大的一家，不但有他们的品牌还有别的品牌。
<br/>
<br/>        只是这几家他基本都认识，如何能找到证据还是一个问题，陈子昂既然说出来，也不可能不信，也不可能全信。
<br/>
<br/>        “陈总，你们为何不一次多批一些商品，反正你们经常用。”凌辰笑着说。
<br/>
<br/>        “哈哈哈，凌总说得很对，一次性多批一些，价格还能便宜，我明天就让公司的人去红砖路批他个几十箱回来。”陈子昂笑着说。
<br/>
<br/>        “几十箱也没有必要。”凌辰感觉他的心思被陈子昂戳破了微微有些尴尬。
<br/>
<br/>        “怕什么，有问题了直接找凌总换，喝不完了找凌总退，就拿几十箱回来。”江俞轩也笑着说。
<br/>
<br/>        “退换都是没有问题的，你们尽管去拿，要不就当是我请客。”凌辰也笑了，和聪明人打交道真的是爽。
<br/>
<br/>        “那可不行，我们在最困难的时候凌总可是出手相助了。”陈子昂露出了小白齿，相互帮助是再正常不过的了。
<br/>
<br/>        “我那是为了宁雅。”凌辰实话实说，当初要不是宁雅，他也许是不会出手的，不过因为他的出手相助，陈子昂是念了他的好。
<br/>
<br/>        而这一次陈子昂也算是帮了他，这一次陈子昂算是还了凌辰的情但确将凌辰和陈子昂紧紧的捆绑在了一起。
<br/>
<br/>        “哈哈哈，无论你是为了谁，你都帮助了我们公司，这份情我们都记得。”江俞轩拿起茶杯和凌辰碰了一下。
<br/>
<br/>        “我听宁雅说伯伯生病了，手术顺利吧？”凌辰主动转变了话题。
<br/>
<br/>        “手术很顺利，谢谢关心，你和宁雅在杭州见面了？”陈子昂已经知道凌辰到杭州了，王律师将宁雅在杭州的一举一动都汇报给她了。
<br/>
<br/>        “见了，很抱歉，让她翘了半天班，陪着我逛了半天街。”凌辰一脸的温柔相。
<br/>
<br/>        “就仅仅是逛街？”江俞轩故作吃惊的看着凌辰。
<br/>
<br/>        “那可不，我拉着手逛了半天街，有些人追了几年估计连手都没有拉过吧？”腹黑如凌辰怎么会不将江俞轩怼回去。
<br/>
<br/>        “你们两人认识？”董亚子听俩个人的对话，感觉他们的老相识。
<br/>
<br/>        “认识好久了，久得我差点都忘记他是谁了。”江俞轩撇撇嘴，凌辰这几天真的是够嘚瑟了，不就是拉拉手有什么了不起，整天情情爱爱的羞不羞耻。
<br/>
<br/>        陈子昂心情还不错，凌辰和江俞轩互怼一下还真能缓和压抑的心情。
<br/>
<br/>        “凌总，明天我安排人去拿商品，你来我公司库房看看如何？”陈子昂看着凌辰，如果明天的货物有问题，那么到是可以帮凌辰一把，货物没有问题凌辰也会安心，最起码可以向厂家交代。
<br/>
<br/>        “哈哈哈，你这是想将那些货物都搬到你的库房？”凌辰哈哈哈大笑。
<br/>
<br/>        “不是不可以啊。”
<br/>
<br/>        “那就明天见。”
<br/>
<br/>        四个人在说笑中将凌辰考察市场的问题解决了，凌辰和陈子昂哪里知道，阴差阳错中他们被绑在了一起，在西城，在甸城，似乎有一根无形的线将他们牵引住了。
