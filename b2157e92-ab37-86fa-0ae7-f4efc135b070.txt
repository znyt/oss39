<br/>　　这一团团的火焰，笼罩在整个山谷之中，每一团火焰之中，都蕴含这狂暴灭世之力。

<br/>　　曾经炎帝最为强横的手段，如今小炎帝继承了炎帝的力量传承。
<br/>　　这一招，也被他继承了下来。
<br/>　　这灭世帝炎一出，轰然间的凝聚起来，短短的瞬息之间，一尊足有千里之长的巨大火龙，被凝聚了出来。
<br/>　　灭世火龙！
<br/>　　在这一条灭世火龙身上散发出来的气息，不亚于真正的九劫古帝，在加上小炎帝，几乎相当于是拥有这两尊九劫古帝的实力。
<br/>　　这才是小炎帝的真正实力，哪怕小炎帝只是八劫古帝的境界，但是他的实力全开，几乎足以媲美两尊九劫古帝。
<br/>　　五行刀祖那样的九劫古帝，在小炎帝的面前，都要被小炎帝所击败。
<br/>　　无数五灵宗圣子，都是瞪大了双眼，似乎是第一次见到小炎帝的真正实力。

<br/>　　在这暴怒之下，小炎帝才展现出了他真正的恐怖实力，也是对楚风眠，起了必杀之心。
<br/>　　“你该死！”
<br/>　　小炎帝突然之间，悍然出手，那灭世火龙，也在同时出手了，猛然之间向着楚风眠冲了过去。
<br/>　　一人一龙，悍然出手，如同两尊真正的九劫古帝，在这一同，向着楚风眠发起了攻击。
<br/>　　不，这一人一龙，甚至是比起两尊九劫古帝同时出手，都要厉害的多。
<br/>　　九劫古帝二人联手，心不齐，自然无法发挥出全部实力，但是这一人一龙，却是同心同意。
<br/>　　一齐出手的威力，比起两尊九劫古帝同时出手，都要更加的可怕。
<br/>　　“灭世火柱！”
<br/>　　小炎帝怒喝一声，在他的手心之中，浮现出了一团火焰，向着楚风眠扔了过来，这火焰在接近到楚风眠的一刻，突然爆炸开来，化为了漫天火海。
<br/>　　众人只见到，这小炎帝一出手，刹那之间楚风眠所在的方圆数十里，都是化为了一片火海。
<br/>　　在这火海之中，哪怕是九劫古帝，仿佛都要被燃烧殆尽。
<br/>　　与此同时，那灭世火龙也动了，从天空之上，向着楚风眠发起了进攻，两道巨大无比的龙爪，对准楚风眠拍了下去。
<br/>　　这两道龙爪轰击而下的瞬间，也化为了纯粹的火焰，一时间楚风眠的身形，都被无数火焰所笼罩。
<br/>　　“这一招之下，九劫古帝都难以逃脱。”
<br/>　　一名五灵宗圣子，喃喃自语道。
<br/>　　“不愧是力量传承，小炎帝继承了炎帝的武道，已经是远远超越了同境界的武者，甚至比起九劫古帝，还要恐怖！”
<br/>　　力量传承，一直以来都是圣龙秘境最大的底牌之一。
<br/>　　那些留下力量传承的强者，几乎都是在半步天人境之中，都是巅峰的存在，比起寻常的半步天人境，还要恐怖。
<br/>　　继承了炎帝的力量武道。
<br/>　　小炎帝的武道，比起九劫古帝，甚至是一些半步天人境的强者，还要可怕。
<br/>　　哪怕现在小炎帝，只是八劫古帝的境界，但是打出的这一招轰击，比起真正九劫古帝打出来的轰击，还要可怕的多。
<br/>　　一时间无数五灵宗圣子都是屏息，看着火焰之中，眼神之中已经是淡然了不少。
<br/>　　哪怕是楚风眠的实力在逆天，承受这样的轰击，只怕也是必死无疑。
<br/>　　可就在这时，那一片火海，突然是被一道龙爪撕裂开来。
<br/>　　这一道龙爪，仿佛是不惧怕任何的火焰，熊熊燃烧的火焰，都被全部撕裂开来，旋即一道身影，从其中走了出来。
<br/>　　“楚风眠？”
<br/>　　“他还没死？”
<br/>　　“不……这怎么可能？”
<br/>　　看着楚风眠的身形走出来的一刻，无数五灵宗圣子的目光都是无比惊骇了，可接下来看清楚的一刻，他们都有愣住了，许久说不出话来，惊骇到了极点。
<br/>　　在楚风眠的身上，没有一点点的伤痕，在这样的火焰之下，如今的楚风眠，不止是全身而退，甚至在身上，都没有一点的伤痕。
<br/>　　“所谓炎帝的灭世帝炎，不过如此。”
<br/>　　楚风眠一步踏出，在他身边的火焰，都是被全部踏灭。

<br/>　　楚风眠目光看向小炎帝，在他的背后，惊天的剑意爆发而出，剑意入芒，冲天而上，化为了一把惊天神剑一般。

<br/>　　足以灭杀九劫古帝的气息。
<br/>　　在刚刚，哪怕是在灭杀金通的时候，楚风眠都没有动用全力。
<br/>　　甚至是在跟小炎帝刚刚交手的时候，楚风眠都仿佛是在戏耍这小炎帝。

<br/>　　小炎帝看到楚风眠拔剑的一刻，也感受到了楚风眠身上恐怖到了极致的力量，眼神之中不由的露出了几分惊骇。

<br/>　　一道剑光，穿越了一切，无数杀招，漫天火海，都无法抵挡这一道剑光，被瞬息毁灭。
