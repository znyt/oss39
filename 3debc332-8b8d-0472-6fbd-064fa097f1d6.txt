范恬一下子张大了嘴巴，她震惊的看着这位花花公子哥孙进，他是不是…脑袋抽风了？
<br/>
<br/>        他竟然给夏夕绾买了包子，还一路揣在怀里送了过来，这可不是他的行事作风。
<br/>
<br/>        夏夕绾羽捷轻颤，澄亮的眸子看着孙进，“孙少，谢谢你的包子，好意我心领了，但是包子我不能收。”
<br/>
<br/>        孙进交往过很多女朋友，但是还从来没有这样讨好过一个女孩，现在他还被拒了。
<br/>
<br/>        “夏夕绾，你为什么不收我的包子，以前的事情是我不对，从现在开始我要正式追求你！”孙进表白道。
<br/>
<br/>        夏夕绾，“…”
<br/>
<br/>        这时一道咳嗽声响起了，是吴校来了，吴校看着孙进，“孙少，我们上课时间快到了，有什么话放学后再找夏同学说吧。”
<br/>
<br/>        孙进坚定的看着夏夕绾，“夏夕绾，我不是说着玩的，你记住我的话，我一定要将你追到手！”
<br/>
<br/>        说完孙进就离开了。
<br/>
<br/>        出了A大的校门，好几个富家公子哥将孙进给围住了，
<br/>
<br/>        靠孙少，你这是什么情况，你怎么一大早就来找夏校花了？
<br/>
<br/>        孙少，你手里的包子是要送给谁啊，我艹你不会…亲自给夏校花送早餐吧？
<br/>
<br/>        孙进将手里的包子塞给了一个公子哥，昨晚他在酒吧泡了一夜，满脑子挥之不去的都是夏夕绾那张纤尘绝丽的小脸，他确定自己是被她给迷住了。
<br/>
<br/>        其实她还没有摘下面纱时，他就动心了，只不过碍于自己的面子，不愿意承认，才会找她的麻烦。
<br/>
<br/>        孙进现在是确定自己的心意了，他要追求夏夕绾，让这样一个纤尘绝丽的小仙女做自己的老婆。
<br/>
<br/>        她本来就是自己的未婚妻，想到这一场婚约被自己作没了，他就想吐血。
<br/>
<br/>        “你们猜的没有错，我就是给夏夕绾送早餐的，从现在开始我把话放下去了，我要追求夏夕绾，夏夕绾是我看上的人，你们最好不要欺负她或者是打她的主意！”孙进宣誓主权道。
<br/>
<br/>        这几个富家公子哥当即沸腾了，
<br/>
<br/>        孙少，你玩真的？
<br/>
<br/>        我看这位夏校花很难追啊，你们之前闹的那么不愉快，她肯定不会答应你的。
<br/>
<br/>        孙进上了自己的跑车，十分自信的丢下一句话，“你们等着瞧吧！”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾和范恬回到了教室里，范恬惊奇道，“天哪绾绾，我看这个孙少是玩真的，他真想追求你。”
<br/>
<br/>        班上其他同学早就围在窗户那里看到了刚才发生的一幕，众人纷纷道，
<br/>
<br/>        夕绾，那个孙少就是一个花花公子哥，你竟然打你的主意了，你千万别被他给骗了！
<br/>
<br/>        不过看着孙少将包子一路揣在怀里送过来，这真的好好笑啊，他也有今天。
<br/>
<br/>        夕绾，你可是我们A大的校花，我们A大帅哥如云，肥水不流外人田，就算找男朋友你也要在我们本校找。
<br/>
<br/>        夏夕绾坐在了椅子上，拿出了自己的书本，她也没有想到孙进那朵烂桃花开到了自己的身上，她当然是不会答应的。
<br/>
<br/>        咳咳。
<br/>
<br/>        这时吴校重重的咳嗽了两声，走了进来，他训斥道，“没听到上课铃声吗，耳朵出问题了？快点回到座位上去！”
