　　不错！这玉简内的炼器之术，对本大师有点儿帮助！”高瘦炼器师满脸笑意，继续道：“你想知道什么？就尽管问吧！”
<br/>　　苏莫闻言，眸中闪过一丝冷笑，只是有点儿帮助吗？
<br/>　　他刚才在玉简内括印的内容，可是欧炼子炼器传承中最高深、最精华的炼器知识。
<br/>　　这些炼器知识虽然不全，只是很少一部分，但也非常珍贵了。
<br/>　　而对方，居然只是说有些帮助。
<br/>　　苏莫也是不清楚对方的炼器级别，所以才会拿出这么大的手笔。
<br/>　　不过，他也不傻，已经看出来了，此人估计也是个炼器学徒，不可能是公会的炼器大师。
<br/>　　“我想问问，成为炼器师公会的炼器学徒或者是炼器师，都有什么要求和待遇？”苏莫沉声问道。
<br/>　　“成为炼器学徒，不需要多高的炼器水平，但需要炼器天赋上佳，而且需要有本公会炼器师愿意收徒才行！”
<br/>　　高瘦炼器师收了苏莫的玉简，满心欢喜，知无不言，继续道：“而且，炼器学徒，终身要为炼器师公会效力！”
<br/>　　“至于公会的炼器师，需要帝级的炼器水平，刚才我已经说过了，你根本不可能成为公会炼器师！”
<br/>　　“不过嘛，你既然问了，本大师当然不会隐瞒，我们炼器师公会，属于特殊势力，是由大量炼器师组成，意在共同研究探讨炼器之术，只要是帝级以上炼器师，无论是何种族、是何势力的人，都能成为公会的一份子，公会不会干涉其自由，炼器师也不用为公会做任何贡献。”
<br/>　　高瘦炼器师的嘴，如同机关炮一般，一阵霹雳啪啦，快速的为苏莫讲解了一番。
<br/>　　他心中暗笑，这些消息根本就不是什么隐秘，随便找个人都知道的事情。
<br/>　　但就这些人人都知道的事情，却是让他轻松得到了珍贵无比的帝级炼器之术死亡精华，这真是天大的喜事啊。
<br/>　　面前的这个家伙，真是个乡巴佬啊！
<br/>　　想到对方之前询问炼器师公会中，有没有武尊强者坐镇，高瘦炼器师就忍不住想要大笑三声。
<br/>　　这简直是他听过的最愚蠢的问题，炼器师公会屹立天荒星域千万载，无人不知无人不晓，居然还有人问出这么愚蠢的问题！
<br/>　　高瘦炼器师，也不怕得罪苏莫，因为苏莫不仅无法成为公会中的炼器师，连成为炼器学徒都不可能，因为公会中的那些炼器大师，目前根本没人想收学徒！
<br/>　　苏莫听闻对方的话，面上露出恍然之时，他算是明白了，这所谓的炼器师公会，并不是真正意义上的那种势力。
<br/>　　只是一个由炼器师组成的松散团体罢了，不受任何种族和势力的限制。
<br/>　　而组成这个团体的目的，就是加强炼器师之间的交流，共同进步。
<br/>　　至于这大殿中售卖的各种兵器，估计也是公会中炼器师探讨交流过程中炼制而出，算是为炼器师公会提供收入。
<br/>　　苏莫不得不感慨，这外界的确不凡，居然还有这种炼器师的圣地。
<br/>　　“这位大师，在下想成为公会的炼器师学徒，不知大师能否帮忙推荐？”苏莫向高瘦炼器师抱拳道。
<br/>　　“现在公会中并不收学徒，你过几个月再来吧！”高瘦炼器师闻言，顿时摇了摇头。
<br/>　　苏莫闻言眉头一皱，他想成为学徒是假，只是想先找个安身之所而已，没想到现在却是不收！
<br/>　　他现在可不愿意离开炼器师公会，在这里足够安全，出去杀机无限。
<br/>　　苏莫咬了咬牙，心中一横，道：“在下要成为公会中的炼器师，希望大师帮忙引荐。”
<br/>　　无法成为学徒，那就只能成为炼器师了，虽然成为炼器师公会的炼器师，需要帝级以上的炼器水平，苏莫完全达不到，但他只能一试了。
<br/>　　“什么？”
<br/>　　高瘦炼器师闻言一怔，随即脸色一沉，冷声道：“本大师不是和你说过了吗？只有帝级以上的炼器师，才能成为公会的炼器师，你没听到吗？”
<br/>　　高瘦炼器师可不相信苏莫会是帝级炼器师，因为苏莫的年纪也就和他相仿，他炼器天赋极佳，目前也只是皇级下品炼器师而已。
<br/>　　以他们这个年纪，能成为皇级炼器师，已经是成就极高了，成为帝级炼器师？那根本不可能！
<br/>　　再者，苏莫刚开始是要成为学徒的，得知不收学徒之后，才要成为公会的炼器师，这就更说明苏莫不是帝级炼器师了。
<br/>　　所以，高瘦炼器师当即便脸色冷了下来，认为苏莫在拿耍他。
<br/>　　“大师，你怎么就能断定，在下没有帝级的炼器水平呢？”苏莫淡淡的说道。
<br/>　　“就你？”高瘦炼器师闻言嗤笑一声，随即无语的大笑了起来：“哈哈哈！你是帝级炼器师？你来搞笑的吧？”
<br/>　　整个炼器师公会，还从来没有出过这么年轻的帝级炼器师，高瘦炼器师压根就不信。
<br/>　　高瘦炼器师的笑声，顿时引起了大殿中其他人的注意，众人纷纷侧目。
<br/>　　帝级炼器师？
<br/>　　不少人打量着苏莫，面带惊疑之色，这个年轻人是帝级炼器师？想加入炼器师公会？这不可能吧？
<br/>　　“不试试怎么知道呢？”
<br/>　　苏莫面色淡淡，毫不在意高瘦炼器师的嘲讽，沉声道：“大师收了在下的礼物，总该为在下引荐一番吧！我若是失败了，也与大师无关！”
<br/>　　“是吗？”高瘦炼器师闻言，耸了耸肩，他看苏莫一脸郑重的模样，心中非常无语。
<br/>　　略一沉吟，高瘦炼器师点了点头，道：“也罢，既然收了你的礼物，本大师自然不能不帮你一把，你随我来吧！”
<br/>　　言罢，高瘦炼器师转身向不愿处的楼梯走去，苏莫见此立刻跟了上去。
<br/>　　“小子，丑话跟你说在前头，你若是待会被柴管事轰了出来，可别怨我！”走在楼梯上，高瘦炼器师向苏告诫道。
<br/>　　他已经猜到了苏莫的结局，只等着看苏莫如何出丑了！
<br/>　　“不会！”苏莫摇了摇头，心中却在急思，如何能成为炼器师公会的炼器师。
<br/>　　既然炼器师公会只要帝级以上的炼器师，那肯定要进行考核了，可他以前最高只炼制过王级兵器，连皇级兵器都没有练过，更遑论帝级兵器了。
<br/>　　就算他有欧炼子的炼器传承，帝级炼器之术的理论知识熟悉无比，但他并没有实践过。
<br/>　　“苏莫，你……？”秦不死跟在苏莫身边，面上也是显出了焦急之色，他也是不太相信，苏莫会是帝级炼器师。
<br/>　　因为苏莫如此年纪能达到如今的修为，必然时间都耗在了修炼上，哪里还会有时间学习炼器！
<br/>　　就算有时间学习炼器，也不可能达到帝级炼器师的水平。
<br/>　　“前辈不必多言！”
<br/>　　秦不死刚刚开口，便被苏莫摆手打断了话语，他现在箭在弦上，不得不发。
<br/>　　反正，他是无论如何，也必须留在炼器师公会。
