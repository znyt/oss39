　　第713章
<br/>　　李玉念被安排在曾经一位高官的府邸中，外面有蛤蟆寨的兵将保护。
<br/>　　纵然程大雷并不觉得住在凉州城会发生什么意外，但万一谁对自己图谋不轨，想暗杀李玉念这位闲散王爷。他假若死在凉州城，自己就真的要和李乐天撕破脸了。
<br/>　　李玉念这几日住在凉州城，没有一刻闲着。每天带着人在城中闲逛，有时候程大雷陪着，有时候程大雷不陪。
<br/>　　他这次是带着任务来到凉州，除了要颁旨令程大雷去长安城，另外一个目的就是刺探程大雷的情况。对于他这个目的，程大雷未必不知，却大开方便之门。
<br/>　　一来，李乐天现在没有实力讨伐凉州；二来，即便他拥有这个实力，先讨伐的也必定不是凉州。
<br/>　　几天时间内，李玉念走了城中的工匠坊，军策府，甚至还赶到蛤蟆城，看了看那里的矿山。
<br/>　　回来的路上，李玉念一直保持着沉默，内心却如波涛翻滚。
<br/>　　帝国之大，憎恶程大雷的人有许多，但畏惧他的人也不少。对于这么一个山贼出身的山大王，连相府都没能收拾掉他，反而令其做大。至于程大雷的势力究竟有多强，大家普遍没有一个正确的估计。
<br/>　　如今，李玉念是亲眼见到了。军中兵强马壮，士兵有极强的纪律性，每天都进行超负荷训练；而工匠坊、铁矿也没有一刻闲着，每日都在生产兵器弓箭，以及各种李玉念想也没想过的新式武器。
<br/>　　而凉州城、朔方城、蛤蟆城三地的居民，堪称安居乐业，这在整个帝国来说，也是罕见的事情。
<br/>　　对民秋毫不犯，对兵严格约束，蛤蟆寨的兵虽不多，但文有文官，武有武将，他能占据凉州，并非是因为侥幸。
<br/>　　“王爷今日所见如何？”
<br/>　　吴用与李玉念两马并行，他的话令李玉念恍然回过神来。稍微定了定神，开口道：“程当家爱民如子，实在是令我辈佩服，长安对程当家有诸多议论，看来是以讹传讹居多。”
<br/>　　吴用点点头：“大当家虽然有时候做事蛮奇怪的，但有些事情做得的确令我们佩服。”
<br/>　　李玉念目光落在吴用身上，眼珠转了转，道：“还未请教吴军师是哪里人？”
<br/>　　“在下青州祁口人。”
<br/>　　“青州是个出圣贤的地方呐，吴军师胸怀锦绣，腹隐山河，即便在长安也常常能听闻吴军师的名字。”
<br/>　　“喔……哈哈。”
<br/>　　吴用脸上挂出笑容，这李玉念没有特意奉承自己的道理，他说长安都流传自己的名声，看来是真话了。
<br/>　　李玉念眼中闪过一道光芒，他这次千里迢迢来到凉州，还有另外一个目的，就是看能不能在凉州内部发展出一个间谍。
<br/>　　徐神机这种人自然不行，刘悲跟随程大雷太久，也几乎没什么希望。倒是这吴用，在长安从没听说过有他这号人，说不得能够用富贵打动他。
<br/>　　“青州与凉州相距甚远，吴军师是如何来了此地？”
<br/>　　吴用轻轻叹口气：“家园破碎，也是没有办法的事，不提也罢，不提也罢。”
<br/>　　“深山藏虎豹，田野埋麒麟，程当家手下当真是人才济济，吴军师在这里一定能施展平生所学，尽展才华。”
<br/>　　“这个……”吴用捻着胡须，他自认为自己是有些本事的，奈何出得很多主意程大雷都不听。至于一身所学，其实并没有施展几分。
<br/>　　李玉念知道什么是尺度，话到这里便不在说。他从身上取下一枚玉佩，在马上递给吴用。
<br/>　　“这枚白鲤玉佩，是先帝所赐。如果吴军师不嫌弃，本王就转赠给吴军师。”
<br/>　　“这……在下万万不敢收。”
<br/>　　“唉，从来香草佩美人，宝剑赠英雄，本王并没有别的意思，只是佩服吴军师的谋略，这枚白鲤玉佩赠给吴军师，也才算是没有蒙尘。”
<br/>　　明珠蒙尘，鲤鱼跃龙门，李玉念觉得自己话点到这里，已经差不多了。如果吴用是个机灵人，自然会明白自己的目的。从言谈华语间也能感觉到吴用的不得意，难道自己一个帝王家能给的，还比不上一个山贼么。
<br/>　　李家的状况很困难，虽然李乐天握住了一些实权，但仍处处被世家豪族掣肘。面对此状况，姓李的皇族不得不抱起团，避免李家将江山丢掉。
<br/>　　毕竟一荣俱荣，一枯俱枯。
<br/>　　赠玉之后，李玉念打马向前，似乎价值连城的宝玉，对他而言并不算什么，只不过随意拍掉肩膀上一粒尘埃而已。
<br/>　　吴用却是感激涕零，看着李玉念的背影，心中对他的好感蹭蹭往上涨。
<br/>　　“这位李王爷是个地道人呐。”
<br/>　　当天夜里，吴用就将这件事显摆出来了。将玉佩挂在腰间，到处晃荡，顿时引起徐神机、刘悲等人的注意。
<br/>　　“哦，这玉佩么，是李王爷送给我的，据说是明帝戴过的东西，小玩意小玩意。”
<br/>　　众人看向吴用的眼神都由几分嫉妒，天子之物，天生就带几分宝气。早知道这样，就该自己陪着李玉念闲逛呐。
<br/>　　就连程大雷都听说了这件事，拿过来把玩一番，又还给了吴用。他有一个藏宝库，搜罗了许多奇珍，也卖不出去，这枚玉佩对他来说不算什么。
<br/>　　唯有崔白玉看在眼里，却也没有说什么。
<br/>　　怪只怪，李玉念做事做得太含蓄。蛤蟆寨有一个算一个，就没有一个出身高的，苏樱是商户之女，理论上穿丝绸的资格都没有。剩下徐神机、刘悲这些人，都是苦出身，半辈子和庄稼地打交道。
<br/>　　如吴用，曾经不过是个穷酸教书匠，有些机智也不过是乡下人阴损缺德的计谋。
<br/>　　如他这种人，怎么想也不会想到明珠蒙尘，鲤鱼跃龙门上面去。
<br/>　　所以李玉念一番话，大概就是冲瞎子抛媚眼，白费力气。在吴用想来，他只以为是自己占了便宜，而想不到其他。
<br/>　　整个蛤蟆寨，也就崔白玉见多识广，大概猜出了李玉念的用意。可她巴不得程大雷一番家业毁掉，自然不会挑明，让程大雷做一次明白人。
<br/>(本章完)
