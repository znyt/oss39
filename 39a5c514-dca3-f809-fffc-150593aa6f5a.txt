么态度跟我们交谈，能听明白吧？”<br/>　　<br/>　　    “呵呵·····”对方擦了下嘴角的血迹，阴霾着眼神说道：“听明白了，你问吧”<br/>　　<br/>　　    “我看你好像没太明白，接着踢他”菜刀文起身说道。<br/>　　<br/>　　    对方顿时懵逼了，这他么的什么路子啊，我都认怂了还不行么？<br/>　　<br/>　　    “你跟我是不认识，但你下手的这个女人我认识，所以我找你也不算坏了规矩，不过规矩什么的我本来也不在乎，坏就坏了能怎么的？”王惊蛰脚尖踢了下对方的胸口，问道：“谁让你祸害她的？”<br/>　　<br/>　　    枯瘦男子咽了口唾沫，说道：“是一个叫吴洋的人找的我，他让我给那个女人下巫·······”<br/>　　<br/>　　    另外一头，陈重来到公司后，秘书拿着一叠资料来到办公室。<br/>　　<br/>　　    “陈先生，你让我查的有谁对我们川南那块地感兴趣，都在这上面了”<br/>　　<br/>　　    陈重接过资料后随意的翻看了几眼，就又甩给了秘书，这些人基本都是蜀地一带的，他多少都有耳闻，他直接断定，这里面的人没有谁是敢捋他的虎须的。<br/>　　<br/>　　    不是一个层次的人，能当对手么？<br/>　　<br/>　　    “今天和明天所有的会议和有关我的活动马上全部取消，不管什么事等我回来再做决定，同时给机场那边打电话，下午之前我要飞申城”陈重跟秘书吩咐完，然后从办公桌上拿起电话拨了出去，蹬那边接通后，他冲着电话说道：“你马上过来找我一趟，尽快，对，就是现在我在公司里等你，你可能得要跟我出一趟门了”<br/>　　<br/>　　    陈重刚和对方说完，身上的电话就响了，他拿出手机见是陈臣打来的，就有些头疼的叹了口气。<br/>　　<br/>　　    “爸，你在哪？”陈臣急促的声音从电话那头传来。<br/>　　<br/>　　    “我马上要出门，去一趟外地，有什么事你等我回家之后再说······”
