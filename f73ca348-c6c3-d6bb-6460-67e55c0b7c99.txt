        “还不止‘一’晚？！”听傅夜沉说这话的时候，童筱颖整个脸色都变了。
<br/>
<br/>        她潜意识里，已经相信了傅昼景所说的关于大哥的那件事情。
<br/>
<br/>        傅夜沉随即在童筱颖的粉唇上啄了一下，接着说道：“与你相处的每一个晚上，都令我难以忘怀。”
<br/>
<br/>        童筱颖顿时尴尬地笑了笑。
<br/>
<br/>        大哥的嘴，还真甜啊！
<br/>
<br/>        但她想知道的，可不是他两之间的事。
<br/>
<br/>        “那在我之前呢？大哥有没有这一辈子都无法忘怀的事？”童筱颖抿了抿唇，依旧尝试着去探问他傅夜沉的过去。
<br/>
<br/>        傅夜沉见童筱颖一副很好奇又认真的样子，忍俊不禁地反问：“你是对我的过去感到好奇，还是你有什么心事瞒着我？”
<br/>
<br/>        “对大哥的过去感到好奇啊！想多了解大哥。”童筱颖莫名其妙地感到心虚起来。
<br/>
<br/>        她其实有些在意傅昼景所说的那件事情，但又不相信大哥以前做过那样的事情，可傅昼景并不会无中生有，所以，她想在大哥这儿求证。
<br/>
<br/>        想知道真正的真相到底是什么。
<br/>
<br/>        傅夜沉搂着童筱颖的肩头，让她的脑袋靠入自己的颈窝，深沉地说道：“得知我父亲过世的那晚，还有我读高一时，被一个女生冤枉的那晚。”
<br/>
<br/>        “被一个女生冤枉？”童筱颖顿时敏感起来，仰面看向傅夜沉，心里揣测着会不会就是傅昼景提及的那件事情。
<br/>
<br/>        傅夜沉的俊容倏地冷了几分，他点头之际，剑眉也跟着微微蹙起。
<br/>
<br/>        他语重心长地接着说道：“而且还是被一个成绩优异、长得很漂亮的女生给冤枉的。”
<br/>
<br/>        “大哥，可不可以，说给我听？”童筱颖深沉道。
<br/>
<br/>        似乎真的与傅昼景所说的那件事情对上了，不过，果然，这件事情另有隐情。
<br/>
<br/>        傅夜沉也毫不忌讳地娓娓道来：“我在读初中之前，一直跟着我父亲过军旅生活，没有去学校里上过一天课。我父亲过世后，我被爷爷接了回去，正式进入学校按部就班的过学生生活。后来，我念完初中，就被爷爷送到这滨江城，念滨江一中的全封闭式寄宿制高中。然后，就发生了那件事……”
<br/>
<br/>        滨江一中，除了每个月月底那两天的月假，其余时间，学生均不得离校，吃喝拉撒学习睡觉，全部都在学校里。
<br/>
<br/>        十二年前。
<br/>
<br/>        自幼跟随父亲过旅行生活的他，最喜欢的玩具就是枪和匕首了。
<br/>
<br/>        所以，他从小就摸过真枪实弹，把玩过各种匕首，更是喜欢射击和近身格斗，而重新组装枪支的速度几乎是他能拿出来跟任何人比拼炫耀的特技。
<br/>
<br/>        而这一下子进入全封闭式寄宿制的高中，压根就无法关住他傅夜沉那颗“贪玩”的心。
<br/>
<br/>        在初中毕业后的那年暑假，他玩了一款3D类国际联网射击游戏，因为操作精湛，国服第一，所以被邀请进入了国内一个赫赫有名的精英战队。
<br/>
<br/>        高一那年，星期天的那天晚上可以不用上晚自习，但不可以离校，因为有战队赛，傅夜沉又拿不到请假条，不得不翻墙离校，去了战队赛指定的网吧。
<br/>
<br/>        那晚的战队赛，他所操作的人物以“30杀”干掉了其他战队的精英，活到了最后，为自己的战队赢得了第一名。
<br/>
<br/>        当年，他最高兴的事情，莫过于，玩了自己喜欢的游戏，还能靠这游戏赚了一笔优厚的奖金。
<br/>
<br/>        比赛结束后，他并未在网吧里逗留，而是直接从原地方翻墙返校。
<br/>
<br/>        结果，他从围墙上跳下去后，突然传来一个女生“啊”地一声尖叫，同时也把他给吓了一大跳。
<br/>
<br/>        围墙角那儿是监控盲区，而且还是小树林绿化带，所以也没有路灯，灰蒙蒙的一片。
<br/>
<br/>        在他还未来得及反应过来，一个黑影迅速从他眼前逃走。
<br/>
<br/>        就在他准备去追那个黑影的时候，突然感觉到有什么东西拽住了他的裤子用力一扯，“唰”地一下髋骨处被裤头的松紧带刮得硬生生的疼，他两腿瞬间一片凉意袭来。
<br/>
<br/>        那个时候的他，穿着一套休闲装，裤子是那种运动型的松紧裤。
<br/>
<br/>        紧接着，有两个保安打着手电筒朝这边跑了过来，手电筒的灯光打向他这边，他才看到，自己的脚边竟然跪坐着一个披头散发、衣衫不整的女生。
<br/>
<br/>        “救命，他要强奸我！”女生抽泣着控诉道。
<br/>
<br/>        当场，他傻眼了，毫无防备的被两个保安按倒在地，像擒拿罪犯一样，反剪着两只手臂，趴在地上动弹不得。
<br/>
<br/>        他当时，完全就是“我是谁，我在哪，什么情况”的一脸懵的状态。
