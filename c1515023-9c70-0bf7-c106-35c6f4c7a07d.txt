　　阿里·伊本有过一次与唐军交手的经验，只不过当时他的对手是尉迟宝琳，而唐军使用的武器也是火铳，那一次他所率领的两万大军在顷刻间被打的落花流水。
<br/>　　不过那时候的阿里·伊本坚信，唐军只是仗着武器之利，如果近战必然不会是大食骑兵的对手。
<br/>　　但现在他知道自己错了，唐军真正擅长的似乎并不是那种远距离攻击的武器，而是近战。
<br/>　　刀光交错，血肉横飞！土著人在唐军面前连一个照面都没撑过去就被杀的落花流水，让他引以为傲的大食军队同样不比那些土著强多少，横刀与弯刀交击，溅起无数火花的同时，另外一侧的唐军便会借机将刀子插进大食人的胸口。
<br/>　　不可否认的是，大食人的确是足够彪悍，手中弯刀也足够唬人，但他们到底还是水军，陆战与李世民带出来的‘海狼’完全没有可比性。
<br/>　　大唐‘海狼’虽然是水军，但其训练却并不是仅仅限于水上，陆战也是他们必修的课程，而且所有的‘海狼’最开始的时候也都是从十六卫里选出来的佼佼者，陆战能力完全在水战能力之上。
<br/>　　这么一来大食人虽然在人数上占优，但是战力上却差了不是一个数量级。
<br/>　　而且这一场战斗也并不是那么简单的，大食人完全是为了躲避海上的大唐援兵才会上岸，心里的那份压力无论如何也挥之不去。
<br/>　　在此之前他们虽然没有对李世民的舰队造成太大的伤害，但优势地位却是显而易见的，也就是传说中的打顺风仗。
<br/>　　但是现在情况变了，大唐援兵一到，大食人也慌了手脚，毕竟现在他们是一支孤悬海外的军队，若是被大唐给围了，立刻就会步上李世民的后尘。
<br/>　　这样的心里压力之下，大食人根本无法集中精神与唐军作战，那些处在队伍里的士兵时不时的就会看向身后，看看自己的后续部队有没有上来，顺便也看看大唐援兵有没有上来。
<br/>　　阿里·伊本同样关心着身后的状况，一边指挥前面的战斗，一边时不时看向自己登陆的方向，希望看到援兵快点赶到。
<br/>　　但就在这个关键的时候，一阵阵闷雷声传来，同时岸边也隐隐传来惨叫之声。
<br/>　　大唐的援兵到底还是到了，阿里·伊本心中暗叹一声，转头对身边的副手命令道：“传我命令，全军压上去，务必把那些唐军打垮，另外，海上没有完成登陆的战船无论如何也要拖住大唐的援兵两天时间，给我们创造机会。”
<br/>　　“将军……”副将有些犹豫。
<br/>　　“怎么？你想说什么？”阿里·伊本有些扭头问道。
<br/>　　副将很清楚眼下的情况，阿里的命令是唯一可以争取时间的办法，但想到大唐舰队那凶残的炮火，他还是提醒道：“将军，水军根本没有办法阻挡大唐舰队两天的，士兵们对大唐战舰的恐惧太深了，让他们去阻拦唐军，搞不好就会哗变！”
<br/>　　阿里·伊本深吸一口气：“本将军不管这些，你传令下去，他们要么光荣的战死在海上，要么耻辱的倒在沙滩上，两者任选其一。”
<br/>　　“可是，万一士兵哗变……”副将再次提醒。
<br/>　　“去传达命令吧，若是他们真的哗变，死在自己人手里也比死在唐人的手中要强上百倍。”阿里伊本无奈的说道。
<br/>　　副将清楚眼下的情况，他同样也清楚眼下的情况，唐军的火炮打过来士兵害怕，他阿里·伊本同样也怕。
<br/>　　可是怕又能如何，仗打到现在这个样子，如果不能彻底将李世民控制在手中，那么等待自己的便是死亡，跟着他的数万士兵也不会有什么好结果。
<br/>　　毕竟海上不比陆地，在大海上所有的补给都是有限的，谁也不会带上数万俘虏，到时候只怕俘虏还没有造反，自己就已经被饿死了。
<br/>　　所以，如果战败，阿里·伊本或许有活着的可能，但那些士兵却死定了，唐军虽然带不走俘虏，但是却可以将所有的俘虏全部杀光。
<br/>　　……
<br/>　　副将的命令传达下去之后，余下那些还没有完成登陆的大食战船立刻开始调头，向着远处围上来的大唐舰队发起绝死反攻，并没有出现如他担心的哗变。
<br/>　　大食的士兵也不傻，同样知道这个时候哗变根本没有任何意义，唐军根本不会放过他们任何一个，拼拿的话或许还有一线生机，但若是哗变或者逃跑，估计只有死路一条。
<br/>　　但是现实总是那么残酷，大食的反击虽然义无反顾，但北海舰队的炮弹也有着一颗铁石心肠。
<br/>　　随着那不断调整的炮口，一颗又一颗炮弹带着阵阵呼啸砸进大食的战船，将敌人的血肉撕扯成无数的碎片。
<br/>　　数百条大食人的战船，数万大食水军，用自己的生命在为岸上的同伴争取时间。
<br/>　　这种行为看在大唐军人眼中，让每个的都露出佩服的神情。
<br/>　　而为了表示对这些大食‘勇士’的尊敬，炮手们不禁加快了装填炮弹的速度，火箭弹部队也开始登上甲板，将中手的武器对准了那些亡命而来的大食人。
<br/>　　鲜血很快染红了整片大海，密集的大食舰队开始变的稀疏，战争之王的声声怒吼彻底的粉碎了大食人的一切希望。
<br/>　　反身攻击大唐援军的舰队别说两天时间，就是两个时辰都没有撑住，便被发了疯一样北海舰队撕成了碎片，数面艘战船最后只余下几十艘，拼了命的向岸上逃去。
<br/>　　“殿下，接下来怎么办？我们要不要登岛？”李震看着已经彻底被打散的大食舰队，沉声问道。
<br/>　　“派出部队登岛，另外，让炮兵再给老子把海岸犁上两遍，确保登岛部队的安全。”李佑看着岛上郁郁葱葱的热带丛林，沉着的下着命令。
<br/>　　老头子眼下就在岛上，但却不知道具体位置，李佑不敢贸然命令部队直接炮击海岛，所以登岛已经是必然，不过出于安全考虑，他最后还是没有吝啬手中那为数的不多的炮弹。
