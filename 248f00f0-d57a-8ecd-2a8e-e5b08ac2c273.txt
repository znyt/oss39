　　对于“暗月”这个组织，最开始赵国邦和赵天宇对它的定义是“恐怖组织”，但是随着跟“暗月”接触的深入，他们又发现这个组织并不是他们所想象中的恐怖组织。
<br/>　　它组织严密，关系复杂，就像是被一层又一层的迷雾包裹着，令人根本看不透它的本质。
<br/>　　而且“暗月”的关系网很复杂，它的触手很长，就目前赵天宇所掌握的资料中就显示，“暗月”跟多国政府都有联系，其中就包括美国。
<br/>　　这不是一个恐怖组织应该有的东西，最重要的是，在“暗月”背后有实力强大的财团做支援，这些财团有一部分是世界知名的公司。
<br/>　　所以，趁着现在这个机会，子午想要从影天使嘴里挖出一些有价值的东西来。
<br/>　　“这个问题，我想我回答不了。”
<br/>　　影天使轻轻摇头，说道：“首先，我不可能出卖‘暗月’；其次，即便是我这样的身份，我也看不清楚这个组织，我这么说，你或许并不相信，但这就是事实，我看不透它，因为它太复杂、太神秘，你们跟‘暗月’作对，无疑是自取灭亡。GHOST，你还有机会，加入我们，以你的实力很快就能得到重用，我们两个人联手，这个世界上谁能跟我们对抗？在虚拟的网络世界中，我们就是无敌的存在！”
<br/>　　“如果你没有睡醒的话，我可以让你先去睡一觉然后再来比试。”子午看着摄像头说道，他拒绝了影天使的提议，如果子午有这方面的想法，他根本就不会加入龙隐部队。
<br/>　　“既然这样，我们之间没什么好说的了。”
<br/>　　电脑屏幕上的倒计时被启动。5……4……3……2……1，键盘的背光亮起，键盘解锁！
<br/>　　子午和影天使没有任何的犹豫，双手飞快的在键盘上敲打起来，两个人都是展现出了惊人的APM，就目前子午和影天使两个人所表现出来的APM，即便是职业的电竞选手都是难以企及，因为实在太快了。
<br/>　　世界各国的黑客都在关注着这场世界黑客界最巅峰的对决，但是这场代表世界黑客巅峰水准的对决没有直播成了世界各国黑客最大的遗憾，也是永远也无法被弥补的遗憾，因为在这场对决之后，SHADOW和GHOST这两个传说级别的黑客只会剩下一个，从此以后也再也不会有这种巅峰级别的对决。
<br/>　　尽管没有直播，无法得知这次对决的情况，可是在世界各大黑客论坛中，却已经是有高手对这次对决进行了分析。
<br/>　　“黑客，有红域和蓝域之分。”
<br/>　　指挥中心房间内，云熙用最简单、最容易理解的方式向在座的人介绍这次子午和影天使之间的对决，“红域主进攻，蓝域主防御，一般来说不同的黑客会根据自己的需求，主修红、蓝域其中一种。当然，在自身技术到了一定水平后，就会进行红蓝双域双修，虽说是双修，可是因为接触时间长短的关系，两者之间还是会有些许的差距，不明显，但是在关键时刻，这些许的差距就会被放大的非常明显。这次老师跟影天使之间的对决，是双方红蓝域能力的双重体现，他们在攻击己方门锁中程序的同时，还要防御对方门锁中的程序，防止被对方攻破。老师说过，只会进攻或者只会防御的黑客还不能算是一个合格的黑客。”
<br/>　　“那子午是红域和蓝域哪一方面更强一点？”赵天宇对云熙问道。
<br/>　　赵天宇跟子午虽然是朋友，但是对于子午的实力，他还真不如云熙这个学生了解的清楚。
<br/>　　云熙微微一笑，回答道：“以前，老师蓝域能力更强一些；现在，老师的蓝域水平有多高，红域水平就有多高。”
<br/>　　云熙的回答，令得指挥中心中的人非常振奋；只是一般在这种振奋人心的时刻都会有一个人出来泼冷水，这个人就是李然。在现场的李然通过卫星通讯技术，跟指挥中心保持着联系，所以刚才云熙所说的话，李然自然也是全都听见了。
<br/>　　李然保持着他特有的冷静，说道：“子午教官有进步，影天使也有进步。别忘了，那家伙背后是‘暗月’！”
<br/>　　子午身上带着战场视频系统，所以子午在房间里跟影天使之间的对话并没有瞒过龙隐部队这边的人，SHADOW是“十二主天使”之一影天使的身份，也是被罗昊、李然所知晓，而在知晓SHADOW真实身份的时候，不管是罗昊还是李然都是对SHADOW这层身份很意外。
<br/>　　“你这家伙怎么涨他人志气灭自己威风？”林晓阳不悦的瞪着李然，说道。
<br/>　　“这不是涨他人志气灭自己威风。”李然轻轻摇头，抬头看着反射着阳光的玻璃墙写字楼，说道：“这是实事求是！”
<br/>　　林晓阳张了张嘴，想要反驳但是却无从反驳。李然说得不错，子午在进步，作为子午老对手的影天使自然也是在进步，因为他们两个人其实谁都想要打败对方！
<br/>　　世界各大黑客论坛上的分析，也跟云熙说的差不多，只是对于影天使和子午红蓝双域的能力，就没有云熙了解的那么详细了。
<br/>　　房间内，子午和影天使的双手还在键盘上翻飞敲打着，但是在速度上却没有之前那么快了，毕竟就是是子午和影天使这样的高手，也不可能始终将APM长时间保持在一个高数值不发生变化的，而且还需要保持自身的状态，应对后面最关键的十五分钟时间。
<br/>　　尽管两个房间内有空调，房间温度也不算高，可是两个人的额头上还是能看到汗水。
<br/>　　“GHOST，你真的不再考虑一下我刚才的提议吗？”影天使这个时候有开口道：“只要你答应，我们之间的较量现在就可以停止，而且我有足够的把握让你跟着我们安全的离开这里。”
<br/>　　“收起你的垃圾话攻击吧，你应该知道这对我并没有任何作用。”子午不为所动，说道：“你有这个力气说垃圾话，还不如想想怎么攻破我写入你门锁中的程序。”
